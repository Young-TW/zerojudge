# l228. 數學運算式(1)

## 內容

小明的數學很差，請你編程一段程序協助他。

## 輸入說明

輸入第一行為一個整數T，代表接下來有T個測試數據。


測試數據描述如下:


第一行為一個整數H，代表該測試數據有H行。


接下來有H行，其中H-1行為變數 格式 : "x(變數)等於?"


最後一行為算式，詳見範例輸入。

## 輸出說明

對應每個測試數據，輸入算式中"?"的符號為甚麼。 格式:"符號 中文符號"


保證每個算式只有一個"?"。


保證所有算式的"?"只有一個解法。


符號和中文符號 (僅只這四個):


+ 加


- 減


* 乘


/ 除

## 範例輸入 #1

```
1
3
a等於3
b等於5
a+b?2=6
```

## 範例輸出 #1

```
- 減
```
