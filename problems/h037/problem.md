# h037. 我恨情人節

## 內容

「一年之中有這麼多情人節幹嘛阿！我明明...」明明的聲音突然哽咽起來，伴隨而來的是他漸漸泛紅的淚眶。


「我明明就只是每天在小學門口，看到喜歡的女生就把情書遞給她，為甚麼...我還是交不到女朋友...」


這是明明單身的第30年了，在這30年間，他做過了無數次的告白，最終都以失敗收場。


「叮咚—叮咚—叮咚———」


明明打開門後，只見一位藍髮碧眼，身著一襲白藍配色水手制服，以彷彿就是明明心目中那最完美的小學生姿態向他走來。


小女孩面帶笑容，她泛紅的臉頰與潔白的肌膚形成了強烈的對比，如同天使下凡一般，使得明明目不轉睛地盯著她。


明明在腦海中進行了地毯式搜索，努力地想要喚起有關眼前這位天使的記憶，卻始終沒有任何頭緒。


「我可以實現你的願望喔！」小女孩以羞澀又輕柔的口吻說道。


「只要你能告訴我，離今天第N近的情人節是哪一天的話。」


※註：離今天第N近的情人節是指，以今天的中午12:00算起，第N近的情人節日期。


※今天就是指你看到題目然後要submit的那一天。


※若今天是日期A，日期B與日期C皆離日期A第K近，則詢問第K近或第K+1近時，日期B及日期C皆可為答案。

## 輸入說明

輸入只有一個整數N，代表小女孩問明明的離今天第N近的情人節是哪一天。


（1≤N≤104）

## 輸出說明

請以格式 YYYY/MM/DD 輸出以今天的中午12:00算起，第N近的情人節日期，若答案有多組，則輸出其中一組

## 範例輸入 #1

```
1
（以2022/02/14為例，此行不會在測試資料中出現，請特別注意）
```

## 範例輸出 #1

```
2022/02/14
```
