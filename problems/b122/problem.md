# b122. 用餐地點 (Lunch)

## 內容

![image](images/problems/b122.gif)

## 輸入說明

測試檔案的第一行有兩個數字 m, n, (1<=m,n<=100)分別代表訓練場的區隔方式（如上圖所示），因此第i, j 區的區域代號就是 (i, j)。接下來的m 行，每一行有n 個整數，這m 行中的第i 行的第j 個整數代表 (i, j) 區的學員數。每兩個整數之間都會有一個空白。

## 輸出說明

請輸出能最快讓所有學員都能集中到達用餐的區域代號。輸出時，不需要輸出刮號或逗點，只需要將代表該區的兩個整數輸出並以空白隔開。

## 範例輸入 #1

```
3 5
5 2 0 2 2
0 2 0 2 0
1 0 0 3 0
```

## 範例輸出 #1

```
1 2
```
