                           5. 辨識碼 (Identification)

問題敘述
  某個國家發明了一套辨識系統，該辨識系統會依據人民的身分證號碼來判定身分，以指
定數字個數將身份證號碼分組，若有兩組以上的數字和相同，則判定為該國國民。

   舉例來說，若指定的分組數字個數為 3，且身分證號為 41924519。從該身分證號之最右
方開始每 3 個數字一組，若數字不夠則將剩餘的數字視為一組。依照規則可依序分出「519」，
         。其中「519」和「924」兩組的數字和都是 15，滿足條件，故該身份證號被
「924」和「41」
判定為國民。

   給定分組數字個數以及身分證號，請你幫忙判斷此人民是否屬於該國家。

輸入格式
  輸入的第一行會有一個數字 X (1≤ X ≤ 199)，代表的是分組數字個數。第二行有一串數字
N，該串數字的長度最多不超過 200 個字元，且保證這串數字不為 0 開頭。

輸出格式
   判別某人民是否屬於該國家，若是，請輸出 Yes；否則，請輸出 No。

輸入範例 1                                 輸出範例 1
3                                      Yes
41924519

輸入範例 2                                 輸出範例 2
2                                      No
134392699

輸入範例 3                                 輸出範例 3
6                                      Yes
555555555555555555555555
