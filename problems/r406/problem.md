# r406. 10690 Expression Again

## 內容

你會得到一個代數表達式，形式如下：
(x₁ + x₂ + x₃ + ... + xₙ) × (y₁ + y₂ + ... + yₘ)，
以及 (n + m) 個整數。
你的任務是使用這些整數來求出這個表達式的最大值與最小值。

例如，若給出的表達式是 (x₁ + x₂) × (y₁ + y₂)，並且提供的整數是 1、2、3 和 4，
那麼最大值為：(1 + 4) × (2 + 3) = 25
最小值為：(4 + 3) × (2 + 1) = 21

## 輸入說明

輸入含有多筆測資(不會超過110筆)。

每組測資以兩個正整數 N、M 開頭（N, M < 51），
接下來一行會提供 (N + M) 個整數，範圍介於 -50 到 50 之間。

輸入以檔案結尾（End of File, EOF）作為結束。

## 輸出說明

對於每一組測試資料，輸出一行：
最大值，接著空格，再輸出 最小值。

## 範例輸入 #1

```
2 2
1 2 3 4
3 1
1 2 3 4
2 2
2 2 2 2
1 3
10 5 -2 29
```

## 範例輸出 #1

```
25 21
24 9
16 16
377 -88
```
