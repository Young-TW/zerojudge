# a267. 太空梭的油箱

## 內容

一架太空梭即將出航了!對於太空梭非常重要的就是有沒有足夠的燃料。為了帶足夠的燃料，太空梭必須攜帶大量油箱來儲存這些燃料。太空梭每個月都會消耗1單位的燃料。每個油箱都最多可以裝5單位的燃料，太空梭本身也可以裝有最多5單位的燃料。然而，為了使油箱正常運作，每個油箱每個月也會消耗1單位的燃料。問題來了，在太空梭出發前，就把太空梭以及N個油箱都裝滿燃料，那這架太空梭可以靠這些燃料航行幾個月呢？聰明的你一定會發現，因為油箱也會耗燃料，我們可以把空的油箱丟掉，這樣能讓每個月的耗油量稍微減少些。

## 輸入說明

每個測資檔包含多筆測資，每筆測資有一個數字N(0<=N<=500000)。當輸入N=-1時結束程式。

## 輸出說明

對每筆測資，輸出太空梭最多能航行幾個月。

## 範例輸入 #1

```
0
1
2
5
7
100
-1
```

## 範例輸出 #1

```
5
7
9
12
13
24
```
