# d540. 老Z的题

## 內容

一日老Z上ZJ刷题，不经意间看到***在ZJ上面发动广大的CODERS来进行**的研究，试图揭开老Z隐藏多年的秘密。要知道，老Z的秘密恰恰就隐藏在一个极大的*里面（大概有*************************位）。老Z一方面担心某一天***研究出来了自己的秘密，另一方面又觉得***的行为是对他的轻蔑，于是决定进行反击！
    可是，问题来了！老Z无法写出一个足够快的程式来击挎***的阴谋。因为，老Z不知道ZJ的测评机何时才是运行的最快的。而且，还有一个问题，那就是如果老Z的程式比***的程式慢，那么***将会运用二者的时间差来侵入老Z的电脑，盗取老Z的秘密。一向谨慎的老Z当然不会允许这样的事情发生！
    万般无奈之下，老Z只好去向万能的MR.W求助。可是因为MR.W不幸的得了甲流，已经被隔离了起来，冒险去见MR.W是非常危险的！老Z站在空无一人的高二十四班门口徘徊，正心中无计可施的时候，突然，他的眼前出现了一个人，那就是POWPOW。他说他可以帮助老Z来击垮***。
    这个方式是启用学校的魔法阵地：百草园！
    但不幸的是，当两人来到百草园的时候，却发现这里由于多年前411寝室的闹鬼事件，已经封印了起来。如果要进入百草园，必须要解开密码锁才可以。
    密码锁是一个奇怪的机器。它有一个屏幕，上面不停地显示着一个整数N。POWPOW从看门的树精那里了解到，这个其实就是一个简单函数f(x)=x^x的一个问题。现在，你需要输入进的是如下的一个东西。首先，计算出使得f(x)的值的位数为N的最小的整数X。然后，将X对915求模。

    这么简单的问题当然难不住POWPOW，他很快就解决了这个问题！当二人进入百草园，他们发现：百草园其实是很多个小的区域。有的区域之间有通道连接，但是经过这些通道会消耗能量。但是也有一些能量场，如果你按照正确的方向通过能量场，那么将会获得一些能量。

    现在POWPOW和老Z想在百草园里先逛一下，同时获得一些能量，为布下魔法阵做出准备。那么，他们CALL你请你帮他们算一算他们究竟能不能达成他们的愿望。

    如果你能帮助他们，他们将保证一个月不去你的开心网花园内偷你的菜！

## 輸入說明

```
第一行：一个整数 N (1 <= N <= 5)，表示共有N组数据。（多组数据测试）
```

```
每组数据：
```

```
第1行：三个整数 N  M  W 分别表示区域数、区域间的通道条数以及能量场的数目
```

```
       第2至M+1行:每行三个整数 (S, E, V),表示在S区域与E区域之间有一个通道，通过需要V(0 <= V <= 10,000) 的能量。
```

```
       第M+2至M+W+1行:每行三个整数 (S, E, V),表示在S区域与E区域之间有一个能量场，通过可以获得V(0 <= V <= 10,000)的能量。
```

```
    注意，一般通道是双向的！能量场是单向的！
```
