# g779. 小億老師的幸運數字

## 內容

一般人的幸運數字可能是某一數字(例如：888或168)，但小億老師不是一般人，只要數字 N (1<=N<=2147483647)可以被某位數字 K (1<=K<=9)整除，或數字 N 中包含有該位數字 K 的數字，都是小億老師的幸運數字。
給你兩個數字 N 與 K ，若 N 為小億老師的幸運數字，請輸出 YES ，否則輸出 NO 。

## 輸入說明

輸入兩個數字 N (1<=N<=2147483647) 與 K (1<=K<=9)

## 輸出說明

若 N 為小億老師的幸運數字，請輸出 YES ，否則輸出 NO 。

## 範例輸入 #1

```
4 2
```

## 範例輸出 #1

```
YES
```

## 範例輸入 #2

```
23 2
```

## 範例輸出 #2

```
YES
```

## 範例輸入 #3

```
17 3
```

## 範例輸出 #3

```
NO
```
