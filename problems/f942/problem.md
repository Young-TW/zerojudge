# f942. 字串判斷

## 內容

給你很多個字串，請你找出相同的字串。

## 輸入說明

讀到 $\text{EOF}$ 結束


每行一個字串，由數字和大小寫字母組成


第 $i$ 行字串為 $s_i$，$1 \leq i \leq 5000$


$1 \leq |s_i| \leq 10^4$

## 輸出說明

請你找出符合 $s_i = s_j$ 的字串對，其中 $i < j$


輸出 $i$ 和 $j$


保證必定且只有一組解

## 範例輸入 #1

```
1234
5678
5677
5678
```

## 範例輸出 #1

```
2 4
```
