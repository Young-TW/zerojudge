# f717. 抽籤

## 內容

文文班上的籤筒不見了，班導發明了一個看似很公平的抽籤方式。

全班每個人選一個非負整數，班導把它們全部加總得到一個數字，然後從一號開始報數，報到最後一號之後再從一號開始。報到這個總和的人就中籤了。

玩了一次以後，班導發現這樣的抽籤方式實在太浪費時間了，於是請你幫忙寫一個程式，讓每個人輸入他所選的數字，然後自動計算出中籤的座號。

## 輸入說明

輸入只有一行，含有全班每位同學所出的數字。

## 輸出說明

輸出中籤同學的號碼。

## 範例輸入 #1

```
4 2 3 1 5
```

## 範例輸出 #1

```
5
```

## 範例輸入 #2

```
0 1 2 3
```

## 範例輸出 #2

```
2
```
