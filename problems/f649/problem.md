# f649. 暐賢的體重

## 內容

暐賢想量體重，但是他前一陣子發生了點意外，膝部的韌帶受傷，再加上最近腳趾又凍甲去開了刀，所以他根本沒辦法自己量體重，於是他來找你幫他量體重。
你有台專門用來秤重的特殊機器，這台機器通常是用來秤特殊物質用的，其重量往往是天文數字的等級，所以機器內有特殊演算法把所秤得的重量轉成至少能以C++的long long變數所表示的形式，也就是轉成N^M的形式來顯示。
一秤下去出現了兩個數字，暐賢當然不懂其表達的意思，所以需要你向他解釋，然而這個數字非常大，你看到時的表情就像天竺鼠車車一樣驚訝，但為了不要讓他太難過，你決定只告訴暐賢他體重位數的位數。

## 輸入說明

測資為多筆輸入，EOF結尾。


每行有兩個非負整數N、M，中間以空白隔開。（N<2^63、M<2^63）

## 輸出說明

對於每個輸入請輸出一個正整數W，代表暐賢體重位數的位數。（每個答案以換行隔開）

## 範例輸入 #1

```
5 3
9 53
95 3
953 953
```

## 範例輸出 #1

```
1
2
1
4
```
