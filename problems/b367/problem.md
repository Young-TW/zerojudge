# b367. 翻轉世界

## 內容

有位勇士排除萬難後,來到了最後一關-----[翻轉世界]

他發現所有物品都翻轉了180度

如果要繼續前進必須做出一項選擇,也就是找到翻轉後不會改變的東西!

 例如:

110

000

011 

若翻轉180度:

110

000

011 

現在給你一張圖 

請你幫忙這位勇士吧,因為除了必須判斷這個問題以外,還必須打倒翻轉世界的怪物才行

所以請你寫一個程式幫他

## 輸入說明

每行有一正整數T(T<11)，代表有幾組測試資料

接下來有T行， 每行有N、M兩正整數

N代表長,M代表寬 (0<N,M<11)

而接下會有N行,每一行會有M個數字 r (0<=r<231-1)

## 輸出說明

對於每個測資,判斷是否可以符合翻轉180度不會改變的圖形是的話請輸出 go forward

否的話請輸出 keep defending

## 範例輸入 #1

```
2
3 3
1 1 0
0 1 0
0 1 1
1 5
0 0 1 0 1
```

## 範例輸出 #1

```
go forward
keep defending
```
