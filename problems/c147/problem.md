# c147. 105北二5搬家規劃問題

## 內容

![image](ShowImage?id=964)

## 輸入說明

測試資料共有三列。第一列有A(1<=A<=1000)個以一個空格隔開的整數W(1<=W<=1000)代表物品重量。第二列同樣有
有A個以一個空格隔開的整數V(1<=V<=1000)代表物品的需求度。第三列有一個整數L(1<=L<=1000000)，代表載重量。

## 輸出說明

輸出１個整數，代表最多可搬運的總需求度。

## 範例輸入 #1

```
範例一
1 1 1 1 2 2 3
1 1 2 3 1 3 5
7
範例二
815 906 127 914 633 98 279 547 958 965
158 971 958 486 801 142 422 916 793 960
5000
```

## 範例輸出 #1

```
範例一
13
範例二
5963
```
