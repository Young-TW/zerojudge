# b576. 北門街

## 內容

念板中別說沒吃過北門街，最近還有由板中生推出的手機軟體「府中小吃趣」，作者好像和之前很夯的詰哥還有b528的加密系統(sin(cos(X*PI/180)))有關，從這個軟體上發現，北門街附近買兩碗義大利濃湯麵和帥乾的小魯，要花49+49+25元，總共123元


 


問題：


一串數字s只含1、2、3，取inv(s)，就是把字串s的所有字元xi變成4-xi


Ex : s = 1123 則 inv(s) = 3321


 


定義數列：an = an-1 + inv(an-1)，遞迴式中的加法是字串加法運算


 


前三項如下


a1 = 123


a2 = 123321


a3 = 123321321123

## 輸入說明

每組測試資料含有多筆輸入


第一行有一個T，代表有T筆測試資料


以下T行，每一行含有一個整數K，詢問apcsh的左數第K個字元，其中pcsh = 102015


 


40%測試資料滿足


1<=K<=106


 


100%測試資料滿足


1<=K<=109


1<=T<=100

## 輸出說明

請輸出apcsh的左數第K個字元，其中pcsh = 102015

## 範例輸入 #1

```
3
4
5
10
```

## 範例輸出 #1

```
3
2
1
```
