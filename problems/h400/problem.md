# h400. 蛋糕店促銷（困難版）

## 內容

※請注意，此題為困難版，簡易版與困難版的唯一差異在於$N$與$T$的範圍。


最近蛋糕店的生意不好，為了吸引客人來光顧，蛋糕店的老闆最近在想要做一個促銷活動，活動內容大致如下。


在蛋糕店裡設置了摸彩箱，摸彩箱裡面有很多紙卡，每個紙卡上面都寫著一個正整數。而來光顧的每個客人都可以進行一次摸彩，假設客人摸彩摸到了數字$M$，他可以使用特定規則找出一個數字$K$，並且用$10M$的價格買走$K$個蛋糕，找出$K$的方法如下：


將數字$M$拆成最少$1$最多$M$個正整數，並且將這些正整數相乘，所得到的值即為$K$。


而老闆為了評估成本，要事前知道所有紙卡最多可以買走幾個蛋糕，希望你能幫助他，但由於答案可能很大，請將答案取$998244353$的餘數後再告訴老闆。

## 輸入說明

輸入的第一行有一個正整數$T$($1≤T≤2×10^5$)，代表老闆要問你$T$個不同紙卡。


接著有$T$行，每行有一個正整數$N$($1≤N≤2^{32}$)，代表老闆問你數字$N$最多可以買走幾個蛋糕。

## 輸出說明

對於每一個$N$，請輸出一行，每行有一個數字$K$，代表數字$N$最多可以買走$K$個蛋糕。

## 範例輸入 #1

```
3
5
4
1
```

## 範例輸出 #1

```
6
4
1
```
