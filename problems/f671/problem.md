# f671. FJCU_109_Winter_Day1_Lab6 最短路徑

## 輸入說明

第 $1$ 行給定兩個整數 $N,M(1 \leq N,M\leq 10)$ 表示為一個 $N\times M$ 的網格，第 $2$ 到 $N+1$ 行每行有 $M$ 個字元，字元可為 `.`(代表路) 或 `#`(代表障礙)，在路上可以上下左右的走，但不能超出邊界。

## 輸出說明

輸出從左上角 $(1,1)$ 走到右下角 $(N,M)$ 最少需要花多少步，如果走不到，請輸出 $-1$

## 範例輸入 #1

```
3 5
.#...
.#.#.
...#.
```

## 範例輸出 #1

```
10
```
