# j362. 因數

## 內容

輸入一個數，輸出這個數的所有因數。用戶可決定是否輸出1和這個數，並且輸出因數的總和及個數(根據用戶的選擇)。


<=1數字<=1258904

## 輸入說明

輸入只有二行，第一行為是一個整數a，第二行則是用戶的選擇。如果用戶想輸出1和這個數，則輸入"print"，否則輸入"no"。

## 輸出說明

若干行，每行一個整數。


輸出完因數後，以"a的因數的個數是xxx，a的因數的總和是xxx"的格式輸出因數的個數及總和

## 範例輸入 #1

```
12
print
```

## 範例輸出 #1

```
1
2
3
4
6
12
12的因數的個數是6，12的因數的總和是28
```

## 範例輸入 #2

```
15
no
```

## 範例輸出 #2

```
3
5
15的因數的個數是2，15的因數的總和是8
```
