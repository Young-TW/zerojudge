# k391. 五子棋好難!QAQ

## 內容

每個人都有棋力值


如果你的對手比你棋力值高


你就會 LOSE


如果棋力值一樣


就TIE


如果你棋力值比他高


就WIN

## 輸入說明

輸入自己名字 棋力值


輸入對手名字 棋力值

## 輸出說明

輸出 win的人,WIN!


如果打平，輸出(tie)

## 範例輸入 #1

```
SomeOne 60
H 30
```

## 範例輸出 #1

```
SomeOne WIN!
```

## 範例輸入 #2

```
S 2
q 2
```

## 範例輸出 #2

```
tie
```

## 範例輸入 #3

```
S 69
g 70
```

## 範例輸出 #3

```
g WIN!
```
