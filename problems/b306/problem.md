# b306. 極．板條大冒險 (二)：比較之門

## 內容

板擦高中是魔法之國首都、可茵城的第一高校學府，其主流學科「魔法」也是在魔法之國最盛行，且最為注重的學科。

在板擦高中裡面，一位天才魔法師老妮可，正在用現代魔法構造一款遊戲－－板擦大冒險 ver 0.0.1！遊戲中的主角稱為 jackyXX，主角必須和 NPC 茵可接許多任務，並且一一克服這些挑戰，等級才能更加提升。例如剛進到遊戲中，jackyXX 必須測試茵可的第一道關卡，關卡內容如下：

「輸入第一行有一個整數 T，代表測試資料組數。對於每組測試資料有兩行，第一行有兩個整數 N、M，N 代表角色的寬度，M 代表這條柏油路的寬度可以被切成 M 段描述，而第二行有 M 個整數 w_i，代表第 i 段柏油路的寬度。對於每組測試資料，如果角色能抵達村莊，請輸出『YEEES!!! INKER!』（不含引號），否則輸出『NOOOO!!! JACKY XX!』（不含引號）。」

設計好第一個關卡之後，時間已經到半夜 3 點，正當心滿意足要離開學校時，可茵城外轟然巨響，老妮可趕到可茵城外，一條由柏油鋪成，道路兩側種滿林木的「柏林大道」上，發現有數個「門」從天上掉下。

這些「門」有大有小，這些「門」的長度由一個奇數 s 來表示，例如下面為一個長度為 5 的「門」：

```
xxxxx...xxxxx
x...x...x...x
xxxxx...xxxxx
x...x...x...x
xxxxx...xxxxx
x...........x
x...........x
x...........x
```

圖中「門」由字元「x」來表示，字元「.」代表空區域。「門」左右對稱，左側由一個長度為 s 的「日」字以及長度為 s-2 的「丨」構成，且左右兩個「日」之間間隔 s-2。

又例如以下是長度為 7 的「門」：兩個長度為 7 的「日」、兩個長度為 5 的「丨」，左右間隔為 5。

```
xxxxxxx.....xxxxxxx
x.....x.....x.....x
x.....x.....x.....x
xxxxxxx.....xxxxxxx
x.....x.....x.....x
x.....x.....x.....x
xxxxxxx.....xxxxxxx
x.................x
x.................x
x.................x
x.................x
x.................x
```

這些「門」散落在柏林大道上，並且「門」與「門」之間不會重疊。

「難道 ......」想起前些日子引起一陣轟動新聞：在紅圓茵可的住處有一條通往異世界的傳送門。

「發生異變了？」老妮可不禁打起冷顫。目前還是先統計出柏林大道上有多少個「門」，以及最大的門的長度到底是多少好了。

## 輸入說明

輸入不超過 N 行，每行不超過 M 個字元，代表柏林大道。

1 <= N,M <= 50
門的長度 s 至少為 5
有不超過 60% 的測資 N,M >= 20
有不超過 20% 的測資 N,M >= 30

## 輸出說明

對於每筆輸入，輸出兩個整數：門的數量以及最大門的長度。若沒有門，則輸出「0」(不含引號)。

## 範例輸入 #1

```
xxxxx...xxxxx
x...x...x...x
xxxxx...xxxxx
x...x...x...x
xxxxx...xxxxx
x...........x
x...........x
x...........x
```

## 範例輸出 #1

```
1 5
```
