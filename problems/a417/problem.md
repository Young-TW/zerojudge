# a417. 螺旋矩陣

## 內容

輸出一螺旋矩陣。

## 輸入說明

每行有一正整數T，代表有幾組測試資料

接下來有T行， 每行有N、M兩正整數

N為矩陣長寬，就是會有N*N矩陣

M為方向，M=1為順時鐘，M=2為逆時鐘

N範圍為1~100之間

## 輸出說明

把矩陣輸出，矩陣值之間寬度為5，就是[00000]寬度

C++可用setw(5)或C的%5d輸出

## 範例輸入 #1

```
2
3 1
2 2
```

## 範例輸出 #1

```
    1     2     3 
    8     9     4 
    7     6     5 

    1     4 
    2     3 
```
