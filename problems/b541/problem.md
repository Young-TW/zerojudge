# b541. 看到這樣的分布，網友們都驚呆了

## 內容

外觀數列，數列的第n項描述了第n-1項的數字分布。它以1開始：


- 　　a1 = 1
- 　　1：讀作1個「1」，即a2=11
- 　　11：讀作2個「1」，即a3=21
- 　　21：讀作1個「2」，1個「1」，即a4=1211
- 　　1211：讀作1個「1」，1個「2」，2個「1」，即a5=111221
- 　　111221：讀作3個「1」，2個「2」，1個「1」，即a6=312211

## 輸入說明

輸入有多筆，每筆一行，包含一個整數n(1<=n<=40)

## 輸出說明

對於每筆輸入，輸出一行，表示外觀數列第n項，保證答案長度小於105

## 範例輸入 #1

```
1
5
```

## 範例輸出 #1

```
1
111221
```
