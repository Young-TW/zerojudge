# d625. 踩地雷真好玩

## 內容

Windows遊樂場的踩地雷真是太好玩了！
可是現在踩地雷掛掉了，
遊戲規則顛倒過來，給你地雷的分佈位置
請你找出遊戲中每一格的"周圍地雷數"！

踩地雷的遊戲是這樣的，
有個5×5個棋盤格如下：
-----
-----
-----
-----
-----

假設現在有個地雷：
-----
-----
---*-
-----
-----

那麼地雷的周圍就會出現提示的數字，告訴你"該格週圍"有幾顆地雷
-----
--111
--1*1
--111
-----

如果說有兩顆以上的地雷，當然也會反應出來
-111-
-1*21
-12*1
--111
-----

當然，提示數字最高到八
12321
2***2
3*8*3
2***2
12321

## 輸入說明

只有一筆測資。
第一行給定正整數n(n<=100)，表示全部的棋盤格是n×n格
接下來的n行，每行有n個字元代表棋盤格
「-」代表空格
「*」代表地雷

## 輸出說明

請按照說明的方式填入各格的"周圍地雷數"
如果地雷數是0的話請維持「-」的樣子。

## 範例輸入 #1

```
7
------*
-*-----
-*-----
-*-----
---***-
----*--
-------
```

## 範例輸出 #1

```
111--1*
2*2--11
3*3----
2*32321
112***1
--13*31
---111-
```
