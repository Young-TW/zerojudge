# b182. 3. 矩形的內部與外部

## 內容

平面上有若干個矩形，任意給定一個點，我們要決定哪些矩形包含這個點，點落在矩形的邊界上也視為被包含，你的程式要計算包含這個點的所有矩形的面積和。

![image](images/problems/b182.gif)

## 輸入說明

第一行是兩個整數m，n，1<m<31，1<n<10，m代表矩形數，格式為x1 y1 x2 y2，其中(x1，y1)與(x2，y2)為一對角線上的兩點，每一個座標值均為非負整數且不大於40000，但是可能是給左上角與右下角，也可能是左下角與右上角，順序不一定，例如：0 0 5 8，指的是左下角在(0，0)而右上角在(5，8)；3 7 2 8，則是右下角(3，7)與左上角(2，8)。

N代表點的數量，每行代表一個點座標(x，y)。

## 輸出說明

對於所給的n 個點， 每一行輸出包含該點的所有矩形的面積和

## 範例輸入 #1

```
4 3
0 0 5 5
3 3 1 1
14 15 20 7
5 8 10 5
5 5
2 4
15 1
```

## 範例輸出 #1

```
40
25
0
```
