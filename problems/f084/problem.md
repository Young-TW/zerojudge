# f084. 斯坦的函數範圍

## 內容

說到「斯坦」，你會想到什麼?


有的人可能會說它是波斯文的後綴，常常用在國家或地區的名稱裡，像是巴基斯坦(Pakistan)。


有些人可能會說它是英文姓名翻譯成中文包含的字，像是愛因斯坦(Einstein)、康斯坦丁(Constantine)。


有些人可能會說它是饒舌歌手Eminem的一首歌(Stan)，後來泛指他的粉絲。


但今天這個斯坦，只是一個小孩。
他的全名叫斯坦·馬什(Stan Marsh)，他與他的家庭住在一個小鎮裡，這是他的大頭照



![image](ShowImage?id=1575)


他原本住在街區，後來爸爸當了農夫而搬到農場，他做過許多事，像是搭乘宇宙飛船、扮演超級英雄阻止小鎮被毀滅、阻止兩個國家的戰爭、成為一個宗教的教主、進入虛擬世界跟自己的帳號大戰……
但是這些都太平常了，真正令他煩惱的是他的作業


今天老師出了「函數」，有一個函數 $\color{black}{f(x)=c_1x+c_2}\ $，$\color{black}{1≤f(1)≤3}\ $，$\color{black}{4≤f(2)≤5}\ $，請問你 $\color{black}{f(3)}\ $ 的範圍?


斯坦想了好久都想不出來，想請你幫忙：

## 輸入說明

你以為題目只有這樣嗎，斯坦除了這一題，還有好多題也不會


第一行有一個數 $\color{black}{t}\ $，代表測資筆數


每筆測資第一行有一個數 $\color{black}{n}\ $，代表 $\color{black}{f(x)}\ $ 為 $\color{black}{n}\ $ 次函數： $\color{black}{f(x)=c_1x^n + c_2x^{n-1} +……+ c_nx + c_{n+1}}\ $， $\color{black}{c_k}\ $ 代表係數(並不會給你 $\color{black}{c_k}\ $ )


接下來 $\color{black}{1\sim n+1}\ $ 行在第 $\color{black}{i}\ $ 行時有兩個數 $\color{black}{a_i,b_i}\ $，代表 $\color{black}{a_i≤f(i)≤b_i}\ $


- $\color{black}{1≤t≤100}\ $
- $\color{black}{1≤n≤20}\ $
- $\color{black}{-100≤a_i≤b_i≤100}\ $

## 輸出說明

輸出 $\color{black}{A,B}\ $，代表 $\color{black}{A≤f(n+2)≤B}\ $

