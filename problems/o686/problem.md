# o686. 喵-數字迷宮

## 內容

給你一個三維數字迷宮
每走一步就會加總該格子的數字


請找出數字最小的路徑並輸出

## 輸入說明

只有一個測資
每筆測資第一行有1個數字N
( 1 <= N <= 60)
接下來有N個區塊
每個區塊有N列N個數字n (1 <= n <= 2147483647)


從起點 (0, 0, 0) 走到終點 (N - 1, N - 1, N - 1)
每走一格就會增加座標中的數字

## 輸出說明

請找出數字總和最小的路徑並輸出

## 範例輸入 #1

```
3

3 1 2 
1 1 2 
3 2 3 

2 2 2 
1 2 2 
3 2 1 

2 1 1 
1 2 1 
3 2 2
```

## 範例輸出 #1

```
11
```
