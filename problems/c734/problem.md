# c734. Sequence

## 內容

有一個神奇的序列


設s1 = 1, s2 = 12, s3 = 123, ..., s10 = 12345678910, s11 = 1234567891011


而這個神奇的序列S就等於s1 + s2 + s3 + ....


所以S=112123123412345123456123456712345678123456789123456789101234567891011...


現在給你一個x，求出S序列上第x個位置的數字是多少

## 輸入說明

第一行有一個數字T，代表測資筆數


接下來T行都有一個數字x，代表詢問S序列上第x個位置的數字是多少


 


對於90%的測資


1 <= x <= 10000


對於100%的測資


1 <= x <= 2147483647

## 輸出說明

輸出T個數字，每個數字用空格隔開，代表每筆測資的答案

## 範例輸入 #1

```
5
1
3
5
7
9
```

## 範例輸出 #1

```
1 2 2 1 3
```
