# f289. 求極限

## 內容

剛上大一理工學院的惡夢想必就是微積分了,諸多令人感覺困惑的名詞,例如極限、連續等等,現在有一位身陷苦海的大一學生向你求助,麻煩你幫他寫一個簡單的程式來求極限。因為只是大一上學期,所以給定的f(x)只有可能是多項式或者是分式型的函數,而且最高次方皆<=2。

## 輸入說明

limit f(x)


x->n


多筆測資


每一行開始會有一個n 代表x趨近於n , n有可能是整數或"inf"(無限大)或"-inf"(負無限大) 其中一種 


再隔一個空格會有1個整數 k 表示x項的係數


再隔一個空格會有x項的次方數 格式為x^2或x


一個函數會以{係數 次方數} 這樣的格式來表示


如果是分式函數則會以兩個函數中間加一個 '/' 表示


請注意:1.x項的次方數=0時只會有x項的係數而不會有次方數


            2.係數是0,次方項也不會寫出來


            3.分式型的函數會寫成 p(x) / q(x) 詳細請看範例

## 輸出說明

對每一個測資輸出計算出來極限的值(保證是整數)


如果極限不存在或值等於無窮大或負無窮大請輸出"limit doesn't exist"

## 範例輸入 #1

```
inf 1 x^2 -2 x 3 / 1 x 3
inf 3
3 2 x^2 -3 x -1
1 1 x^2 -1 / 1 x -1
0 1 / 1 x

```

## 範例輸出 #1

```
limit doesn't exist
3
8
2
limit doesn't exist
```
