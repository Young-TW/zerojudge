# r580. 10432 - Polygon Inside A Circle

## 內容

給你圓的半徑, 和內接正多邊形的邊數, 你要算出正多邊形的面積

 

![image](ShowImage?id=4863)

## 輸入說明

輸入含有多組測試資料。每組測試資料一列包含兩個數字,  r ( 0 < r < 20000 ) 代表圓半徑， n ( 2 < n < 20000 ) 代表內接正多邊形邊數。

## 輸出說明

對於每一列測試資料，輸出圓內接正多邊形的面積，請四捨五入到小數點後3位。

## 範例輸入 #1

```
2 2000
10 3000
1 4 
0.7071 4
3.2 5 
```

## 範例輸出 #1

```
12.566
314.159
2.000
1.000
24.347
```
