# b518. 樹葉節點到根節點之路徑-商競103

## 內容

這是103學年度商業類學生技藝競賽模擬題 ，原題對樹的介紹頗長，略！


給一棵樹，就「一」棵而且一定是「樹」，問葉節點往上走到根節點的路徑，葉子及根不算。

## 輸入說明

第1列一個數字 N, 1<=N<=5, 代表有N棵樹，接著每棵樹的第1列為一個數字m, 1<=m<=80 ，代表這棵樹有 m 個節點，而且編號由 0 ~ m-1，接著 m列   「 i,j 」， i的父節點編號為j，若i為根節點，則 j=99 。

## 輸出說明

每棵樹的輸出不一定幾列，依葉節點而定{編號由小至大}，樹與樹之間空一列。


每個葉節點一列，先輸出葉節編號接冒號，以{ }括住路徑，由下至上的編號，以逗號隔開，但若沒有中間節點，則在冒號後只輸出N

## 範例輸入 #1

```
3
7
0,99
1,3
2,3
3,5
4,6
5,0
6,5
4
0,99
1,0
2,0
3,0
1
0,99

```

## 範例輸出 #1

```
1:{3,5}
2:{3,5}
4:{6,5}

1:N
2:N
3:N

0:N
```
