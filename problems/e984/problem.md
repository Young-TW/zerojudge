題目:


一個正整數$X$若滿足下列條件：


在$X$用$10$進位表示時，若$X$的每位數字都與前/後一位數字相差不超過1，則$X$是一個$Lunlun Number$ (好lun的名字(X))


給你一個正整數$K$，找出第$K$個$Lunlun Number$

## 輸入說明

一正整數$K$($K<=10^6$)

## 輸出說明

第$K$個$Lunlun Number$

## 範例輸入 #1

```
1
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
15
```

## 範例輸出 #2

```
23
```

## 範例輸入 #3

```
100000
```

## 範例輸出 #3

```
3234566667
```
