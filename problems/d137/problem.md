# d137. 绝对值

## 內容

从初中就已经学了绝对值的定义与应用，今天我们就要编一个程序，输入一个数（只要是数就行了），输出它的绝对值。

## 輸入說明

每行一个数，可能是实数，可能是虚数，可能是复数。实数一定是整数，复数表示为a+bi（i^2=-1），其中a和b都是整数，虚数ai中a也是整数。

## 輸出說明

输出这个数的绝对值。保留3位小数。

## 範例輸入 #1

```
0
1
-1
+1
i
-i
5+2i
4+i
-3-i
-3-4i
+3+4i
```

## 範例輸出 #1

```
0.000
1.000
1.000
1.000
1.000
1.000
5.385
4.123
3.162
5.000
5.000
```
