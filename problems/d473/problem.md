# d473. 秒杀求幂题(求幂系列题4)

## 內容

为了求幂挑战的顺利进行，请先来这儿做准备。

## 輸入說明

每行一个a n，(a,n∈Z)当输入0 0时输出‘All Over.’和多余行数结束。 (注意样例)

## 輸出說明

每行一个a的n次幂的值。-2147483648<=an<=2147483647, a,n∈Z。

## 範例輸入 #1

```
6 5
567 3
2 25
-1 4
0 -0
-9 7
00000000000000 111111111111111111111111111
0 0
32 6
86 9
0 1
0 0
4 5
6 9
```

## 範例輸出 #1

```
7776
182284263
33554432
1
0    {不是'0 0',不能退出;输出'0'是本题的唯一“规定”}
-4782969
0

```
