# l014. 棋譜

## 內容

小明剛剛下完了棋，想在棋記錄在棋譜裡，請你編寫一段程序，協助小明解決難題。

## 輸入說明

輸入第一行為x，y，代表棋盤有多少行(x 橫)和列(y 縱)。


接下來有y行，繪制出一個棋盤，棋盤裡的數字代表步數，沒有棋子的地方為"-"，詳見範例輸入。

## 輸出說明

輸出共有若干行(總步數行)，為每一步的坐標。


格式: "第?步為第y行，第x列"


y值為第幾列(縱)，x是指第幾行(橫)。


註:這裡的行和列皆以"-"或數字為單位。

## 範例輸入 #1

```
10 10
----1-----
----------
----------
----------
----------
------2---
--------5-
---4------
----------
---------3
```

## 範例輸出 #1

```
第1步為第1列，第5行
第2步為第6列，第7行
第3步為第10列，第10行
第4步為第8列，第4行
第5步為第7列，第9行
```

## 範例輸入 #2

```
4 5
--1--
2----
-----
-----
```

## 範例輸出 #2

```
第1步為第1列，第3行
第2步為第2列，第1行
```
