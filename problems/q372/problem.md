# q372. 7. 變異數

## 內容

給定正整數 $N$，以及包含 $N$ 個正整數的序列 $a$。
　　有 $Q$ 筆詢問，對於每筆詢問 $l, r$，請輸出 $a_l\sim a_r$ 的變異數 $\text{Var}_{[l, r]}$。


\begin{aligned}
\mu_{[l, r]} =& \frac{1}{r-l+1}\sum_{i=l}^{r}a_i\\
\text{Var}_{[l, r]} =& \frac{1}{r-l+1}\sum_{i=l}^{r}({a_i}-μ_{[l, r]})^2
\end{aligned}

## 輸入說明

輸入的第一行有兩個正整數 $N, Q$（$1\le N, Q\le 10^6$），代表序列 $a$ 的長度與詢問數量。
　　第二行有 $N$ 個正整數 $a_i$（$1\le a_i\le 4\times 10^6$），為序列 $a$ 的元素內容。
　　接下來有 $Q$ 行，每行有兩個正整數 $l, r$（$1\le l\le r\le N$），代表每筆詢問的區間範圍。

## 輸出說明

對於每筆詢問，請將 $\text{Var}_{[l,r]}$ 四捨五入到整數，並輸出於一行。

## 範例輸入 #1

```
5 4
3 2 4 1 5
2 2
1 3
3 4
1 5
```

## 範例輸出 #1

```
0
1
2
2
```
