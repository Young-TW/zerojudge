# l003. 棋盤

## 內容

小明想下棋，但是他身邊沒有棋盤，因此他想把坐標記錄下來，再呈現圖像出來。請你編程一段程序，幫助小明解決問題。


棋盤為10*10。

## 輸入說明

輸入共有n行，以EOF結束。


格式: (y值,x值)


y值為第幾列(縱)，x是指第幾行(橫)。

## 輸出說明

輸出共有十行，詳見範例輸出。棋盤裡的數字代表第幾步。

## 範例輸入 #1

```
(1,5)
(6,7)
(10,10)
(8,4)
(7,9)
```

## 範例輸出 #1

```
----1-----
----------
----------
----------
----------
------2---
--------5-
---4------
----------
---------3
```

## 範例輸入 #2

```
(1,1)
(3,7)
(2,8)
```

## 範例輸出 #2

```
1---------
-------3--
------2---
----------
----------
----------
----------
----------
----------
----------
```
