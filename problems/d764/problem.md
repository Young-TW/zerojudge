# d764. 10606 - Opening Doors

## 內容

主角：Dima
題目敘述的時間：發生在即將要放暑假....
某間學校有N個學生，每個學生都有自己的衣物櫃(有幾個學生就有幾個衣物櫃)，假設一開始所有的衣物櫃都是關的，當第i個學生走過所有的衣物櫃時，會順手改變櫃子編號可被 i 除盡的衣物櫃狀態，即關閉原本開啟的櫃子，或開啟原本關閉的櫃子。

## 輸入說明

每一行有一個整數Ｎ，當Ｎ為０時，請不要處理並結束，學生人數範圍：(1 ≤ N ≤ 10100)。

## 輸出說明

求當所有學生都走過一遍後，開啟的櫃子中，編號最大的是幾號？

## 範例輸入 #1

```
１０１００
９０
１
０
```

## 範例輸出 #1

```
１００００
８１
１
```
