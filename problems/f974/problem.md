# f974. 我看你…完全是不懂喔

## 內容

阿偉和彬彬去網咖玩遇到杰哥


杰哥：「不好意思，我剛剛聽到你們兩個說肚子餓。我這個剛好有個麵包，我還不餓，請你們吃。」
阿偉：「先吃先吃。」
杰哥：「對了，我叫阿杰，我也常來這裡玩，他們都叫我杰哥。」
阿偉、彬彬：「杰哥好！」
杰哥：「你們好，我一個人住，我的房子還滿大的，歡迎你們來我家玩，玩累了就…直接睡覺，沒問題的。」


阿偉：「你覺得咧？」
彬彬：「我覺得怪怪的。」
阿偉：「看起來就是一個奇怪的人啊！不要理他。」
彬彬：「不要去不要去。」


杰哥：「我常常幫助一些翹家的人，如果你們不要來的話，也沒有關係。如果要來的話，我等一下可以帶你們去超商，買一些好吃的喔！」
阿偉：「有東西可以吃耶，要不要去啊？」
彬彬：「好啦，不然去好了。」


阿偉：「去一下好了啦。那杰哥，我跟我朋友今天就去住你家囉！」
杰哥：「好啊，沒問題啊，那走啊。我們現在就去超商買一些吃的。」
阿偉：「好啊！」
杰哥：「走走走。」


-------------------------


阿偉：「你去那邊，你去那邊。」
阿偉：「泡麵！」
彬彬：「小泡芙耶！」
杰哥：「都可以拿。」
阿偉：「謝謝杰哥！」


阿偉：「好多飲料喔，有酒耶！」
彬彬：「不要看酒了啦，先買飲料啦！」
彬彬：「什麼都可以拿嗎，杰哥？」
杰哥：「都可以拿。」
阿偉：「真的假的？」
杰哥：「隨便拿你們隨便拿。」
阿偉：「真的可以嗎？」
杰哥：「可以拿，都拿。」
阿偉：「謝謝杰哥！」


-------------------------


杰哥：「你看，你看這個彬彬，才喝幾罐就醉了，真的太遜了。」
阿偉：「這個彬彬就是遜啦！」


杰哥：「聽你這麼說，你很勇喔？」
阿偉：「開玩笑，我超勇的好不好，我超會喝的啦。」


杰哥：「超會喝？很勇嘛，身材不錯喔，滿結實的啊。」
阿偉：「杰哥，你…幹嘛啊？」
杰哥：「都幾歲了，還那麼害羞，我看你…完全是不懂喔～」
阿偉：「懂什麼啊？」

杰哥：「你想懂？我房裡有一些好康的。」
阿偉：「好康，是新遊戲喔？」
杰哥：「什麼新遊戲，比遊戲還刺激，還可以教你登大郎喔！」
阿偉：「登大郎？」


杰哥：「對啦，來啦，來看就知道了，來啦。」
 阿偉：「哎呀。」

阿偉：「幹嘛啦？杰哥，你有好多遊戲喔！」
杰哥：「唉唷，那沒什麼，來，來看這個好康的。」
阿偉：「杰哥，這是什麼啊？」
杰哥：「唉唷，你臉紅啦？」
杰哥：「來，讓我看看！」

阿偉：「不要啦！」
杰哥：「讓我看看！」
阿偉：「不要啦，杰哥，你幹嘛啊！」
杰哥：「讓我看看你正不正常啊！」
阿偉：「杰哥，不要啦！」
杰哥：「聽話！」


杰哥：「讓我看看！」

阿偉：「不要！」

阿偉：「杰哥不要啦，杰哥不要！杰哥，杰哥不要杰哥，杰哥不要！」


彬彬突然出現！！！

彬彬：「只有今天限定，彬彬姐，泡湯 ✌ ！」
彬彬：「第一次，我在台北這個地方叫作什麼呀…杰哥家，杰哥家。」
彬彬：「啊～～你看這麼多讚，是看到我的身材讚。」
彬彬：「肥嫩肥嫩，謀跨怕勝。」


彬彬：「今天彬彬姐在這，喔～～享受兩人世界。太棒了，挖嘎哩共，哩喀豐滿齁，哩愛喀挖賊。」
彬彬：「哩那挖幾勒，ㄆㄧㄚˇ，哩耨莫秋。」
彬彬：「不是啦，你不能側面，側面就會拍到。」
杰妹：「不會啦，安捏妹阿。」


彬彬：「真的嗎？」
杰妹：「不會啦。」
彬彬：「拍你喔拍你喔，哈哈哈哈哈。這一個，比我大隻的。」
彬彬：「唉唷，疑，這樣拍不就不會拍到…疑，拍到我的腳。」


彬彬：「我們在這邊泡湯，這個湯真的是很棒的齁，叫做無色無味。」
彬彬：「啊，我把…蹲進來。無色無味，這個地方叫做杰哥家。」
彬彬：「啊，然後這個房子，叫作杰哥家。我覺得一定是跟那個"杰哥"是有關係的啊。」


彬彬：「因為我們實在沒有別的地方可以拍給你們看。啊，ㄏㄧˋ 安奈就不會ㄏㄧˋ 豆。」
杰妹：「欸喔欸喔欸喔。」
彬彬：「不會不會不會，ㄏㄧˋ 安奈就不會。」


彬彬：「齁這是我們的外面。」
彬彬：「啊，阿係賣冒險，掰掰，不能冒險。」

事情是這樣的，彬彬是一個女生，跟杰哥已經結婚三年了，這是他們新買的房子，所以彬彬才會說是第一次泡湯。彬彬在阿偉面前假裝不認識杰哥是想要給阿偉一個驚喜，然後杰哥想讓阿偉給他看看的是他手機遊戲上的角色。


彬彬得到很多讚，數量是 1+x+x^2+x^3+…+x^n，給你 x 跟 n，請你求出彬彬的讚數。

## 輸入說明

每行輸入正整數 x 和 n


x=0 和 n=0 代表結束


1≤ x ≤10^6，1≤ n <2^63

## 輸出說明

輸出 1+x+…+x^n


因為答案可能會很大，請把答案 %1000000007 後再輸出

## 範例輸入 #1

```
2 1
3 2
4 4
1 100
5 1000
1000000 2147483648
0 0
```

## 範例輸出 #1

```
3
13
341
101
211933906
67476844
```
