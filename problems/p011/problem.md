# p011. 我愛偶數 (之偶數大道)

## 內容

文文愛偶數，而且他就住在偶數大道的原點。在偶數大道 1 ~ 𝑚 之間 (含) 的每個正整數座標上有一個偶數，而且由小到大排列。


文文每天早上都會出門散步，他會先用電腦亂數產生一個正整數 𝑥，然後出門在偶數大道上散步，一路欣賞那些令人賞心悅目的偶數，一直到他看到第一個 ≥ 𝑥 的偶數時，再心滿意足地從那個座標折返走回家。


例如偶數大道上的偶數依序為：


座標 1  2  3  4  5  6  7  8 偶數248812141620
第一天的亂數 𝑥 為 16，他會走到 16 的所在位置座標 7 再走回來。
第二天的亂數 𝑥 為 5，第一個 ≥ 5 的偶數是 8，他會走到座標 3 再走回來。
第三天的亂數 𝑥 為 25，他會走到座標 9 才發現已經沒有偶數了，然後失望地走回家。


𝑛 天後，文文要你幫忙計算一下他總共走了多少的距離。

## 輸入說明

輸入的第一行含有兩個整數 𝑚, 𝑛 (1 ≤ 𝑚, 𝑛 ≤ 105)，代表偶數大道上有 𝑚 個偶數，文文一共走了 𝑛 天。


第二行含有 𝑚 個由小到大的偶數 𝑒𝑖 (2 ≤ 𝑒𝑖 ≤ 109)，依序在偶數大道上的整數座標上。


第三行含有 𝑛 個正整數 𝑥𝑖  (1 ≤ 𝑥𝑖 ≤ 109)，是文文每天早上所得到的亂數 𝑥。

## 輸出說明

輸出這 𝑛 天文文所走的總距離是多少。

## 範例輸入 #1

```
8 3
2 4 8 8 12 14 16 20
16 5 25
```

## 範例輸出 #1

```
38
```
