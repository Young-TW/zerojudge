# e990. 奇怪的隕石

## 內容

![image](ShowImage?id=1521)


Caido有一天晚上在散步，突然看到一顆隕石從天上掉下來，Caido走到游標處，發現隕石就落在那。


Caido看到隕石發著光，心裡想著這應該可以賣很多錢吧，於是就把隕石帶回家，隕石足足有20公斤重，可把Caido累壞了。


Caido隔天早上起來，迫不及待地要把隕石拍賣掉，拿起了隕石，結果發現隕石變輕了，拿去秤，發現只剩10公斤。


「咦，怎麼會這樣?」Caido不解地看著隕石，決定尋找隕石專家米特瑞亞。


米特瑞亞說這個隕石是放射性物質，會進行衰變，放射性衰變通常都有一定的周期，並且一般不因物理或化學環境而改變，原子的某些衰變會產生出另一種元素，並放出某些粒子，半衰期代表此元素在這個期間過後，未衰變的原子會剩下原來的二分之一。



![image](ShowImage?id=1522)


米特瑞亞仔細端詳了這塊隕石，發現其轉變後的元素是氧氣，會逸散掉，所以半衰期過後，隕石的質量會變為原本質量的一半，Caido睡覺時間的8小時就是這顆隕石的半衰期。


Caido恍然大悟，於是他想要問你，一顆隕石的半衰期為T小時，轉變後的元素都會逸散掉，請問從原本一顆完整的隕石變成n倍重(0<n<1)要花多久呢?

## 輸入說明

輸入有多行，每行有兩個三位浮點數T、n


0<T≤10^6，0<n<1

## 輸出說明

請輸出半衰期為T的隕石質量變成n倍要花多久


答案請四捨五入到小數第三位

## 範例輸入 #1

```
8.000 0.500
12.345 0.250
6.666 0.666
```

## 範例輸出 #1

```
8.000
24.690
3.909
```
