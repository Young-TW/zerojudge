# g256. 考拉茲猜想

## 內容

考拉茲猜想（英語：Collatz conjecture），又稱為奇偶歸一猜想、3n+1猜想、冰雹猜想、角谷猜想、哈塞猜想、烏拉姆猜想或敘拉古猜想，是指對於每一個正整數，如果它是奇數，則對它乘3再加1，如果它是偶數，則對它除以2，如此循環，最終都能夠得到1。

這個猜想已被驗證在n小於等於268以下的正整數都成立，現在我想委託你寫一個程式計算對於一個正整數n，需要變換多少次才可以達到1

## 輸入說明

每一行有一個正整數n(n<=106)


讀到EOF結束

## 輸出說明

對每一行的n輸出變換到1所需要的次數

## 範例輸入 #1

```
1
2
3
```

## 範例輸出 #1

```
0
1
7
```
