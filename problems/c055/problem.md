# c055. 00568 - Just the Facts

## 內容

請寫一個程式求出 N! 從右邊算來最後一個不是0的數字.(0<=N<=10000)

   

 NN! 0 1 1 1 2 2 3 6 4 24 5 120 10 3628800
    例如：

  3!=6 所以你應該要輸出 6. 

  5!=120 所以你應該要輸出 2.

  10!=3628800 所以你應該要輸出 8

## 輸入說明

輸入包含多筆資料，每筆資料一列，包含一個整數N。(0<=N<=10000)

  對每筆測試資料輸出一列，格式：第1到第5欄為輸入的N（靠右對齊），第6到第9欄為” -> “，第10欄為N! 從右邊算來最後一個不是0的數字。請參考Sample Output。

## 範例輸入 #1

```
1
2
26
125
3125
9999
```

## 範例輸出 #1

```
    1 -> 1
    2 -> 2
   26 -> 4
  125 -> 8
 3125 -> 2
 9999 -> 8
```
