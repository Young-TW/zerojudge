# j030. 11945 - Financial Management

## 內容

Michael 今年從 ITESO 畢業，終於找到了一份工作。 他賺了很多錢，但不知何故似乎永遠不夠。 Michael 已經決定他需要抓住他的金融投資組合併解決他的融資問題。 第一步是弄清楚他的錢是怎麼回事。
Michael 有他的銀行賬戶對賬單，想看看他有多少錢。


請幫助 Michael 編寫一個程式，從過去十二個月的每個月中獲取他的期末餘額並且計算他的平均賬戶餘額。

## 輸入說明

第一行為一個數字 N (1 ≤ N ≤ 100)，代表數據數量


每個數據集由十二行組成。


每行將包含他的銀行賬戶在特定月份的期末餘額。

## 輸出說明

對於每組數據，請輸出數據編號，空格，字符$，十二個月平均值(四捨五入到美分)，並且使用逗號分隔千位數

## 範例輸入 #1

```
1
100.00
489.12
12454.12
1234.10
823.05
109.20
5.27
1542.25
839.18
83.99
1295.01
1.75
```

## 範例輸出 #1

```
1 $1,581.42
```
