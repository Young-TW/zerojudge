# c914. 導盲磚

## 內容

天龍國為照顧視障朋友行路的安全，準備重新鋪設行人道的導盲磚，已知導盲磚鋪設區域為一寬度2單位，長度 n 單位的區塊，
今規劃使用兩種導盲磚鋪設，一種為2*1矩形磚，一種為1*1正方形磚，試問可以有多少種不同的鋪設方式？
例如：

![image](ShowImage?id=1244)

## 輸入說明

第一行一個整數 K，代表接下來有 K 筆測資，每筆測資一行一個整數 n
(n <= 30)

## 輸出說明

每行輸出對應 n 的組合數。

## 範例輸入 #1

```
3
1
2
3
```

## 範例輸出 #1

```
2
7
22
```
