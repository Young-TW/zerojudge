# b352. 相似

## 內容

小A有3n根小棍，他把每三根小棍分为一组，编号为1，2，...，n。
第i组小棍的长度分别为Ai，Bi，Ci。
小A把每一组小棍组成三角形，有几组小棍不能组成三角形，所以小A把这些组的木棍先放在一边。

现在，小A的桌子上有m个三角形，编号为1，2，...，m。
对于第i个三角形，将除其本身外，这m个三角形中与它相似的三角形个数记为Xi。
由于小A刚学了相似三角形，所以他想知道，Xi（1≤i≤m）的最大值。

## 輸入說明

输入第一行仅一个正整数n。
接下来的n行，每行有三个整数Ai，Bi，Ci。
意义如题目所述。

## 輸出說明

输出文件仅一行，包含用空格隔开的两个数，m和Xi的最大值。
意义如题目所述。

## 範例輸入 #1

```
3
1 1 1
2 2 2
1 2 3
```

## 範例輸出 #1

```
2 1
```
