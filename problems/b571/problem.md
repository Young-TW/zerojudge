# b571. 不要慌，這是孔明的陷阱！

## 內容

話說某一天turtlebee不小心把她唯一的男裝忘在學校了，當她想回學校拿的時候，發現XX在學校各地布滿了孔明的陷阱。「叭叭叭叭叭叭叭叭～叭叭～～叭叭叭叭叭叭叭～yee」XX不知道在說著什麼。總之turtlebee勢必是要把男裝拿回來的。她發現孔明的陷阱有三種，就稱為A型陷阱、B型陷阱、跟C型陷阱好了，這三種陷阱會輪流啟動，當一個陷阱啟動時若turtlebee剛好在那個位置上就會發生很恐怖的事情。「妮可妮可妮~~」XX不知道在說著什麼。turtlebee只能往上下左右移動，一開始只有A型陷阱是啟動的，走一步之後變成只有B型陷阱是啟動的，再走一步之後只有C型陷阱是啟動的，再走一步又換A型啟動，以此類推。若下一步某位置的陷阱會啟動，那turtlebee下一步就不能在那位置上。

## 輸入說明

第一行有兩個數字n, m(1<=n, m<=500)。
 接下來有n行，每行有m個字元，代表這張地圖。
 # - 牆壁
 . - 空地
 A - A型陷阱
 B - B型陷阱
 C - C型陷阱
 X - 起點
 Y - 終點

## 輸出說明

輸出從X到Y最少需要幾步。若X無法走到Y則輸出-1。

## 範例輸入 #1

```
3 10
##########
#X.CABC.Y#
##########
```

## 範例輸出 #1

```
9
```
