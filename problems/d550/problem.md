# d550. 物件排序

## 內容

Excel 的排序，使用上真是有點麻煩
現在請你先對第一列先做排序，
若還是相同，再比較第二列 ...類推 ( 由小→大)

## 輸入說明

每組輸入的第一行會有兩個數字 N M  ( 1 ≦ N ≦ 1,0000 , 1 ≦ M ≦ 200 ) 

分別代表接下來有 N 行 , 每行上有 M 個數字 P ( 0 ≦ P ≦ 2147483647 )

## 輸出說明

請輸出排序後的結果。

## 範例輸入 #1

```
8 3
1 1 2
1 0 1
1 1 1
1 3 2
1 3 5
2 1 1
2 3 2
2 1 4
```

## 範例輸出 #1

```
1 0 1
1 1 1
1 1 2
1 3 2
1 3 5
2 1 1
2 1 4
2 3 2
```
