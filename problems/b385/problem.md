14階以上的 A陣列 同10階先建m/2階的奇階幻方每格 -1後*4，然後每格分成4小格，而B陣列由m-4階的B陣列複製而得，m階B 由m-4階的最上2列複製往上增2列、最下2列複製往下增2列、最左2行複製往左增2行、最右2行複製往右增2行，當m階的B陣列由m-4階的B陣列往四周擴增完成後，再與m階的A陣列相加即得m階幻方。 

例如14階的A陣列就是先建 7 階幻方，再將每1格值 -1 後再*4，然後每1格分成4格即成 14*14的A陣列，而14階的B陣列即由10階的B陣列複製而成，最上2列複製往上擴增、最下2列複製往下擴增即得14列，然後最左2行複製往左擴增(皆是b2)、最右2行複製往右擴增(皆是b3)即得14行。

依題意規定建好m階的幻方後，請輸出第r列第c行的值，例如6階的第5列第3行值為36、10階的第1列第10行值為67。

## 輸入說明

第1列輸入一正整數 t, { 1<= t <= 10 }，接著 t 列、每列 3 個正整數 m , r , c 以空白隔開、其中m=4*n+2 { 1<=n<=1000 }且{1<= r,c <=m}

## 輸出說明

共輸出 t 列，每列1個正整數 k , { 1<= k <=m^2 }

## 範例輸入 #1

```
3
6 5 3
10 1 10
4002 4002 4002
```

## 範例輸出 #1

```
36
67
8004002
```
