# b540. 這個六邊形，看完後我都驚呆了

## 內容

六個角都是120度的六邊形的框框，可以用若干個邊長為1正三角形來填滿，想要知道符合上述條件的六邊形框框，需要預備幾個邊長為1的正三角形來填滿

## 輸入說明

輸入有多筆測試資料，每行有六個整數a1,a2,a3,a4,a5,a6(1<=ai<=1000)代表一個六邊形框框的六邊長(依照順時針順序)，並保證這個六邊形符合每個角都是120度

## 輸出說明

對於每個六邊形輸出一行，代表這個六邊形需要預備多少個正三角形來填滿

## 範例輸入 #1

```
1 1 1 1 1 1
1 2 1 2 1 2
```

## 範例輸出 #1

```
6
13
```
