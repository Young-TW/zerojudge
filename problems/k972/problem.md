# k972. 12293 - Box Game

## 內容

有2個一樣的盒子，其中一個盒子有 n 顆球，另一個盒子則只有1顆球。Alice和Bob發明了一個遊戲，玩法如下：


 Alice和Bob輪流搬球，由Alice先。每次玩家把兩個盒子中球較少的那個盒子中的球倒掉（倒掉的球永遠都不會再出現），然後玩家重新 分配另一個盒子中的球到這2個盒子中（每個盒子中至少要有1顆球）。假如某一個玩家無法做一個有效的搬球，他就輸了。下面是一個例子：n＝5



![image](ShowImage?id=3352)


當這2個盒子都僅有1顆球時，Bob無法再搬球了（因為2個盒子中的球一樣多），所以Bob輸了，而Alice是贏家。


假如Alice和Bob都很聰明，都是超厲害的玩家（就是只要遊戲有可能贏，他就一定會贏），給你 n ，請問誰會是贏家？

## 輸入說明

輸入含有多組測試資料。每組資料一列，含有一個整數 n ( 2 <= n <= 109)。


當n=0時，代表輸入結束。

## 輸出說明

請每組測試資料輸出誰會是贏家。

## 範例輸入 #1

```
2
3
4
0
```

## 範例輸出 #1

```
Alice
Bob
Alice
```
