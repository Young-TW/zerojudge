# m775. 超級細菌

## 內容

一種超級細菌繁殖速度驚人，每天可以生出 p 隻相同的細菌，
第1天 --->  1 隻
第2天 --->  增加  p  隻
第3天 --->  增加  p^2  隻
第4天 --->  增加  P^3  隻
第5天 --->  增加  p^4  隻
.....
試問第 k 天細菌的總數為何？
 



![image](ShowImage?id=3728)

## 輸入說明

第一行輸入一個整數T，代表共有 T 筆測資 (T<=1000)，之後共有 T 行，每筆測資兩個整數 p、k (1<=p<=2*10^9 ,  1<=k<=2*10^9 )

## 輸出說明

每筆測資輸出一行，代表第 k 天細菌的總數。數字很大，因此 mod 1000000007 ，輸出格式如範例。

## 範例輸入 #1

```
4
1 5
2 3
5 4
123 123
```

## 範例輸出 #1

```
Case 1: 5
Case 2: 7
Case 3: 156
Case 4: 868208613

```
