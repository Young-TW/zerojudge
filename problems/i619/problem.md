# i619. 我看你…完全是不懂喔 - Extreme

## 內容

原題



![image](ShowImage?id=3447)


這個男人實在是太狠了，面對五米高的壯漢，輕輕一推竟然就把他推倒在地。


男人名叫老鼠，是一名住在起士國的學生，這天他與兩名好友大壯和小美走在路上，結果不小心撞到同樣也走在路上的黑老大，這時不出意外的話就出意外了，黑老大把老鼠叫住，你絕對想不到他接下來做的事會改變老鼠的一生。


你有見過這樣子的男人嗎？只見老鼠撞到了走在路上的黑老大，黑老大不但沒有生氣，還對老鼠說要帶他去見一個人。老鼠隨著黑老大進入一個陌生的房間，裡頭走出一名男子，男子自稱杰哥，杰哥對著老鼠的第一句話就是：「讓我看看！」老鼠對著杰哥大喊：「不要！」就把五米高的杰哥推倒在地上了，這時 FBL 衝了進來，把杰哥和黑老大抓住，並把老鼠帶回去審訊。


你有見過這麼奇葩的審問方式嗎？警官大白面對老鼠，拿出一張字條，上面寫著 $1+x+ x^2 + \dots + x^n\ (\bmod m)$，給你 $x, n, m$ 請你算出這條算式的結果。老鼠人狠話不多，一下就算出來了。

## 輸入說明

第一行有一個正整數 $t$，代表測資筆數。


接下來 $t$ 行，每行有三個正整數 $x, n, m$，代表要請你算出 $1+x+ x^2 + \dots + x^n\ (\bmod m)$，注意 $m$ 不一定是質數。


- $1 \leq t \leq 10^4$
- $1 \leq x < m \leq 2 \times 10^9$
- $1 \leq n \leq 10^{18}$

## 輸出說明

每行輸出一個數字代表答案。
