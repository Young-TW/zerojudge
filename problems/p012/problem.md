# p012. 收集盒子

## 內容

在一條長長的直線走道上，依次擺放著若干個正方體盒子。每個盒子有一個邊長。小明只能從左到右走一次去收集盒子。他手上只能拿一個盒子，但是如果遇到一個比較大的盒子，可以將手上較小的盒子放入那個較大的盒子中，再拿起那個較大的盒子。


請幫助小明計算，他最多可以收集幾個盒子？

## 輸入說明

第一行是一個整數 𝑛 (1 ≤ 𝑛 ≤ 100），代表盒子的數量。


第二行是 𝑛 個整數，代表依序擺放在走道上的盒子的邊長。

## 輸出說明

輸出一個整數，代表小明最多可以收集的盒子數量。

## 範例輸入 #1

```
5
2 3 1 4 6
```

## 範例輸出 #1

```
4
```

## 範例輸入 #2

```
9
2 3 4 8 9 1 5 6 7
```

## 範例輸出 #2

```
6
```
