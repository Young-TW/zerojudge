# b841. 104北二5.骨牌遊戲

## 內容

5．骨牌遊戲 (104北二區 桃竹苗 )


問題描述


骨牌遊戲是一個很有興趣的益智遊戲，其目標是找到棋盤上越多的骨牌越好。


範例一：以下左為一個輸入範例：    　而以下右為骨牌的例子：「骨牌」是數字相同且相鄰的兩格綁在一起。


https://i.imgur.com/DGSFy1R.jpg

![image](ShowImage?id=1974)


以這個例子來說，我們最多可以找到8個骨牌。現在請你寫一個程式來幫忙求出骨牌最多可以找到的個數。

## 輸入說明

輸入的第一列的兩個正整數 H，W為棋盤的高度和寬度(1<=H,W<=6)，從第2列開始為棋盤第一列由左到右的數字，依此類推，輸入值皆為1~9的整數。數字間均用空格(white space)隔開。

## 輸出說明

輸出資料為一個正整數或0，表示骨牌最多可以找到的個數。

## 範例輸入 #1

```
4 5
1 1 2 1 2
3 1 2 2 2
1 1 1 2 3
4 4 1 3 3
```

## 範例輸出 #1

```
8
```
