# n693. 10721 - Bar Codes

## 內容

條碼符號(bar-code)由交替的深色條和淺色條組成，從左邊開始是一個深色條。每個條的寬度以單位表示。Figure 1顯示了一個由4個條組成的條碼符號，這些條總共延伸了1 + 2 + 3 + 1 = 7個單位。



![image](ShowImage?id=4009)


一般而言，條碼符號(bar-code) BC(n, k, m) 是所有符號的集合，這些符號由k個條組成，總共延伸了正好n個單位，每個條的寬度最多為m個單位。例如，Figure 1中的符號屬於 BC(7, 4, 3)，但不屬於 BC(7, 4, 2)。Figure 2顯示了 BC(7, 4, 3) 中的所有16個符號。每個“1”代表一個深色單位，每個“0”代表一個淺色單位。



![image](ShowImage?id=4010)

## 輸入說明

每個輸入將包含三個正整數 n、k 和 m（1 ≤ n, k, m ≤ 50）。

## 輸出說明

對於每個輸入，輸出 BC(n, k, m) 中符號的總數。輸出將適合64位有符號整數。

## 範例輸入 #1

```
7 4 3
7 4 2
```

## 範例輸出 #1

```
16
4
```
