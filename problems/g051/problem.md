# g051. 最多點的線

## 內容

給你 $Ｎ$ 個點，請你找出最多有幾個點在同一條直線上。

## 輸入說明

第一行一個整數 $N$，代表總共有 $N$ 個點


接下來有 $N$ 行，每行一個點 $(x_i, y_i)$，保證點不重複出現


保證最多點的那條直線上有超過 $\frac{N}{4}$ 個點


$2 \leq N \leq 2 \cdot 10^5$


$-2^{31} <= x_i, y_i <= 2^{31}-1$

## 輸出說明

輸出一個整數，最多有多少點在同一條直線上。

## 範例輸入 #1

```
10
1 1
2 2
5 123
5 922
5 1982730
5 12280
3 3
98 1111
2222 3333
4444 98
```

## 範例輸出 #1

```
4
```
