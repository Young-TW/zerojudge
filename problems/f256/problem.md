# f256. 宇辰的平均成績

## 內容

段考結束後又到了揭曉成績的時刻，宇辰迫不及待的打開信封想看看自己的平均成績有沒有及格。然而他在拆信封的時候不小心打翻了桌上的咖啡，把平均成績的部分弄髒了，現在他只好自己算平均成績。只是這次考試的科目太多了，他實在無法計算，所以他想請你幫他寫程式計算他的平均成績。


 


平均成績只要在60以上就算是及格；反之則為不及格。


平均成績的計算是由各科目的成績乘以該科目學分數並加總後除以總學分數再無條件捨去至整數得來的。例：


宇辰考了2個科目，分別為數學與國文。


數學共4學分，宇辰考了35分，則為35×4=140分。
國文共3學分，宇辰考了70分，則為70×3=210分。


所以宇辰的平均分數就是(140+210)÷(4+3)=50分，因此宇辰這次段考又不及格了。

## 輸入說明

每筆測資有N行輸入，以EOF結尾。(0<N<=5×10^6)


每行有G,C兩個整數，分別代表宇辰該科考試成績與學分數。(0<=G<=100、0<C<=4)

## 輸出說明

如果宇辰的平均成績有及格，請輸出「Oh wow! You pass it!」(不含引號)，並在下一行輸出平均成績。


如果宇辰的平均成績又不及格了，請輸出「YEEEEEE!!!」(不含引號)

## 範例輸入 #1

```
35 4
70 3
```

## 範例輸出 #1

```
YEEEEEE!!!
```

## 範例輸入 #2

```
60 2
75 3
```

## 範例輸出 #2

```
Oh wow! You pass it!
69
```
