# f776. 方形數字 （ 基礎題 ）

## 內容

![image](ShowImage?id=1729)


原點在左下角。


x 方向左而右由 1 ~ N, y 方向下而上由 1 ~ N 的一個方形數字。


請算出每組 （x1, y1）to（x2, y2）的數字合計。

## 輸入說明

輸入的第一列有一個數字 N 表示橫向及縱向的最大值。 0 < N < 10^50


第二列有一個數字 Ｔ 表示詢問的筆數。


接著有 Ｔ 列 x1, y1, x2, y2    N >= x2 > x1, N >= y2 > y1


請計算從 （x1, y1）到 （x2, y2）間數字的總和。

## 輸出說明

每列詢問輸出一個數字。

## 範例輸入 #1

```
10
5
3 3 5 7
2 5 8 8
3 1 8 3
3 2 7 5
2 2 9 8
```

## 範例輸出 #1

```
79
192
99
104
364
```
