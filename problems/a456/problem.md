# a456. 子集合

## 內容

輸出一集合中的所有子集合

## 輸入說明

每行有一正整數T，代表有幾組測試資料

接下來有T行， 每行有一正整數N

為1~N個集合，1 <= N <= 15

 ex: 5

{1, 2, 3, 4, 5}

 找出其所有子集合

## 輸出說明

輸出該集合所有子集合

空集合請用{0}輸出 

順序為集合元素由少至多，相同時，以數字小的先輸出

每筆資料間隔一空行  

以3為例是:

{0}

{1}

{2}

{3}

{1,2}

{1,3}

{2,3}

{1,2,3}

## 範例輸入 #1

```
2
1
2 
```

## 範例輸出 #1

```
{0}
{1}

{0}
{1}
{2}
{1,2}
```
