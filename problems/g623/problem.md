# g623. 泊愷愛排組

## 內容

泊愷是個熱愛排組的小孩，有一天泊愷出給與紅一個排組問題，但是與紅解不出來，想請你幫忙！！


給你一個長度為N的整數數列A1,A2,A3.....AN，請問有多少組(i,j,k)，(i<j<k)使得Ai,Aj,Ak相異

## 輸入說明

第一行有一個整數T，代表有T組側資。


每組側資第一行有一個整數N，第二行有N個整數A1,A2,A3...AN。


T<=20


3<=N<=10^5


1<=Ai<=10^9

## 輸出說明

輸出T行，代表每筆測資中有幾組(i,j,k)，(i<j<k)使得Ai,Aj,Ak相異

## 範例輸入 #1

```
4
3
1 2 3
5
1 2 3 4 5
4 
1 2 3 3
4
3 3 3 3
```

## 範例輸出 #1

```
1
10
2
0
```
