# b940. 聖誕廢文

## 內容

上週是一年一度的聖誕節，雖然今年的聖誕節熱的跟什麼一樣，但仍然不難看到路上被雪白紅綠所妝點著，大街小巷無論男女老少都在歡慶耶誕，除了出題者以外。


在這樣一個歡樂溫腥的日子裡，本應是和家人朋友出遊的阿！然而出題者卻必須獨自一人待在家裡，作出題這種麻煩的事情。看著FB上別人在外打卡PO文放閃，心情更是鬱悶。


不過還好，待在家裡仍然可以玩電腦，沒有電腦仍然可以玩手機，就像非洲人沒有電腦還是能玩踩地雷，沒有手機還是能玩部落衝突一樣的小確幸。


        不過在這之前，出題者還是必須把題目出完，不然你們就不知道這題怎麼寫了。雖然那樣好像沒差，因為這樣大家都沒有分，對於結果沒什麼影響，


除非你們用輸入說明和輸出說明就可以判斷出這題在說什麼。但其他人都出了，如果出題者不出就好像沒做什麼事，只好來打打廢文刷一點存在感。


        這題的題目是這樣的：「給你一個正整數n，請問在小於等於n的正整數中，共有多少個數與n互質呢？」雖然只是這樣短短的一句，但沒辦法，


出題者還是必須把他巧妙的藏進一堆屁話之中，真的是有夠麻煩，不知道是誰開始這種風氣的，明明打一行簡潔又有力，解題者也不用看出題者的牢騷。


        看看字數大概也差不多了，那這題就差不多到這裡囉！YA！

## 輸入說明

每筆測資包含1行1個正整數n。


-測資點1滿足n≤106
-測資點2滿足n<231
-測資點3滿足n<264
-測資點4滿足n<264

## 輸出說明

對於每筆測資請輸出1行一個整數代表有多少個小於等於n的正整數與n互質。

## 範例輸入 #1

```
5
30
100
```

## 範例輸出 #1

```
4
8
40
```
