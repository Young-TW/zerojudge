# b369. [福州19中]因子和阶乘

## 內容

输入正整数n，把阶乘n!=1*2*3*...*n分解成素因子相乘的形式，从小到大输出各个素数的指数。

  例如120=2^3*3*5可以表示成(3,1,1)，表示分别有3、1、1个2、3、5。你的程序应该忽略比最大素因子更大的素数（否则末尾将会有无限多个0.）

## 輸入說明

一行，一个数，表示n。

## 輸出說明

一行，若干个数，表示n的每个素因子的个数，具体参见题目说明。请注意输出的格式。

## 範例輸入 #1

```
53
```

## 範例輸出 #1

```
53!=49 23 12 8 4 4 3 2 2 1 1 1 1 1 1 1
```
