# r476. Zzz的數列儀式

## 內容

在Zzz王國中，傳說有一個神秘的「數列儀式」
給定一個正整數 n，Zzz會依照以下規則生成一個儀式數列：

- 
初始數列僅有一個元素：n

- 
若目前數列最後一項為 x，則下一項依下列規則決定：

- 
若 x 為偶數，則下一項為 x/2

- 
若 x 為奇數，則下一項為 3x+1

- 
若產生過重複的數字（即數列中出現過同樣的數），儀式立即停止

Zzz想知道：
這個儀式數列的「長度」與「最大值」各是多少？

## 輸入說明

輸入一個整數 n

- 
1≤n≤10^6

## 輸出說明

輸出兩個整數：

- 
數列的長度 L

- 
數列中的最大值 M

## 範例輸入 #1

```
6
```

## 範例輸出 #1

```
9 16
```
