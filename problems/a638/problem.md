# a638. Closest-pair problem

## 內容

在二維座標平面上有 N 個點，

請找出最近兩點的距離。

距離公式如下：

![image](ShowImage?id=462)

## 輸入說明

每個測資點只有一組測資

第一行只有一個整數 N (1 < N ≤ 3,000,000) 代表有幾個點

接下來 N 行每行有兩個整數 x, y (0 ≤ x, y ≤ 10,000,000) 代表每個點的座標

座標已排序且不會有重複的點

## 輸出說明

請輸出最近點對的距離

答案請四捨五入到小數點後第四位

## 範例輸入 #1

```
3
1 7
7 10
10 9
```

## 範例輸出 #1

```
3.1623
```
