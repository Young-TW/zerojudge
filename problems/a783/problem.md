# a783. 5. Houston Skyline

## 內容

給一些建築物的配置和高度，請寫一個程式描繪出郝斯敦( Houston )天際線的陰影。

## 輸入說明

提示1：每棟建築物的起始點(從左開始)[1-60]、寬度[1-40]、高度[1-20]，建築物的數據是以室內來測量而非室外，所以寬度1、高度6的建築物實際上有 寬度3x高度8 的陰影。

提示2：輸入多組測資，每組第一行為一個整數n代表n( 1 ≤ n ≤ 16 )棟建築，接下來n行，每行有三個數字，分別代表建築物的 起始位置、寬跟高，天際線總共有50行，最大高度為20

## 輸出說明

輸出最後的陰影，較小的建築物會被較大的建築物遮住。

## 範例輸入 #1

```
4
3 3 3
10 5 2
4 3 7
10 1 10
```

## 範例輸出 #1

```
         +-+
         | |

   +---+ | |
   |   | | |
```
