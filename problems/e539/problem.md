# e539. 00967 - Circular

## 內容

循環質數：當每個最左邊的數字(最高有效數字)又移到右側時，仍保持質數的性質。
例如，數字19937是循環質數，因為序列19937、99371、93719、37199、71993中的所有數字皆是質數。
您的目標是寫一個程式，在給定的範圍計算該範圍內的循環質數。

## 輸入說明

每個輸入有兩個整數i和j。
100 <= i,j < 1000000，且 i <= j。
如果該行只有一個"-1"代表輸入結束。
請計算i和j之間(包括i和j)，有幾個循環質數。

## 輸出說明

對於每對輸入
如果範圍內沒有循環質數
輸出"No Circular Primes."
如果範圍內只有1個循環質數
輸出"1 Circular Prime."
如果範圍內有n個循環質數
輸出"n Circular Primes."

## 範例輸入 #1

```
1000 1100
100 120
100 1000
-1
```

## 範例輸出 #1

```
No Circular Primes.
1 Circular Prime.
12 Circular Primes.
```
