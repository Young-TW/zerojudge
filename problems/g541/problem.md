# g541. 類題:兔子跳躍(TOIP)

## 內容

這次新手組有一題兔子跳躍，問:兔子一開始在位置0，每次可以往前跳N或M步，能否跳至位置D，被電神秒殺(2分鐘)


今天，臨末把題目稍微加強一點點，交給各位秒殺囉!!!

## 輸入說明

第一行有3個正整數N,M,Q，表示兔子可往前跳N或M步，Q表示詢問數(1<=N,M<=200)(1<=Q<=1000000)


第二行有Q個整數D，詢問你兔子是否有可能跳至位置D(1<=D<=2147483647)

## 輸出說明

針對每筆詢問，如果兔子可能跳到位置D，輸出"YES"，否則輸出"NO"，然後換行

## 範例輸入 #1

```
2 3 5
1 2 3 4 5
```

## 範例輸出 #1

```
NO
YES
YES
YES
YES
```

## 範例輸入 #2

```
7 6 2
15 17
```

## 範例輸出 #2

```
NO
NO
```
