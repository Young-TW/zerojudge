# e355. Squares and Rectangle in Rectangle

## 內容

單位格子整齊排列的$N \times M$矩形，請問有多少正方形及長方形?


 


#這裡的長方形指鄰邊不相等的矩形。

## 輸入說明

輸入包含很多行$(<900,000)$，以EOF結束。


每行包含兩正整數$N, M$。


$N \leq 10,000$


$M \leq 10,000$

## 輸出說明

請輸出正方形及長方形個數。

## 範例輸入 #1

```
1 2
3 4
5 6
```

## 範例輸出 #1

```
2 1
20 40
70 245
```
