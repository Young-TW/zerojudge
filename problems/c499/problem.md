# c499. ♋大耳神下麵真好吃♋

## 內容

有一天, 有一個耳朵很大的神在吃晚餐時,突然心血來潮,想要出個Zerojudge題目, 於是他決定用他的大耳朵聽聽看平民百姓聊天的內容, 看看能不能因此想到有趣的故事來當題目敘述. 結果, 這位大耳神只聽到了一些不是太無聊就是太低級的話. 他一直想不到好點子, 最後受不了了, 就很生氣的把他吃的麵條往大地丟.


地球上各洲各個國家的人民看到麵條從天上掉下來時都覺得很驚訝,一邊叫 "餌神下麵給我們吃!" 一邊吃大耳神下的麵. 有一個信仰大耳神的少女艾希知道, 大耳神一定是生氣了, 才會發生這種事. 她開始向大耳神祈禱, 希望祂可以原諒世人. 同時, 地球的另一邊也有一個少男迪克也是大耳神的信徒, 他也開始祈禱了. 大耳神用他的大耳朵聽到他們的祈禱聲, 但因為兩個人同時說不同的東西, 所以大耳神並聽不太清楚. 兩位信徒知道了這件事, 發現了他們的禱詞分別可以被字串M,B表示, 而清晰度就是這兩個字串的最長共同子字串的長度. 大耳神要聽到清晰度T以上的祈禱才會讓他冷靜下來, 不詛咒全世界的人.


現在給你艾希和迪克的禱詞和大耳神能夠接受最小的清晰度, 請判斷大耳神是否會詛咒全世界的人.

## 輸入說明

本體輸入只有單筆測資.
前兩行分別有字串 M 和 B.
第三行有一非負整數 T.


|M|,|B|<=1000
T<min(|M|,|B|)


M 和 B 只有英文大寫字母

## 輸出說明

如果大耳神會詛咒全世界的人請輸出 "sitini na tisa"
如果大耳神不會詛咒全世界的人請輸出 "kwa nini unaendesha"

## 範例輸入 #1

```
THREETIMES
TWENTYTHREE
6
```

## 範例輸出 #1

```
sitini na tisa
```
