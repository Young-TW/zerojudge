# d785. 二、正邊形

## 內容

已知長度的m根棍子，利用棍子連成一正n邊形。連接的條件如下：
一、僅允許以端點連接
二、不能折斷
三、所有棍子必須使用完畢(m>=n)

## 輸入說明

第一行的數字，代表有幾組測試資料，第二行開始的每一行即為一筆測試資料。
每一筆測試資料以空格分開數字：
第一個整數為正n邊形，其範圍為[4,20]的整數；
第二個整數為棍子數目m，其範圍為[4,30] 的整數；
接下來的m個整數，分別代表每根棍子的長度，其範圍為[1,100]的整數。

## 輸出說明

對每一筆測試資料，以一行輸出，如可連成正n方形，則輸出1。否則輸出0。

## 範例輸入 #1

```
2
4 5 1 3 3 4 5
4 8 2 5 6 7 1 4 4 3
```

## 範例輸出 #1

```
0
1
```
