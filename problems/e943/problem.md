# e943. pD. 最大群問題

## 內容

如果有一群人兩兩之間都是朋友，那麼我們稱他們是一個朋友群。
現在有編號 1 ~ N 的 N 個人，和他們之間的朋友關係，請輸出最大朋友群的人數。

## 輸入說明

第一行有兩個整數 N 和 M（ 1 ≤ N ≤ 22, 0 ≤ M ≤ min( N*(N-1)/2, 22 ) ）
分別代表有 N 個人和 M 組朋友關係

接下來有 M 行，每行有兩個整數 a, b（1 ≤ a, b ≤ N）
表示 a 和 b 互相是朋友

## 輸出說明

最大朋友群的人數

## 範例輸入 #1

```
4 4
1 2
2 3
3 1
1 4
```

## 範例輸出 #1

```
3
```

## 範例輸入 #2

```
4 4
1 2
2 3
3 4
4 1
```

## 範例輸出 #2

```
2
```
