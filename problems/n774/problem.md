# n774. 10678 - The Grazing Cow

## 內容

一頭牛正在田野中吃草。田野中的一根繩子被繫在兩根柱子上。牛被繩子通過一個環扣住，所以可以認為牛被繫在繩子的任意一點。你的任務是計算牛能到達並吃草的田地面積。如果需要，假設
![image](ShowImage?id=4067)
（這裡的角度是以弧度為單位）。你還可以假設繩子的厚度為零，牛是一個點對象，環的半徑和柱子的厚度可以忽略不計。請使用雙精度浮點數類型進行浮點數計算。

## 輸入說明

輸入文件的第一行包含一個整數 N（N ≤ 100），表示有多少組輸入數據。接下來的 N 行中，每行包含兩個整數 D（0 ≤ D ≤ 1000）和 L（D < L ≤ 1500）。第一個整數 D 表示兩根柱子之間的距離（以英尺為單位），第二個整數 L 表示繩子的長度（以英尺為單位）。

## 輸出說明

你的程序應該產生 N 行輸出。每行包含一個浮點數，該浮點數保留小數點後三位。這個浮點數表示牛可以到達並吃草的區域面積。

## 範例輸入 #1

```
3
10 12
23 45
12 18
```

## 範例輸出 #1

```
62.517
1366.999
189.670
```
