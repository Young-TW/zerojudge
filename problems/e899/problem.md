# e899. 畢業舞會

## 內容

有 N 位男同學、N位女同學要參加畢業舞會，舞會即將開始，男女同學排成一列對伍準備進入會場；
男同學進入會場後如果一時沒有女伴，男同學會暫時坐在椅子上等待，直到女舞伴出現 (這是風度表現 ^-^)；
女同學進入會場後如果一時沒有男伴，則會非常生氣，立刻轉身回家 (事情大條了 *o*)；
校方希望舞會能成功舉行，每一位同學都能配到舞伴，因此進場時的排隊隊伍是重要的關鍵。
請幫忙寫一程式計算一下，如果要成功配成 N 對舞伴，試問有多少種的隊伍排列方式？
(假設男女同學我們只考慮性別的差異)
 



![image](ShowImage?id=1488)

## 輸入說明

多筆測資 (<=100000 筆)，
每筆測資一行，一個整數 N
(N <= 10000)

## 輸出說明

每筆測資一行輸出，即排隊的方案總數
數字龐大，請輸出 % 10007 的結果

## 範例輸入 #1

```
1
2
10
```

## 範例輸出 #1

```
1
2
6789
```
