# o726. 衛生紙特賣

## 內容

某超商一包衛生紙賣20元，一串有10包衛生紙賣180元，一箱有5串衛生紙賣850元。請問給你衛生紙的需求包數，你可以寫一個程式算看看最少要花多少元採購呢?

## 輸入說明

輸入只有一行，代表你需要的衛生紙 n 包，1 ≤ n ≤ 1000。

## 輸出說明

請輸出一個數字，代表你至少要花費的金額。

## 範例輸入 #1

```
5
```

## 範例輸出 #1

```
100
```

## 範例輸入 #2

```
15
```

## 範例輸出 #2

```
280
```

## 範例輸入 #3

```
350
```

## 範例輸出 #3

```
5950
```
