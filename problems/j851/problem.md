# j851. Rook

## 內容

有一個大小為 n * m 的棋盤，上面有一些城堡，請計算出有幾對城堡可以互相攻擊。


城堡可以朝上下左右攻擊，只是不能穿越。

## 輸入說明

每筆測資的第一列為 n m 表示為 n * m 大小的棋盤。


每個格子裡如為 o 表示空格，


# 表示此格為城堡。

## 輸出說明

請輸出有幾對城堡可以互相攻擊

## 範例輸入 #1

```
3 5
ooooo
oo#o#
###o#
```

## 範例輸出 #1

```
6
```

## 範例輸入 #2

```
5 5
o#ooo
#oooo
o#o#o
ooooo
oo##o
```

## 範例輸出 #2

```
4
```
