# d729. 10593 - Kites

## 內容

給你一個正方形的紙，本身可能有很多洞。你的任務是算出這張紙可以剪成多少種箏形和正方形(任意大小，要超過1x1) ，當然剪下來的圖形不能有洞。

```
 
```

```
                        x
```

```
           x           xxx           xxx           xxx
```

```
          xxx         xxxxx          xxx           x.x         x
```

```
           x           xxx           xxx           xxx
```

```
                        x
```
上面一排左邊三個剪下來圖形是符合的，右邊兩個是不符合的。

## 輸入說明

輸入第一個數為n (n ≤ 500)，代表正方形紙張的邊長。接下來n行每行有n個字元('x' or '.')。'.'代表紙張上的洞 。檔案以EOF結束。

## 輸出說明

輸出為一行，代表能剪下幾種圖形。

## 範例輸入 #1

```
4
.xx.
xxxx
.xx.
.x..
3
xxx
xxx
xxx
```

## 範例輸出 #1

```
4
6
```
