## 輸入說明

輸入由不超過 20測試資料。每個測試資料第一行包含兩個正整數 n（≤40）和k（≤1000000）。其次是N行，每一個包含N個非負整數，給矩陣 A

輸入是終止條件：n = 0。這種情況下不需要進行處理。

## 輸出說明

對於每個測試資料, 你的程式只要計算A + A2 + A3 + ... + Ak . 算出來的答案可能很大, 你只需要輸出最後一個數字就好. 輸出一行空白列在每筆測試資料後。

## 範例輸入 #1

```
3 2
0 2 0
0 0 2
0 0 0
0 0
```

## 範例輸出 #1

```
0 2 4
0 0 2
0 0 0
```
