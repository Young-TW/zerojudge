# e571. 詞語接龍

## 內容

[哥~~~來玩詞語接龍嘛~~~]


[不要！我很忙！]


[陪我玩嘛~~~看在這麼可愛的弟弟份上>3<]


[你是妹妹我就陪你玩]


[~!@#$%^&*()_+]


[你在這樣，我告老師喔！]


[好啦，我給你一張單詞表，只能從上面選詞，一個詞只能用一次，就這樣玩]


[YA~~~]

[真是可惡，該怎麼辦才能讓這遊戲玩最久呢？？？]


大神大神~~~幫幫我~~~~

## 輸入說明

輸入有多組測資。


每組測資第一行有一個整數N (2 <= N <= 100)，代表單詞表上有幾個單詞。


接下來N行，每行一個單詞 (2 <= 單詞長度 <= 10)。


注意：對於每張單詞表，所有詞語的開頭皆不相同，且所有詞語的結尾皆不相同。


詞語接龍只能一個字接續一個字，例如："火上加油"不能接"加油站"


相同字才能接龍，破音字也可接龍(範例測資一)。

## 輸出說明

對於每組輸入，輸出兩行


第一行為詞語接龍可以接多長(單詞數量)


第二行為能夠接出最長詞語接龍的開頭單詞(可以拿來當開頭的單詞皆要輸出)


輸出請按照單詞表順序


如果最長的詞語接龍長度為1(根本不能玩)


請輸出"什麼爛表"

```

```

```

1
什麼爛表
3
火上加油 油燈 燈火
```
