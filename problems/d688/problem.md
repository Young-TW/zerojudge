# d688. 无向图中子图的个数

## 內容

给你一张无向图，求出其连通子图的个数。

## 輸入說明

每组测资第一行为n，m，（1<=n<=20，1<=m<=n*(n-1)/2）用一个空格隔开，分别表示该无向图的节点个数（其节点序号为1...N），该无向图的边数。

接下来有m行，x，y，用一个空格隔开，表示x和y是互相连通的。题目保证不会有重复的路径，而且节点没有连通自己的边。

读到 0 0 这组测资，结束程序。

## 輸出說明

对于每组测资，输出它的子图的个数。

## 範例輸入 #1

```
4 5
1 2
1 3
1 4
2 3
3 4
0 0
```

## 範例輸出 #1

```
14
```
