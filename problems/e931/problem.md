# e931. pA. 直式乘法

## 內容

給定 a, b 兩個正整數，請印出 a 和 b 直式乘法的過程
舉例來說，當 a = 23, b = 204，需要印出：


   23
  204
 ----
   92
   0
 46
 ----
 4692

## 輸入說明

正整數 a, b （1 ≤ a, b ≤ 9999）

## 輸出說明

a 和 b 直式乘法的過程

## 範例輸入 #1

```
23
204
```

## 範例輸出 #1

```
  23
 204
----
  92
  0
46
----
4692
```

## 範例輸入 #2

```
23
12
```

## 範例輸出 #2

```
 23
 12
---
 46
23
---
276
```
