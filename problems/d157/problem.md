# d157. 序章-永夜抄-B

## 內容

在紫與靈夢進行漫長的猜拳對決的同時，紅魔館的主人－吸血鬼蕾米莉亞·斯卡蕾特也察覺到這次滿月的異變

於是將家事交給女僕長十六夜咲夜後就自行出發了 

雖然十六夜放不下心讓她獨自前去，但面對眼前堆積如山的家事卻也莫可奈何

 正當十六夜忙的焦頭爛額之際，她突然想到為何不請人來幫忙呢？

於是就請紅魔館的看門人紅美鈴和圖書館長帕秋莉·諾雷姬來替她分擔工作

但兩者似乎都對此事避而遠之

所以十六夜只得以放她們假三天為籌碼，用猜拳來決定誰要來做家事

## 輸入說明

每筆輸入中有三筆資料，第一筆為十六夜出的拳，第二筆為紅美鈴的，最後為諾雷姬的，中間以逗號間隔

她們三個除了出(Scissors,Stone,Paper)來代表(剪刀，石頭,布)之外

也有可能是以任意一串符號，如(!@#$%^&*)來代表心中的不滿 

(由於她們都不易怒，故不可能在一局中同時不滿) 

除了對每筆測資輸出之外，還需計算誰贏的勝場數最多

輸入的最後為"Game Over"和"End"

請對"Game Over"輸出比賽結果及對"End"輸出結局

## 輸出說明

(以下輸出範例都以X當人名，i代表該局)
  

假如有人出的不是拳，而是如輸入所述的一串符號的話
 
為了平息她的憤怒，請讓她當該局的唯一贏家，輸出"X贏了第i局的比賽"
 
假如三者出的拳剛好平手的話，則輸出"第i局不分勝負"
 
 假如有贏家，且贏家為兩人，則輸出該局的輸家，如"X輸了第i局的比賽"
 
 假如有輸家，且輸家為兩人，則輸出該局的贏家，如"X贏了第i局的比賽"
 

並計算每個人的勝場數(輸了不倒扣，平手不計算)
 
最後對"Game Over"輸出比賽結果(分成三行)
 
第一行為十六夜的勝場數，第二行為紅美鈴的勝場數，第三行為諾雷姬的勝場數
 
並對"End"輸出結局 
 
假如是十六夜贏最多的話輸出"趕上旅程"，如不是的話則輸出"繼續做家事"

## 範例輸入 #1

```
Paper,Scissors,Scissors
Scissors,Stone,Paper
Stone,Scissors,Scissors
@#$%,Scissors,Stone
Paper,Paper,Scissors
Stone,Scissors,Scissors
Game Over
End
```

## 範例輸出 #1

```
十六夜輸了第1局的比賽
第2局不分勝負
十六夜贏了第3局的比賽
十六夜贏了第4局的比賽
帕秋莉·諾雷姬贏了第5局的比賽
十六夜贏了第6局的比賽
十六夜總計贏了3局比賽
紅美鈴總計贏了1局比賽
帕秋莉·諾雷姬總計贏了2局比賽
趕上旅程
```
