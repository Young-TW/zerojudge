# i158. 謝爾賓斯基三角形

## 內容

輸出謝爾賓斯基三角形的前 $n$ 層。

## 輸入說明

輸入只有一個整數 $n$。


$1 \leq n \leq 2048$

## 輸出說明

輸出謝爾賓斯基三角形的前 $n$ 層

## 範例輸入 #1

```
1
```

## 範例輸出 #1

```
*
```

## 範例輸入 #2

```
7
```

## 範例輸出 #2

```
*
**
* *
****
*   *
**  **
* * * *
```

## 範例輸入 #3

```
10
```

## 範例輸出 #3

```
*
**
* *
****
*   *
**  **
* * * *
********
*       *
**      **
```

## 範例輸入 #4

```
16
```

## 範例輸出 #4

```
*
**
* *
****
*   *
**  **
* * * *
********
*       *
**      **
* *     * *
****    ****
*   *   *   *
**  **  **  **
* * * * * * * *
****************
```
