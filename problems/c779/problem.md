# c779. 106北二4.盆栽展覽

## 內容

![image](ShowImage?id=1160)

## 輸入說明

測試資料有兩行。
第一行有 A (1 ≤ A ≤ 30) 個以一個空格(white space)隔開的整數 B (1 ≤ B ≤ 2000)代表盆栽高度。
第二行有一個整數 C (1 ≤ C ≤ 2000),代表相鄰最大高度差。

## 輸出說明

請輸出 1 個整數,代表擺放這排盆栽所需的最低展示架總高度。

## 範例輸入 #1

```
範例輸入一：
401 284 254 412 125
50

範例輸入二：
1010 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000 1000
5
```

## 範例輸出 #1

```
範例輸出一：
412

範例輸出二：
5
```
