# d732. 二分搜尋法

## 內容

給你一個嚴格遞增的數列$A_{1},A_{2},A_{3}.....A_{N}(1\leq N\leq 100000)$,


以及$K(1\leq K\leq 100000)$筆詢問,


每筆詢問給定$X(-2147483648 \leq X\leq 2147483647)$,求數列中是否存在一個$A_{i}(1\leq i\leq N)$的值與$X$相等?

## 輸入說明

第一行包含兩個整數$N$, $K$分別表示數列長度以及詢問數,


第二行包含$N$個整數,第$i(1\leq i\leq N)$個整數依序為數列中$A_{i}$的值,


第三行包含$K$個詢問的整數$X$.

## 輸出說明

對於每個詢問整數$X$對應一行輸出:


輸出$i$的值


其中$1\leq i\leq N$且$A_{i}=X$


若沒有這樣的$i$值請輸出$0$代替.

## 範例輸入 #1

```
5 5
1 3 4 7 9
3 1 9 7 -2
```

## 範例輸出 #1

```
2
1
5
4
0
```
