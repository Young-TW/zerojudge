# d647. 2的次方表示数

## 內容

任何一个正整数都可以用2的幂次方表示。例如 137=2^7+2^3+2^0 .同时约定次方用括号来表示，即a的b次方可表示为 a(b) .由此可知，137可表示为 2(7)+2(3)+2(0) ，进一步 7=2^2+2+2^0 ( 2^1 用 2 表示)；3=2+2^0. 最后 137 表示为 2(2(2)+2+2(0))+2(2+2(0))+2(0). 同理有 1315=2(2(2+2(0))+2)+2(2(2+2(0)))+2(2(2)+2(0))+2+2(0).

## 輸入說明

每行一个自然数n，在 longint 范围内。

## 輸出說明

对于每个n，输出符合约定的n的"0","2"表示，不能有空格。

## 範例輸入 #1

```
2
137
1315
```

## 範例輸出 #1

```
2
2(2(2)+2+2(0))+2(2+2(0))+2(0)
2(2(2+2(0))+2)+2(2(2+2(0)))+2(2(2)+2(0))+2+2(0)
```
