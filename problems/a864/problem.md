# a864. 4. Stellar Classification

## 內容

古時候的人們在描述恆星的”顏色”時大多指肉眼所見的色彩，因此一顆恆星可能同時被形容成淡橙色或是月白色。而天文學家則以”色”來描述恆星的光譜種類，基本種類包含O、B、A、F、G、K和M，其中O是最藍的，M是最紅的，光譜種類由B-V色彩指標(B‐V Color Index)決定，該色彩指標為UBV系統中B波段和V波段的差值：

          B‐V Color Index = mB ‐ mV

  B波段涵蓋了藍色頻率，V波段涵蓋了綠色及黃色頻率。

   

  請寫一個程式，藉由B波段和V波段的值將該恆星分類

![image](ShowImage?id=566)

## 輸入說明

每筆輸入包涵恆星名稱、mB、mV，中間以空白格開，恆星名稱只含英文字母，mB、mV為浮點數精準至小數點以下第二位，讀至”END”結束

## 輸出說明

對於每筆輸入請輸出該恆星名稱、B-V色彩指標和光譜種類，中間以空白格開

## 範例輸入 #1

```
VEGA 0.00 0.00
PROCYON 0.80 0.38
SOL 5.48 4.83
END
```

## 範例輸出 #1

```
VEGA 0.00 A
PROCYON 0.42 F
SOL 0.65 G
```
