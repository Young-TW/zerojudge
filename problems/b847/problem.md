# b847. 一些字串

## 內容

給你很多的字串，請你幫忙統計每個字串中各字母的出現次數及該字母在所有字母所占的比率。

## 輸入說明

很多的字串，每個字串一行，檔案EOF結尾。

## 輸出說明

針對每一個字串輸出兩行數字，每行26個數字，分別代表a~z(不分大小寫)，第一行為個數，第二行為比率(四捨五入至小數點後第二位)。

## 範例輸入 #1

```
apple
banana
cat
dog
```
