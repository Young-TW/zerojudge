# e172. 圍棋入門 - 棋譜

## 內容

阿光：佐為~~現在要下哪裡？


佐為：4之4，星位


阿光：1、2、3，是這裡嗎?


佐為：不是，那是三三


阿光：你是說?對!!


佐為：阿光你再這樣子，我就要附身喔


阿光：對不起，我會乖乖


 


"棋譜"，是初學者最容易進步的地方，觀察別人怎麼下棋，將技巧學習起來，就知道如何應對


你能幫助阿光，將棋譜記下來嗎？

## 輸入說明

第一行有一個數字n，代表下了多少步，n = 0代表輸入結束


0 <= n <= 500


接下來n行，每行有兩個數字x、y代表佐為說的棋盤上的位置x之y


1 <= x,y <= 19


注意！！！棋盤皆為正規棋盤19*19


不同步數的落子位置(x, y)有可能相同


黑子為先手


注意：(1, 1)為最左下角座標，(19, 19)為最右上角座標

## 輸出說明

輸出棋譜


每個棋譜間空一行


著子處，請以步數表示


沒有著子處，請以0表示


如果有發生不同步數在同一點的狀況下


請標明第幾步和第幾步相同


每組一行，如果有多組，請按照第一次著子步數排列

## 範例輸入 #1

```
3
4 4
16 4
4 16
8
1 1
1 2
2 1
2 2
3 2
3 1
2 1
1 1
0
```
