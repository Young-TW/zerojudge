# i863. 12403 - Save Setu

## 內容

達卡大學 CSE 第 13 批（第 12 卷）的 Rahaduzzaman Setu 病得很重。
他長期患有耐多藥結核病。
現在，他的左肺已經受損，無法修復。沒有藥物對他的身體起作用以減輕他的疼痛。急需對他的左肺進行手術，以免疾病擴散到他的右肺。它可以通過手術切除或移植。
Rahaduzzaman Setu 來自一個普通的家庭，他們已經很難也不可能承擔他的醫療費用了。由於移植所需的資金（1200 萬 BDT），他的家人決定進行手術（300 萬 BDT）。
我們必須通過籌集資金在經濟上幫助他們。
我們的目標是收集盡可能多的東西來幫助我們的朋友。
如果有人現在想貢獻，請將您的貢獻發送給我或與我聯繫。
請根據您的能力做出貢獻。
我們的共同努力可能會挽救生命。
有關更多信息，請參閱以下鏈接。
http://supportsetu.com/
在這個問題中，您必須撰寫一個可以計算捐款的程式。
初始總金額為0，每次都會有兩種操作。


1)  "donate K" (100 ≤ K ≤ 10^5) 在帳戶中增加 K
2)  "report", 報告目前帳戶有多少錢

## 輸入說明

第一行有一個數字 T (1 ≤ T ≤ 100)，代表有 T 筆操作


接下來 T 行包含兩個給定的操作事件 "donate K" 和 "report"

## 輸出說明

針對每個 "report" 操作，報告目前帳戶有多少錢

## 範例輸入 #1

```
4
donate 1000
report
donate 500
report
```

## 範例輸出 #1

```
1000
1500
```
