## 輸入說明

每一行有一個整數n，代表有幾張手牌


1 <= n <= 100，n = 0代表輸入結束


接下來有n張牌


每張牌以2個字元代表，第一個字元為點數(A=Ace, 2-9, T=10, J=Jack, Q=Queen, K=King)


第二個字元為花色(C=Clubs♣, D=Diamonds♢, H=Hearts♡, S=Spades♠)


如果兩個字元為GG，代表牌為鬼牌


只有一副撲克牌，可能有很多張鬼牌，重複的鬼牌不可棄


棄排順序為(C=Clubs♣, D=Diamonds♢, H=Hearts♡, S=Spades♠)，如同範例二

## 輸出說明

輸出整理好的手牌


請按照數字由小到大排列，鬼牌排最後


如果手上的牌剛好棄光，輸出"No card"

## 範例輸入 #1

```
5 AS AD 5D 5H TS
5 AS AD AC 5H TS
4 AS AD AC GG
8 AS AD 5D 5H TS TC TD TH
0
```

## 範例輸出 #1

```
TS 
AS 5H TS 
AS GG 
No card
```
