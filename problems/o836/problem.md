# o836. 火柴棒數字

## 內容

火柴棒是用來表達數字不錯的工具。一種常見表達數字的方式如下:

![image](ShowImage?id=3353)

如同你在一般電子錶上看到的數字。

給你一些火柴棒，你能排出一些數字來。本題要問你的是: 使用所有的火柴棒，你能排出最大的整數是多少? 請注意: 每個阿拉伯數字最多只能出現一次。

## 輸入說明

輸入含有多筆測資。

每筆測資一列，含有一個整數 n ( 1 <= n <= 50)，代表你有的火柴棒數量。

## 輸出說明

對每筆測資輸出一列，使用所有的火柴棒，你能排出最大的整數是多少? 每個阿拉伯數字最多只能出現一次。

如果使用所有火柴棒無法排出數字，請輸出 -1

## 範例輸入 #1

```
1
2
3
4
5
49
```

## 範例輸出 #1

```
-1
1
7
4
71
9876543210
```
