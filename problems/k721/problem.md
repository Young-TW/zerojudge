# k721. 肥余歷險記---(數學城2)

## 內容

肥余在問了一百多個村民後 終於找到了商店


商店裏有一名商人叫做肥李


肥余說:'你好，我要買木劍'


肥李說'你有多少錢? 木劍一把a元'


肥余說'我有b元'

## 輸入說明

輸入a和b

## 輸出說明

輸出能否買到木劍

## 範例輸入 #1

```
8 9
```

## 範例輸出 #1

```
能
```

## 範例輸入 #2

```
9 8
```

## 範例輸出 #2

```
不能
```

## 範例輸入 #3

```
-2 -1
```

## 範例輸出 #3

```
能
```

## 範例輸入 #4

```
2000000 0
0 -1
```

## 範例輸出 #4

```
不能
能
```
