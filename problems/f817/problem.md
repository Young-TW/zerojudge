# f817. TOI_y21m4_a03枯枝

## 內容

TOI練習賽2021年4月_潛力組-3枯枝(Branch)  原始題目檔
樹的枯枝程度 = max( 左子樹的枯枝程度 ,  右子樹的枯枝程度) + 根樹枝的枯枝值

## 輸入說明

第一行輸入 1 個正整數 N (1 ≤ N ≤ 1,000),表示園藝用造景樹有 N 段樹枝。
第二行包含 N 個非負整數 di (0 ≤ di ≤ 10^9) 的序列,表示園藝用造景樹依「前序走訪」記錄各段樹枝其枯枝值的序列。

## 輸出說明

為兩個整數值,分別代表對於滿足條件的園藝用造景樹中,其可能的枯枝程度最小值與最大值。

## 範例輸入 #1

```
3
1 2 0
```

## 範例輸出 #1

```
3 3
```

## 範例輸入 #2

```
7
3 1 1 0 2 0 5
```

## 範例輸出 #2

```
8 12
```

## 範例輸入 #3

```
10
1 2 3 0 2 3 3 0 2 5
```

## 範例輸出 #3

```
6 21
```
