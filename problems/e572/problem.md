# e572. 連連看

## 內容

燈燈燈燈~~~燈燈燈燈~~~(國小上課鐘聲)


老師：小朋友們大家好~~


學生：老師好~~


老師：大家真棒！


學生：棒在哪？


老師：~!@#$%^&*()_+....今天的課程為認識文字，大家把學習單傳下去。


(悉悉簌簌，一陣騷動)


老師：最高品質靜悄悄。


學生：吼~~又是學習單，這學習單不就是連連看= ="。


老師：沒有綽啦，把相同文字連起來，讓你們認識文字。


學生：怯，那麼簡單，我還能用膝蓋連，叫我阿嬤連都連得出來。(ㄩㄩㄩ下台下台)


老師：造反啦(拍講桌)？？不然這樣，每連一條線加學期總平均1分，如果有線段交叉，那交叉線段都不給分。


(學生眼睛發亮開始振筆疾書)

## 輸入說明

輸入有多組測資。


每組測資第一行有一個整數N (1 <= N <= 100)，代表學習單上排文字和下排文字的數量。


N = 0代表輸入結束。


接下來兩行，代表上排文字和下排文字。


文字的長度皆為1。

## 輸出說明

對於每組測資，請輸出最多能得幾分。

## 範例輸入 #1

```
5
測 試 資 料 一
一 測 試 資 料
5
一 二 三 四 五
五 四 三 二 一
0
```

## 範例輸出 #1

```
4
1
```
