# a275. 字串變變變

## 內容

每一組輸入都有兩行不含空白且 ASCII 碼介於 33~126 之間的字串，請告訴我，經過適當的順序調整後，兩個字串會變得一模一樣嗎？
例如，e83k 可以調整順序成 38ek 或 e3k8 等等共 15 種其他排列方式。

## 輸入說明

輸入如題目描述，當讀到「STOP!!」時結束。
字串長度最長為 1000000 個字元。

## 輸出說明

如果兩個字串可以變得一樣，輸出 yes，否則輸出 no。

## 範例輸入 #1

```
e83k
38ek
asdfghjkl;'
';lkjhgfdsa
1234
4521
_01=1
_=110
SToP!!
stop!!
STOP!!
```

## 範例輸出 #1

```
yes
yes
no
yes
no
```
