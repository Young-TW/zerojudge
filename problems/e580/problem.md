# e580. 00696 - How Many Knights

## 內容

騎士是在國際西洋棋中使用的棋子，他的攻擊為走馬步式的攻擊。



![image](ShowImage?id=4053)

在此問題中，您將計算可以放置在具有M行N列的西洋棋盤上最大數量的騎士，以使任何騎士都不會攻擊其他騎士。
M和N皆不大於500。

## 輸入說明

輸入有多行，每行兩個整數M和N (1 <= M,N <= 500)。
如果M = N = 0代表輸入結束。

## 輸出說明

對於每行，輸出可放置最大數量的騎士。
請參考範例輸出之格式。

## 範例輸入 #1

```
2 3
5 5
4 7
0 0
```

## 範例輸出 #1

```
4 knights may be placed on a 2 row 3 column board.
13 knights may be placed on a 5 row 5 column board.
14 knights may be placed on a 4 row 7 column board.
```
