# e678. 12024 - Hats

## 內容

倫敦皇家劇院衣帽間的服務生John Hatman想知道解決以下問題的方法。
演出結束後，劇院裡的觀眾都急忙去觀看歐洲冠軍聯賽的決賽。
他們跑到衣帽間取回帽子，但是他們中有些人總是會戴錯帽子。
John Hatman想知道每個人戴錯帽子的可能性有多大？

## 輸入說明

輸入第一行包含一個整數t，代表測資數量。
每組測資一行，包含一個數字n (2 ≤ n ≤ 12)，代表人數和帽子數。

## 輸出說明

對於每組測資，輸出所有人都戴錯帽子的數量"/"所有人戴帽子的情況。

## 範例輸入 #1

```
3
2
3
4
```

## 範例輸出 #1

```
1/2
2/6
9/24
```
