# l749. 數學運算式(2)

## 內容

續上題， l228. 數學運算式(1)，這次小明遇上了更難的問題，請你編程一段程序協助他。

## 輸入說明

第一行為一個整數n，n是由很多整數連起來組成的，不過沒有空格。


第二行為一個符號s，為"+" 或"-"  或 "*" 或 "/"，分別代表加減乘除。


第三行為一個整數t，代表需要多少個s。


第四行為一個整數a，代表將t個s連接在n的不同位置所成立的算式的計算結果。

## 輸出說明

輸出將t個s連接在n的不同位置所成立的算式，詳見範例輸出。

## 範例輸入 #1

```
328924
*
2
68352
```

## 範例輸出 #1

```
32*89*24=68352
```
