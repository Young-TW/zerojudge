# e556. 00846 - Steps

## 內容

有一條直線，從一個整數點到另一個整數點需要一步。
每一步的長度必須為非負數，並且每一步可以比上一步的長度多一、少一或者一樣。
第一步和最後一步的長度必須為1。
給定兩個位置x和y，從x到y的最小步數是多少？

## 輸入說明

輸入第一行有一個整數n。n代表有幾組測資。
接下來每一行有兩個數字x和y (0 ≤ x ≤ y < 2^31)。

## 輸出說明

對於每組測資，輸出從x到y的最小步數。

## 範例輸入 #1

```
3
45 48
45 49
45 50
```

## 範例輸出 #1

```
3
3
4
```
