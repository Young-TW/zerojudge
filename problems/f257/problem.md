# f257. 君王的處刑遊戲

## 內容

有一個古代君王喜歡觀看處刑，但是看到後來覺得太無聊了，所以他想出了一個遊戲來讓處刑增添更多樂趣。這個遊戲是這樣的，他命令囚犯們排成一個 n x n 的方陣，囚犯初始的驚恐值都是 0，君王會指出將要被殺的囚犯的座標，每個被殺的囚犯周圍的囚犯驚恐值都會上升 1。不過君王單看完處刑還不夠，還想要看各個囚犯的驚恐值來記錄他每一次的處刑過程，現在君王命令你給出一份奏表，上面是處刑完最終各個囚犯的驚恐值。

## 輸入說明

多筆測資


每筆測資第一行會有一個整數 n (0<n<=100)


第二行有一個整數 k 表示有幾個犯人要被君王處決


再來會有 k 行 每一行會有兩個整數 (x,y) 表示囚犯的座標位置

## 輸出說明

輸出要給君王的奏表


已經被處死的人請用 'x' 來表示

## 範例輸入 #1

