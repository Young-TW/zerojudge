# s140. 排隊買飲料

## 內容

學校有兩個飲料販賣機，販賣機 A 每 $a$ 秒可以製作一杯飲料，販賣機 B 每 $b$ 秒可以製作一杯飲料。兩個販賣機同時從第 $0$ 秒開始運作，且可以同時製作飲料。

現在有 $n$ 個學生要排隊買飲料，每個學生只買一杯飲料。請問第 $n$ 個學生最少要等多少秒才能拿到飲料？由於答案可能很大，請輸出答案除以 $10^6+7$ 的餘數。

## 輸入說明

輸入只有一行，包含三個整數 $n, a, b$，分別表示學生人數、販賣機 A 的製作時間、販賣機 B 的製作時間。

- 
$1 \leq n \leq 10^9$

- 
$1 \leq a, b \leq 10^5$

## 輸出說明

輸出一個整數，表示第 n 個學生拿到飲料的最少等待時間除以 $10^6+7$ 的餘數。

## 範例輸入 #1

```
5 2 3
```

## 範例輸出 #1

```
6
```

## 範例輸入 #2

```
10 4 6
```

## 範例輸出 #2

```
24
```
