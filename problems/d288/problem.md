# d288. Parallelogram! And triangle!!

## 內容

在平行四边形ABCD中，E在AD上，a*ED=b*AE，连EC、AC、BE交于O。

若平行四边形ABCD的面积=k，求△AOE的面积、△EOC的面积、△COB的面积。

## 輸入說明

每行k，a，b（小于2^31的整数，当然不会是负数啦）。

## 輸出說明

每组测试数据有三行输出。

第一行输出△AOE的面积。

第二行输出△EOC的面积。

第三行输出△COB的面积。

若答案是整数，就直接输出；若答案是分数，就输出化简之后的分数（含假分数）。

答案中分子和分母化简后都小于2^31。

## 範例輸入 #1

```
1 3 2
140 3 4
```

## 範例輸出 #1

```
9 / 80
3 / 16
5 / 16
9
21
49
```
