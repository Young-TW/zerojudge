# k647. 肥余歷險記---(1)

## 內容

肥余穿越到數學王國，迷路了(...)


他要解開一個問題才能，就是:


1*1+2*2+3*3+......+n*n =?


n為輸入

## 輸入說明

輸入n

## 輸出說明

輸出1*1+2*2+3*3+......+n*n的結果

```
1
```

## 範例輸出 #1

```

```
