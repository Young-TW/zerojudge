# e217. 同分異構物(國二理化)

## 內容

在延平,考試數要以 googolplex 來計算


在考理化週考時,自詡為理化鬼才的少年π碰到了難題


改編自延平國二第二次理化週考其中一題:


同分異構物指的是擁有相同分子式，但結構式卻不相同的多種分子。


exp: C6H14 有 5 種同分異構物


現在給你某烷類分子的碳數,求同分異構物的數量

## 輸入說明

輸入一正整數N(N<=50),N=0 時結束

## 輸出說明

CNH2N+2同分異構物的數量(unsigned long long)

## 範例輸入 #1

```
1
6
0
```

## 範例輸出 #1

```
1
5
```
