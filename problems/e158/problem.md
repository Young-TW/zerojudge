# e158. 星際聯邦

## 內容

星年 $3$∂ 年，星際間爆發了一場能源戰爭，這場戰爭將星系間的智慧生物分成了兩派。


你現在是其中一派的軍師，現在上級需要你在他的許多名單中挑出一些部隊組成一個軍隊來進攻$\zeta \eta S$；一個不錯的戰略太陽系。


一個好的軍隊需要有防護、修復、遠程、技術、後勤、娛樂、coding、動漫...等雜七雜八的屬性，而哪些部隊有這些屬性你的有良心上級已經幫你分好了，你只需要在那些名單中選出部隊組成軍隊。


近日因為某大火導致經費拮据，所以你的軍隊需要"精簡化"，即是需要最小量的部隊種類數量


PS.你的上司有個奇怪的習慣，名單上的部隊編號的是連續的

## 輸入說明

第一行有一個數字$n$，代表接下來有$n$份名單，每份名單都代表不同的屬性。


接下來有$n$行，每一行都有$a_i, b_i, c_i$，分別代表第 $i$ 個名單裡有部隊編號$a_i \sim b_i$，以及你需要在這份名單中至少選幾個部隊。


$1 <= a_i <= b_i <= 50000$


$1 <= c_i <= b_i - a_i+1$

## 輸出說明

請輸出在此$n$份名單內選$c_i$個部隊，所能組成最小軍隊的大小。

## 範例輸入 #1

```
5
3 7 3
8 10 3
6 8 1
1 3 1
10 11 1
```

## 範例輸出 #1

```
6
```
