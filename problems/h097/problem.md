# h097. 大型魔法（一）

## 內容

"MP Too Low" 是一個聚集了許多魔法師玩家的公會。

某天，這些玩家們正在進行"世界BOSS討伐計畫"的會議，成員們正為了要使用哪些魔法來戰鬥吵得不可開交，原因是不同的魔法所需要的魔力消耗不同，隨便亂用魔法很可能會 MP too low 。
身為公會長的你很清楚知道，需要知道所有可用魔法的魔力消耗才有辦法繼續主持會議，但因為成員實在太多了，能用的魔法也是五花八門，所以想寫個程式來計算所有魔法的魔力消耗。


 


魔法：魔法師用來戰鬥的常見手段，其強度可量化為一正整數 P。

## 輸入說明

測資為多筆輸入，以EOF結尾。( 測資數量不超過25000筆 )


每筆測資有一正整數 P 於一行，代表該魔法的強度。（P < 231）

## 輸出說明

針對每個魔法，請輸出一個正整數 C 於一行，代表該魔法的魔力消耗。

## 範例輸入 #1

```
1
5
10
```

## 範例輸出 #1

```
1
6
18
```
