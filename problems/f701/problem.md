# f701. 連接字詞

## 內容

請用指定的連接字詞連接所有的單字。

## 輸入說明

輸入只有二行。第一行含有所要連接單字，單字之間均以空白隔開，單字可能包含字母及/或符號，但不含空白。第二行為連接字詞。

## 輸出說明

輸出一行，含有所有的單字，單字與單字間均有一個連接字詞。

## 範例輸入 #1

```
apple banana cantaloupe
and
```

## 範例輸出 #1

```
apple and banana and cantaloupe
```

## 範例輸入 #2

```
you he I
as well as
```

## 範例輸出 #2

```
you as well as he as well as I
```
