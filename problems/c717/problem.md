# c717. You can say that again!

## 內容

在英文的世界裡，當你同意對方的說法時，會要求對方再講一次。(誤

## 輸入說明

輸入只有一行，含有要輸出的敘述。

## 輸出說明

輸出所輸入的敘述兩次於同一行。

## 範例輸入 #1

```
Snail is handsome.
```

## 範例輸出 #1

```
Snail is handsome. Snail is handsome.
```

## 範例輸入 #2

```
Do that to me.
```

## 範例輸出 #2

```
Do that to me. Do that to me.
```
