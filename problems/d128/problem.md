# d128. 三、速算游戏

## 內容

小牛会算术，尤其精通加法和乘法，这得益于它们从小玩的一个“速算游戏”。游戏随机产生三个0到9的数字排成一行，然后小牛们在不改变这三个数的顺序的情况下，用一次或两次加法或乘法求出运算所得到的最大结果，谁算得快，谁就取得优胜。例如游戏机出的三个分别为3、9、0，那么用一次加法可以得到3+90=93，也可以是39+0=39；用一次乘法则可以得到3*90=270，也可以是39*0=0；用两次运算可以有：3+9+0=12，3+9*0=3，……，最大的结果是270。长期的游戏训练使小牛练就了超高的算术本领。

现在请你编一个程序模拟这个游戏，输入三个数a、b、c（它们均是从0到9的整数），求出由这三个数求得的最大结果。

## 輸入說明

输入档有多组数据，每行有三个数a、b、c。

## 輸出說明

输出由这三个数求得的最大结果。

## 範例輸入 #1

```
3 9 0
```

## 範例輸出 #1

```
270
```
