# e628. 10494 - If We Were a Child Again

## 內容

有一個聰明的大學生說："噢噢噢噢噢噢！如果我能像我上小學的時候那樣做簡單的數學！我可以保證，我不會犯任何錯誤！！"
但是他的老師更聰明，他說："好！我就指定那種計劃給你做。 別難過了。"
"真的嗎？！"學生高興的說道。他感到非常高興，以至於看不到老師臉上陰險的微笑。
這個計畫是要學生做一個計算機來做基本數學運算。
但是像其他大學生一樣，主人公不喜歡自己實作這個計畫，他只想從網路上直接抄別人的程式。
由於你是他的朋友，他要求你寫程式。但是，你也是個聰明人知道該如何應付這類人。
你只同意為他編寫整數除法(division)和求餘數(mod)。

## 輸入說明

輸入包含多組測資。
每組測資一行，每行將包含一個輸入數字m，一個符號(/或%)，另一個輸入數字n (n < 2^31)。
兩個輸入數字均為非負整數。 第一個數字m大小無上限。

## 輸出說明

對於每組測資，輸出一個整數，該整數為運算後的答案。

## 範例輸入 #1

```
110 / 100
99 % 10
2147483647 / 2147483647
2147483646 % 2147483647
```

## 範例輸出 #1

```
1
9
1
2147483646
```
