# b869. Wilson

## 內容

威爾森自從某一天早上清醒後就發現被困在孤島上，為了逃離這座孤島，威爾森必須在孤島上生存並收集四大零件後前往啟動傳送門，才能解救困在島上的其他冒險者並脫離孤島。


然而在這座島上生存並不容易，一天被切割為三個時段，依序為白天、黃昏和黑夜，其中黑夜非常險惡，如果周遭不夠明亮便會遭到黑夜的襲擊。


威爾森研擬了一條路徑，預計在未來的某一天完成路徑上的探索，由於威爾森遭到怪物追殺，因此不能夠在路徑上走回頭路，只能直直往前走。在這一條路徑上會隨意分布三種香菇，紅香菇、綠香菇和藍香菇，紅香菇只在白天冒出地面，綠香菇只在黃昏冒出地面，藍香菇只在黑夜冒出地面，香菇只有在冒出地面才能採集。幸運的威爾森早就撿到了靈子分解仗，這把法仗能夠把時間暫停在白天、黃昏或黑夜。現在我們知道威爾森一路上遇到的香菇順序，請問威爾森最多能採集多少香菇?

## 輸入說明

每組測試資料包含數比測資，每筆測資第一行包含一個整數N代表威爾森會遇到幾個香菇，第二行有N個字元，依序表示威爾森會遇到的香菇，其中'r'代表紅色香菇、'g'代表綠色香菇、'b'代表藍色香菇


測試資料範圍

40%測試資料滿足  1≤N≤1000

100測試資料滿足   1≤N≤50000

## 輸出說明

輸出威爾森最多能採集多少香菇

## 範例輸入 #1

```
7
rbggrbr
```

## 範例輸出 #1

```
4
```
