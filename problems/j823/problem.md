# j823. 編程從此變得方便

## 內容

在編程時，我們通常都要複製一大堆資料，請你編程一段程序，解決問題。

## 輸入說明

輸入一堆東西(可以是字串或數字)，輸入的最後一行須為"stop"，代表輸入的結束，這個文字請勿做任何處理。

## 輸出說明

將輸入的東西以print包住。

## 範例輸入 #1

```
hi
hi
hi
stop
```

## 範例輸出 #1

```
print("hi")
print("hi")
print("hi")
```

## 範例輸入 #2

```
1
2
3
4
5
6
7
stop
```

## 範例輸出 #2

```
print(1)
print(2)
print(3)
print(4)
print(5)
print(6)
print(7)
```

## 範例輸入 #3

```
1
2
hi
stop
```

## 範例輸出 #3

```
print(1)
print(2)
print("hi")
```
