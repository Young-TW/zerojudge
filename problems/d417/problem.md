![image](ShowImage?id=287)

現在我們的問題是：給你 k，請你寫一個程式找出所有的 x 和 y。

## 輸入說明

輸入含有多組測試資料（不會超過100組）。每組測試資料一列，有 1 個正整數 k （ 0 < k <= 10000）。

## 輸出說明

對每一組測試資料輸出一列，輸出共有多少組（x,y），然後輸出這些解答。輸出格式請參考Sample Output。

## 範例輸入 #1

```
2
12
```

## 範例輸出 #1

```

8
```
