# g504. 109北二7.消毒殺菌

## 內容

黃先生幫忙為實驗室的桌子噴灑消毒劑殺菌。已知殺菌力與噴灑後所經過的時間呈如下關係：
當噴灑了 x 單位殺菌力的消毒劑後，經過 t 單位時間，該次噴灑的殺菌力剩下 (x - t)。
在任何時間點，總殺菌力為迄今噴灑所遺留殺菌力的加總。


請你寫一個程式，計算在首次噴灑消毒劑後經過 y 單位時間，該桌子上的殺菌力。
其中中途可能再有數次噴灑。

## 輸入說明

測試資料只有一行，
第一個數字為噴灑次數 N ( 1 ≤ N ≤ 10000)，第二個數字為上述之時間單位 y，
接著兩兩一組，分別為噴灑的時間點及該次噴灑的殺菌力單位(依時間順序排列)。
各數字之間用空格(white space)隔開。


所有輸入值皆可以 32 位無號正整數表示

## 輸出說明

輸出資料為一個數值，為在首次噴灑消毒劑後經過 y 單位時間，該桌子上的總殺菌力。

## 範例輸入 #1

```
2 2 0 3 4 2
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
2 3 1 2 5 1
```

## 範例輸出 #2

```
0
```

## 範例輸入 #3

```
2 3 0 3 2 3
```

## 範例輸出 #3

```
2
```
