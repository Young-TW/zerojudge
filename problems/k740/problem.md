# k740. 楊輝三角形

## 內容

```
            1
          1   1
        1   2   1
      1   3   3   1
    1   4   6   4   1   (圖1)
```

如圖1所示，一個楊輝三角形的其中一個值等於它左上方與右上方的和；


請寫一個程序，輸入三角形的高度n，輸出高度為n的楊輝三角形。

## 輸入說明

輸入僅有一行，為一個正整數n (1<=n<=20)。

## 輸出說明

輸出高度為n的楊輝三角形。

## 範例輸入 #1

```
5
```

## 範例輸出 #1

```
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```
