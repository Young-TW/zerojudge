# k571. pB. 關於第二道題這件事

## 內容

「好耶！是校內賽第二題！」

為了慶祝順利寫到第二題，你想要用盡可能多的 2，也就是 222.... 來表達你的喜悅之情；
其中你有最多 K 次機會，能夠把任意位置的數字改變成你想要的值。


給定 N 個數字，
在最多只能改變其中 K 個數字的情況下，最多可以找到幾個「連續的 2」呢？

## 輸入說明

第一行有兩個整數 N 和 K，代表數字總數 和 可改變次數
1 ≤ N ≤ 8*104
0 ≤ K ≤ N


第二行由左至右有 N 個整數 xi，代表數字內容
0 ≤ xi ≤ 2

## 輸出說明

至多只能改變 K 個數字的情況下
最多可以找到的「連續的 2」的長度

## 範例輸入 #1

```
14 2
2 2 0 2 0 0 0 2 2 2 0 0 2 2
```

## 範例輸出 #1

```
7
```

## 範例輸入 #2

```
6 2
2 2 0 2 2 0
```

## 範例輸出 #2

```
6
```
