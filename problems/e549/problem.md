# e549. 00737 - Gleaming the Cubes

## 內容

身為Starship Interprize的首席工程師，因此修復hyperstellar, cubic, transwarpedout software的重責大任，就交給你了。
簡而言之，您必須計算2到1000個立方體彼此重疊的體積。

## 輸入說明

輸入有多組測資。
每組測資第一行包含一個數字n (2 <= n <= 1000)，n代表立方體數目。
如果n = 0代表輸入結束。
接下來n行每一行包含四個整數x, y, z, L。 
前三個整數是立方體角落的(x, y, z)坐標，而第四整數是立方體在三個方向(平行於x，y，z軸)上延伸的正距離。
立方體彼此重疊的體積不超過10^6單位。

## 輸出說明

對於每組測資，輸出立方體彼此重疊的體積。

## 範例輸入 #1

```
2
0 0 0 10
9 1 1 5
3
0 0 0 10
9 1 1 5
8 2 2 3
0
```

## 範例輸出 #1

```
25
9
```
