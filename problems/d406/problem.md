# d406. 倒水時間

## 內容

水由上而下的流，現在給你水管之間的地圖
水可以由上而下，可以往右流，往左流。

可是呢 ... 有時候也可以往上流
現在從地圖的最上方開始倒水，請輸出到的時間。

※  開始倒的地方只有 1 個且只在第一列倒

## 輸入說明

每組測資的第一列有一個數字 S，若 S = 2  代表水不能往上流， S = 1  代表水可以往上流。
              第二列有兩個數字 N M， N 代表接下來有 N 列，M代表每列上有多少數字。
                                         ( 1≦ N M ≦ 100 )
              接下來會有 N 列，每列上有 M 個數字， 1 代表有水管， 0 則代表沒有。

## 輸出說明

對每個地點輸出到的時間。
