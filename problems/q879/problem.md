# q879. I love physics

## 內容

HJ開始準備段考了，他前幾天看到一題理化會考考古題，如下(改編自109年會考理化46,47題):



潛水員可在不施力划水的情形下於海面下維持同樣的深度，此種調整背心的充氣量而能夠在海面下維持同樣深度的技術，稱為「中性浮力」。當潛水員維持在海面下 10 m 的深度以「中性浮力」活動時，給定未攜帶裝備潛水員的體積為V人、質量為M人，潛水員所攜帶的所有裝備體積為V裝、質量為M裝，海水的密度為D海 其中四者，求第五者？

## 輸入說明

每有五個用空白隔開之整數，若該整數為-1，該項即為要求知項目。
五項依序為未攜帶裝備潛水員的體積V人、質量M人，潛水員所攜帶的所有裝備體積V裝、質量M裝，海水的密度D海(單位相同，不會有轉換上之問題)

EOF結束

## 輸出說明

每行輸出缺少那項之值(輸出到小數點後第二位)

## 範例輸入 #1

```
100 80 150 70 -1
-1 90 140 80 1
120 -1 180 100 1
```

## 範例輸出 #1

```
0.60
30.00
200.00
```
