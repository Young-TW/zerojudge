# i230. Subset Sum

## 內容

$a, b$ 為兩個長度為 $N$ 的整數數列 $a_0, a_1, ..., a_{N-1}$ 和 $b_0, b_1, ..., b_{N-1}$


計算 $c_0, c_1, ..., c_{N-1}$，並對每一項 $\bmod 998244353$，其中


$c_k = \sum_{k'\subseteq k} a_{k'} \cdot b_{k \backslash k'}$

## 輸入說明

第一行為整數 $N$。


第二行與第三行各有 $N$ 個整數 $a_i$ 和 $b_i$。


$N = 2^k$, $1 \leq k \leq 18$


$0 \leq a_i, b_i \leq 998244352$

## 輸出說明

輸出 $N$ 個數字

## 範例輸入 #1

```
4
1 2 3 4
5 6 7 8
```

## 範例輸出 #1

```
5 16 22 60
```

## 範例輸入 #2

```
8
3 14 15 9 26 53 58 979
27 18 28 18 28 45 90 45
```

## 範例輸出 #2

```
81 432 489 959 786 2426 2984 31751
```
