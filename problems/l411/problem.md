# l411. 時差

## 內容

小明成功考上了機師! 因此，他經常要到各到國家，請你編程一段程序，協助他解決問題。

## 輸入說明

第一行為小明的常居地和時間，以空格隔開。


格式 : "地區 時間" 時間採二十四小時制


時間 : xx:yy 


接下來有一個整數n，代表接下來有多少行。


接下來有n行，格式 : "其他地區 其他地區對應常居地時間"


時間 : xx:yy 


最後一行為一個整數t，代表接下來有t行。


接下來有t行，為其他地區的時間(時間隨機)。

## 輸出說明

輸入共有t行，格式為 : "其他地區時間 常居地對應時間"。

## 範例輸入 #1

```
Taiwan 12:23
4
Beijin 14:35
Africa 16:22
America 0:13
Korea 17:55
4
Beijin 16:35
Africa 15:21
America 13:13
Korea 18:00
```

## 範例輸出 #1

```
Beijin 16:35 Taiwan 14:23
Africa 15:21 Taiwan 11:22
America 13:13 Taiwan 1:23
Korea 18:00 Taiwan 12:28
```
