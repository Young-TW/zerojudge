# b539. 分數運算-3

## 內容

本來上次出 系列-2 時的想法是預定 加指數再加四則且多個運算符號，後來決定先單獨出指數部份，是否出系列-4，等12月之後再說吧！



![image](ShowImage?id=884)

## 輸入說明

每個測資檔有多列以EOF結束，每列四個整數 1 <=a,b,c,d <=9999999以空白隔開
 代表兩個分數的指數運算  (a/b) ^ (c/d) ，其中^次方。

## 輸出說明

![image](ShowImage?id=885)


保證 m,n,h,t,u <= 10^9，但計算過程自己處理哦。

## 範例輸入 #1

```
1684800 2323139 4 10
125 32 35 21
135 12 2 4
125 64 35 21
3 5 3 5
123 3 5 5
1234567 1234567 6543 6543
```

## 範例輸出 #1

```
12/7_/5(67500/1900087)
3125/256_/3(1/2)
3/2_/2(5)
3125/1024
_/5(27/125)
41
1
```
