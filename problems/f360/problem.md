# f360. Shady的複製品

## 內容

![image](ShowImage?id=1620)


'Cause I'm Slim Shady, yes, I'm the real Shady
All you other Slim Shadys are just imitating
So won't the real Slim Shady please stand up
Please stand up, please stand up?


總共有n個slim shady，但只有一個是real slim shady，剩下都在imitating，請問有多少人在imitating呢?

## 輸入說明

輸入不超過100行，每行有n(2≤n≤10^1000)，EOF結束

## 輸出說明

輸出有多少在imitating

## 範例輸入 #1

```
2
10
100000
```

## 範例輸出 #1

```
1
9
99999
```
