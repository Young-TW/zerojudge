# d311. 數學少女的難題

## 內容

雖然蒂蒂並不是那麼般的精通於數學，但是這仍然無法減低她對於數學的熱忱。

某一天，她遇到了一題題目：

設Pr表示1, 2, 3, ..., 10中任取r個相異的數的乘積之和(例如P2 = 1*2+1*3+...+1*10+2*3+2*4+...+2*10+3*4+...+3*10+...+9*10一共45項)，那請問P5是多少？

看到這樣的題目，蒂蒂拿起筆奮戰了一整個下午卻是沒有收穫，所以她拿著題目去找米爾迦。

看到題目後，米爾迦輕輕閉上了眼睛開始沉思，過沒幾分鐘就露出自信的微笑，把想法以及答案寫了下來。

然而這種程度的題目實在是無法滿足米爾迦的求知慾，所以她稍微修改了題目，然後拿給同樣熱愛數學的你：

設P(n, m)代表從1到n個數字之中任取m個相異的數的乘積之和，請問P(n, m)是多少？

## 輸入說明

兩個正整數n及m

我們保證： 

至少30%的測試資料中m<=n<=10

至少60%的測試資料中m<=n<=100

所有的測試資料中m<=n<=1000

## 輸出說明

一個整數，代表P(n,m)除以1572869後的餘數

## 範例輸入 #1

```
10 3
```

## 範例輸出 #1

```
18150
```
