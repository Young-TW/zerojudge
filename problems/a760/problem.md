# a760. 四、世界末日預言

## 內容

世界末日是何日，有許多預言紛紛發表，隨著日子的來到，流言也一一破解。

  現在，有一個末日預言的傳言又來了，內容是這樣子的：

  將特定文章的內容裡，大、小寫英文字母的字元轉換為ASCII碼，接著把該ASCII碼數字(10進位)除以3，收集餘數成為一長串的數字字串。

  接著，由數字字串裡，取出14碼的數字，分別代表著：

  西元年(4碼)+月(2碼)+日(2碼)+時(2碼)+分(2碼)+秒(2碼)

  這其中，可能有許多的合理日期會出現，而最靠近今日 (2013/09/18 13:30:00) 且是未來的，就是世界末日預言的日期。

  例如：字串  Monday's process 

  M的ASCII碼是77，除以3的餘數是2

  o的ASCII碼是111，除以3的餘數是0

  n的ASCII碼是110，除以3的餘數是2

  其他的字母處理方式是相同的，而空白與 ' 是忽略不計的，只處理大小寫英文字母即可。

  得到的數字字串是： 20211111000211

  代表的日期是： 2021/11/11 00:02:11

## 輸入說明

輸入檔中含有一至數行(段)英文句子。

  每行(段)是獨立的字串，不可以合併處理。

## 輸出說明

僅一行，日期，格式為YYYY/MM/DD HH:MM:SS。

## 範例輸入 #1

```
Monday's process was the first step in a plan to remove and scrap the 952-foot ship.
```

## 範例輸出 #1

```
2021/11/11 00:02:11
```
