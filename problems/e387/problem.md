# e387. Bill 的超級電腦

## 內容

有 n 台超級電腦放在機房內，現在要求由你選定一台，用共 n − 1 條線從這台機器開始一台接一台地依次連接他們，最後接到哪個以及連接的順序也是由你選定的，為了節省材料，網線都拉直，因此線的長度就等於兩點之間的平面距離。求總共最少需要購買多長的網線。 （簡單來說，就是找出 n 的一個排列P1P2P3..Pn 然後找出 |P1P2| + |P2P3| + ... + |Pn−1Pn| 長度的最小值）

## 輸入說明

第一行 n，下面 n 行，每行分別為機器的坐標 (x, y)


 


30%的測資 2 <= n <= 5


60%的測資 2 <= n <= 10


100%的測資 2 <= n < 20


100%的測資 x, y 為實數 且 −100 ≤ x, y ≤ 100

## 輸出說明

最小的總長度，四捨五入到小數點後第二位。

## 範例輸入 #1

```
3
0 0
1 1
1 -1
```

## 範例輸出 #1

```
2.83
```
