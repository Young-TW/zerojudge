# a494. 清一色麻將

## 內容

看過d341: 11210 - Chinese Mahjong嗎

http://zerojudge.tw/ShowProblem?problemid=d341

由於自己是個弱者

所以這題對我來說太難了...

為了建立自己的信心

我把問題簡化:僅考慮清一色手牌

手牌就是吃碰槓後剩下的牌

因此可能為16,13,10,7,4,1張

我們希望從剩下的牌中得知自己所聽的牌有哪些

## 輸入說明

每一行代表一副手牌

其中僅包含1到9的數字

其中每一個數字皆代表同一種花色

同一個數字輸入的次數代表那張牌的張數並且不會超過四次

例如:2234 代表有兩張2 一張3 一張4

此時25可成胡

輸入保證一定有聽牌也不會有相公的情形發生

## 輸出說明

輸出每一行手牌所能成胡的所有牌

並由小到大輸出

## 範例輸入 #1

```
2234
1234
1122
1
1112345678999

 
```

## 範例輸出 #1

```
25
14 
12
1
123456789

 
```
