# h557. pA. 迷宮入口(entrance)

## 內容

完整題目:https://drive.google.com/file/d/1RSq17g00V-ygkC8x37iLhtNteoqLAwZg/view?usp=sharing


在2維座標平面上，有 $N$ 個神奇的圖騰，這些圖騰要滿足特定條件才會被啟動，你可以在平面上選任意的一個格子點，然後在上面發出奇怪的聲音，以你在的座標半徑為 $r$ 畫圓，在這個圓裡面的所有圖騰就會接收到你的聲音，如果有奇數個圖騰接收到聲音，那圖騰就會被啟動，請問有多少個格子點可以使得圖騰被啟動。


限制:


$1 \leq N \leq 2500$


$1 \leq r \leq 10$


$ | x_i |, | y_i | \leq 10^6$


圖騰位置兩兩相異

## 輸入說明

第一行輸入兩個數 $N$、$r$ 分別代表圓的數量以及他們的半徑。


再來有 $N$ 行，每行兩個數 $x_i$ $y_i$ 代表圓的位置 (皆為整數)。

## 輸出說明

輸出能啟動圖騰的點的數量。

## 範例輸入 #1

```
3 2
0 0
-1 0
2 0
```

## 範例輸出 #1

```
17
```
