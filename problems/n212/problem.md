# n212. DD成為化學電神2

## 內容

在化學工廠裏，有一台奇怪的機器。這台機器的運作理念如下：


它只接受兩種化合物 A 和 B 。


給它一串長度 N 且只有 A 和 B 的字串（代表你按照順序給這個機器的化合物），它會隨機取K個 (1<=K<=N) 並把它們按照順序融合成為新的一個化合物。


舉例來説，如果你給他 ABA 這個字串，他可能會做出：


A（取第一個 或 取第三個）


B（取第二個）


AB（取第一個、第二個）


BA（取第二個、第三個）


AA（取第一個、第三個）


ABA（取第一個、第二個、第三個）


一共有6種可能的化合物。


你現在想要讓這個機器恰好可能做出 X 種不同的化合物，請問要給他什麽字串呢？

## 輸入說明

輸入一個數 X


X<=200,000

## 輸出說明

輸出 S


任何符合題目敘述的輸出都會過，只要S的長度小於1,000,000。

## 範例輸入 #1

```
6
```

## 範例輸出 #1

```
ABA
```

## 範例輸入 #2

```
3
```

## 範例輸出 #2

```
BA
```
