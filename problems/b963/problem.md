# b963. GIIA v.s. 死亡之握

## 內容

明天就是元旦連假了，大家都很期待這次的假期，不過既然連假讓大家這麼高興，就一定會有邪惡的壞人出來搗亂，就像飛天小女警每次都必須維護小鎮村一樣。


今年鬼島情報中心(Ghost Iland Intelligence Agency, GIA)獲得情報，擁有死亡之握的怪獸將要襲擊鬼島，所以鬼島情報中心決定要派出鬼島的所有戰力去對抗牠。


但是鬼島是個國際融合的大村落，有巴西及法國的1進位、埃及的2進位、丘馬什語系的4進位、蘇美人的60進位。在作戰時的基礎作戰單位是以部隊為主，


但是每個部隊是由兩種族群所組成的，所以鬼島情報中心為了統整併計算每個種族的兵力，於是決定要利用程式來計算每個部隊的兵力總和。

## 輸入說明

每筆測資包含三行，前兩行分別有一個正整數Ni與一個Ni進位的數字(英文大小寫皆有可能)，分別代表其中兩國人以他們的進位制表示他們的兵力，


第三行有一個正整數m，為鬼島情報中心所使用的進位制度。


所有測資滿足2≤Ni,m≤36，且所有運算過程皆不超過十進位的2147483647

## 輸出說明

對於每筆測資，請輸出一行為兩國兵力加總後換成鬼島情報中心所使用的進位制度(若為英文請輸出大寫)。

## 範例輸入 #1

```
15 124
30 C
6
22 67
24 99
11
36 j
5 1413
18
```

## 範例輸出 #1

```
1131
301
E0
```
