# f700. 劃重點

## 內容

文文讀書時喜歡在重點內容下面劃上波浪底線。現在他又看到一句重點了，麻煩在它下面劃上等長的小波浪。

## 輸入說明

輸入只有一行，含有文文看到的重點。

## 輸出說明

輸出文文所看到的重點，並在下面劃上等長的小波浪。

## 範例輸入 #1

```
The quick brown fox jumps over the lazy dog.
```

## 範例輸出 #1

```
The quick brown fox jumps over the lazy dog.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
```

## 範例輸入 #2

```
Important! Important! Important!
```

## 範例輸出 #2

```
Important! Important! Important!
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
```
