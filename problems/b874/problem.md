# b874. 又是一封信

## 內容

接續 b855: 一封信，小明又收到了一封奇怪的信，這次是一個數學題目(? 給你一堆的數字，還有一個數字M，問你能不能用這堆的數字組成一個能被M整除的數，用的數字數量不限，但不得重複，一個數字只能用一次。

## 輸入說明

第一行的數字T代表測資筆數，接下來每筆會給數字N,M，N代表有幾個數字可以使用，請問是否能用這些數字組成可被M整除的數。



```
n<=1000000,m<=10000
```

## 輸出說明

如果可以輸出"YES"，不行就輸出"NO"

## 範例輸入 #1

```
1
2 5
2
3
```

## 範例輸出 #1

```
YES
```
