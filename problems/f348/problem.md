# f348. 完全偶數

## 內容

文文愛偶數，他甚至有一套自己的「偶數風水」。如果一個地方的座標是「完全偶數」，那麼這個地方就是「九陰聖地」。所謂「完全偶數」是每位數都是偶數的正整數，如 6、84、428 等。


現在，給你文文所在位置的座標，請問離他最近的「九陰聖地」的距離為何？

## 輸入說明

輸入只有一行，含有一個正整數 n (1 ≤ n ≤ 106)，代表文文所在位置的座標。

## 輸出說明

輸出文文所在位置和離他最近的「九陰聖地」的距離。

## 範例輸入 #1

```
246
```

## 範例輸出 #1

```
0
```

## 範例輸入 #2

```
2346
```

## 範例輸出 #2

```
54
```
