# k846. best friends

## 內容

nathan, nelson, neil, wisley, hayden是五個好朋友，彼此也是最好的朋友。但是，因為nathan犯了錯，導致neil,


wisley, hayden被人罵，他們有點不開心，所以對nathan的態度變得冷淡。nelson想幫nathan，於是他建議


nathan可以送他們一份禮物以表示歉意。 請你幫助nathan和nelson，製作一份禮物吧。禮物是一個簡單的加


減乘除及乘方計算機，每次能計算兩個數字之間的加減乘除或乘方，還會顯示nathan和nelson精心製作的語句。---(真人故事改編)


加:"加起來就是團結，團結就是力量，我們五個是最好的朋友!"


減:"減並不負面，減只是加一個負數，失去的，終究會回來。我們之間失去的友誼，也終究會回來。生命是美好的，友誼是寶貴的!"


乘:"正數乘起來會變大，我們五個好朋友在一起，能量也會翻倍!"


除:"除法就是分，我們五個是能平分東西，同甘共苦的好朋友!"


乘方:"乘方就是能力翻倍，我們自己能力翻倍的話，我們五個好朋友的能力就會更強!"

## 輸入說明

總共有n行運算式，分為加減乘除，以用户輸入"0"結束


運算式可以是"x+x"的形式，也可以是"x + x"的形式(不一定是加)

## 輸出說明

輸出有n行，對應各個運算式，輸出答案。輸出答案的下一行就是nathan和nelson精心設計的語句。


如果答案是浮點數，就四拾五入，結果一定是正數。

## 範例輸入 #1

```
1+3
5 + 7
8/4
7/4
5 * 5
0
```
