# d395. 10519 - !!REALLY STRANGE!!

## 內容

Raju 剛獲得大學理學士的學位。他出社會的第一個工作是到某地區調查該地被劃分為多少塊不同的區域。他發現那裡的人用圓來畫區域，每 2 個圓一定相交於 2 點且不會有三個圓相交於同一點。由於人數實在太多了，Raju 需要你的幫忙計算出這些圓把這地區切割成多少塊不同的區域。以下的圖為 2 個圓的情況，共有 4 個不同的區域。

![image](ShowImage?id=284)

## 輸入說明

每組測試資料一列

含有 1 個整數 n（0 =< n <= 10100）

## 輸出說明

對每組測試資料輸出一列

這 n 個圓把這地區切割成多少塊不同的區域

## 範例輸入 #1

```
3
4
```

## 範例輸出 #1

```
8
14
```
