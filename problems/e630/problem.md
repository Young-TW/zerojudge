# e630. 中文數字

## 內容

關於數字轉中文想必大家都不陌生(請參考"e379: 數字轉中文")


那中文轉數字呢？


其實並不難(可能比數字轉中文還要簡單很多？！)


現在給你一串中文數字，可以請你幫我轉回阿拉伯數字嗎？

## 輸入說明

包含多組輸入，每組輸入一行


每行有一串中文數字。


該中文數字小於一京，以下為此中文數字會出現的中文字。


{"零", "一", "二", "三", "四", "五", "六", "七", "八", "九"}
{"十", "百", "千"}
{"萬", "億", "兆"}

## 輸出說明

對於每行輸入，請把該中文轉為數字。

## 範例輸入 #1

```
二
二十五
二百零五
一千零五十
二十萬零五百
二十萬五千
二億零五千
二十一億四千七百四十八萬三千六百四十七
```

## 範例輸出 #1

```
2
25
205
1050
200500
205000
200005000
2147483647
```
