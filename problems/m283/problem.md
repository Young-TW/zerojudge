# m283. 螞蟻的擴散

## 內容

座標萍面上，當螞蟻在(a,b)時，鉈可隨機移動到點(a-1,b), (a,b-1), 或(a-1,b-1)之中的一點，且移動到上述任一點的機率均為1/3，又接下來每次移動都與前次移動無關。現有一螞蟻從 (x,y) 開始移動，直到鉈第一次遇到任意座標軸即停止移動，請問此螞蟻最後一次停止的點不是在原點 (0,0) 機率為何?

## 輸入說明

多筆測資，每筆測資輸入兩正整數 x,y 


(x,y<=10)

## 輸出說明

答案以最簡分數表示，輸出("分子/分母")

## 範例輸入 #1

```
3 3
2 2
1 1
```

## 範例輸出 #1

```
70/81
22/27
2/3
```
