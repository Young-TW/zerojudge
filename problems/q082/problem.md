# q082. 小明的次方和(簡單版)

## 內容

小明在六年級的數學進階課堂上學習了立方和和立方差的計算公式，他發現，凡是奇數次方和，都是有公式的，而偶數次方和就沒有。請你用編程幫他判斷此n次方和有沒有公式。

## 輸入說明

兩行，第一行輸入一個正整數n，第二行輸入以空格分隔的兩個數a和b


a, b, n <= 50

## 輸出說明

兩行，第一行是能或不能，第二行是兩個數的n次方和，如果第一行是不能則第二行不用輸出

## 範例輸入 #1

```
3
9 10
```

## 範例輸出 #1

```
能
1729
```

## 範例輸入 #2

```
2
9 10
```

## 範例輸出 #2

```
不能
```

## 範例輸入 #3

```
5
6 9
```

## 範例輸出 #3

```
能
66825
```
