# b914. 神奇寶貝1.聯盟賽

## 內容

神奇寶貝xy最近完結了，塔西塔覺得不過癮，聯盟賽只播了9集就結束，播完聯盟賽
，塔西塔不只覺得空虛，更感到憤怒，小智依然的不到冠軍......
"不!!!為什麼?編劇不給小智冠軍!!!"塔西塔心中想著
幾天後，塔西塔在一個論壇上，得知聯盟賽集數是由一群內部的高層投票決定，把每人的投票內容相加
除於投票人數在四捨五入，得到的數字就是那季的聯盟賽集數

## 輸入說明

本題為EOF版，每一行有個整數n(0<=n<=200)，代表高層的人數，接下來有n行，每行有個m(m<=20000)，代表每一位高層的答案

## 輸出說明

輸出那季的聯盟賽集數(四捨五入到整數)

## 範例輸入 #1

```
5
2
3
6
8
4
1
8
```

## 範例輸出 #1

```
5
8
```
