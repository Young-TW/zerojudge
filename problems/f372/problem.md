# f372. 崑棋的臭豆腐

## 內容

崑棋在夜市賣臭豆腐，是個臭豆腐職人，對於炸臭豆腐很有心得。他很清楚知道炸臭豆腐的時間會影響到臭豆腐的美味度，美味度越高就越好吃（廢話）。完全還沒炸過的生臭豆腐美味度是1，每炸1分鐘美味度就會變成3倍，但是美味度不是那麼容易就能掌控的，只要某塊臭豆腐的美味度超過10006，它的美味度就會在瞬間內不斷減少10007，直到不超過10006。這些道理崑棋都明白，但他實在懶得再算了，於是他想請你幫他寫個程式計算現在這塊臭豆腐的美味度是多少。

## 輸入說明

測資為多筆輸入，以EOF結尾


每行一個正整數M，代表這塊臭豆腐炸了M分鐘。（0<M<2147483648）

## 輸出說明

請在每一行輸出該臭豆腐的美味度。

## 範例輸入 #1

```
1
2
3
```

## 範例輸出 #1

```
3
9
27
```
