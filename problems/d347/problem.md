# d347. 00847 - A Multiplication Game

## 內容

Stan和Ollie在玩一個乘法的遊戲。他們輪流對一個正整數 p 乘上2到9中的一個數。遊戲開始時，p=1且由Stan先開始。Stan乘完後換Ollie乘，然後再換Stan乘，如此一直下去。遊戲一開始時他們也訂了一個整數 n （1 < n < 232-1），誰先乘上一個2到9的數，使得 p>= n，他就是遊戲的贏家。

## 輸入說明

每組測試資料一列，含有1個整數 n。

## 輸出說明

每組測試資料輸出一列，輸出贏家是誰。


你應該要假設這兩個人都是很厲害的玩家（就是說，如果可以贏的話，絕對不會有失誤）。

## 範例輸入 #1

```
3
38
168
162
17
34012226
```

## 範例輸出 #1

```
Stan wins.
Stan wins.
Ollie wins.
Stan wins.
Ollie wins.
Stan wins.
```
