# b528. 板條編年史 (二)：長頸鹿與妮可妮可妮的逆襲

## 內容

時間一天又一天、一年又一年的過去了，紅圓茵可從一個見習魔法師，逐漸蛻變為一個獨當一面的魔法師。紅圓茵可從大魔導 nanj 中學習到「爆搜魔法」、「DP 魔法」，但感於自己在「板條界」的時候是一個遊戲設計師，因此將魔法用程式語言的形式表達，因此開創了嶄新的流派－－稱為現代魔法，時間正值板擦瑪吉曆 426 年。


現代魔法在短短數十年間迅速散播，帶動了新一代的革新，以往以城鎮做為基礎自治單位的世界發生了變化，城鎮與城鎮之間利用現代魔法，以魔法為動力的高速交通工具串起「板擦界」中約 81000 個大大小小的城鎮；為了因應魔法人才的空缺，城鎮之間相繼成立高等學校；除此之外，城鎮和城鎮聯繫變密切了，開始產生跨城鎮的組織，例如：後出後進研討會 (Last Out Last In CONference, LOLICON)、東方邏輯魔法物件導向政府 (East Logically maGical Object Oriented Government, ELGOOG)、...... 等不計其數。


但其中最有影響力的組織稱為「進階城市管理組織 (Advanced City Management, ACM)」，起源於現代魔法最鼎盛的都市－－可茵城，儘管名字中還保留著「城」的字樣，但經過了數十年的發展，已然是千萬人居住、都會區綿延三百公里的大城市。


ACM 組織採用和其他城市相比先進十年的現代魔法，短時間內迅速整合所有城鎮的管理系統，並且在板擦瑪吉曆 489 年 1 月 1 日建立前所未有新的組織－－魔法之國，而紅圓茵可因為其創造了現代魔法，而被封為大魔法師 (La alta Magia Del Magister)。


正當整個板擦界正在慶祝魔法之國的誕生之際，駭人聽聞的事情發生了－－一切事物都停擺了，不管是交通運輸、工廠、... 等等，都停止了。魔法之國的管理部門第一時間就發現，這一切起因於現代魔法的運作將魔法依附在金屬石頭上，這些金屬石頭有個特別的名稱－－伺服器，而伺服器被人利用了一種加密方法，使得系統的資料全部大亂。


「到底是誰亂搞呢？」紅圓茵可苦惱著。


「嗚哈哈哈哈哈！！我已經得到 GEASS，只要再拿到『板中大冒險』的 CODE，我就能統治世界了，哇哈哈哈哈哈！！」一隻外號稱為「許胖」的長頸鹿走來。


原來這一切是因為古老的魔法流派不滿現代魔法的推廣所策動的計劃，同時魔法世界最高學府－－板擦高中－－被天才魔法師老妮可掌握在手中，因此茵可只能靠自己的力量來解決問題！原來這些資料原本都被當作一個介於 0 到 180 度的浮點數 X 做處理，而加密後的資料為 sin(cos(X*PI/180))，大魔法師－－紅圓茵可只好用新一代的現代魔法去解決！

## 輸入說明

輸入有數筆測試資料，每筆測試資料只有一行浮點數 Y，代表被加密過後的資料。


 


不超過 50% 的測試資料筆數 <= 100


不超過 80% 的測試資料筆數 <= 10000


不超過 100% 的測試資料筆數 <= 100000

## 輸出說明

對於每筆測試資料，輸出原來資料至小數點後兩位，若該筆資料不可能發生，則輸出「FAIL!」（不含引號）。

## 範例輸入 #1

```
0.776453
3.000000
```

## 範例輸出 #1

```
27.25
FAIL!
```
