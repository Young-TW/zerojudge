# c410. 二、見面會(Meeting)

## 內容

小佳犯罪，被關進監獄裡面，小佳的家人想去見小佳，於是前往監獄一年一度的見面會。


　　為了避免不必要的事情發生，見面時犯人與家屬必須在不同的房間以電話連絡，而電話之間的線路是一對一且不能改變的！但是獄卒並沒有排好時間表，隨便就讓先到達的犯人與家人先進去見面室使用電話（見面室裡不可以有沒使用電話的人，否則就得在外面等），以至於有一些家屬必須待在電話前裡等待自己要找的犯人進來才能聯絡完並且離開，反之亦然。


　　假設犯人與家屬都必須要見到自己要聯絡的人才能離開，並且家屬與犯人數量相同，也不會有兩個人都要見同一個的人的狀況，請你撰寫一個程式，在得到犯人與家屬的到達順序名單後，計算出見面室最少需要的電話對數量。

## 輸入說明

第一行有一個整數N代表有幾個家屬與犯人，第二行有N個整數以空白隔開代表犯人來的順序編號K1~KN，第三行有N個整數以空白隔開代表從家屬來的順序編號J1~JN。
家屬與犯人想要見的人的編號相同，且都介於1~N之間。

## 輸出說明

最少需要準備幾對電話。

## 範例輸入 #1

```
3
1 2 3
3 1 2
```

## 範例輸出 #1

```
2
```
