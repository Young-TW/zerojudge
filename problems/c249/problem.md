# c249. 《Φ》序章·II ~ 戲如科學

## 內容

「


　　『二進制世界』不只是在表面上感受到的與『α世界』有所差異，在科學方面，雖然大多的物理與數學規則不變，譬如『1與0的作用力定律』－－在『α世界』稱作『牛頓第三運動定律』或『作用與反作用定律』－－在多數的世界都是通用的，基本上只有名詞上的差異，所以對您在日常的生活並不會有太大的影響，大可放心。所謂名詞上的差異，在此就以『物質』來做舉例，在『α世界』中『物質』分為２種，一種是由您所熟悉的『電子』－－在『二進制世界』稱為『一子』－－等基本粒子組成的『正物質』，而構成『正物質』的基本粒子則稱做『正粒子』，另一種則是『反物質』，同理構成『反物質』的基本粒子則稱為『反粒子』，『正粒子』與之對應的『反粒子』在所有內部相加性量子數皆同值異號；對應到『二進制世界』也是如同上述的方式在運作，只不過『正物質』、『反物質』在這裡稱為『｜物質』、『Ｏ物質』，同理『正粒子』、『反粒子』則稱作『｜粒子』、『Ｏ粒子』，單純僅有如此差異。儘管這些名詞在日常生活幾乎用不到，但由於『二進制世界』的人們都來自不同的世界，雖然在『二進制世界』已經備有良好的翻譯功能，為了讓科學能夠順利進步，目前仍然使用統一的名詞較為方便。


　　說到『｜粒子』和『Ｏ粒子』，在此為您科普一下關於『真空』的知識，您可能會認為『真空』之中什麼都沒有，但是依照現今科學觀點來看，『真空』並非什麼都沒有，在『真空』中充滿了各種不斷波動的粒子，不過在出現的瞬間就會瞬間配對『湮滅』，『｜粒子』會配對１個『Ｏ粒子』，同理『Ｏ粒子』會配對１個『｜粒子』，但是在同一個時間點可能會產生多個『｜粒子』和『Ｏ粒子』，至於『｜粒子』和『Ｏ粒子』會如何配對就要看各個粒子間的親和力來做決定，親和力的強弱與粒子種類、粒子電荷、粒子間距離...等等眾多因素有關，不過值得注意的是，『｜粒子』對『Ｏ粒子』的親和力不一定等於『Ｏ粒子』對『｜粒子』的親和力，並且不是『｜粒子』和『Ｏ粒子』任意配對就好，這需要考慮到各個『｜粒子』和『Ｏ粒子』彼此間的親和力來形成『穩定湮滅配對』，所謂的『穩定湮滅配對』是指在此配對方案中任意的２組配對『｜粒子』和『Ｏ粒子』－－以下以Ａ組配對的『｜粒子．Ａ』和『Ｏ粒子．Ａ』、Ｂ組配對的『｜粒子．Ｂ』和『Ｏ粒子．Ｂ』作區別－－不能產生以下條件：『｜粒子．Ａ』對『Ｏ粒子．Ｂ』的親和力大於『｜粒子．Ａ』對『Ｏ粒子．Ａ』的親和力並且『Ｏ粒子．Ｂ』對『｜粒子．Ａ』的親和力大於『Ｏ粒子．Ｂ』對『｜粒子．Ｂ』的親和力，若產生上述情況明顯『｜粒子．Ａ』與『Ｏ粒子．Ｂ』進行配對會更加穩定，意即此配對方案並不穩定，須排除上述的情況才能稱作『穩定湮滅配對』。如果覺得剛剛說得令您難以理解，簡單點來說，想像世界是一片大海，平時正常的物質世界是大風大浪，『真空』則是平靜的，但是當然絕對平靜的大海是不存在的，偶爾會翻起幾朵浪花，就算是在『真空』中，也會有粒子出現，只不過隨之又『湮滅』消失，這樣的說明可以離理解嗎？不能理解也沒關係，畢竟這跟日常生活相距太遠了。


　　雖然是說大多數的物理與數學規則在各個世界幾乎都是通用的，但是不同世界之間物理與化學的本質是有可能有所不同的，在此指的本質並不是指某個物體有著異常的狀態或可以違反物理規則，而是指宇宙基本恆定的常數，比如有著不同的光速、不同的『一子』電量、不同的熔沸點... ...等等，連帶會影響到比如『質能交換』後所得的能量會有些微差距，必須提醒的是，這個差異並不是『單位』所帶來的差距，而是從根本上確確實實有增加或減少，不過由於『二進制世界』與『α世界』在本質上的差異小於±2-64％，所以就算在巨觀的世界中誤差連帶被放大增加，但這影響對您來說依舊是不會有所察覺的，甚至可以說完全不會影響到日常生活，所以聽聽當個知識便足夠了。


　　對您將來生活會有實際上影響的『單位』的問題，因為這不只是單單在名稱上的差距，而是在原本定義上就有所不同，以『長度』來說，在『α世界』所定義的基本單位是『公尺』，其完整定義為：『光在真空中於２９９７９２４５８分之１秒內行進的距離』，但在『二進制世界』所定義的『長度』基本單位稱作『℄』，其完整定義為：『氫原子1s1與2s1能階間躍遷輻射在真空中的波長的220倍長』，如果您稍微有點化學知識，暫且先不考慮光速在『α世界』與『二進制世界』有著±2-64％的差距，您應該能清楚發覺『公尺』與『℄』本身長度是有差距的。其它諸如『秒』對應到的是『Ꭶ』、『公斤』對應到的是『₲』... ... 等等，在『α世界』中您所熟悉的『單位』在『二進制世界』是不適用的，不過為了讓您能快速適應『二進制世界』，在先前所說的『免費程序』之中就有『單位轉換程序』，稍後您即可下載並且使用，『單位轉換程序』可以將來自不同世界的『單位』進行轉換。雖然有『程序』的幫助之下您不用特地重新熟悉『單位』，但這對日常生活或多或少會有些許影響，原因是因為來自不同世界的人各自使用他們世界的『單位』，這樣每個人對於數字漂不漂亮的標準會不一樣，在此以『α世界』的電腦當作舉例，您可能會認為１０００這個數字很漂亮，因為在『α世界』是以１０進位為主；但是換作在電腦內部，電腦會認為１０００是個不怎麼漂亮的數字，因為電腦內部是以２進位來進行運算，所以電腦看成的數字是『１１１１１０１０００』；類似的事情也會發生在『單位』上，譬如剛才講的『公尺』與『℄』，經過換算１℄≒０.１２７４公尺，所以如果有人跟您要求５℄的物品時，您可能在『長度』方面上就需要傷腦筋了，所以建議若能盡早習慣『二進制世界』的『基本單位』的話，以長遠的角度來看對您或是其它人都是較為方便的。


　　另外就是在數學方面，比較麻煩的是不同世界的進制可能有所不同，像是在『α世界』中雖然主要使用的是『１０進制』，但也有極少數的民族使用的是『１２進制』或是『６０進制』的情況這樣，這會有問題，假設你只單單寫作『１０』，在不同進制可以被當作不同的數字，在整數方面就已經如此，不知您可否想像在不同進制的情況下表達小數、無理數、超越數... ...等是多麼混亂的情況了，這不只對於數學領域上會成為一個相當大的阻礙，日常生活有關於數字的交流都會有問題，在剛剛大人的『１＋１問答』時我想您應該已經充分體驗到了，所以為了方便數學的部分也能順利發展，所以任何人在『新手程序』結束之後都會強制安裝『進制轉換程序』，『進制轉換程序』要求您設置您所習慣的進制，這樣每個人皆可依照自己的習慣來表達數字，之後透過『進制轉換程序』就可以無障礙的跟他人使用數字做交流，您大可放心，由於這也是為了增進『二進制世界』的發展，所以『進制轉換程序』也是屬於『免費程序』，不必支付任何費用即可使用。在此必須先強調，為了配合您理解方便，所以本『導航狐狸』對您所說的任何數字，除非像是剛剛『１１１１１０１０００』有特別強調之外，皆是以『「十」進制』作為主要表達方式，畢竟如果真的以各個進制來表達，那每個人都能主張他所使用的是『「１０」進制』，不知只用說您是否能夠明白，不明白也無所謂，您只要記得您在數學方面交流不會有問題就行了。


」

## 輸入說明

輸入第一行有一個整數 T 代表測資筆數。


每筆測資的第一行只有一個正整數 N ，代表在某個時間點同時有 N 對的「｜粒子」和「Ｏ粒子」。


接下來的 1~N 行，代表編號 1~N 的「｜粒子」對「Ｏ粒子」的親和度；每行有 N 個整數，分別代表「Ｏ粒子」的編號，排在越前面的「Ｏ粒子」代表該「｜粒子」對此「Ｏ粒子」有較高的親和度。


接下來的 N+1~2N 行，代表編號 1~N 的「Ｏ粒子」對「｜粒子」的親和度；每行有 N 個整數，分別代表「｜粒子」的編號，排在越前面的「｜粒子」代表該「Ｏ粒子」對此「｜粒子」有較高的親和度。


測資保證所有在同一行的整數皆由一個空格做分隔，且行頭以及行尾並無多餘空格。


 


測資範圍：
第00筆測資(01分, 1.0s): 同範例輸入。
第01筆測資(16分, 1.0s): T = 22 , N ≤ 22 。
第02筆測資(22分, 1.0s): T = 23 , N ≤ 23 。
第03筆測資(13分, 1.0s): T = 24 , N ≤ 25 。
第04筆測資(17分, 1.0s): T = 24 , N ≤ 27 。
第05筆測資(31分, 1.0s): T = 24 , N ≤ 210 。
對於所有的測資，「｜粒子」和「Ｏ粒子」的編號 k ， 1 ≤ k ≤ N ，且沒有重複的編號。

## 輸出說明

每筆測資輸出 N 行。


如果存在「穩定湮滅配對」方案，則每行輸出２個整數 o , z，分別代表「｜粒子」和「Ｏ粒子」的編號，由此２個粒子所組成的一組配對。
２個整數中間以一個空格隔開，且行頭及行尾不要有多餘的空格；若存在多種「穩定湮滅配對」方案，輸出任一解即可。


如果不存在「穩定湮滅配對」方案，請輸出一行的 "-1"(不包含引號) 。
