## 輸入說明

每一行有兩個數字n、k，代表當前桌面上有幾張牌，以及需要翻牌的次數


0 <= n <= 10000，0 <= k <= 10^9


接下來一行，每行n個字元i代表牌面數字


-10^5 <= i <= 10^5

## 輸出說明

輸出所有牌數字總和的最大值

## 範例輸入 #1

```
3 1
1 2 3
4 3
0 -1 2 3
3 2
-1 -2 -3
```

## 範例輸出 #1

```
4
6
4
```
