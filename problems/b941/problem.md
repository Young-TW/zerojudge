# b941. 火焰王者塔洛斯

## 內容

在冒險故事中，總會有一些益智問題等待主角去解決，現在英雄柯尼必須向火焰王者塔洛斯挑戰西洋棋，塔洛斯的西洋棋有很多種，


柯尼選了一種有很多的皇后、城堡和主教的西洋棋，遊戲規則要在邊長為棋子數總和的棋盤上放上所有棋子並且不能讓其中任一個被吃掉。


請幫助柯尼想出有多少種擺法能使他的旗子不被吃掉，並打敗火焰王者塔洛斯吧!

## 輸入說明

每筆測資包含一行三個非負整數n,m,k，分別為該種西洋其中皇后、城堡和主教的數量。


-測資點1滿足m=0,k=0
-測資點2滿足n=0,k=0
-測資點3滿足n=0,m=0
-測資點4滿足k=0
-測資點5滿足m=0
-測資點6滿足n=0
-測資點7滿足n+m+k≤8
所有測資保證三者不同時為0且n+m+k<=10

## 輸出說明

對於每筆測資，請輸出一行一個數字，為符合題目要求的可行解數量。

## 範例輸入 #1

```
6 0 0
0 4 0
0 0 5
3 1 0
4 0 2
0 2 3
2 3 2
```

## 範例輸出 #1

```
4
24
3368
8
364
872
41480
```
