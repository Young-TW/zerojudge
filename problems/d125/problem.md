# d125. 阿尼亞的煩惱

## 內容

話說在魯魯修R2最後面華蓮在講說「當然，有些問題還沒解決」時，

正好有照到阿尼亞和ORANGE在採橘子的畫面，

可是之後也沒有說明就結局了。

其實他們在處理橘子時遇到很麻煩的問題了。

他們的橘子樹所產的橘子每個月會增長一次，

可能是ORANGE名子的緣故，

所以長的非常快，

短短一個月內分成5期，

每一期時間內所有橘子園內的樹的枝幹量會是上一期枝幹樹的2倍減去前一期和上一期的枝幹數的差額，

(因為第一個月的前2期為準備階段,所以總枝幹數都為1,真正生長是從第3期開始)

每根支幹在月底時會各結出一顆橘子，

而且神奇的是就算已經摘過了橘子(不會拔掉枝幹)，

在下個月還是會長出橘子，

並且每個月底會採收一次橘子，

因為長的速度很快,$賺很快，

所以阿尼亞只打算做10年就不做了，

但是因為生長太快了，

所以阿尼亞也不知道一個月會產生多少顆橘子(即採多少顆橘子)，

她的「手機」也只能紀錄不能計算，

這讓阿尼亞十分煩惱，

請你幫阿尼亞寫一個程式計算每個月會採收多少顆橘子，

而且因為可能會有一個月要採收很多的情況，

所以只要採收的橘子數是>=10^19的數量的話，

她就會駕駛mordred，

並且找朱雀(駕駛lancelot)、華蓮(駕駛紅蓮2世)和所有黑色騎士團團員一起來採，

所採的天數會是所有橘子數的位數和/(2*第幾個月)也請你的程式算一算需要幾天。

## 輸入說明

輸入的每一行是一個整數n,0<=n<=120,代表這是第幾個月,(第一年1月就是1,2月就是2....,

            假如到第2年1月會以13輸入2月14輸入....,第3年1月會以25輸入...以此類推).當n為0時,

            代表輸入結束,不必對這筆測資作任何處理.

## 輸出說明

對於每一個n分別輸出2行,第一行為這個月所採的橘子數,第2行為請人來幫忙所要花的天數,

       並且數字只要取到整數就好了,(如10/4主要輸出2),如果根本沒請人(即數量<10^19),就輸出0.

## 範例輸入 #1

```
1
2
3
0
```

## 範例輸出 #1

```
5
0
55
0
610
0
```
