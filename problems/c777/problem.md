# c777. 106北二2.規律的數列

## 內容

![image](ShowImage?id=1157)

## 輸入說明

測試資料的只有一行,有兩個正整數 n 與 m,
這兩個數字之間用空格(whitespace)隔開,
其中 1 ≤ n ≤ m ≤ 1,000,000,000,000,000,000。

## 輸出說明

輸出資料為一個數值,為在輸入兩個正整數 n 與 m 所形成的範圍內,屬於上述數列的數值個數。

## 範例輸入 #1

```
範例輸入一：
1 6

範例輸入二：
20 50

範例輸入三：
2 2500000000000000
```

## 範例輸出 #1

```
範例輸出一：
4

範例輸出二：
2

範例輸出三：
58
```
