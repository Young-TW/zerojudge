# d048. 11309 - Counting Chaos

## 內容

Wolfgang Puck 的對手，Emeril Lagasse，最近以 2 公分的大小破了最小的舒芙蕾的廚藝紀錄。Wolfgang 輸人不輸陣，打算自己創一個廚藝紀錄：世界上最對稱的大理石蛋糕。這顯然不是易事。從 Wolfgang 的暢銷自傳中我們知道，他是一個非常迷信的廚師。當他試著要做對稱的蛋糕時，他宣誓只在「迴文時間」把蛋糕從烤箱拿出來，也就是這個時間必需是從左讀到右或從右讀到左都相同。現在的時間不算，下一個他可以把蛋糕拿出來時間是什麼時候？

## 輸入說明

第一行輸入會給你一個 n，代表 Wolfgang 試著做對稱蛋糕的次數。接下來的 n 行每行有一個格式為 "HH:MM" 字串代表二十四小時制的現在時間。(因此，0 ≤ HH ≤ 23，0 ≤ MM ≤ 59，"00:00" 在 "23:59" 之後。

## 輸出說明

對於每一次嘗試，輸出一個表示下一個迴文時間 (不含現在的時間) 的字串於個別的一行，格式為"hh:mm"。當判斷 HH:MM 是否為迴文時，請忽略 HH 所有的前導零。如果 HH 為零，請忽略 MM 的前導零。

## 範例輸入 #1

```
3
00:00
23:30
14:59
```

## 範例輸出 #1

```
00:01
23:32
15:51
```
