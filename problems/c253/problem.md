# c253. 從零開始的異世界生活【序】-異世界の失竊的象徵-

## 內容

這下真的遭了...
身為家裏蹲(引きこもり)的少年菜月昴
在買完晚餐後回家的路上走著走著
突然一陣暈眩 醒了過來
「我說這難道就是所謂的異世界召喚嗎!?」


身上..只有快沒電的手機,錢包..裡面剩10元日幣 
可是我說這世界的貨幣是金銀銅阿...
再來..只剩下便利商店買的杯麵和玉米餅乾,還有身上的衣服,鞋子
「好歹也給我來一把王者之劍阿,我說..這下該這麼辦啊!!」


「路上都是亞人阿..話說看向我的視線連0.1秒都不到....好歹吐槽一下好不...」
一轉頭就看到拿著發著光的飾品的女孩往我這衝
「閃開阿你這混蛋! 給我滾邊去」
我一個瞬間移動 
雖然只是往右一步
但漂亮的躲開了女孩的衝撞
「你這傢伙! 幹甚麼東西」
結果 女孩就這樣跑了 頭也不回的
「可惡..這傢伙真是沒禮貌」



正走入一個巷子
巷子口站著一名少女
而且還是美少女,及腰的銀髮,充滿知性的藍紫色雙瞳凝視著我
「現在我還能原諒你,畢竟是我的疏忽在先 所以高尚的將偷竊之物還來吧」
「慢著,你認錯人了吧?」
我驚訝著看著她
「剛才你說東西被偷了吧?我剛遇到一個全力亂竄的女孩,也許是你要找的人」
「恩...所以說,小偷在另外一頭?得趕快追上去才對」
相信了..?真是單純的女孩...
「我來幫妳吧」
「不用了 這是我的事情」
不行...在弄清楚這異世界之前 必須找個夥伴
「我知道小偷的樣子喔」
「...恩,好吧」
這麼簡單就答應了!!...


於是昴為了幫助女孩就展開了
聖物奪回大作戰!!


這個世界有V個王國 每兩個王國之間是互通的 也就是說
A國可以到B國 B國也可以到A國
但是 有些路...不是看不見盡頭...就是死胡同
這女孩知道王國間的一些情報 有些王國組成聯盟
聯盟之間有【星門】 通過【星門】的時間很短
找一條路(起點終點任意)使得該道路經過【星門】且時間最短


為了更加了解這個神秘的異世界
菜月昴還要找出所有聯盟的情報
畢竟他對這裡一無所知


為了方便計算最少所耗時間的路徑 時間採用異世界的標準單位

## 輸入說明

有多筆測資
第一行有3個integers
V(2<=V<=1000)個王國
E(1<=E<=V*(V-1)/2)道【星門】
T(1<=T<=10)代表走過【星門】所需的時間單位是T
除了【星門】以外的路
都要耗費100時間單位
接下來有E行包含2個integers
a和b(1<=a,b<=V)
代表路在a王國和b王國之間
當V=E=T=0時 輸入結束

## 輸出說明

每個測資 輸出最小所需時間和最大聯邦的國家數

## 範例輸入 #1

```
5 3 1
1 2
1 3
4 5
4 4 1
1 2
1 4
2 3
3 4
0 0 0
```

## 範例輸出 #1

```
Case 1: 103
3
Case 2: 4 
4
```
