# d329. 00713 - Adding Reversed Numbers

## 內容

把一個數字反轉的意思是把各阿拉伯數字的順序反果來。例如：1245反轉之後變成5421。請注意反轉後最左邊的0應該被刪去。例如：2100反轉後變成12，而不是0012

## 輸入說明

數入的第一列有一個整數N，代表以下有多少組測試資料。每組測試資料一列，有2個正整數 ，就是要做反轉相加的數。

## 輸出說明

對每一列輸入，把2數分別反轉再相加，相加後的結果再反轉後輸出。以Sample Input中的第1組測試資料為例說明：305 和 794 經反轉後變成 503 和 497。503+497=1000，再把1000反轉後得到1，就是答案了。

## 範例輸入 #1

```
3
305 794
24 1
4358 754
```

## 範例輸出 #1

```
1
34
1998
```
