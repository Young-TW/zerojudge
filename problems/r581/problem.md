# r581. 10489 - Boxes of Chocolates

## 內容

小皮皮在她6歲的生日收到許多巧克力。她是一個非常好又有魅力的女孩，並且她總是把她的東西與朋友分享。現在她想要把她所有的巧克力分給朋友們。她試著公平的分，讓每個朋友拿到的巧克力的數目一樣多。不過有時候分完之後還有剩（不夠每個朋友一顆了），所以她就把這些給她的狗Kittu吃。

小皮皮收到的巧克力都是放在用漂亮的紙包裝起來的盒子。有時候盒子內沒有直接放巧克力，而是放有一個較小的盒子。而這個較小的盒子有時也是沒有直接放巧克力，而是放一個更小的盒子。只有最小（最內層）的盒子中才真正放有巧克力。

## 輸入說明

輸入的第一列有一個整數T, 代表以下有幾組測試資料。

每組測試資料的第一列，含有2個整數N、B，N代表小皮皮朋友的數目，B代表她收到的巧克力有多少盒。接下來的B列每列代表一個巧克力盒，每列的開始有一個正整數K，然後接著有K個整數a1, a2, a3,......, ak。其中ai代表第i層盒子中含有較小的盒子數目（0 < i < K）。而ak則代表最內層盒子中含有的巧克力的個數。

以第一組測試資料為例說明：小皮皮要分巧克力給5個朋友，她共收到2盒巧克力。第一盒巧克力內有2盒，這2盒內又各有3盒，而最小的盒子內每盒有4個巧克力。第二盒巧克力內有5盒，這5盒內又各有2盒，這2盒內又各有3盒，而最小的盒子內每盒有1個巧克力。所以皮皮總共有2*3*4+5*2*3*1=54顆巧克力。

請參考範例輸入。

## 輸出說明

對每組測試資料輸出一列，在小皮皮平分巧克力給朋友後，剩下多少顆可以給 Kittu。

## 範例輸入 #1

```
2
5 2
3 2 3 4
4 5 2 3 1
6 1
4 5 6 7 8
```

## 範例輸出 #1

```
4
0
```
