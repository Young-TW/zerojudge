# e656. 11137 - Ingenuous Cubrency

## 內容

Cubeland(又稱為立方國)，立方國的人們使用立方幣。
不僅貨幣單位稱為立方，而且硬幣的形狀是立方體，就連其面額也為數字的立方倍。
立方幣的最大面額為9261 (213^3)，也就是代表立方國有面額為1、8、27、...、9261的立方幣。
您的任務是計算立方國的立方幣有多少種支付給定金額的方式。
例如：有3種支付面額21立方幣的方式：
1. 21個面額1立方幣。
2. 1個面額8立方幣和13個面額1立方幣。
3. 2個面額8立方幣和5個面額1立方幣。

## 輸入說明

包含多組輸入，每組輸入一行。
每行包含要支付的整數金額n (0 < n < 10000)。

## 輸出說明

對於要支付的整數金額，輸出有多少種使用立方幣的支付方式。

## 範例輸入 #1

```
10
21
77
9999
```

## 範例輸出 #1

```
2
3
22
440022018293
```
