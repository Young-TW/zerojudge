# d741. 复制贴上——加强版

## 內容

当我们在复制、贴上时,都希望按按键的次数越少越好!! 

当初在学程式语言的时候,一直用for去跑星星? 

现在请你回归传统使用暴力!! 

 

一开始会有一颗星星,请你告诉我,最少的次数,以及有多少种方式,以及怎么按!! 

因为C离Ctrl比较近所以C优先于V 

而且一复制就是复制全部

Ctrl + C 是复制  Ctrl + V 是贴上

每行会有一个n (2<=n<=2147483647) 代表你要复制到几颗星星。

## 輸出說明

请输出最少的按键次数、以及最少次数有几种。

测资中: (1000笔输入而已)

```
50
```
