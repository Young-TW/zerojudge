# b301. 板條大冒險 (三)：JACKY 的路

## 內容

「遊戲設計師之道，到底在哪裡呢 ......」放學後的夕陽，照射在 jackyXX 的側臉。想當初當上幹部之後，天天被茵可抓去測試遊戲，還要接受各種奇奇怪怪的考驗(例如遊戲中的 NPC 們：廖氏如神、老妮可、長頸鹿許胖、...等人的任務)，難道程式設計師的工作就是不斷的測試、不斷的輪迴嘛？

正在找尋答案的 jackyXX，毅然決然跑去購買剛推出的遊戲 SAO (Soil and Asphalt Online)，這個遊戲係藉由高週波頭盔界面，和人腦接觸後將意識輸入伺服器中，由磁場偵測大腦活動，並且由微波刺激大腦額葉皮質，進而模擬真實觸感的遊戲。然而遊戲者的軀體會因頭盔由脊髓輸出訊號而呈現假死狀態，因此常有家屬誤以為去世而將之火化的案例，是非常不安全的遊戲。

jackyXX 進入後，發現整個世界都是土(Soil)和柏油(Asphalt)所構成，還有幾個路人正在遊走，不到五分鐘便想離開。

「咦？我的登出鍵呢？」jackyXX 手舞足蹈一陣，目錄選單內並沒有登出按鈕。想起知名輕小說劇情的 jackyXX 暗叫不妙，倏然天空一陣巨響，jackyXX 眼前閃起一道白光，轉瞬間到了一座廣場，廣場上陸陸續續有人被傳送進來。

(10 小時後)
 
 「喂！喂！這位先生起床了，遊戲設計師出場了！」睜開惺忪雙眼，看見一旁的男士親切叫醒他，再看看空中一團黑影，原來是 SAO 遊戲設計師－－Almighty Lu。

驚！

「難道你是故意將我們封鎖在這裡，要破一百層關卡才能回到現實世界嘛！！」jackyXX 大喊。

「不是、不是，這是 bug，我們的工作團隊正在搶修中，估計還要兩年才會修好 ...」Almighty Lu 說。

...... 什麼？！

「我這兩年不就是只能吃土和玩柏油了嘛 ...」jackyXX 暗忖，後悔這時候應該要在茵可那邊測試遊戲，想到不禁簌簌地流下眼淚。

忽然，jackyXX 看見一人戴著黑布小帽，穿著黑布大馬褂，深青布棉袍，蹣跚地走到廣場邊，慢慢探身下去，尚不大難。可是他穿過柵欄，要爬上這邊的廣場，就不容易了。他用兩手攀著上面，兩腳再向上縮；他的身子向左微傾，顯出努力的樣子。這時我看見他的背影，我的淚很快地流下來了。我趕緊拭乾了淚，怕他看見，也怕別人看見。我再向外看時，他已抱了朱紅的橘子走來了－－是茵可！他先將橘子散放在地上，自己慢慢爬下，再抱起橘子走。到這邊時，我趕緊去攙他。他和我走到廣場正中央，將橘子一股腦兒放在我的皮大衣上。於是撲撲衣上的柏油，心裏很輕鬆似的，過一會說，「我走了，到那邊來信！」

「等等！茵可怎麼會在這！！！」jackyXX 不可置信地看著橘子。

「沒，就這遊戲的 bug 有點多，原本只是來看看沒想到就進來了。」茵可回。

「那 ...... 這堆橘子是怎麼回事？」

「無心插柳柳橙汁。」茵可停頓一下，又繼續說：「他的伺服器只有一個，為了處理那麼大量的遊戲人潮，只好先將各個使用者組織成樹狀，最後再和伺服器連接，因此只要知道哪一條連結，是我跟你之間連線會通過的連結，我們就有辦法出去了。」

「是喔！那我們開始 coding 吧！」jackyXX 花了五秒鐘想到正姐。

「...... 你忘了嘛？這裡只有土和柏油ˊ_>ˋ」茵可一臉淡定。

板條高中資訊社經由 ICMP 連絡到茵可與 jackyXX，得知連線分布。為了新的遊戲誕生，板條高中的學生們是否能營救出茵可和 jackyXX 呢？

## 輸入說明

輸入第一行有兩個整數 N、Q，N 代表使用者人數，依序編號為 1 到 N，而伺服器編號為 0；Q 代表詢問次數。接著有 N 行，每行有兩個整數 A 和 B，A、B 同為編號，代表 A 和 B 間有連結。緊接著有 Q 行詢問，每次詢問有四個整數 W、X、Y、Z，W 代表茵可的編號，X 代表 jackyXX 的編號，Y、Z 代表一條連結。
 
題目保證所有使用者都會相互連通，並且 Y、Z 一定是一條連結，且 W、X、Y、Z 兩兩相異。

1 <= N <= 10000
1 <= Q <= 1000
0 <= A,B <= N
1 <= W,X <= N
0 <= Y,Z <= N

有不超過 50% 的測試資料 N >= 1000，Q >= 100
有不超過 10% 的測試資料 N >= 9000，Q >= 500

## 輸出說明

對於每個詢問，如果 Y、Z 間斷開連結，會使得 W、X 不連通，則輸出「GO」(不含引號)；否則，輸出「STOP」(不含引號)。

## 範例輸入 #1

```
3 2
0 1
0 2
1 3
1 3 0 2
2 3 1 0
```

## 範例輸出 #1

```
STOP
GO
```
