# n768. 10370 - Above Average

## 內容

據說90%的新生期望在班級中表現超過平均水準。你需要提供一個現實檢查。



![image](ShowImage?id=4066)

## 輸入說明

標準輸入的第一行包含一個整數 C，表示測試案例的數量。接下來是 C 組數據。每組數據開始於一個整數 N，表示班級中的人數（1 ≤ N ≤ 1000）。接下來是 N 個整數，這些整數由空格或換行符分隔，每個整數表示班級中某個學生的期末成績（成績範圍在0到100之間）。

## 輸出說明

對於每組數據，你需要輸出一行，給出成績高於平均成績的學生百分比，精確到小數點後三位。

## 範例輸入 #1

```
5
5 50 50 70 80 100
7 100 95 90 80 70 60 50
3 70 90 80
3 70 90 81
9 100 99 98 97 96 95 94 93 91
```

## 範例輸出 #1

```
40.000%
57.143%
33.333%
66.667%
55.556%
```
