# s086. Atrium

## 內容

傳統羅馬住宅的中庭，與現代的中庭非常相似，是一個完美的正方形房間，供住戶和客人聚集，享受從上方傾瀉而下的陽光。或者，在不列顛尼亞的例子中，是享受從上方傾瀉而下的雨水。

傳統羅馬建築，尤其是在現代，一個主要問題是房間之間缺乏有效的隔間牆。你來到了義大利，現在將受政府委託重建這些隔間牆，首先從你發現的一棟破敗不堪的建築的中庭開始。

要完全封閉建築物的中庭，您需要攜帶多長的預製牆體？

## 輸入說明

輸入由一個整數 a (1 ≤ a ≤ 10^18) 組成，表示中庭的面積（以平方公尺為單位）。

## 輸出說明

輸出中庭所需牆體的總長度(小數後10位)，單位為公尺。長度的絕對誤差或相對誤差應不超過 10^-6 。

## 範例輸入 #1

```
64
```

## 範例輸出 #1

```
32.0000000000
```

## 範例輸入 #2

```
2
```

## 範例輸出 #2

```
5.6568542495
```
