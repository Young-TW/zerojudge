# n757. 12578 - 10:6:2

## 內容

孟加拉國的國旗是瓶綠色的矩形，長度 (L) 與寬度的比例為 10:6。旗幟上有一個紅色圓圈，圓圈背景為綠色。長度 (L) 與圓圈半徑的比例為 5:1（如果長度為 10，則寬度應為 6，半徑應為 2）。背景的綠色代表了孟加拉國的綠色植物，而紅色圓圈象徵著旭日東升和在自由戰爭中犧牲的生命。

## 輸入說明

輸入的第一行將包含測試案例的數量，T ≤ 100。接下來有 T 行，每行包含一個正整數 L ≤ 1000，代表旗幟的長度。

## 輸出說明

對於每個測試案例，輸出一行，包含兩個以空格分隔的小數，精確到小數點後兩位。這兩個數字分別代表紅色部分和綠色部分的面積。 注意：Pi 被認為是 arccos(-1)。

## 範例輸入 #1

```
1
10
```

## 範例輸出 #1

```
12.57 47.43
```
