# e641. 10260 - Soundex

## 內容

Soundex編碼根據拼寫將聽起來相似的單詞組合在一起。
例如："can"和"khawn"，"con"和"gone"。
Soundex編碼將每個單詞轉換為一組數字，其中每個數字代表一個字母。
以下為Soundex編碼表：
1：B、F、P、V
2：C、G、J、K、Q、S、X、Z
3：D、T
4：L
5：M、N
6：R
Soundex編碼中未表示字母A、E、I、O、U、H、W、Y。
具有相同編碼字母如果相鄰的重複出現僅以一個數字表示。
具有相同編碼的單詞視為相同單字。

## 輸入說明

輸入的每一行都包含一個單詞，全部為大寫字母。
單詞長度小於20個字母。

## 輸出說明

對於每行，輸出此單詞的Soundex code。

## 範例輸入 #1

```
KHAWN
PFISTER
BOBBY
```

## 範例輸出 #1

```
25
1236
11
```
