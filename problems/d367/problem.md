# d367. 我要最小值——加强版

## 內容

给定三角形ABC

在三角形ABC内取一点P

想必一定有个点P能使得PA+PB+PC有最小值

所以给你三角形的三边长a、b、c

请输出能找到的PA+PB+PC的最小值

## 輸入說明

每行输入三个正整数a,b,c

1<=a,b,c<=100

并且测资保证a,b,c可组成一个三角形

以a=b=c=0时代表输入结束

不需处理这行输入

## 輸出說明

在三角形ABC内的所有P点当中有些点能使PA+PB+PC有最小值

输出这个最小值，并以小数以下两位方式输出(%.2f)

## 範例輸入 #1

```
65 68 83
68 80 66
0 0 0
```

## 範例輸出 #1

```
123.83
123.03
```
