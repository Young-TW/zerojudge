# i212. 三則運算

## 內容

計算三則運算式的結果，包含加 $(+)$、減 $(-)$、乘 $(*)$

## 輸入說明

輸入由兩個數字 $a$ 和 $b$ 和一個運算符號 $x$ 組成


數字和運算符號之間有一個空格


$a$ 和 $b$ 不會有前綴 $0$


$0 \leq a, b \leq 10^{10^6}$


$x \in \{+, -, *\}$


若 $x$ = $-$，則保證 $a \geq b$

## 輸出說明

請輸出運算結果

## 範例輸入 #1

```
123 + 456
```

## 範例輸出 #1

```
579
```

## 範例輸入 #2

```
12 * 2
```

## 範例輸出 #2

```
24
```

## 範例輸入 #3

```
100 - 1
```

## 範例輸出 #3

```
99
```

## 範例輸入 #4

```
0 - 0
```

## 範例輸出 #4

```
0
```
