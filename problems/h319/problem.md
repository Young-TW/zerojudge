# h319. 超級常數測試

## 內容

n 個人站在位置 1 到 n。 您必須執行兩種類型的查詢：


"- x" — 位置 x 的人坐下；
"? x" — 找到離自己右邊最近的仍然站立的人。

## 輸入說明

第一行有兩個整數n,q(<=5e6)代表n個人和q筆詢問


接下來q行詢問


詢問格式參考題目內容。

## 輸出說明

對每個"?"的詢問輸出一個數字代表答案。

## 範例輸入 #1

```
5 10
? 1 
- 3 
? 3 
- 2 
? 1 
? 2 
- 4 
? 3 
- 5 
? 3 
```

## 範例輸出 #1

```
1
4
1
4
5
-1
```
