# a686. 蝸牛往上爬

## 內容

傳說中彰中大樓的牆壁上住著一隻大蝸牛，有一天他突然想爬到頂樓去，
大蝸牛從一樓高開始爬，牆壁高X公尺，每天從早上爬到傍晚可以爬Y公尺，
晚上休息時會下滑Z公尺。給你x,y,z，請問這隻大蝸牛在第幾天能爬到頂樓?

## 輸入說明

第一行有一個正整數n，代表有幾組測資
接下來2～(n+1)行，每行有3個正整數代表x,y,z。

## 輸出說明

輸出共n行，每行輸出一個答案
如果大蝸牛爬得到頂樓，請輸出天數
如果爬不到的話，請輸出"Poor Snail"

## 範例輸入 #1

```
3
10 3 2
13 4 1
10 2 2
```

## 範例輸出 #1

```
8
4
Poor Snail
```
