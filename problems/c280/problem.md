# c280. 三角形又來囉~

## 內容

今天小瑤發現了一件驚為天人的事情--給定一個長度為n的正整數序列，一定能選出三個不同的元素，當邊長構出一個三角形，而只要邊長的值域小於某個數值A[n]，這件事情就一定會是對的，很神奇吧XD。另外由A[1], A[2], A[3]... 構成的數列<A> 和我們熟知的某個數列居然長的一模一樣，你知道是哪一個嗎OuO，對於一個給定的n你能算出A[n]是多少嗎? (貼心提醒n小於3時，你要輸出那個長的一模一樣的數列的第n項喔:3)
Sea Food: 信仰成力量
小A: 最後你會發現這個數列是個遞迴數列，應該開個Array逐一慢慢推出每一項就可以了，而且這個數列跟兔子(大兔子生小兔子)和東東(東東爬樓梯)有關呢:3
小W: 這根本就只要解出一般式硬算就可以啦~
小C: 只要丟空白Code就會過了呀...
小T: 直接寫個遞迴跑下去就沒問題了
小R: 開個大小為1000000000000的陣列肯定就沒問題了!

## 輸入說明

第一行有一個正整數n，0 < n < 81。

## 輸出說明

If n >= 3, 請輸出A[n]。
Else, 輸出那個長的一模一樣的數列的第n項

## 範例輸入 #1

```
範例輸入1:
1

範例輸入2:
2

範例輸入3:
3

範例輸入4:
4
```

## 範例輸出 #1

```
範例輸出1:
1

範例輸出2:
1

範例輸出3:
2

範例輸出4:
3
```
