# f866. DD排磁磚

## 內容

DD人物介紹 : https://zerojudge.tw/ShowProblem?problemid=f754 


DD平時沒什麼朋友，放學後也不知道要幹嘛，沒人會跟他一起打電動，而他也不想寫作業，他每天回家的路上會經過一個工地，外面放了許多印有不一樣花紋磁磚，原本都已經快排好了，但DD經過時便會無聊把他弄亂，使工頭非常生氣，並把DD叫去罵一頓。不過，有時DD弄亂後，竟然恰好與工頭希望的順序相同，這會使工頭非常開心，當天就不會找DD來訓話。


  工地的磚頭，分為「待用區」與「廢棄區」，DD會把他們混在一起，弄亂後再把跟原本待用區一樣數目的磁磚放回待用區，其他的則丟回廢棄區。


  DD的同班同學們，很喜歡看DD被罵，甚至開始打賭DD今天是否會再被工頭罵，身為那其中的一份子，請算算他今天被罵的機率是多少吧!

## 輸入說明

第一行輸入一個整數T，代表有幾筆輸入。


每筆輸入第一行有一個整數M和一字串S，代表磁磚種類數目與工頭希望DD在待用區排成的樣子。接下來有M行，每行有一字元C與整數N，代表C樣式的磁磚有N個


0<M≦10，S長度≦12


C為大小寫英文字母


所有磁磚總數≦30

## 輸出說明

輸出DD被罵的機率 -> 以整數(有可能是0或1)或最簡分數表示


如果DD今天不被罵的機率不低於1/2，輸出"DD真幸運!!我要下注!"。

## 範例輸入 #1

```
3
2 xoxox
x 3
o 2

3 xdddd
d 3
x 2
a 5

2 dd
d 10
e 2
```

## 範例輸出 #1

```
9/10
1
7/22 DD真幸運!!我要下注!
```
