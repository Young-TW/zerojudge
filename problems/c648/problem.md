# c648. 免洗平台

## 內容

你現在和一堆人在一起


你現在在進行忍耐任務


你現在在51區


你現在必須要跨過一堆漂浮平台到52區


你現在有危機


 


OK 問題來了


 


這些平台是消耗品


而且一次只能站一個人


而且沒辦法修復


 而且每次讓一個人經過就會扣一滴血


 也就是說 A->平台,平台到->B , 平台血量-1


 


51區到52區總共W-1 個平台


起點編號0,終點編號W


則平台有W-1,編號1~W-1


 


每個人最多可以一次跨過L個平台 (你目前在X可以到X+L


)


結論:


有多少人可以挺過這一關到52區

## 輸入說明

第一行輸入T:代表測資數


每筆測資輸入 W,L


代表 總長W, 一次最多可以跳L平台


還有W-1個i代表平台的血量 Hi


 


 


 測資範圍:


第00筆測資:T=102,1≤L<W≤102,0≤Hi≤50


第01筆測資:T=103,1≤L<W≤102,0≤Hi≤104


第02筆測資:T=50,1≤L≤min(100,W-1),L<W≤105,0≤Hi≤104


第03~04筆測資:T=50,1≤L<W≤105,0≤Hi≤104


第05筆測資:T=30,1≤L<W,W=105,0≤Hi≤104

## 輸出說明

輸出最多有幾人可以過關

## 範例輸入 #1

```
1
10 5
0 0 2 0 1 0 0 2 1
```

## 範例輸出 #1

```
3
```
