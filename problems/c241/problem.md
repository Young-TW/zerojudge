# c241. 藍燄你……有點太奇怪了喔

## 內容

在一個黑夜籠罩、萬籟俱寂的森林之中，藍燄走到了在黑夜之中相當耀眼的營火旁，盤腿坐下，並面向著他身旁的少女，少女的臉上充滿著期待，期待著藍燄繼續敘說著他的故事。為何他要說故事呢？因為對他來說……也許對所有的世界也是如此吧，東風即將到來(East wind is coming)，一股可以撼動所有世界的東風。而藍燄，想要在自己的生命正式劃下句點之前，再全部回憶一次這一生的經歷，很長很長的一生。
 
 
「吶，藍燄。」少女戳了戳躺在地上並仰望著星空的少年，她內心似乎有些許疑惑。「怎麼了？莉莉絲」藍燄起身面向這位名叫莉莉絲的少女。「你有什麼事情是辦不到的嗎？」莉莉絲道。「痾……你是在問我超能力的極限嗎？」藍燄露出了疑惑中帶有一絲厭惡的神情。「對啊怎麼了，你似乎有點不太高興，我踩到地雷了嗎？那樣的話，抱歉。」少女道。「不不，不是你想的那樣。理論上來說，我可以做到任何事，甚至比起你們信任的神明還要更為強大，我不太喜歡這種無限的力量，我很討厭這樣。」藍燄嘆了口氣如此說。
「為啥，你什麼事都做得到耶？可以創造無限多的冰淇淋來吃，多好呀！」「正因為什麼都做得到才可怕不是嗎？我可以隨心所欲的毀滅我不想要的世界，或是創造一個我理想的世界。這令我相當地害怕，害怕哪一天我突然發瘋把一切都消滅。『正因為做得到，你便這麼做了。』這是我某個朋友對別人說過的話，我至今仍不時地想起這句話。」語畢，藍燄垂頭並又更哀怨地嘆了氣。
莉莉絲這時抱住了藍燄並說「別哭啦，因為你沒那麼做不是嗎？況且你會忌憚著這樣的事，說明你不早就是你想要當的『真正的』好人嗎？」「我沒哭，而且我也不是好人，好人的手上可不會沾染過無數生命的血。」藍燄把莉莉絲推開後，神色確實有平復了許多。良久，莉莉絲突然這麼問：「那位開著TARDIS到處飛的『博士』，你覺得是好人嗎？」「是阿，而且是我少數看過的『真正的』好人。」藍燄道。
「可是……那位『博士』可是有結束過別人的生命喔。可是你把他當成了好人，卻不把自己的人格歸類於『善』，你認真的？」莉莉絲有點生氣地指著藍燄。「你不是活了很久嗎？應該有想過這樣的問題吧？這不太像你喔，藍燄。至少不像你那些故事所告訴我的『你』。起碼你想成為好人不是嗎，而且還努力地去實現這樣子的目標，那樣就夠稱作好人了吧？」藍燄沉默了。
良久過後。「……那個……可以變出冰淇淋給我吃嗎？」莉莉絲突然一臉興奮地兩手各搭在藍燄的肩膀上搖著他。「等……別搖啦！而且為啥突然提出這樣子無理的要求？我心情還在低潮耶！」藍燄試圖以言語制止莉莉絲繼續晃動他，但是失敗了。正當他想要把莉莉絲的手挪開時，她自己便停了下來。
「這要求是作為我成功安慰藍燄你的獎勵。」莉莉絲雙手撐著腰、挺著胸膛，並自信地向藍燄這麼說道。「你剛剛那算哪門子安慰啊？而且根本沒成功不是……嗎？」一聽到藍燄的話，莉莉絲鼓起臉頰並嘟著嘴表示不滿。「好好，我變出來給你總行了吧？那你要什麼口味？」藍燄對於少女的詭異行為舉止實在招架不住，於是答應了她的要求。「那我要瑞士巧克力口味的，嘻嘻！」莉莉絲高興地笑著說。
藍燄心想：嘻嘻個頭呀？唉，自願當工具人的我也沒辦法說什麼。於是，藍燄右手一擺，便憑空出現了一個瑞士巧克力口味的冰淇淋在他手上。「拿去吧。」藍燄把上手的冰淇淋地給了少女。「我還是看不出來你是怎麼『創造』出東西的，原理是啥？」莉莉絲舔著冰淇淋道。「只是質能互換而已，我可以操縱能量的流向，然後將能量轉換成我想要的物質。」「你可以操縱能量！太誇張了吧？」莉莉絲驚訝到冰淇淋從手上滑落了下來。「把冰淇淋給我拿好啊喂。」藍燄右手一指，冰淇淋便在半空中驟停了。
「所以這也是『操縱能量』的延伸？」莉莉絲把停在半空的冰淇淋拿了回去，繼續津津有味地舔著。「是沒錯啦……但是操控能量的去留不是我真正的能力。」藍燄走到一旁，倚著樹幹道。「你真正的能力是啥呀？」莉莉絲繼續舔著冰淇淋。「不想說。」藍燄簡短地表示。「什麼嘛！明知道人家會有興趣知道，不想談還講給我聽，哼！」莉莉絲一氣之下直接把剩下的冰淇淋連帶餅乾一起大口吞下。
「沒冰淇淋吃了，好無聊喔……可以幫我做暑假作業嗎？我看你做。」莉莉絲從地上的背包裡拿出許多的厚厚的試題本。「原來你是學生嗎，完全看不出來啊！話說，自己的作業自己做啊喂！請不要把我當工具人好嗎？」藍燄用手大力地指著莉莉絲。「什麼嘛，我看起來有那麼老嗎？應該比較像普通的高中女生吧？」莉莉絲轉了轉身體表示忿忿不平。「你頭上長著兩個角，膚色偏暗紅色，瞳孔還是紅色的，我還以為你是惡魔之類的東西呢。」藍燄挑著眉頭說。「是惡魔沒錯啊，惡魔也是可以上學的好嗎！」莉莉絲用力地跺了腳。「就那樣子上學？」藍燄表示。「對阿，怎麼了？我……我看起來那裡很怪嗎？」「倒是沒有。只是我得知了原來這世界是人類跟惡魔共存的啊，厲害厲害。抱歉，剛剛都只是我的刻板印象而已，哈哈。」藍燄將手掌平舉到與肩同高，表示無奈。
「所以說，為啥不自己做作業？」藍燄指了指莉莉絲還拿在手上的本子。「因為我都已經會了嘛！而且一直做那樣的題目有什麼意思呢？還不如學習新的知識還比較實際一點。」莉莉絲把手上的簿子硬塞給了藍燄。「好好好，讓我看看你們現在在學些什麼……哦？是機率概論喔，無聊透頂，嗯？旁邊這些一坨數字和運算是幹啥的，這幾頁不是在講機率的嗎？」藍燄皺著眉頭道。「那是我們程式設計的比賽的題目，才不是『機率概論』好嘛！往前幾頁才是我要你做的習題。」莉莉絲不耐煩地說。
「你們有程式的比賽，所以這些是考古題？」「對啊，不過跟我沒關係，快翻到前面啦！」莉莉絲更不耐煩地催促著藍燄。「原來是大數題啊？而且還全部是整數的運算。哈哈笑死，這樣我用Python不就秒……喔等等原來是要算進位借位的次數。什麼嘛，還是水題啊……呃啊啊好痛啊，別抓著我耳朵啦！」莉莉絲面露凶光地抓著藍燄的耳朵並怒吼：「快！給！我！翻！到！前！面！去，你這個工具人！」「喂喂，你的個性設定跑掉了吧啊啊啊，別更用力啊！」
藍燄日常……？

## 輸入說明

/*


你是一個在只有藍燄看得到的攝影機後方的觀眾，很剛好地，你對於程式設計也頗有心得。因此想寫寫看藍燄提到的大數題。


*/  (？


每一行的輸入會有兩個數字(Number1、Number2)和一個運算符號(+ 或 -)。


格式為 "Number1 +/- Number2" (不含括號"")，數字跟運算符號之間有空白隔開。


 


測資範圍：


Number1, Number2 < 101000


且保證 Number1 - Number2 ≥ 0

## 輸出說明

針對每一行的輸入，輸出一行表示加法進位或減法借位的次數。

## 範例輸入 #1

```
1 + 1
2 + 8
87 + 87
456 + 144
9999999999 + 1

87 - 0
12 - 3
88 - 29
188 - 99
789456123 - 456789123
```

## 範例輸出 #1

```
0
1
2
2
10

0
1
1
2
3
```
