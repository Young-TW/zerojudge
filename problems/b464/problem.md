# b464. 聲音時間長度

## 內容

播放器都會顯示聲音時間長度。給你一段聲音，請你求出總時間。

## 輸入說明

一段聲音：兩個整數 R N (8000 <= R <= 48000) (1 <= N <= R*100)，是取樣頻率與訊號數量。下一行有 N 個整數，是訊號數值 (-32768 ~ +32767)。

取樣頻率的意思是：錄製聲音時，每秒鐘擷取的訊號數量；播放聲音時，每秒鐘處理的訊號數量。

## 輸出說明

時間格式為 M:SS。M是分，至少顯示一位數；S是秒，固定顯示兩位數。秒數若非整數，請四捨五入。

## 範例輸入 #1

```
8000 1
0
```

## 範例輸出 #1

```
0:00
```
