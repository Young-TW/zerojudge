# o361. 計算多邊形面積

## 內容

計算多邊形的面積


請寫一個程式，可以讀入多邊形的頂點座標，並計算出該多邊形的面積。

## 輸入說明

第一行輸入數n，表示接下來有多少組頂點座標。之後n行，每行包含一組頂點座標，格式為 x y。


其中 n <= 10000,  x <= 10^8, y <= 10^8。

## 輸出說明

輸出該多邊形的面積，四捨五入至整數。

## 範例輸入 #1

```
4
0 0
4 0
4 3
0 4
```

## 範例輸出 #1

```
14
```

## 範例輸入 #2

```
4
1 1
1 4
4 4
4 1
```

## 範例輸出 #2

```
9
```
