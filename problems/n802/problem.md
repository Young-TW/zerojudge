# n802. 11991 - Easy Problem from Rujia Liu?

## 內容

雖然劉汝佳通常會為比賽設置困難的題目（例如，西安2006、北京2007和武漢2009的區域比賽，或者UVa OJ比賽如劉汝佳的Presents 1和2），但他偶爾也會設置簡單的題目（例如UVa OJ中的“the Coco-Cola Store”），以鼓勵更多的人來解決他的問題


給定一個數組，你的任務是找到整數v的第k次出現（從左到右）。為了使問題更加困難（和有趣！），你需要回答m個這樣的查詢。

## 輸入說明

有若干個測試案例。每個測試案例的第一行包含兩個整數 n 和 m (1 ≤ n, m ≤ 100,000)，分別表示數組中的元素數量和查詢的數量。接下來的一行包含 n 個不大於 1,000,000 的正整數。接下來的 m 行中的每一行包含兩個整數 k 和 v (1 ≤ k ≤ n, 1 ≤ v ≤ 1,000,000)。輸入以文件結束符 (EOF) 結束。

## 輸出說明

對於每個查詢，輸出該元素從左到右第 k 次出現的位置（基於 1 的位置）。如果沒有這個元素，則輸出 ‘0’。

## 範例輸入 #1

```
8 4
1 3 2 2 4 3 2 1
1 3
2 4
3 2
4 2
```

## 範例輸出 #1

```
2
0
7
0
```
