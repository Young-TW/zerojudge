九九(撲克牌)規則：


A：累計數字歸零
2：累計數字加2
3：累計數字加3
4：迴轉，即更改出牌的順序，順時針改為逆時針，反之亦然
5：指定，隨便指定一位玩家出牌
6：累計數字加6
7：累計數字加7
8：累計數字加8
9：累計數字加9
10：累計數字加10或減10，減10的時候如果小於零以零計算
J：pass，即由下一位玩家出牌
Q：累計數字加20或減20，減20的時候如果小於零以零計算
K：累計數字變成99

[到底要玩多久，都半夜了]

[我來檢查，抓到作弊的人請客一個月]

## 輸入說明

每一行有兩個數字n、m，n代表有n個人編號1~n，m代表已經出了幾張牌


2 <= n <= 100、1 <= m <= 1000


接下來有m張牌，牌可能是A、2、3、4、5(x)、6、7、8、9、10-、10+、J、Q-、Q+、K


其中5(x)，x為指定編號(1 <= x <= n)


"10-"代表減10、"10+"代表加10


"Q-"代表減20、"Q+"代表加20


一開始出牌方向為順時針


n個人按照編號順時鐘坐，編號1的人先出牌

## 輸出說明

如果抓到誰爆點請輸出


"x cheated!"，x為爆點人編號


如果有多個爆點人，請輸出第一個


否則輸出


"The sum is x"，x為當前總和

## 範例輸入 #1

```
5 5
K Q- Q+ J 4
3 5
A A A A A
4 7
K K J 5(1) Q+ Q- A
```

## 範例輸出 #1

```
The sum is 99
The sum is 0
1 cheated!
```
