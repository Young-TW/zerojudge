# a506. 走投無路的國王

## 內容

西洋棋，又稱歐洲象棋或國際象棋，是一種二人對弈的戰略棋盤遊戲。
黑白棋子各16個，多用木或塑膠製成，也有用石塊製作；較為精美的石頭、玻璃（水晶）或金屬製棋子常用作裝飾擺設。
西洋棋是世界上最受歡迎的遊戲之一，全世界有5億西洋棋人口。

<以上摘自維基百科>

婷婷最近迷上了西洋棋

在經過幾天的的訓練後，連原本教導她西洋棋的文文都不是她的對手了
不甘心的文文決定發明新的遊戲，來重拾他的自信心

新遊戲是在一個N*M的棋盤上玩
文文知道婷婷最喜歡西洋棋裡的國王，所以新遊戲的棋盤上只有一個國王棋子
就像西洋棋一樣，文文跟婷婷輪流移動棋盤上那唯一的棋子
每一次移動棋子都可以往 上 下 左 右 左上 左下 右上 右下 8個方向移動 (當然移動後不可以超出棋盤)
但是已經被走過的格子就不能再走
而且在遊戲一開始，棋盤上有些格子髒掉了
為了不要弄髒婷婷最愛的國王棋子，那些髒掉的格子是一定不能走到的
沒辦法再走下一步的人就輸了

基於紳士風度，文文讓婷婷先走，請問誰會必勝？

P.S.文文跟婷婷都很聰明~

## 輸入說明

每個測資檔的第一行是一個正整數T，代表接下來有T筆測試資料(1<=T<=10)

每筆測資第一行有兩個整數N，M，代表棋盤的大小(1<=N,M<=50)
接下來N列，每列M個字元，代表一個棋盤的初始樣子
'K'代表國王一開始的位置，'#'代表髒掉的格子，'.'代表可以走的格子

輸入保證每個棋盤恰有一個國王

## 輸出說明

每筆測試資料輸出一行必勝者的名字
若婷婷必勝輸出"Ting"
文文必勝輸出"Wen"

## 範例輸入 #1

```
2
2 3
K..
.##
3 4
.#.#
#.#.
K#.#
```

## 範例輸出 #1

```
Ting
Wen
```
