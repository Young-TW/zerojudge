# q869. 逐步建立的密碼

## 內容

在某間情報機構中，密碼是由一個長字串逐步建立而成的。每次密碼更新時，會在現有密碼的尾端新增幾個字元，並紀錄下每一次更新後的密碼樣貌。現在你手上有這些歷次密碼的紀錄，但它們是無序的。你的任務是，根據密碼長度由短到長的順序，將這些紀錄依序排列，並以右對齊的方式輸出，模擬密碼從無到有逐步建立的過程。

## 輸入說明

第一行輸入一個整數 n，代表接下來有 n 行的密碼紀錄。
接下來 n 行，每行一個字串，代表某次更新後的密碼樣貌。

## 輸出說明

輸出 n 行，每行為一個密碼紀錄，依照字串長度由短到長排序，並且右對齊輸出，使得最長的密碼在最右邊，其他密碼與它對齊。

## 範例輸入 #1

```
3
abcd
a
abc
```

## 範例輸出 #1

```
   a
 abc
abcd
```
