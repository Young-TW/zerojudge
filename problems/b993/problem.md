## 輸入說明

輸入共有兩行，


第一行為一個正整數，代表共有 N 隻史萊姆，


第二行為每隻史萊姆的擊殺分數 P，數字間以空白隔開，


程式需要重複輸入資料。

## 輸出說明

輸出全部史萊姆中分數最高的擊殺分數 Pmax。

## 範例輸入 #1

```
5
1 7 69 87 0
1
66
```

## 範例輸出 #1

```
87
66
```
