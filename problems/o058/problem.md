# o058. DD的計算機

## 內容

DD在考數學前，密謀想用計算機作弊，而他的好同學紅林，早就看穿次陰謀並偷偷將DD的換成一台他自製的計算機，以下是此計算機的特殊構造：


- 數字只有7 8 9能按
- 運算子只有+ - * / =能用
- 一旦按下任一運算子，計算機會自動顯示之前運算是的結果，即無先乘除後加減
- 只有4個位數可以使用
- / 是整數位無條件捨去，無小數點
- 若按下上述未出現的符號，或是任一時刻計算機上顯示的數字超過6位數，或是除以0，計算機會當場撥放此音效："DD爛! DD爛! ......"，使DD當場因作弊被逮捕。

身為DD的真正的摯友，你告訴了DD此訊息，但由於DD頭腦簡單，請你順便提示他，若要讓計算機顯示出某數字，至少要按幾下按鍵。

## 輸入說明

輸入T，代表有T比測資。


之後T行，每行1個數字N，代表DD想按出N這個數字。


1 <= T <= 100


1 <=N<= 10000

## 輸出說明

輸出至少要按幾下

## 範例輸入 #1

```
5
25
78
2
1000
243
```

## 範例輸出 #1

```
6
2
4
8
8
```
