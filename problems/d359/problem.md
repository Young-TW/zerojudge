給你三角形ABC的面積S

點 D, E, 及 F 分別為 BC, CA, 及 AB 三個邊上的點

並且AF/FB=x；BD/CD=y；CE/EA=z

並且AD、BE、CF相交得三角形PQR  如圖

![image](ShowImage?id=282)
 

 

聰明的你想必知道要做什麼了XDD

給你S、x、y、z

求三角形PQR的面積

## 輸入說明

每行四個正整數S、x、y、z

且1<=S<=5000，1<=x、y、z<=10

當S=x=y=z=0時代表輸入結束，不需處理這行輸入

## 輸出說明

對於每行輸入

輸出PQR的面積，計算到小數以下第二位(以%.2f的形式輸出)

## 範例輸入 #1

```
7 2 2 2
10 1 2 3
0 0 0 0
```

## 範例輸出 #1

```
1.00
0.99
```
