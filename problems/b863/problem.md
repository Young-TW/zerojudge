# b863. 榴槤

## 內容

黏黏榴槤是一種黏黏的榴槤，年年喜歡吃黏黏榴槤，所以年年種了很多黏黏榴槤，年年將黏黏榴槤種在黏黏榴槤園，年年有很多黏黏榴槤園。年年想去黏黏榴槤園採收黏黏榴槤，各黏黏榴槤園的黏黏榴槤要經過一段時間才會變黏，變黏的黏黏榴槤才能採收，年年的黏黏榴槤園連成一直線，年年從一個黏黏榴槤園移動到隔壁的黏黏榴槤園，要花一單位時間，年年採收一個黏黏榴槤園不計時間。年年從第一個黏黏榴槤園出發，採收完所有黏黏榴槤再回到第一個黏黏榴槤園最少需要多少時間？

## 輸入說明

第 1 行有一正整數 T(T <= 10)，接下來有 T 筆輸入。
每筆輸入第 1 行有一正整數 N(N <= 100000)，代表有幾個黏黏榴槤園。下一行有 N 個不超過 10^7 的正整數，代表各黏黏榴槤園要經過多少時間才會變黏，由左至右，最左邊是第一個，最右邊是最後一個，年年由最左邊的黏黏榴槤園出發。


 


有 40% 的測資 N <= 8

## 輸出說明

對於每筆輸入，輸出年年採收完所有黏黏榴槤再回到第一個黏黏榴槤園所需的最少時間。

## 範例輸入 #1

```
2
3
1 2 3
3
6 5 4
```

## 範例輸出 #1

```
5
6
```
