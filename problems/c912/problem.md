# c912. 天龍國積水的馬路坑洞

## 內容

天龍國的道路長久失修，一下雨就5000個坑洞，坑洞中積水嚴重，天龍國的行政單位想量測一下可能的積水量有多少，
假設道路坑洞的形狀簡化如下圖所示，藍色區域為積水區域，總共的積水量(面積)換算為：
A = 5 (左) + 11 (中) + 0 (右) = 16
(註：右側因底部已連通下水道，故沒有積水情況)

![image](ShowImage?id=1242)

## 輸入說明

一筆測資，一個由 H、U、D 三個字碼構成的字串，其中 H 代表向右橫向1單位， U 代表垂直上升1單位， D 代表垂直下降1單位，字串長度<=10000 個字元。

## 輸出說明

輸出總積水量(面積) A。  (A<100000)

## 範例輸入 #1

```
HUUUUHDHDDHHUHDDHUHUUHDHDHUUUHDDDDH
```

## 範例輸出 #1

```
5
```
