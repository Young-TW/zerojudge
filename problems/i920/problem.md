# i920. 12136 - Schedule of a Married Man

## 內容

我們瀟灑的英雄丹尼最近結婚了，這給他帶來了很多問題，至少他的朋友們是這麼認為的。如此多的失信，如此多的錯過約會和晚餐。
呃！丹尼，現在連最簡單的計算都忘記了，所以你必須幫助他決定他是否可以參加他的會議。丹尼一天中的大部分時間都在忙於和他的妻子在一起。這大部分由開始時間和結束時間表示。
然後丹尼在一天中有一個重要的會議，如果它重疊或接觸，他將會錯過這場會議


例如，如果丹尼與妻子的時間在 18:00 結束，然而會議在 18:00 開始，那麼兩個日程安排衝突，丹尼將錯過會議


鑑於丹尼為他的妻子分配的時間以及與您會面的時間，您必須確定丹尼是否錯過了那次會議。

## 輸入說明

第一行包含一個整數 N (0 < N < 2001)，代表測試資料數量


每組測試資料包含兩行


第一行代表陪伴妻子的開始時間和結束時間


第二行代表會議的開始時間和結束時間


輸入為 24 小時制，且開始時間和結束時間會在一天內，不會跨日

## 輸出說明

對於每組測試資料，輸出一行


第 i 組測試資料


如果可以參加會議


輸出: Case i: Hits Meeting


否則


輸出: Case i: Mrs Meeting

## 範例輸入 #1

```
3
17:47 22:40
06:18 17:04
10:44 17:05
01:11 01:27
03:36 19:02
14:33 15:24
```

## 範例輸出 #1

```
Case 1: Hits Meeting
Case 2: Hits Meeting
Case 3: Mrs Meeting
```
