# m129. pA. 街道改造

## 內容

「猴猴共和國」是由兩千多隻猴猴組成的王國，猴猴之王非常重視城市的整齊美觀，但猴猴共和國的「成功街」上建築物高度參差不齊，因此猴猴之王很生氣，決定對成功街上的大樓進行改造。


成功街上有$N$棟大樓，由左至右第$i$棟大樓的樓層數為$a_i$。猴猴之王希望成功街上的大樓高度由左到右遞增，每棟建築物不得比前一棟矮（但可以一樣高）。猴猴之王請了猴猴工程隊來蓋房子，猴猴工程隊每天施工可以選擇一棟大樓，將其增高一層或兩層。求猴猴工程隊最少需要幾天，
才能將成功街改造成功？

## 輸入說明

輸入第一行為一個整數$N$，代表成功街上的大樓數目。
第二行為$N$個整數，第$i$個整數為$a_i$，代表由左至右第i棟大樓的樓層數。


測資限制


$1\leq N\leq 10^6$
$0\leq a_i \leq 10^9$

## 輸出說明

輸出一行包含一個整數，代表猴猴工程隊最少需花費的施工天數。

## 範例輸入 #1

```
5
6 4 10 10 2
```

## 範例輸出 #1

```
5
```

## 範例輸入 #2

```
5
5 2 12 3 11
```

## 範例輸出 #2

```
8
```
