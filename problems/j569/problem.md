# j569. 無預警小考 ( 下 )

## 內容

( 接續 j568. 無預警小考 ( 上 ) )


現在老師連答案都懶得算了 !


( 畢竟每張考卷都有100題梔多 )


看來，又需要你的幫忙了。


請寫一個程式，幫助數學老師完成 " 無預警小考 " 的考試卷解答。

## 輸入說明

輸入第一行為一個整數t (t<=129), 代表老師要出幾張考卷，接下來每筆測資第一行有一整數n,代表每張考卷的題目數量。接下來輸入每題的三個係數 ( 分別為x^2,x,常數項 )。|係數|<=10000。


*x^2的係數不會是0

## 輸出說明

對於每題輸出"題號. "後輸出答案並換行


- 若答案無解，輸出"無解"
- 若答案為重根，輸出"(答案) (重根)"
- 若答案為兩個整數或分數，輸出"(答案1) 或 (答案2)"
- 若答案有根式，先輸出分母的內容再換行輸出分子的內容 :"分母..."空行"分子..."。(兩行請對齊)

*根號請用√，正負號請用±表示


*若答案為分數則負號一律放在分子


*答案都要約成最簡，根式也要


*從較大的根開始輸出

## 範例輸入 #1

```
1
5
2 3 4
1 2 1
1 -2 -3
1 4 1
29 9410 1919
```

## 範例輸出 #1

```
1. 無解
2. -1 (重根)
3. 3 或 -1
4. -2±√3
5. 分母29
   分子-4705±3√2453486
```
