# e785. 12592 - Slogan Learning of Princess

## 內容

王子出門買午餐


王子遇到群眾喊口號


王子跟著一起喊口號


王子忘記回家


公主出門找王子


公主發現王子


公主跟著王子一起喊口號

## 輸入說明

第一行數字N (0 < N < 20)


接下來 N 個 pair，總共 N * 2 行


每個 pair，第一行為"呼喊口號"，第二行為"對應口號"


接下來一個數字Q (0 < Q < 100)


接下來 Q 行，"呼喊口號"

## 輸出說明

請輸出"對應口號"

## 範例輸入 #1

```
3
ko te kader molla
tui rajakar tui rajakar
tumi ke ami ke
garo chakma bangali
jalo re jalo
agun jalo
2
jalo re jalo
ko te kader molla
```

## 範例輸出 #1

```
agun jalo
tui rajakar tui rajakar
```
