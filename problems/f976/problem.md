# f976. The Adventure of Morty and Jessica

## 內容

![image](ShowImage?id=1792)


Morty 和 Jessica 被機器狗狗大軍包圍了


無奈之下Morty打開傳送門，結果他們被傳送到高空中


兩人在下落時受到重力造成的向下加速度為定值 a，空氣阻力造成的向上加速度為 kv ( v 為當前速度 )


整體來說的向下加速度就是 a - kv


Morty 需要知道 t 單位時間後的位移、速度和加速度，才有機會活下來，請你幫幫他


過程中皆無落地且初速為 0

## 輸入說明

輸入有多行


每行有整數 a, k, t


1≤ a,k ≤1000


1≤ t ≤ 10000000

## 輸出說明

輸出三行


第一行輸出 x(t)=...


第二行輸出 v(t)=...


第三行輸出 a(t)=...


四捨五入到小數第 2 位

## 範例輸入 #1

```
1 4 5
2 4 3
1 1 1
```

## 範例輸出 #1

```
x(5)=1.19
v(5)=0.25
a(5)=0.00
x(3)=1.38
v(3)=0.50
a(3)=0.00
x(1)=0.37
v(1)=0.63
a(1)=0.37
```
