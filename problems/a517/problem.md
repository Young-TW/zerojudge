# a517. Freak大危機!!!

## 內容

EVE ( 星戰前夜 ) ，一款在大陸轟動一時的MMORPG。  

 

在一片晴朗的星空下，一艘駕駛員為Freak的旗艦型工業艦正在進行著例行性採集。就在此時，敵對陣營的爪牙剛好出來溜達溜達，三艘戰鬥巡洋艦絕對不是工業艦所能抵擋的，Freak很明智地選擇了三十六計走為上策，開啟了躍遷引擎，在對方開火前溜了。偏偏星圖壞了，Freak想知道最近的空間站 ( 座標原點 ) 與工業艦 ( x, y, z ) 的距離 d。

## 輸入說明

題目輸入座標 ( 三個 unsigned long long ) x, y, z ( 三數中的最小值與其他兩數的差為完全平方數 )，且最小值的倒數會等於其他兩數的倒數和。

EOF 輸入。

## 輸出說明

輸出距離 d ( unsigned long long )。

## 範例輸入 #1

```
1 2 2
2 3 6
3 4 12
```

## 範例輸出 #1

```
3
7
13
```
