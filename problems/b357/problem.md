 在一個 N 個城市的國家，每個城市之間用 M 條邊相連。隨著時間，它們開始分裂，彼此會開始斷訊，無法聯絡到相鄰的城市。定時回報某個城市可以藉由間接或直接連絡的城市個數。

 操作：

 - D i：移除輸入順序 i 的連接邊。
- Q i：輸出城市 i 能聯絡的城市個數。

## 輸入說明

輸入有多組測資。

每一組第一行會有兩個整數 N, M，表示一開始國家有 N 個城市，並且由 M 條邊相連。

接下來會有 M 行，每行上會有兩個整數 x, y，表示城市 x 和城市 y 之間有一條邊相連。

接下來會有一行整數 Q，表示會有 Q 組詢問。接下來則會有 Q 行詢問，格式如題目描述。

(0 < N, M, Q < 65536, 1 <= x, y <= N) 

保證不會重複刪除編號，並且兩個城市間最多一條邊直接相連。

## 輸出說明

對於每組詢問輸出一行。
