# k076. 簡易棒球模擬

## 內容

我們來模擬一個簡易的棒球賽。給你壘上有人的情況，以及打者打了幾壘安打，請計算該支安打的得分及跑壘後壘上有人的情況。


為了簡化題目的複雜度，壘上跑者前進的壘數與打者相同，不會有人被封殺或觸殺出局。

## 輸入說明

輸入只有二行，第一行含有三個整數，分別代表三、二、一壘是否有人，1 代表壘上有人，0 代表空壘。第二行含有一個整數，代表打者擊出幾壘安打，0 代表出局，4 代表全壘打。

## 輸出說明

先輸出得分於一行，第二行則輸出跑壘後三、二、一壘上是否有人，1 代表壘上有人，0 代表空壘。

## 範例輸入 #1

```
1 0 1
1
```

## 範例輸出 #1

```
1
0 1 1
```

## 範例輸入 #2

```
1 1 1
4
```

## 範例輸出 #2

```
4
0 0 0
```
