# f985. 攗皥蒽快給我去工作！！！

## 內容

皥蒽是一個青少年，年紀輕輕就已經有老婆 Dora，和兩個嗷嗷待哺的小孩。


皥蒽非常辛苦，每天都要去數學研究所工作。美其名是 "數學研究所"，但皥蒽做的工作其實非常耗費體力，每天只能拿到 0.05 ZAR 的工資，還要看研究所主人的臉色。如果主人不開心，皥蒽就會被打。


皥蒽唯一能休息的時候就是主管機關來視察時。皥蒽需要向他們展示矩陣乘法，他們會給皥蒽兩個 3*3 的矩陣 A, B，皥蒽要算出 C ，也就是 A 和 B 相乘後 (AB) 的矩陣。


有一天皥蒽算出了 C，正沾沾自喜時，他碰過的水不小心灑出來，把 B 蓋住了，只剩下 A, C


皥蒽心想完蛋了，你願意幫皥蒽嗎？ 我個人是不願意啦

## 輸入說明

多筆輸入


每筆輸入第一行是 A:


接下來三行每行有三元素，第 i 行第 j 個表示矩陣 A 中的元素 aij


再來輸入是 C:


接下來三行每行有三元素，第 i 行第 j 個表示矩陣 C 中的元素 cij


aij, cij 皆為整數，-1000≤ aij, cij ≤1000

## 輸出說明

你可以像我一樣輸出 皥蒽回家吃自己吧！，得到一個 WA，皥蒽就要回家面對他的老婆 Dora 和兩個嗷嗷待哺的小孩，他們已經五天沒飯吃了，皥蒽又沒了工作，他們真的不知如何是好。


但是你真的忍心這樣嗎？ 我是忍心啦


如果你想 AC 讓皥蒽保住工作 不建議


就要輸出 B: 後，再輸出矩陣 B 中的元素 bij，四捨五入到小數第 4 位


ex：12.34445→ 12.3445，12.34444→ 12.3444
0.00005→ 0.0001，0.00004→ 0.0000
-12.34445→ -12.3445，-12.34444→ -12.3444
-0.00005→ -0.0001，-0.00004→ 0.0000


保證不會有多解或無解的狀況
