# o626. 檔案順序 (Files)

## 內容

題目連結如下：
https://msqrt1024.github.io/TOIproblems/潛力組/202409/p1/files (zh_TW).pdf

## 範例輸入 #1

```
5 3
1 100 1 24
11 2015 3 1024
30 3030 1 256
2 366 3 4096
34 2533 2 65536
1 3 3
```

## 範例輸出 #1

```
2 366 3 4096
11 2015 3 1024
34 2533 2 65536
1 100 1 24
30 3030 1 256
```
