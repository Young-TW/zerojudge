# k980. 求包含最多因數的數

## 內容

小明想在一個數字裡，擷取任意部分數字(從哪個位開始，哪個位結束都可以)，並求出擷取哪個部分才能找出包含最多因數的數。請你編程一段程序，協助他擷取數字。

## 輸入說明

輸入只有一行，為一個整數n。


1<=n<=36042067211

## 輸出說明

輸出有若干行。


如果只有一個數含最多因數(包括1和該數)，輸出:


該數


該數的所有因數


如果多於一個數含最多因數(包括1和該數)，輸出:


那些數(隔行，以升序排列)


那些數的最大公因數


那些數的最小公倍數

## 範例輸入 #1

```
108409
```

## 範例輸出 #1

```
840
```

## 範例輸入 #2

```
36042067211
```

## 範例輸出 #2

```
360420672
```

## 範例輸入 #3

```
23
```

## 範例輸出 #3

```
2
3
23
1
138
```
