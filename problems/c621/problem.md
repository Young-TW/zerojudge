# c621. Reverse this and that !!

## 內容

給定一個字串，字串是由大小寫字母及數字組成。
例： 123abc456de7fgh890i
請依下列2個條件處理後輸出：
1)  字母的部分原地反轉。
2)  數字的部分依數字於整個字串的位置反轉。
處理後如下輸出： 098cba765ed4hgf321i

## 輸入說明

123abc456de7fgh890i

## 輸出說明

098cba765ed4hgf321i

## 範例輸入 #1

```
123abc456de7fgh890i
FGwxjgdoCaT24txLeGkehc958KP
```

## 範例輸出 #1

```
098cba765ed4hgf321i
TaCodgjxwGF85chekGeLxt942PK
```
