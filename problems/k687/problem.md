# k687. 排名統計

## 內容

小明製作了一款遊戲，每當遊戲結束時，玩家能看到他們的帳戶名稱+分數。小明想製作分數排行榜，從大到小排序，還要附上排名，請你編程程式幫助他。

## 輸入說明

第一行有一個數字n，代表接下來有多少行


接下來有n行，每行都會有帳戶名稱+分數

## 輸出說明

輸出n行


格式為"排名+帳戶名稱+分數" 用空格隔開


每行按照分數高低從大到小排列


記住同分的話分數要相同，由先輸入的帳戶在頭，後輸入的帳戶在後面，如此類推


就算有同分的情況出現


整份排名的排名都比上面一個的排名大1

## 範例輸入 #1

```
3
sam10
tom799
john28
```

## 範例輸出 #1

```
1 tom 799
2 john 28
3 sam 10
```

## 範例輸入 #2

```
4
小明0
小光88
小美40
小平40
```

## 範例輸出 #2

```
1 小光 88
2 小美 40
2 小平 40
3 小明 0
```
