# k182. 資訊科技小考成績

## 內容

學期末快結算成績了，文文擔心他的資訊科技會被當。請你幫他算一下他的小考平均成績，讓他看看是不是需要花點時間狂刷習題來拉高成績。

## 輸入說明

輸入只有二行。第一行含有一個整數 n，代表小考的次數。第二行有 n 個整數，則是文文歷次的小考成績。

## 輸出說明

輸出文文小考的平均成績，四捨五入到小數點以下第二位，並去掉後面不需要的 0 或小數點。

## 範例輸入 #1

```
5
100 88  77 66 55
```

## 範例輸出 #1

```
77.2
```

## 範例輸入 #2

```
7
100 0 0 0 0 0 0
```

## 範例輸出 #2

```
14.29
```

## 範例輸入 #3

```
6
100 100 70 60 50 10
```

## 範例輸出 #3

```
65
```
