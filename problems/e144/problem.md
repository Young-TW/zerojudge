# e144. throw it away!

## 內容

拿到基物2就有很多人開始煩惱這第一道難關，斜拋。


而你做為友善的物理好朋友，你決定幫他們一把，負責驗算答案正確與否。


然而200*38題的基礎題計算量是非常巨大的，所以你必須寫個程式好讓你的工作輕鬆一點。



![image](ShowImage?id=1300)

## 輸入說明

第一行有一個數字t，代表接下來有t行。


每行有三個正整數v0 (m/s), D (O), T (s)，代表在位置(0,0)投擲的初速度,仰角,以及經過幾秒。


1<=t<=30


0<=v0<=1000


0<=D<=360


0<=T<=2000

## 輸出說明

請輸出這顆球在T秒時的(x,y)位置(一單位為一公尺)，以及它的y軸速度正處於下降,上升或靜止何種狀態


格式: x: x, y: y  at T(s) falling/rising/rest

## 範例輸入 #1

```
3
25 53 4
25 53 20
2 2 0
```

## 範例輸出 #1

```
x: 60.18, y: 1.46  at 4(s) falling
x: 300.91, y: -1560.68  at 20(s) falling
x: 0.00, y: 0.00  at 0(s) rising
```
