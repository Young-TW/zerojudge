# l179. One More Grid Task - Extreme

## 內容

原題，和本題的差異只有執行時間限制 $\text{(TL)}$ 以及 $A_{i, j}$ 的大小。


有一個 $N \times M$ 的網格，其中第 $i, j$ 格裡面的數字為 $A_{i, j}$，現在需要你從這裡面找一個長方形區塊 $R$，並且定義 $f(R)$ 代表「區塊 $R$ 裡面所有數字中的最小數字 $\times$ 區塊 $R$ 中的數字的總和」，請你讓 $f(R)$ 盡可能的大，並求出這個 $f(R)$ ?

## 輸入說明

第一行有兩個正整數，代表 $N, M$ 。


接著有 $N$ 行，每行有 $M$ 個數，第 $i$ 行中的第 $j$ 個數代表 $A_{i, j}$。


同一行的多個整數中間都以空白隔開。


- $ 1 ≤ N, M ≤ 300$
- $ 1 ≤ A_{i, j} ≤ 10^6$

## 輸出說明

輸出一個正整數，代表 $f(R)$ 的最大值。注意答案可能超過 $2^{31}$。

## 範例輸入 #1

```
3 3
5 4 3
4 3 2
3 2 1
```

## 範例輸出 #1

```
48
```

## 範例輸入 #2

```
4 5
3 1 4 1 5
9 2 6 5 3
5 8 9 7 9
3 2 3 8 4
```

## 範例輸出 #2

```
231
```

## 範例輸入 #3

```
6 6
1 300 300 300 300 300
300 1 300 300 300 300
300 300 1 300 300 300
300 300 300 1 300 300
300 300 300 300 1 300
300 300 300 300 300 1
```

## 範例輸出 #3

```
810000
```

## 範例輸入 #4

```
2 2
48763 48763
48763 48763
```

## 範例輸出 #4

```
9511320676
```
