# e827. 2.道路鋪設 (Roads)

## 內容

y19m10_a2道路鋪設 (Roads)   2019年10月TOI練習賽 潛力組  {試題連結}


 


問題敘述


給你一條長度為 L 的道路和長度為 1, 2, 3, …, L 的磁磚各無限個，請問將道 路用磁磚鋪滿共有幾種鋪法？ 以長度 3 為例，總共有四種鋪法：


 
![image](ShowImage?id=1446)
 


評分說明 此題目測資分成三組，每組測資有多筆測試資料，需答對該組所有測試資 料才能獲得該組分數。各組詳細限制如下。


第一組 (10 分) : 1<=L<=30


第二組 (30 分) : 1<=L<=10^4


第三組 (60 分) : 1<=L<=10^9

## 輸入說明

第一列有一個正整數 T（1<=N <= 10^4），代表有 T 條路要鋪。接著 T 列為道 路長度，每列有一個正整數 L（1<=L<=10^9），為道路的長度。

## 輸出說明

對每筆資料請輸出 T 列，每列皆有一個正整數，為鋪法總數除以 10^9＋7 後 的餘數。

## 範例輸入 #1

```
1
3
```

## 範例輸出 #1

```
4
```

## 範例輸入 #2

```
2
50000
353880108
```

## 範例輸出 #2

```
509082729
849175862
```
