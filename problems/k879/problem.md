# k879. DD癡迷玩原神結果被困在迷宮！？

## 內容

故事
DD是一個勇敢而聰明的冒險家，他對謎題和冒險充滿熱情。今天，DD決定進入一個傳說中的迷宮，據說裡面蘊藏著無數的寶藏和神秘的生物。


 


DD穿著一套輕便的冒險裝備，手持一把明亮的探照燈，開始進入迷宮的入口。他立即感受到了迷宮的複雜性。牆壁上充滿了細小的刻痕和符號，似乎是指引著他的方向。DD繼續往前走，努力記住每一個轉彎和岔路口。


 


迷宮內部黑暗而陰森，充滿了神秘的氛圍。DD不斷遇到了各種巧妙的機關和謎題。有時他需要解開數字謎題，有時他需要找到特定的鑰匙來開啟門。每一個階段都考驗著他的智慧和觀察力。


 


在迷宮的旅程中，DD遇到了一隻奇怪的生物。她是個穿著紅色連衣裙和紅色帽子的小女孩，有著金色的捲發和大大的紅色眼睛。她的裙子上裝飾著黃色和白色的蝴蝶結。手中總是拿著她最喜歡的玩具炸彈。DD的直覺告訴他這隻生物還未成年，剛好DD喜歡這種類型。這隻蘿莉自稱為可莉，它是迷宮的守護者，也是迷宮的一部分。可莉告訴DD，只有解開所有謎題並通過所有的考驗，他才能得到迷宮的最終寶藏。


 


DD與可莉成為了朋友，他們一起努力解決每一個挑戰。有時候，可莉會給出一些提示，有時候則會提供寶貴的資訊。這個冒險不僅僅是一個單打獨鬥的旅程，而是一個互相合作的歷程。


 


經過了幾個小時的努力，DD和可莉終於到達了迷宮的最深處。那裡散發著耀眼的光芒，數不清的珠寶、寶石和黃金散落在地上。DD感到無比的興奮和滿足，因為他終於找到了傳說中的機器人，一個可以跟他聊原神的機器人。


 


DD和可莉一起將機器人帶出了迷宮，分享了這個巨大的成就。他們成為了迷宮冒險的英雄，被人們津津樂道。但最重要的是，他們的友誼在這個冒險過程中變得更加深厚。


 


這次的冒險經歷讓DD學到了很多，他體會到了智慧和勇氣的價值。他發現，只要擁有堅定的意志和堅強的信念，他就能克服任何困難。DD決心將這些經驗帶到未來的冒險中，繼續追尋更多的寶藏和新的挑戰。


題目
請幫助DD通過迷宮中的關卡，DD會得到一張迷宮的地圖，你的任務是幫助DD從起點到終點找到一條路徑。你可以向左、向右、向上和向下走。


輸入第一行有兩個整數 n 和 m ：地圖的高度和寬度。


接著有n*m的地圖，地圖涵蓋 '.' (可以走的路)、'#'(牆壁)、'A'(起點)、'B'(終點)。輸入中正好有一個 'A' 和一個 'B' 。

## 輸入說明

輸入第一行有兩個整數 n 和 m ：地圖的高度和寬度。


接著有n*m的地圖，地圖涵蓋 '.' (可以走的路)


、'#'(牆壁)、'A'(起點)、'B'(終點)。輸入中正好有一個 'A' 和一個 'B' 。


測資範圍：


1≤n,m≤1000

## 輸出說明

如果有路徑，請先輸出"YES"，否則輸出“NO”。


如果有路徑，請將最短路徑的長度及其描述列印為由字元 L （左）、 R （右）、 U （向上）和 D （向下）組成的字串。如果有多組解，DD喜歡先L，再R，再D，最後U的走法。

## 範例輸入 #1

```
5 8
########
#.A#...#
#.##.#B#
#......#
########
```

## 範例輸出 #1

```
YES
9
LDDRRRRRU
```
