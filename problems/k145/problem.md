# k145. 街燈管理員

## 內容

文文是街燈管理員，他管理某街道上的 𝑛 盞街燈。每天傍晚，文文都會玩一個遊戲：這 𝑛 盞街燈的編號依序為 1 ~ 𝑛，每盞燈都有自己的獨立開關，他會在這個街道來回走 𝑛 趟，第 𝑖 趟時會去撥動所有編號為 𝑖 的倍數的街燈開關，也就是關變開、開變關。


假設一開始時所有的街燈都是關著的，請問這 𝑛 趟走完以後，有幾盞街燈是開著的？

## 輸入說明

輸入只有一行，含有一個整數 𝑛 ( 1 ≤ 𝑛 ≤ 231-1)，代表街燈的數量。

## 輸出說明

輸出一個整數，代表文文走完 𝑛 趟後，有幾盞街燈是亮著的。

## 範例輸入 #1

```
3
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
5
```

## 範例輸出 #2

```
2
```
