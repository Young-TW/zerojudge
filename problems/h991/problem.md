# h991. 確實大師 (Chesh Master)

## 內容

確實小子 ($\text{Chesh Boy}$) 平常遇到什麼事都會說「確實」，像是有人對他說他很煩，他會說「確實」；有人問他晚餐要吃什麼，他也會說「確實」。


只是如果所有問題他都說確實就不太好了，如果要表達否定，確實小子會說「不確實」。像是有一次肯肯肯在練習砍樹，他問確實小子他砍得如何，確實小子說「不確實」，於是肯肯肯發憤圖強，之後拿到 $\text{OIO (International Olympiad in Tree Chopping)}$ 的滿分金。


確實小子的夢想是成為「確實大師」($\text{Chesh Master}$)，只是在此之前，他得去「確實賣場」($\text{Chesh Mart}$) 買東西。他在「確實賣場」遇到了他的好友「確實怪」($\text{Chesh Monster}$)，「確實怪」身上有很多點讓他感到很確實，如果把這些點以三維座標表示，每個點都可以寫作整數點 $(x, y, z)$，而「確實怪」的「確實度」則是「確實怪」身上所有點中兩個點曼哈頓距離的最大值，也就是找得到兩個不同編號 $i, j$，$|x_i - x_j| + |y_i - y_j| + |z_i - z_j|$ 為最大。


確實小子可以在確實怪身上加入更多點，或是移除一些點，每次加入或移除時，請你幫確實小子找出確實怪的確實度。


如果你幫了確實小子的忙，他會對你說「確實」的。

## 輸入說明

第一行有一個整數 $n$，代表有幾筆操作。


接下來 $n$ 行，每行有四個整數 $ty, x, y, z$，若 $ty = 1$，代表確實怪身上多了 $(x, y, z)$ 這個點，保證確實怪在此操作前，身上沒有 $(x, y, z)$ 這個點；若 $ty = 2$，代表確實怪身上少了 $(x, y, z)$ 這個點，保證確實怪在此操作前，身上有 $(x, y, z)$ 這個點。


- $1 \leq n \leq 2\times 10^5$
- $1 \leq ty \leq 2$
- $-10^8 \leq x, y, z \leq 10^8$

## 輸出說明

對於每筆操作，輸出目前確實怪身上兩個點最遠的曼哈頓距離。若確實怪身上的點數小於 $2$，輸出 $-1$。
