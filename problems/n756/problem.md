# n756. 12611 - Beautiful Flag

## 內容

提林是一個小男孩。他正在嘗試繪製孟加拉國國旗。作為一個聰明的孩子，他知道在繪製國旗時必須保持正確的比例和尺寸。你知道如何繪製國旗的規則，對嗎？如果不知道，也沒關係，提林在這裡幫助你：


孟加拉國的國旗由一個綠色的矩形和裡面的紅色圓形組成。矩形的長寬比為100：60（即如果長度是100單位，那麼寬度將是60單位）。圓的半徑是長度的20%（即如果長度是100單位，那麼圓的半徑將是20單位）。要獲得圓心，你需要畫一條水平線將寬度等分，再畫一條垂直線將長度按45：55的比例分割（即如果矩形的長度是100單位，那麼左側45單位，右側55單位）。兩條線的交點就是圓的中心。下面是示意圖以便更好地理解。

現在提林已經開始繪製國旗了。他已經在一個2D坐標系的原點畫了一個半徑為R的圓。現在他需要確定矩形的角點，這樣他就可以將它們連接起來完成國旗。你能幫助他嗎？

## 輸入說明

輸入的第一行將包含一個整數 T (T < 101)，表示測試用例的數量。接下來的 T 行，每行將包含一個整數 R (R < 1001)，表示圓的半徑。

## 輸出說明

對於每個輸入，輸出五行。第一行將包含案例編號。接下來的四行將分別表示旗幟矩形的左上角、右上角、右下角和左下角的座標。每行你需要打印 x 座標和 y 座標，並以空格分隔。你可以假設輸入是給定的，使得矩形的角總是整數座標。請參閱示例輸入輸出以獲取詳細信息。

## 範例輸入 #1

```
2
20
100
```

## 範例輸出 #1

```
Case 1:
-45 30
55 30
55 -30
-45 -30
Case 2:
-225 150
275 150
275 -150
-225 -150
```
