# f394. 心愛數質數

## 內容

智乃在學校剛學會質數的概念


但是還不是很熟 所以就去找心愛幫忙寫回家作業



![image](ShowImage?id=1667)


雖然心愛說交給姐姐吧 但他只會背到4位數的質數 所以正煩惱著


看到心(ㄌㄠˇ)愛(ㄆㄛˊ )煩惱的樣子 所以你決定幫老婆們寫一個程式判斷一個數是否為質數

## 輸入說明

每行有一個大於0小於2^63的數字


大概有500行

## 輸出說明

如果是質數的話就輸出Yes


不是的話就輸出No

## 範例輸入 #1

```
154590409516822759
2305843009213693951
1436697831295441
```

## 範例輸出 #1

```
Yes
Yes
No
```
