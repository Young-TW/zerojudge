# e613. 13216 - Problem with a ridiculously long name but with a ridiculously short description

## 內容

我們所有人都討厭"loooooong"的題目。(改成題目很"loooooong")
所以這裡簡短介紹一下：對於給定的n計算(66^n) mod 100。

## 輸入說明

輸入第一行包含一個整數T (1 ≤ T ≤ 5000)，代表測資數量。
每組測資有一個整數n (1 ≤ n ≤ 10^1000)，n如題目敘述。

## 輸出說明

對於每組測資，輸出(66^n) mod 100，也就是66的n次方除以100的餘數。

## 範例輸入 #1

```
4
0
1
2
9999999999999999999999
```

## 範例輸出 #1

```
1
66
56
36
```
