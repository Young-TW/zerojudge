# a865. 5. Greek Numerals

## 內容

古希臘人用他們的字母來表示數字，解碼這些數字是翻譯古文獻中不可或缺的一環，這個系統如右圖所示，稱為愛奧尼亞數，使用了27個字母來表示十進位的1~9、10~90、100~900

例如：27的愛奧尼亞數為KZ，由K和Z總和而得27

![image](ShowImage?id=568)
 

 

請寫一個程式來轉換愛奧尼亞數至十進位

## 輸入說明

每筆輸入含一個愛奧尼亞數，最後以＂.”為結尾

## 輸出說明

輸出該愛奧尼亞數的十進位

## 範例輸入 #1

```
KZ
UQE
.
```

## 範例輸出 #1

```
27
495
```
