# e174. 圍棋基礎 - 整地

## 內容

阿光：三谷！！！你是不是又偷作弊！！！


三谷：哪有？歐北共！！


阿光：我都看到你再整地時偷動棋子了，還狡辯！！！


三谷：不然你想怎樣？就沒有咩！


阿光：佐為你看啦，他都這樣


佐為：阿光你別吵，這演唱會重播正精彩呢！阿阿阿，莎朗嘿！莎朗嘿！莎朗嘿！


阿光：.........


 


"整地"，在圍棋棋局結束後，為了方便人員計算，會將雙方地形整理成長方形


以便於計算自己到底佔了多少地，你可以幫助阿光幫忙算地嗎？

## 輸入說明

第一行有一個數字n，代表下列有幾個棋盤


1 <= n <= 100


接下來有n組棋盤，棋盤大小為19*19個字元i代表棋盤狀況


i = '.'代表沒棋子，i = 'O'代表黑棋，i = 'X'代表白棋


盤面一定為下完的盤面，其中不會有死棋


注意：(1, 1)為最左下角座標，(19, 19)為最右上角座標

## 輸出說明

輸出


黑棋佔地多少，O:??


白棋佔地多少，X:??


誰贏了，Black win!! / White win!!
