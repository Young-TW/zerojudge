# d256. 11388 - GCD LCM

## 內容

兩個正整數的最大公因數 (GCD) 就是可以整除這兩數的最大整數。兩個正整數的最小公倍數 (LCM) 就是可以被這兩整數整除的最小整數。一個正整數有可是好幾對整數的 GCD。相同地，它也可能是好幾對整數的 LCM。本題給你兩個整數，你要求出以第一個數為 GCD、第二個數為 LCM 的兩個整數。

## 輸入說明

第一行的輸入有一個正整數 T，代表有幾個測試。接下來的 T 行每行有兩個正整數 G 和 L。

-           T ≤ 100

-           G 和 L 都小於 231。

## 輸出說明

每個測試有一行輸入。它會有兩個正整數 a 和 b，a ≤ b，它們的 GCD 為 G，LCM 為 L。如果有多對整數符合這個條件，請輸出 a最小的那一對。如果無解，請輸出 -1。

## 範例輸入 #1

```
2
1 2
3 4
```

## 範例輸出 #1

```
1 2
-1
```
