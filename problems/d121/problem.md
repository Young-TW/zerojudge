# d121. 00583 - Prime Factors

## 內容

因數分解是個常遇到，但是卻不是那麼容易的問題，特別是大數字的時候。
 某一個數字n，可以寫成因數分解式：

 n = $f_1$ x $f_2$ x … x $f_k$


其中對所有的i來說 $f_i$>1,且對所有的i<j來說 $f_i <= f_j$

## 輸入說明

輸入包含好幾個資料，每個資料一行，含有1個整數n。（$-2^{31}$ < n < $2^{31}$ ）
 n=0代表輸入結束。

## 輸出說明

對每個測試資料輸出n的因數分解式。請參考sample output

## 範例輸入 #1

```
-190 
-191 
-192 
-193 
-194 
195 
196 
197 
198 
199 
200 
0
```

## 範例輸出 #1

```
-190 = -1 x 2 x 5 x 19
-191 = -1 x 191
-192 = -1 x 2 x 2 x 2 x 2 x 2 x 2 x 3
-193 = -1 x 193
-194 = -1 x 2 x 97
195 = 3 x 5 x 13
196 = 2 x 2 x 7 x 7
197 = 197
198 = 2 x 3 x 3 x 11
199 = 199
200 = 2 x 2 x 2 x 5 x 5
```
