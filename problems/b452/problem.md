經過上次「b450: 傻傻地幫人數錢」之後，我依然是個窮人。這次還是要麻煩各位大德幫我數一下錢錢，唯一的差別是：一元硬幣可能互相接觸。

為什麼呢？上次我七歲的姪女剛好來我家玩，我就叫她設計測試資料，順便把硬幣通通分開；今天她沒來我家，所以我就懶的分開了。


 


拍攝環境

1. 一元硬幣平攤在我家客廳的桌子上，不互相重疊、不互相接觸。
2. 一元硬幣下面墊著我家的日曆紙，白色的那面朝上。
3. 日曆紙上面只有一元硬幣，沒有其他東西了。
4. 我手持手機，垂直空拍。距離遠近不一定，但是總是拍到完整的硬幣。
5. 照片的邊界，我會修圖成不超過日曆紙範圍。
6. 照片太大張，我會等比例縮小。
7. 不使用閃光燈，使用我家客廳的日光燈。
8. 本題附圖皆符合上述拍攝環境。同時也是測試資料的一部分。

## 輸入說明

一張圖片：兩個整數 W H (1 <= W, H <= 256)，是圖片的寬和高；接下來的 H 行，每行有 W*3 個整數，是每個像素的 RGB 值 (0 <= R, G, B <= 255)。

## 輸出說明

請輸出金額：一個整數 N (0 <= N <= 100)。礙於本人財力不足，N沒辦法太大。
