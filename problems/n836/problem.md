# n836. 八卦傳遞

## 內容

文文班上很愛傳八卦。同學們只要聽到什麼八卦，儘管不敢在班群上公開地傳，卻會在私底下密給他在班上的所有好友。


你有一則八卦，想要讓他們全班同學都知道。給你他們班上同學們加好友的狀況，請問最少要傳給哪幾個人，這個八卦就會傳遍全班？

## 輸入說明

輸入的第一行有兩個整數 𝑛, 𝑚 (1 ≤ 𝑛 ≤ 104, 0 ≤ 𝑚 ≤ 105)，代表班上有 𝑛 個人，存在 𝑚 組好友關係。


接在下的 𝑚 行，每行有兩個由空白隔開的整數 𝑎, 𝑏 (1 ≤ 𝑎, 𝑏 ≤ 𝑛, 𝑎 ≠ 𝑏)，代表這兩個座號的人有加好友。

## 輸出說明

參考範例輸出，列出文文需要把八卦傳給哪些人？請輸出最小字典順序的組合於一行，每兩個座號間以「, 」隔開，前後再用一對中括號「[]」包起來。

## 範例輸入 #1

```
6 3
1 2
2 3
4 5
```

## 範例輸出 #1

```
[1, 4, 6]
```

## 範例輸入 #2

```
5 4
1 2
1 3
1 4
1 5
```

## 範例輸出 #2

```
[1]
```
