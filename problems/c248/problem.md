# c248. 《Φ》序章·I ~ 遊玩須知

## 內容

「欲前往吾之世界之人，汝為何擇至此地？」


　　在一個全白的空間之中，前方疑似有個矇矓的身影在對我說話，明明很確信自己的視覺一切正常，但目光總是無法對焦在那個身影上。我想任誰都能看得出來這裡與原來的世界不同，難道說我現在身處異世界？話說在跟我對話的這人是誰啊？居然還說還說什麼「吾之世界」？我想除非真的是這個世界的神，不然講出這種話的八成是中二病吧... ...


　　「我不知道... ...我隱約只記得我好像喝了碗湯，然後... ...然後醒來就在這裡了... ...」


　　覺得自己的回答實在有夠蠢的，但無奈的是，事實就是如此... ...


　　「汝能否億起至此地前之事？」


　　喂喂喂，這麼扯的回答也相信，這人腦袋是不是有點問題啊？


　　「我連我自己是誰都不記得了，我只記得一些生活常識和一堆感覺不必要的知識，如果我連基本的事物和道理都忘了的話，我現在大概就跟嬰兒差不多，應該沒辦法站在這跟你好好對話對吧？」


　　呃... 我這樣的講話風格真不像我，因為面對的是一個看不清形體的身影的緣故？嘛～既然話都說了就... 算了吧... ...


　　「汝入此地之前，吾終問汝一事。」


　　這傢伙好麻煩啊... 沒由來的開始問人事情，對現況又不做解釋... ...


　　「隨便啦，反正我也不知道我現在還能做什麼。」


　　這是實話，在這全白的空間之中，我甚至分不清這空間是有邊界還是無邊無際的。在這種環境之下，反而會倍感壓力，看來把人丟進空無一物的房間就會發瘋這一說似乎是真的，不過至少現在還有一個似人非人的身影願意陪我對話，或許我就該謝天謝地了，話說現在這裡還有宗教中所謂的天地概念嗎？


　　「１又１同等於何數？」


　　如果我國文還有救的話，這是在問１＋１等於多少對吧！原來我已經被看扁到這種程度了嗎？


　　「２... ？」


　　誰知道這人在玩什麼花樣？明知中規中矩的回答百分之百會被坑，但我又能回答什麼？難道我要回答「王」嗎？


　　「汝須知，世上分為『１０』種人，一為知二進制之人，另為不知二進制之人。」


　　啥？在說啥呢？等等等，我越聽越不懂了。


　　「那其他『８』種人呢？」


　　嗯... ？不知為何，雖然看不見形體，但是我卻能感受到對方對我的回答飽受訝異。


 　　「哈哈哈... 以吾觀之汝確然不知曉二進制。」


　　聽到這宏偉又爽朗的笑聲莫名覺得火大... 難道不懂二進制錯了嗎？


　　「所以呢？答錯總該有什麼處罰的吧？」


 　　反正現在也不能幹嘛，在這如同虛擬世界的空間，我想如果真的要發生什麼事情，我大概也就只能接受吧... ...


　　「勿急，吾對汝方才之答以為甚趣，故特讓汝允問吾一事。」


　　這人剛剛說了「勿急」對吧？我沒聽錯吧？果真有懲罰？


　　「真的什麼問題都能問？包括私人或是現今科技尚未解答的世紀問題？」


　　這種事要先確保是否所有事情都能回答，比如問說「我要如何才能交到女朋友？」，結果得到的回覆是「汝絕無交女友可能。」這種玩笑回答那我可受不了。


　　「哈哈哈... 看來吾為汝輕視，答：世間大小事、未解之謎皆可，且吾諾答覆必無誤。」


　　又是這不討喜的笑聲，算了... 如果真的那麼神通廣大的話，感覺這個提問的機會千載難逢，我應該要好好考慮問什麼才好，問對問題或許能改變我接下來的人生，是說我還有接下來的人生嗎？我總覺得... 不管啦！先思考問什麼對自己最有利吧！是要問科學尚未探詢的領域幫助人類進步呢？還是問如何讓人類永續經營呢？但是我現在所處的地方值得我這樣問嗎？這裡怎麼看都不像我熟悉的世界... ... 看來我還是問... ...


　　「那我要問的是... ...」


　　我才正要講問題就被打斷了... ...


　　「且慢，汝方才即問，不可二問。」


　　所以... 這個意思是... 我剛才就算是提問了!?


　　「這不是擺明坑我嘛！」


　　第一次感受到自己距離人生勝利只差一步之遙，然後又夢想破滅，哎～人生命苦啊... ...


　　「只怪汝不足慧。」


　　這擺明在說我笨對吧！看來我真的已經被看扁了，大概扁到可以去做奈米薄層的那種程度了吧！


　　「... ...」


　　無言... 真的無言... 感覺現在不管我做什麼都會被輕視，那乾脆什麼都不做好了。


　　「閒談止於此，汝行入此地前尚有一事。」


　　我知道，我還沒忘記，就是剛剛的懲罰嘛！


　　「懲罰嘛～我還沒忘啦～」


　　到底為什麼我要受這種苦啊？一醒來就在陌生的地方不說，被眼前這不知是不是人的傢伙玩弄，然後現在還要遭受懲罰!?我是造了什麼孽啊？我覺得我平時為人不錯啊，只是有點懶而已，難道這樣就錯了嗎？


　　「非也，汝將行之事為入此地者皆必然之；雖非吾自為之，惟須使汝知曉吾之世界，方可入此地。」


　　等等等，該不會這人真的是這世界的神吧？那我剛剛愚蠢的行徑不就... ...


　　「噠！」


 　　突然一道彈指聲，響亮的聲音在這個空間內迴盪，與之同時在那模糊的身影上方出現了「Ｌｏａｄｉｎｇ...」的字樣，蛤？我這是在玩啥最新的ＶＲ遊戲嗎？


　　「吾離前，汝有何話欲對吾說？」


　　不是吧，在我還是滿頭霧水的時候就要走了？這人還有沒有良心啊？不過真要說有什麼想說的話... ...


　　「從一開始就很想吐槽了，你非得用這種講話方式嗎？我聽得很痛苦欸！」


　　話說我現在才注意到一件事，雖然我不記得我有學過文言文一事，但是我卻能聽得懂文言文，果然就算忘記了事實經過，但知識卻確實還保留著。


　　「嘻嘻嘻～」


　　這次的笑聲明顯與前兩次雄偉的笑聲不同，很明顯能感受到對方還帶有童心未泯的稚氣，彷彿是在嘲笑我一般。


　　「怎麼？我說錯話了嗎？」


　　明明就是你說有話可以說的，不會我又被坑了吧？


　　「你果然是個有趣的人呢！真要說為什麼那樣說話... 因為覺得好玩？嘻嘻嘻～」


　　這明顯是不同人吧！我想不管是誰都很難想像這兩者的聲音是來自同一人吧！而且「因為覺得好玩」又是哪招？起碼講個正當一點的理由嘛，唬爛說「這世界很流行古文」什麼的我都會相信啊... ...


　　「那現在到底要做什麼啊？」


　　我看著那身影上方「Ｌｏａｄｉｎｇ...」顯示的百分比逐漸上升，我心中越是感到不安，這該不會真的是個遊戲吧？


　　「接下來你將進行進入我的世界前必須完成的『程序』，所謂的『程序』就是讓你事先瞭解這個世界，避免你進入到我的世界後便成了廢人。嘻嘻嘻～難得遇見這麼有趣的傢伙，讓我都想親自帶你參觀我的世界呢！因為時間有限，我必須去見下一個來到這世界的人了，不過你別擔心，我親自幫你準備的『新手導航程式』你一定會滿意，至少對於已經單身１８年的你來說～嘻嘻嘻～那麼期待能再相見的一天～」


　　在那以少年般的語氣說完話的同時，「Ｌｏａｄｉｎｇ...」正好到達１００％，瞬間，以那身影為中心發出了刺眼的白光，然後... 整個空間染上了色彩！正確點來說，是我看見了這世界的樣貌，從高空１００００米的高度俯瞰整個世界，剛剛所處的空間框架似乎還未消失，所以目前還能安穩站在空中，而那身影如同他所預告一樣，已經消失得無影無蹤了。


　　「『愚者』專屬『新手導航程式』建構完畢，代號為『導航狐狸』，立即啟動。」


　　一個少女的聲音從「Ｌｏａｄｉｎｇ...」字樣傳來，那個「愚者」該不會是指... ...


　　「瞭解，奧沃大人。即將開始進行『新手程序』。」


　　一位狐耳少女突然自顯示「Ｌｏａｄｉｎｇ...」的平面跳出，目測年紀大淤１５歲上下，身著日系女巫的服裝，有著卡哇伊的聲音，金色長髮及腰，隨風飄逸時彷彿會飄來陣陣迷人香氣，可惜的是有著那麼可愛的外表卻面無表情。


　　「確認身分『愚者』，現在開始進行『新手程序』，以下所說內容將會帶您快速了解這個『二進制世界』，開時構築『新手語音程序』，『新手語音程序』構築完畢，執行『新手語音程序』。」

「


　　首先您必須認知到，您現在所處的世界已不是您原本的世界－－為方便說明以下簡稱『α世界』，這個世界被稱作『二進制世界』，顧名思義，象徵著這個世界的兩個重要事務即是『１』與『０』，在『二進制世界』裡『１』與『０』決定了一切，當然不只包括二進制本身而已，運用二進制的『程序』也是在『二進制世界』裡扮演著舉足輕重的地位，反舉生活大小事若您能自行開發出一套合適對應的『程序』，那將會大大提升平時做事的效率，這在稍後會詳細為您介紹。

　　接著將會從不同的面向來比較『α世界』與『二進制世界』的不同，首先最大的不同即是世界的格局，雖然『二進制世界』的人們同樣也是生活在一顆環繞著恆星的行星，在這方面與『α世界』並無太大差異，但是由於『二進制世界』的人口只有不到216人，所以基本上絕大多數的人們都居住在『城牆』內以及其周圍的數個『新手村』中，在此之外的地方，與未被人類開發的『α世界』相似，保有原生生態的樣貌，動植物昌盛，依著大自然的規則永續循環。需要注意的是，在『二進制世界』的生態與『α世界』的生態並不大相同，畢竟是經過不同的演化方式，所以造成每個世界都有其屬於自己的生態，很幸運的是，雖然或多或少有些許不同，但是跟其它世界相比，『二進制世界』的生態已經與『α世界』算是較為接近的了。


　　值得一提的是，『二進制世界』除了有著多數世界共有的生物比如貓、狗之外，一定要介紹的是『二進制世界』獨有的特殊植物－－『二叉樹』！以下簡單介紹『二叉樹』的特性，『二叉樹』是一種沿著牆壁向上生長的木本植物，『二叉樹』的主要生長範圍在『城牆』的外圍，『二叉樹』自扎根後，第一天會自根部長出一節枝幹，接著從第二天起，『二叉樹』會每天隨機從其中一節枝幹的頂端左側或右側長出新的一節枝幹，但是每節枝幹最多只有２個空間－－也就是頂端左側及右側－－能生長出新的枝幹，所以當某個枝幹頂端左右兩側都有生長枝幹時，『二叉樹』會選擇其它尚有生長空間的枝幹生長新的枝幹，因為有這一特性，所以若您想知道某株『二叉樹』的年齡時，可以數該『二叉樹』總共有多少的枝幹就等於該『二叉樹』的生長天數；另外，雖然『二叉樹』的生長規則看似很隨機，但是仔細思考就會發現，其實就算是擁有相同枝幹數量的『二叉樹』，各個不同模樣的『二叉樹』生長機率卻大不相同，比如說都只往某一側生長的『二叉樹』並不常見，因為依照『二叉樹』的生長特性要生長出那種模樣的『二叉樹』機率太低了。


　　『二進制世界』有個任何人都能使用的核心功能－－開發『程序』，所謂的『程序』即是使用者能夠自行撰寫程式來解決決生活大小事，包括物品分類、自動排定行程... ...等等各式各樣的功能，為了讓所有人能夠馬上適應『二進制世界』並與其他人來自不同世界的人們交流，在『二進制世界』中會提供幾項『免費程序』給所有人，像是『語言翻譯』、『導航』... ...等等基本功能，在初期基本上妥善使用這些『免費程序』就足夠生活了。若您不想自行開發『程序』而且『免費程序』無法滿足您的需求時，您可以選擇前往『線上商城』，在『線上商城』中有著『二進制世界』裡所有開發者公開的『程序』，您可以隨時下載來使用，不過使用他人的『程序』會依照『程序』所使用之資源向使用者收費回饋給開發者，所以當您自行撰寫的『程序』有一定的使用率時，是可以以『程序』當作穩定收入來源的。提醒一點，在『線上商城』是有可能存在『惡意程序』的，在『二進制世界』有相當完善的防毒檢測，所以在此指的『惡意程序』並不是電腦病毒那類的，指的是在運行『程序』時會進行不必要運算的『程序』，由於『程序』是依照所使用之資源收費的，所以會有開發者惡意在『程序』內增加不必要的運算增加『程序』所需的資源，藉此提高『程序』的收費費用，所以在『線上商城』下載『程序』時請務必謹慎挑選。


」

## 輸入說明

輸入第一行有一個整數 T 代表測資筆數。


每筆測資只有一行，並且只有一個正整數 N 。


 


測資範圍：
第00筆測資(01分, 1.0s): 同範例輸入。
第01筆測資(11分, 1.0s): T = 23 , N ≤ 23 。
第02筆測資(11分, 1.0s): T = 25 , N ≤ 25 。
第03筆測資(16分, 1.0s): T = 27 , N ≤ 27 。
第04筆測資(20分, 1.0s): T = 210 , N ≤ 210 。
第05筆測資(28分, 2.0s): T = 215 , N ≤ 215 。
第06筆測資(13分, 1.0s): T = 220 , N ≤ 220 。

## 輸出說明

每筆測資輸出一行。


輸出一個整數，代表第 N 天過後「二叉樹」可能生長成的所有模樣的個數。
