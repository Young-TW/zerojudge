# j940. 科學記號

## 內容

科學記號 (英語：scientific notation，英國則稱為：standard form)，又稱為科學記數法或科學記法，是一種數字的表示法。科學記號最早由阿基米德提出。


在科學記號中，一個數被寫成一個實數 𝑎 與一個10的 𝑛 次冪的積：[1]


𝑎 × 10𝑛


其中：


- 𝑛 必須是一個整數。
- 1 ≤ |𝑎| < 10 (如果 |𝑎|  是一個小於 1 的小數，或 |𝑎| 大於等於 10，皆可通過改變 𝑛 來表示)。
- 𝑎 是一個實數，可稱為有效數或尾數 (英語：mantissa)。

請輸入一個科學記號表示的數字並轉換為一般表示法後輸出。

## 輸入說明

輸入只有兩行。


第一行為一個正整數，代表去掉小數點的有效數 𝑎 (10 ≤ 𝑎 < 1063)。


第二行為指數 𝑛 (0 < 𝑛 ≤ ⌊𝑙𝑜𝑔10(𝑎)⌋，也就是沒有需要在前面或後面補 0 的情況)。


例如：
1.234 × 102
輸入為：
1234
2

## 輸出說明

用一般表示法來輸出這個數字。

## 範例輸入 #1

```
12345
3
```

## 範例輸出 #1

```
1234.5
```

## 範例輸入 #2

```
6250
1
```

## 範例輸出 #2

```
62.50
```
