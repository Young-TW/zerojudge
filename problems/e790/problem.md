# e790. a1.倉庫整理 (Warehouse)

## 內容

Y19m12_a01_倉庫整理 (Warehouse) TOI練習賽2019/12


問題敘述：{試題連結}


   簡單的說就是河內塔問題，有N環，要從1號塔搬至3號塔，請問第i步是將幾號環從哪一塔搬至哪一塔？ {TOI練習內原題是將環改箱子，塔改為倉庫}


 


 


評分說明


此題目測資分成三組，每組測資有多筆測試資料，需答對該組所有測試資料才能獲得該組分數。各組詳細限制如下。


第一組 (10 分) : 1 <= N <= 4


第二組 (30 分) : 1 <= N <= 20


第三組 (60 分) : 1 <= N <= 60

## 輸入說明

輸入只有一列，共兩個正整數N 與 i
（1 <= N<=60、1<=i < 2^N），以一個空白隔開，
N 代表箱子總數，i 代表當前是第幾次移動。

## 輸出說明

對每筆測試資料請輸出「move A from B to C」。代表將編號為A 的箱子從倉庫B 移動到倉庫C。

## 範例輸入 #1

```
3 3
```

## 範例輸出 #1

```
move 1 from 3 to 2
```

## 範例輸入 #2

```
20 524288
```

## 範例輸出 #2

```
move 20 from 1 to 3
```
