# b949. 3rd CPSC Problem 3－－「障」量土地

## 內容

你聽過西障自治區嗎？西障自治區位於忠國，其土地位於喜雅拉馬山的北部，土地面積相當於8787個臺灣那麼大。


　　由於幅員廣大，住在西障自治區的障族人們為了能輕鬆地測量各式各樣的距離，他們所使用的長度單位相當地特別，卻不使用其他我們一般使用的長度單位，例如：「丈」（大約等於3.33公尺）；他們使用的長度單位為「障」，相當於我們熟知的5公尺。


　　在西障自治區，因為土地資源一點兒也不缺乏，所以地價相當地便宜。住在臺北市信義區的日本人，廿一田八阝章，早已厭惡了住在「好窄」裡的生活，決定移民到西障去，以最低的成本，享受擁有200坪大平臺的生活。


　　根據《西障自治區房地產法》第87條中所提到的「在西障自治區（以下簡稱本地區），為了維持『棋盤式街道的整齊排列』，本地區所有的土地皆須為正方形，且其邊長皆須為整數障。」及第187條所提到的「為防止不動產公司集體進行壟斷，房價上漲，民不聊生，所有土地的價格限制必須為其土地之邊長的障數」。


　　在房地產界打滾數十年的廿一田八阝章，遇到這特別的制度也慌了手腳。他只好請你幫他代購，在西障自治區買塊土地。請你以他所擁有的M元，在不違法的情況下，買下所能買的最大塊土地，並把其面積換算成他所熟悉的單位──平方公尺給他。

## 輸入說明

輸⼊有多筆測資，每筆測資占一行，包含一正整數M。測資以EOF結尾。


- M < 231

## 輸出說明

對於每筆測資，請輸出一整數，為題目中所要求的值（請以平方公尺為單位表示）。

## 範例輸入 #1

```
2
10
```

## 範例輸出 #1

```
100
2500
```
