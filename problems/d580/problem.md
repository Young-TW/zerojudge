# d580. 末日預言

## 內容

聽說人類在西元2012年將會因為看太多動畫而毀滅？！姑且不論這則謠言的真實性，有一種生物理論指出，生物的大毀滅是有週期性的，而且這個週期會因為生物的進化而越來越長，它的週期如下：1, 1, 2, 3, 5, 8...也就是說，第0次生物毀滅的週期為1年(!!!)，第1次也為1年，第2次則為2年。更一般的說，第n次生物毀滅的週期(除了第0次和第1次)，是第n-1次和第n-2次的週期總和。現在請你你算出第n次生物毀滅的週期。

## 輸入說明

一堆數N，每行一個，代表你要算哪一次的週期。(0 <= N <= 10000)

## 輸出說明

對於每個N，輸出第N次生物毀滅的週期。另外因為週期可能會很長，超過了宇宙曆可以表示的年份(2147483647)，故請輸出週期除以2012的餘數。

## 範例輸入 #1

```
1
2
3
4
10
```

## 範例輸出 #1

```
1
2
3
5
89
```
