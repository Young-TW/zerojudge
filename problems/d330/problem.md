# d330. 尋找質數

## 內容

讓我們來找找質數吧~

不過那些小質數已經不合各位的程度了是吧XD

所以來找點大質數吧

爲了方便

只考慮2n-1形式的數

輸入n (不要太大好了.. n<=1500000)

告訴我2n-1是不是質數就好

## 輸入說明

每行輸入一正整數n

1<=n<=1500000

n=0時代表輸入結束

不需處理這行輸入

## 輸出說明

對每行輸入n

輸出一行判斷2n-1是不是質數

是質數就輸出"yes"  (不含雙引號)

不是質數就輸出"no" (不含雙引號)

## 範例輸入 #1

```
1
2
3
4
0
```

## 範例輸出 #1

```
no
yes
yes
no
```
