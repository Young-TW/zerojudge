# b872. 又是一窩螞蟻

## 內容

接續 b848: 一窩螞蟻，因為螞蟻的繁殖實在是太快了，你決定要消滅一些螞蟻，防止螞蟻稱霸地球，你拿到了一個能解決螞蟻的武器，可是那個武器雖然方便，不過攻擊範圍是一條直線，而且最糟的是只剩下一發，也就是說你只能選好一條線再按下武器，請你找出一次能解決最多螞蟻的那條線，並輸出能解決的螞蟻數量。

## 輸入說明

第一行只有一個數字T，代表測資筆數，接下來每筆測資的第一行有一個數字N，代表螞蟻的數量，接下來輸入N隻螞蟻的座標(x,y)。



```
n<=1000
```

## 輸出說明

請你針對每一筆測資輸出最多能消滅的螞蟻數量。

## 範例輸入 #1

```
1
5
0 0
1 1
2 2
3 3
4 5
```

## 範例輸出 #1

```
4
```
