# a266. 校內賽

## 內容

今天是校內賽前夕！為了知道自己究竟有沒有機會成為校隊，你必須刺探一下各個參賽者的實力。在經過與老師聊(ㄉㄚˇ)天(ㄐㄧㄠ ㄉㄠˋ)之後，你得到了一些關於參賽者的資訊。而你最想要知道的，當然是參賽者的排名。

## 輸入說明

每個測資檔僅含一筆測資。

    首先輸入參賽者的個數N(1<=N<=100)，以及你知道的資訊數M(1<=M<=1000)，每個資訊代表著兩名參賽者的實力強弱。接下來M行，每行兩個正整數A B代表編號A的參賽者實力優於參賽者B的實力。注意實力的強弱是有傳遞性的，也就是如果A強於B且B強於C，則A也強於C，並且測資保證不會有循環不合理的事情發生。注意其中參賽者編號為(1~N)

## 輸出說明

請根據目前已知的資訊，計算一下有哪些編號是可以"確定"自己的排名的。

  請將能夠確定排名的編號由小到大依序輸出，如果都沒有能確定排名的，請輸出"non"(不含引號)。

## 範例輸入 #1

```
範例輸入一：
5 5
5 4
4 3
3 1
2 3
5 3

範例輸入二：
20 0
```

## 範例輸出 #1

```
範例輸出一：
1 3

範例輸出二：
non
```
