# e175. 圍棋基礎 - 氣

## 內容

佐為：阿光，你知道你現在剩下多少氣嗎?


阿光：我想想，我今年10歲，我可能會活到100歲.....然後...


佐為：等一下!!!!不是那個氣，我是問你現在棋子剩下多少氣


阿光：喔喔喔~~早說嘛....


佐為：不行了，看來你需要接受特訓了(握拳


 


"氣"，指棋子四周和幾方棋子所聯通的交叉點，並且無對手棋子


如果當前四周和幾方棋子所聯通的交叉點皆是對手棋子，稱之"氣絕"


"氣絕"，表示當前棋子已無氣，需要從場上被提出


佐為今天想幫阿光做特訓，你能幫助阿光判斷當前佐為指的棋還剩下多少氣嗎？

## 輸入說明

第一行有一個數字n，代表佐為的考試棋盤部份的長寬


n = 0輸入結束，5 <= n <= 19


接下來n行，每行n個字元i代表棋盤狀況


i = '.'代表沒棋子，i = 'O'代表黑棋，i = 'X'代表白棋


接下來有一個整數q，代表佐為的詢問次數


1 <= q <= 100


接下來q行，每行兩個整數x、y代表棋盤座標


1 <= x,y <= n


注意：(1, 1)為最左下角座標，(n, n)為最右上角座標


棋盤上不會有已氣絕的棋子

## 輸出說明

輸出當前佐為指的棋子剩下多少氣


如果當前佐為指的地方沒有棋子，輸出0

## 範例輸入 #1

```
5
.X...
.....
.OO..
.....
.XOX.
4
1 1
2 5
2 3
3 1
0
```

## 範例輸出 #1

```
0
3
6
1
```
