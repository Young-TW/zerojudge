# q870. 指揮中心的數據決策

## 內容

在未來的戰術指揮中心，AI 系統每日會收到來自各地區戰場的資料報告。這些報告以一個 n × n 的數據矩陣呈現，每列（橫行）代表某個戰術單位在一段時間內的行動效能，每欄（直行）則代表某個地點在該時間段內的防禦表現。


為了判斷未來的戰略決策方向，AI 需要分析兩個面向的最高表現：


- 
橫行最大總效能值 p0 ~ p(n-1)：代表各戰術單位的最大總輸出。



- 
直行最大總效能值 m0 ~ m(n-1)：代表各地點的最大總防禦效能。




AI 會依照以下準則作出調整策略：


- 
比較每對對應位置 (mi, pi)：


- 
若 mi > pi，則地點表現優於單位。



- 
若 pi > mi，則單位表現優於地點。





- 
若直行表現較佳的位置數量較多，表示地點的配置效率更強，AI 將以地點為基礎重組部隊，輸出以「直行為橫行」的方式重新排列，並依直行總和由小到大排序輸出。



- 
若橫行表現較佳的位置數量較多，表示單位執行力更強，AI 將以單位為主導重組任務，輸出以原橫行為基礎，依橫行總和由小到大排序為直行輸出。

## 輸入說明

第一行輸入一個整數 n（1 ≤ n ≤ 100），代表接下來會有 n 行的矩陣資料。
接下來 n 行，每行有 n 個整數，代表資料矩陣的每一橫行。

## 輸出說明

輸出一個 n × n 的矩陣，依照策略判定方式進行重組後的結果。

## 範例輸入 #1

```
3
1 2 3
7 5 6
3 3 5
```

## 範例輸出 #1

```
2 5 3
1 7 3
3 6 5
```

## 範例輸入 #2

```
3
4 2 6
1 3 3
3 1 5
```

## 範例輸出 #2

```
1 3 4
3 1 2
3 5 6
```
