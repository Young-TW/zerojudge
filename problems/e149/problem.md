# e149. 五子棋

## 內容

小楊和小揚是五子棋初學者，他們現在正在一個6*6的棋盤上下五子棋，對規則還不熟悉的他們，常常犯規，你可以寫個程式幫忙檢查當前盤面是否為在符合規則下下出來的


以下是五子棋規則：
黑棋先手，之後換白棋方下棋，之後輪替
當一方一直排，一橫排或一斜線連成五個以上棋子，即獲勝
(此非正式規則，可以雙三可連四可長連XD)

## 輸入說明

輸入第一行有一個正整數n (n≤10)，表示你要檢查幾個棋盤


接下來有6行，每行有6個字元，字元X表示這邊還沒有人放棋子，B代表黑棋，W代表白棋
每一個棋盤後有一個空行。

## 輸出說明

對於每個詢問輸出'Y'如果棋盤是可能在符合規則下下出的局面，輸出'N'反之

## 範例輸入 #1

```
2
XXXXXX
XXXXXX
XXXBXX
XXXXXX
XXXXXX
XXXXXX

XXXXXX
XXXXXX
XXXWXX
XXXXXX
XXXXXX
XXXXXX
```

## 範例輸出 #1

```
Y
N
```
