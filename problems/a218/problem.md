# a218. 連猴子都會的小case

## 內容

數數跟排序大概是我們從兩三歲起就開始訓練的技能，

現在該是你好好發揮這個技能的時候了！

## 輸入說明

輸入含多組測資。

每組測資的第一行含一個整數n，1<=n<=1000，

接下來有n個範圍為0-9的整數a1,a2,...,an。

## 輸出說明

把輸入的數字按照出現次數由多到少排序，

若次數相同，優先輸出數值較小的數字。 

參考範例輸出。

## 範例輸入 #1

```
5
1 4 5 1 4
10
0 0 0 0 9 9 9 2 2 8
```

## 範例輸出 #1

```
1 4 5
0 9 2 8
```
