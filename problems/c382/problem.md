# c382. 加減乘除

## 內容

文文剛學完了加減乘除四則運算，作業簿上有些練習題，他寫完了練習題可是不知道對不對。麻煩你提供正確解答讓他對一下答案，謝謝！

## 輸入說明

輸入只有一行，含有兩個非負整數，兩數中間有一個運算符號 + (加)、- (減)、* (乘)、或 / (除)。整數與運算符號之間沒有空白。

## 輸出說明

請輸出算式的結果，答案均為整數。

## 範例輸入 #1

```
12+3
```

## 範例輸出 #1

```
15
```

## 範例輸入 #2

```
101-11
```

## 範例輸出 #2

```
90
```

## 範例輸入 #3

```
13*7
```

## 範例輸出 #3

```
91
```

## 範例輸入 #4

```
111/3
```

## 範例輸出 #4

```
37
```
