# j168. 青蛙

## 內容

青蛙在一張 $n \times m$  的方格紙醉左下角的格子中。每一次，牠可以從牠目前的所在的位置往右跳一格、往上跳一格或往右上跳一格。則青蛙跳到最右上角格子的方法總數為多少？


請輸出答案模 $1000000007$。

## 輸入說明

第一行有一個數字 $t$ 代表測資數，接下來 $t$ 行，第 $i$ 行會有一個數字 $n_i, m_i$ 代表第 $i$ 個詢問方格紙的大小。


測資限制：


$1 \le n_i, m_i\le 10^6$


$\sum n_i+m_i \le 2\times 10^6$

## 輸出說明

輸出 $t$ 行，輸出第 $i$ 個詢問的答案數除以 $1000000007$ 的餘數。

## 範例輸入 #1

```
1
5 5
```

## 範例輸出 #1

```
321
```
