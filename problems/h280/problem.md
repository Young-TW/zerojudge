# h280. 小明的胖手指

## 內容

小明有著笨拙的胖手指 (fat finger)，在鍵盤上打字很慢又容易出錯。這次他在打字時不小心按到了 Enter 鍵，把原本一行的文字斷成了兩行。你發現了這個問題，所以幫他把這兩行接成一行。



![image](ShowImage?id=2018)

## 輸入說明

輸入含有兩行文字。

## 輸出說明

輸出兩行文字於一行。

## 範例輸入 #1

```
s
nail
```

## 範例輸出 #1

```
snail
```

## 範例輸入 #2

```
This is a sculpture from the Hell
enistic age.
```

## 範例輸出 #2

```
This is a sculpture from the Hellenistic age.
```
