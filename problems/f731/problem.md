# f731. 老鼠的直播間

## 內容

老鼠希望能夠知道直播期間最高同時有幾個人在線上觀看，於是他找了會寫程式的你幫忙。

## 輸入說明

單筆測資


第一行 $n$ 代表接下來有 $n$ 個人


接下來每行有兩個整數 $a_i, b_i$ 分別代表這個人進入和離開直播間的時間點


每個時間點只會有一個人進入/離開


$1 \leq n \leq 2 \cdot 10^5$


$1 \leq a_i < b_i \leq 10^9$

## 輸出說明

請你輸出直播期間最高同時有幾個人在線上觀看

## 範例輸入 #1

```
3
1 3
5 6
2 4
```

## 範例輸出 #1

```
2
```
