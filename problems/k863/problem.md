# k863. 清朝年號轉換

## 內容

文文讀清末歷史：「光緒 20 年，中日爆發甲午戰爭...」文文覺得那個年代好麻煩，換個皇帝就要換年號，搞得他的頭很大。於是他想寫個程式，輸入一個清朝的年號，可以自動換算成西元年份。


清朝一共有 10 個皇帝，其年號依序是：順治 (ShunZhi, 1644)、康熙 (KangXi, 1662)、雍正 (YongZheng, 1723)、乾隆 (QianLong, 1736)、嘉慶 (JiaQing, 1796)、道光 (DaoGuang, 1821)、咸豐 (XianFeng, 1851)、同治 (TongZhi, 1862)、光緒 (GuangXu, 1875)、宣統 (XuanTong, 1909)。

## 輸入說明

輸入只有一行，含有一個年號及年份。

## 輸出說明

將該年號年份轉成西元年份後輸出。

## 範例輸入 #1

```
GuangXu 20
```

## 範例輸出 #1

```
1894
```

## 範例輸入 #2

```
YongZheng 1
```

## 範例輸出 #2

```
1723
```
