# e928. pB. 多項式相乘

## 內容

輸入兩個一元多次多項式，請計算兩多項式相乘後的結果。
例如：(x^2 + 2x) * (2x^3 + 3x + 1) = (2x^5 + 4x^4 + 3x^3 + 7x^2 + 2x)


其中所有多項式皆以各項係數以及最高冪次表示，舉例來說
(x^2 + 2x) 各項係數為 (1, 2, 0)，最高冪次為 2
(2x^3 + 3x + 1) 各項係數為 (2, 0, 3, 1)，最高冪次為 3
(2x^5 + 4x^4 + 3x^3 + 7x^2 + 2x) 各項係數為 (2, 4, 3, 7, 2, 0)，最高冪次為 5

## 輸入說明

輸入共有四行

第一行有一個整數 N1 (0 ≤ N1 ≤ 100)，
代表第一個多項式的最高冪次

第二行有 N1 + 1 個整數，
代表第一個多項式由冪次高至低的各項係數 w1 (-100 ≤ w1 ≤ 100)

第三行有一個整數 N2 (0 ≤ N2 ≤ 100)，
代表第二個多項式的最高冪次

第四行有 N2 + 1 個整數，
代表第二個多項式由冪次高至低的各項係數 w2 (-100 ≤ w2 ≤ 100)

## 輸出說明

輸出共有兩行
第一行為相乘後多項式的最高冪次
第二行為相乘後多項式由冪次高至低的各項係數

## 範例輸入 #1

```
2
1 2 0
3
2 0 3 1
```

## 範例輸出 #1

```
5
2 4 3 7 2 0
```
