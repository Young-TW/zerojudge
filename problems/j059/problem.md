# j059. 11547－Automatic Answer

## 內容

上個月，愛麗絲漫不經心地在抽獎中輸入了她的名字，抽獎抽獎抽了 Tapmaster 4000。今天檢查她的郵件時，她發現了一封信，上面寫著：
“恭喜你，愛麗絲！您贏得了 Tapmaster 4000。要領取您的獎品，您必須回答以下技能測試問題。”
愛麗絲最初的驚訝喜悅很快變成了沮喪。她一生的技能測試題記錄是糟糕的 3 對和 42 錯。
領先的技能測試問題開發公司 Mad Skills 受聘為這次特定的 Tapmaster 4000 抽獎提供技能測試問題。他們決定為每個獲勝者創建一個不同的技能測試問題，以便獲勝者無法合作回答問題。
你能通過解決技能測試問題幫助 Alice 贏得 Tapmaster 4000 嗎？

## 輸入說明

輸入第一行有一個整數 t (1 ≤ t ≤ 100)，代表測試資料的數量。


每組測試資料一行，包含一個整數 n (−1000 ≤ n ≤ 1000)。

## 輸出說明

對於每組測試資料，將以下技能測試問題的答案單獨輸出一行：
"將 n 乘以 567，然後除以 9，然後加 7492，然後乘以 235，然後除以 47，然後減去 498。請問此時數字的十位數字是多少？"

## 範例輸入 #1

```
2
637
-120
```

## 範例輸出 #1

```
1
3
```
