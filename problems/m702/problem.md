# m702. 傑出校友票選活動

## 內容

文文母校適逢百年校慶，校友會舉辦「傑出校友票選活動」。在回收了所有的校友選票之後，交給你來協助統計票數。

## 輸入說明

輸入的第一行含有兩個整數 𝑛, 𝑚 (1 ≤ 𝑛 ≤ 10, 1 ≤ 𝑚 ≤ 105)，代表一共回收 𝑛 張有效選票，要選出 𝑚 個傑出校友。


接下來會有 𝑛 行，每行有一個不含空白的名字，代表該張選票所投給的對象。

## 輸出說明

依得票數高低，輸出得票最高的 𝑚 個傑出校友於一行。前 𝑚 名的傑出校友中不會有得票數相同的情況。

## 範例輸入 #1

```
10 3
Peter
James
John
John
James
James
Peter
Peter
Peter
Paul
```

## 範例輸出 #1

```
Peter James John
```
