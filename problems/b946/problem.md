# b946. 3rd CPSC Problem 1－－配「分」法

## 內容

藍瘦國中的數學老師－－香菇老師負責這次學校段考數學科出題的工作。為了不要讓學生們太藍瘦香菇，他決定這次的段考題目只有選擇題。


　　但是只有選擇題實在是太簡單又太無趣了，因此香菇老師決定將選擇題分成簡單及困難兩種難度，並分別給它們不同的配分：S1及S2分。在全部的題目中，有Q1題簡單選擇題、Q2題困難選擇題，共K題。除此之外，為了防止學生太早寫完考卷，趴在桌上睡覺，導致睡覺時流下的口水汙損答案卡，造成無法辨識答案，使學生只能藍瘦香菇地拿零鴨蛋，老師決定將考卷的總分改變成T分，將防護措施做好做滿。


　　在決定配分及總分後，接著就是決定題數了，但香菇老師早已累得趴在桌上睡覺。現在，請你寫出一個程式，幫助老師決定簡單及困難選擇題的題數吧！

## 輸入說明

輸入有多筆測資，每筆測資一行，測資以EOF結尾。


　　每行包含四個正整數： S1 、 S2 、 T 、 K ，分別以一個空白隔開。


- 1 ≤ S1 < S2 ≤ 50
- 100 ≤ T ≤ 109
- 2 ≤ K ≤ 109

## 輸出說明

對於每筆測資請輸出一行，包含Q1、Q2，以空白隔開。若Q1或Q2的值不是正整數，請輸出「impossible」（不含引號）。

## 範例輸入 #1

```
4 5 100 23
2 3 100 42
3 7 100 10
3 15 225 15
10 12 101 11
```

## 範例輸出 #1

```
15 8
26 16
impossible
impossible
impossible
```
