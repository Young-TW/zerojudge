# i232. LCM

## 內容

$a, b$ 為兩個長度為$N$的整數數列 $a_1, a_2, ..., a_N$ 和 $b_1, b_2, ..., b_N$


計算 $c_1, c_2, ..., c_N$，並對每一項 $\bmod 998244353$，其中


$c_k = \sum_{lcm(i, j) = k} a_i \cdot b_j$

## 輸入說明

第一行為整數 $N$


第二行與第三行各有 $N$ 個整數 $a_i$ 和 $b_i$。


$1 \leq N \leq 10^6$


$0 \leq a_i, b_i \leq 998244352$

## 輸出說明

輸出 $N$ 個數字

## 範例輸入 #1

```
6
1 2 3 4 5 6
6 5 4 3 2 1
```

## 範例輸出 #1

```
6 27 34 65 42 125
```
