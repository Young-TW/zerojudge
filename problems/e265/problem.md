拱豬(撲克牌)規則：


出牌和吃牌如同橋牌出牌和吃牌


牌點大小順序（遞減排列）：A，K，Q，J，10，9，8，7，6，5，4，3，2。


特殊牌：在52張牌中，有16張被定義為特殊牌，其中15張分牌和1張特殊作用牌。


♡：A為負50分，K為負40分，Q為負30分，J為負20分，10-5各為負10分，4-2為0分。


♠：Q為豬，負100分。


♢：J為羊，正100分。


♣：10為double，所有分數兩倍


負分最多者輸，如果拿到所有的紅心牌則發生"豬羊變色"，豬變正100分、羊變負100分、所有紅心變正分

## 輸入說明

每一行為一個整數n，代表有幾回合(1 <= n <= 50)


每回合有4行，代表第幾人吃到的牌


每張牌以2個字元代表，第一個字元為點數(A=Ace, 2-9, T=10, J=Jack, Q=Queen, K=King)


第二個字元為花色(C=Clubs♣, D=Diamonds♢, H=Hearts♡, S=Spades♠)


每行的最後，會有一個字元'#'代表此行結束


吃到的牌只會有特殊牌

## 輸出說明

對於每回合，輸出當回合每個人的結果(如範例輸出)


在最後輸出總結果(如範例輸出)

## 範例輸入 #1

```
2
AH 2H 3H 4H #
5H 6H 7H 8H #
9H TH JH QH KH QS #
JD TC #
AH 2H 3H 4H 5H 6H 7H 8H 9H TH JH QH KH QS #
#
#
JD TC #
```
