# k867. 筆記本 notebook

## 內容

小明將一些人名寫在他的筆記本上，這些人名維持着一種他自己排出來的亂序，但是筆記本不小心被小明弄丟了。幸運的是，小明還記得每個名字在順序中的位置，請你編程一條程序，重新以升序排序所有名字。

## 輸入說明

輸入只有兩行，第一行輸入所有名字。第二行輸入上方各個名字分別對應的位置，位置由1開始。

## 輸出說明

輸出只有一行，以輸入給定的位置重新以升序排列所有名字。

## 範例輸入 #1

```
tom john mike johnson paul
1 5 3 2 4
```

## 範例輸出 #1

```
tom johnson mike paul john
```

## 範例輸入 #2

```
bob issac kyle andy alan bill steve
7 3 4 5 6 1 2
```

## 範例輸出 #2

```
bill steve issac kyle andy alan bob
```
