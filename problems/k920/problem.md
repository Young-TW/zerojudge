# k920. 青蛙跳井-綜合版

## 內容

續上兩題，青蛙跳井(1)、青蛙跳井(2)，如果輸入"A"，求t，輸入"B"求h。

## 輸入說明

輸入共兩行，第一行是一個字母，可以為"A"、"B"。


第二行分別對應第一行的字母


A : h x y


B : t x y


t ≥1 x > 1 y>1


x=y+1

## 輸出說明

輸出只有一行，對應輸入的字母


A : t


B : h

## 範例輸入 #1

```
A
10 4 3
```

## 範例輸出 #1

```
7
```

## 範例輸入 #2

```
B
7 4 3
```

## 範例輸出 #2

```
10
```
