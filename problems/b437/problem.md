# b437. 11476 - Factorizing Larget Integers

## 內容

給你一個正整數$N(1<N\leq 10^{18})$，請你把$N$質因數分解。

## 輸入說明

第一行是一個正整數$T(1\leq T\leq 800)$，表示測資數量。

接下來$T$行，每行一個正整數$N(1<N\leq 10^{18})$。

## 輸出說明

對於每個$N$，輸出N的質因數分解。格式參見範例。

## 範例輸入 #1

```
3
60
36
10007
```

## 範例輸出 #1

```
60 = 2^2 * 3 * 5
36 = 2^2 * 3^2
10007 = 10007
```
