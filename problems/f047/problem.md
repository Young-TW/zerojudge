# f047. 三路縱隊

## 內容

基於戰術需求，百變兵團最近經常行軍。行軍時會依路面寬度來決定成幾路縱隊前進。


今天一早出發時，因為路面較窄，所以成一路縱隊。來到了一條寬大的馬路時，指揮官下令成三路縱隊前進，以縮短隊伍的長度。這時候原隊伍中的第一個人留在第一縱隊，第二人及第三人則成為第二及第三縱隊的排頭，第四人留在第一縱隊，第五及第六人則分別依序排到第二及第三縱隊，以此類推。


給你原先一路縱隊的名單，請依序輸出改成三路縱隊時每個縱隊的名單。

## 輸入說明

輸入只有一行，含有原先一路縱隊的名單，每兩個人中間以一個空白隔開。3 ≤ 兵團人數 ≤ 100。

## 輸出說明

輸出三行，依序分別為第一、第二、第三縱隊的名單。

## 範例輸入 #1

```
John David Steve Andrew Mary Leah
```

## 範例輸出 #1

```
John Andrew
David Mary
Steve Leah
```

## 範例輸入 #2

```
810123 810234 810345 810567 810678 810789 810890
```

## 範例輸出 #2

```
810123 810567 810890
810234 810678 
810345 810789 
```
