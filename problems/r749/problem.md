# r749. Dominoes

## 內容

Dominoes are tile pieces used in various games. A domino piece consists of two squares. Each square is marked with dots, but may or may not be marked with dots. The number of dots is determined by the size of the set. In a domino set of size N, the number of dots is greater than or equal to 0 and less than or equal to N. If two dominoes have the same number of dots, they are equal. For example, a domino with 2 dots and 8 dots is the same as a domino with 8 dots and 2 dots.

A domino set of size N contains all possible dominoes with N or fewer or equal points, and each domino is unique. The following is a domino set of size 2.

![image](ShowImage?id=4991)

After receiving N as input, write a program that calculates how many dots are in a domino set of size N.

## 輸入說明

The first line contains the size N of the domino set (1 ≤ N ≤ 1000).

## 輸出說明

Print the number of dots printed on a set of dominoes of size N.

## 範例輸入 #1

```
2
```

## 範例輸出 #1

```
12
```

## 範例輸入 #2

```
3
```

## 範例輸出 #2

```
30
```

## 範例輸入 #3

```
15
```

## 範例輸出 #3

```
2040
```
