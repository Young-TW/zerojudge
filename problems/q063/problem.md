# q063. 喵-三維迷宮

## 內容

給你一個迷宮


請輸出最短路徑長

## 輸入說明

只有一筆測資
每筆測資第一行有三個數字l n m(5<=l, n, m<=205)
l是高，n是長，m是寬


接下來有l組
每組有n行
每行有m個字元
#代表牆壁
" "代表路


從起點(1, 1, 1)走到終點(l - 2, n - 2, m - 2)

## 輸出說明

請輸出最短路徑長

## 範例輸入 #1

```
5 5 5
#####
#####
#####
#####
#####

#####
#   #
# # #
# # #
#####

#####
# # #
#####
### #
#####

#####
# # #
#####
#   #
#####

#####
#####
#####
#####
#####

```

## 範例輸出 #1

```
6
```
