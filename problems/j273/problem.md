# j273. 進德教育

## 內容

「進德教育」是蝸牛所任教的學校數十年的一個傳統，透過正當、合理且符合教育目的之方式，達到積極正向管教、教育、輔導學生之目的。給予犯規輕微者再一次自省自新的機會，避免動輒警告、小過存檔。


一個星期結束了，學務處正在統計需要參加進德教育的名單及每個人的次數。由於本週違規的人太多了，學務處要麻煩你寫個程式加快統計作業。

## 輸入說明

輸入只有二行，第一行含有一個整數 𝑛 (1 ≤ 𝑛 ≤ 100)，代表本週登記了多少人次的進德教育。第二行含有 𝑛 個五位數的班級座號，代表該生被記進德一次。

## 輸出說明

由小到大依序輸出所有的班級座號及該生被登記的次數。

## 範例輸入 #1

```
7
31203 10124 22011 10606 10124 10124 22011
```

## 範例輸出 #1

```
10124 3
10606 1
22011 2
31203 1
```

## 範例輸入 #2

```
6
11234 31223 11234 31223 11234 31223
```

## 範例輸出 #2

```
11234 3
31223 3
```
