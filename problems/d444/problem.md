# d444. 排容原理

## 內容

現在從 1~N 連續整數中，刪除掉 S 的所有倍數 (包蓋 S )
請輸出沒有被刪除的個數

## 輸入說明

輸入的每一行有兩個數字 N M，代表接下來有 M 個數字S，必須將所有S的倍數都刪掉。
若 N=0 M=0 ，代表輸入結束。
( 1 < N , S < 2147483647 , M ≦ 15 )

※ ∏S<=2147483647

補充 ∏i=0nYi 表示所有 (n+1)-元組 (y0,…,yn)。 ∏n=13R = Rn

## 輸出說明

請輸出沒有被刪除的個數

## 範例輸入 #1

```
100 4
2 3 5 7
1000 5
6 8 11 13 15
0 0
```

## 範例輸出 #1

```
22
603
```
