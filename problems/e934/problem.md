# e934. pD. DNA 轉錄

## 內容

根據 DNA 轉錄為 RNA 的規則：
T → A，A → U，G → C，C → G
因此對於 DNA 片段 TAGGAC，會轉錄為 AUCCUG。


給定片段 DNA，請將所轉錄的 RNA 印出。

## 輸入說明

片段 DNA（20 ≤ DNA 的字串長 ≤ 1000）

## 輸出說明

所轉錄為的 RNA

## 範例輸入 #1

```
TAGGAC
```

## 範例輸出 #1

```
AUCCUG
```

## 範例輸入 #2

```
ATCGATCGATCGATCGATCG
```

## 範例輸出 #2

```
UAGCUAGCUAGCUAGCUAGC
```
