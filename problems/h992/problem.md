# h992. 給測驗們的電神

## 內容

今天有一群測驗們在討論一個題目，我太弱了，但是為了能夠幫助測驗們，決定找一位與測驗們一樣電的人 (你) 來幫忙。題目如下：


費氏數列 $F_1 = F_2 = 1，F_i = F_{i - 1} + F_{i - 2}$，給你 $a, b$，請你輸出 $F_a^{F_{\ b}}$ ($\text{mod }1000000007$)

## 輸入說明

第一行有一個正整數 $t$，代表測資筆數。


接下來 $t$ 行，每行有兩個正整數 $a, b$，代表請你
求 $F_a^{F_{\ b}}$ ($\text{mod }1000000007$)。


- $1 \leq t \leq 10^5$
- $1 \leq a, b \leq 10^9$

## 輸出說明

對於每筆測資，輸出一個答案。
