# c489. kevin 的啃得雞

## 內容

在 kevin 玩完水桶之後，雞餓的 kevin 又餓又累，於是他打電話叫了啃得雞外送


他點了他最愛的大旺大吉紙包雞派對餐當作他的下午茶
派對餐裡面有 M*K 個紙包雞，一共有 K 種口味，每個口味剛好有 M 塊


而當啃得雞的外送員將餐點送達準備離開時，他看到了外送員嘴角邊的一抹微笑
kevin 聞到了 K 種不同的口味紙包雞，其中 K - 1 種都完好如初
卻只有「一種」紙包雞少了好幾塊 kevin 懷疑是外送員吃了他的紙包雞
而外送員即將離開，kevin 來得及找回他失去的義式香草紙包雞嗎？


 


因為 kevin 只有兩隻手，翻找紙包雞很麻煩，所以內存不足 (注意本題開不下 10^6 個int)


本題記憶體限制 5MB
請勿    include <iostream>
請使用 include <stdio.h> 使用 scanf , printf 輸入輸出

## 輸入說明

第一行有一個數字 N


代表 kevin 訂購的紙包雞派對餐剩下的紙包雞數量


接下來有 N 個數字，代表 N 個紙包雞的口味

## 輸出說明

請輸出 kevin 到底少了哪種紙包雞

## 範例輸入 #1

```
7
1 2 1 3 4 3 4
```

## 範例輸出 #1

```
2
```
