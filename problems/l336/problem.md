# l336. 集印花

## 內容

小明是一位教師，只要在他的課程上表現好就能獲得一枚印花。達到一定數量的印花就能換大獎，印花有多種，因此搞得他的學生很頭痛。請你設計一段程序，為他設計一套系統解決問題。

## 輸入說明

第一行為一個整數n，代表有多少個印花種類，1<=n<=10。


第二行為印花種類，以空格隔開，以其大小升序排列。(最左邊的為最低級的種類，最右邊的為最高級的種類)


第三行為一個整數t，代表印花之間對換的數量 (如t個a換一個b，t個b換一個c)，1<=t<=10。


第四行為要換的印花種類a+其數量b，和被印花種類c。


格式 : "ba c"


詳見範例輸入。

## 輸出說明

輸入ba可換多少個c，如果有餘的話，則輸入 格式 : "多少個c (餘數)+(印花種類)"  餘數由最小等級+數量 空白隔開，餘數種類大於一種時 由等級小先輸出。


如果不夠，則輸入"-1"


保證不會換最左邊的種類。

## 範例輸入 #1

```
3
bronze silver gold
3
27bronze gold
```

## 範例輸出 #1

```
3gold
```

## 範例輸入 #2

```
4
red blue yellow green
5
1red green
```

## 範例輸出 #2

```
-1
```

## 範例輸入 #3

```
4
D+ C+ B+ A+
4
1025D+ A+
```

## 範例輸出 #3

```
16A+ 1D+
```
