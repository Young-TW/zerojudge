# e168. 演唱會記行 - 周邊商品確認

## 內容

在慢長時間的排隊之後，終於...終於...終於...！！！


買到啦~~~~~~~~~灑花灑花


購買前，工作人員會發給你商品DM，你只需用勾選想購買的商品，所以不會講韓文或者日文也沒有關係喔~


在買到商品後，工作人員會請您現場做確認，確保商品沒有瑕疵，如果商品有瑕疵才能退換貨


每樣商品需要拆開最外層包裝紙，來確認內部是否有瑕疵


由於小粉絲腦波太弱，一個不小心買太多了，你能幫助小粉絲確認商品是否有問題嗎？

## 輸入說明

第一行為一個整數n，代表小粉絲買了幾樣商品


1 <= n <= 1000


接下來n行為小粉絲買的商品，每樣商品皆有包裝


包裝紙只會有()[]{}這三種


注意！！！拆下包裝紙後商品不可為空，且所有包裝紙必須要成對配好(包括商品)

## 輸出說明

對於每一樣商品


如果商品完整，請輸出商品拆開最外層包裝後的模樣


如果商品有毀損，請輸出Product Broken!!

## 範例輸入 #1

```
6
({[]})
([])([])([])
([{}]{})[()]
)(
()()()
([)]
0
```

## 範例輸出 #1

```
{[]}
[][][]
[{}]{}()
Product Broken!!
Product Broken!!
Product Broken!!
```
