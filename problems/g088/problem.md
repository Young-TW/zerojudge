# g088. 生成樹數量

## 內容

給你一個無向圖，請你計算有幾個生成樹

## 輸入說明

第一行兩個整數n和m，代表有n個點(0~n-1)和m條邊


接下來m行每行兩個整數a和b代表a到b之間有一條邊


1 <= n <= 10


0 <= a, b <= n - 1


保證不會有重複的邊

## 輸出說明

輸出這個圖有多少個生成樹

## 範例輸入 #1

```
4 4
0 1
0 2
0 3
2 3
```

## 範例輸出 #1

```
3
```

## 範例輸入 #2

```
5 7
0 1
1 2
3 2
3 4
1 4
0 2
1 3
```

## 範例輸出 #2

```
21
```
