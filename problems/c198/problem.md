# c198. 保持穩定供電

## 內容

某城市出現電力緊張, 尤其在晚間的高鋒期, 為了令發電機不會超出負荷, 他們想出了一個奇怪的方案, 就是把當時的用電者進行隨機由1--N編號, 設定一個M(M>=2)值, 由1開始順序數M人, 數到M的就暫停對他供電, 一直循環地數下去, 直到指標回復正常, 為了確保市長能一直都得到供電, 該方案要選出一個最小的M值, 令到即使前面N-1人都停止了供電, 也不會影響到市長的工作, 本方案假設第一個被停止供電的一定是1號。

## 輸入說明

有多行輸入, 每行兩個數N和X, 分別表示有N( 3<=N<=500)戶人, 其中第X(2<=X<=N)戶人為市長, 輸入0  0表示結束, 行數不超過50

## 輸出說明

輸出對應最小的M值

## 範例輸入 #1

```
3 2
12 2
0 0
```

## 範例輸出 #1

```
2
16
```
