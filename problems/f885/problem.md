# f885. 加總

## 內容

給你兩個整數 $a$ 和 $x$，請你找出 $n$ 使得：


$$\sum_{i=a}^{n} i \geq x$$


若有多組解，請輸出 $n$ 最小的解。

## 輸入說明

單筆測資


第一行一個整數 $t$。


接下來有 $t$ 行，每行兩個整數 $a$ 和 $x$。


$a \leq x \leq 2^{31} - 1$

## 輸出說明

輸出 $n$ 最小的解答

## 範例輸入 #1

```
5
2 3
1 10
1 11
5 5
5 18
```

## 範例輸出 #1

```
3
4
5
5
7
```
