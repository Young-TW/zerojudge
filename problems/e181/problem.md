## 輸入說明

每一行有一個數字n，代表長頸鹿當前有多少錢袋


1 <= n <= 10000


接下來一行，每行n個字元i代表錢袋內有多少錢


1 <= i <= 100


接下來一行有一個數字m代表PD提出的金額


(all i) <= m <= 100

## 輸出說明

輸出長頸鹿最後剩下多少錢


如果錢不夠付回收金額，輸出QQ
