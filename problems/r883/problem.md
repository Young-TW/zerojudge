# r883. PB. 排列

## 內容

Seonggwan 喜歡滿足以下條件的安排：

- 數組的長度為 N。
- 數組中填充的數字是大於或等於 1 且小於或等於 K 的自然數。
- 當陣列中的連續數字為 A 和 B 時，必須滿足 A <= B 或 A % B != 0。

例如，當 N = 4 且 K = 7 時，[1, 7, 7, 2] 是 Seonggwan 喜歡的陣列。這是因為所有連續的數字都滿足條件 1 <= 7, 7 <= 7, 7 % 2 != 0。

給定 N 和 K，寫一個程式來找出 Seonggwan 喜歡的排列組合的數量。

## 輸入說明

第一行給定 N 和 K。（1 ≤ N ≤ 10，1 ≤ K ≤ 100,000）

## 輸出說明

第一行輸出 Seonggwan 喜歡的陣列數量，對 10^9+7 取餘數後的結果。

## 範例輸入 #1

```
2 2
```

## 範例輸出 #1

```
3
```

## 範例輸入 #2

```
9 1
```

## 範例輸出 #2

```
1
```

## 範例輸入 #3

```
3 3
```

## 範例輸出 #3

```
15
```

## 範例輸入 #4

```
1 107
```

## 範例輸出 #4

```
107
```

## 範例輸入 #5

```
2 10
```

## 範例輸出 #5

```
83
```
