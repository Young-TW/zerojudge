# d284. 漂亮數碼

## 內容

一個「漂亮數碼」指的是對於一個n位數的數碼

這n位數碼的"0"的個數為偶數(包含沒有0的情況也算)

例如：098334908就是一個n=9的漂亮數碼

而098330908就不是漂亮數碼。

現在給你n值(1<=n<=100)

請你求出所有n位的數碼中是漂亮數碼的個數

例如：n=1時

有1    2    3    4    5    6    7    8    9   這幾種可能

所以你應該輸出9。

注意0983340908是n=10的非漂亮數碼

而983340908為n=9的漂亮數碼。

## 輸入說明

輸入為n的值。

當n=0時代表輸入結束，不必處理這行輸入。

## 輸出說明

輸出n位數碼中，為漂亮數碼的個數。

## 範例輸入 #1

```
1
2
3
0
```

## 範例輸出 #1

```
9
82
756
```
