# f479. 質數成長

## 內容

連續質數列表(包含1)為： 1 2 3 5 7 11 13 17 19 23 ......。
今有一種奇怪的植物，其每天的成長速度是依照以上連續的質數規則進行的，例如：
第1天 --> 長 1 節 -------->   A
第2天 --> 增長 2 節 ----->   AAB
第3天 --> 增長 3 節 ----->   AABABC
第4天 --> 增長 5 節 ----->   AABABCABCAB
第5天 --> 增長 7 節 ----->   AABABCABCABABCABCA
第6天 --> 增長 11 節 --->    AABABCABCABABCABCAABCABCABCAB
.....
試問第 N 天時植物的第 K 節為何？

## 輸入說明

多筆測資(<=10000)，每筆測一行，整數 N, K
(1 <= N <= 10000   1 <= K <= 1000,000,000)

## 輸出說明

每筆測資一行，輸出一個字母
(如範例)
如果 K 超出第 N 天的節數，則輸出 # 字母

## 範例輸入 #1

```
1 1
4 3
6 17
2 5
```

## 範例輸出 #1

```
A
B
C
#
```
