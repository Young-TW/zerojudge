# d214. What's the next number?

## 內容

Triangle, pentagonal, 和 hexagonal 的數字是依照下列定義:

Triangle      Tn=n(n+1)/2   1, 3,  6, 10, 15, ... 
Pentagonal  Pn=n(3n-1)/2   1, 5, 12, 22, 35, ... 
Hexagonal   Hn=n(2n-1)     1, 6, 15, 28, 45, ...
