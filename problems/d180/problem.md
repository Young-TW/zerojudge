# d180. 拿硬幣

## 內容

有天楓火，在路上遇到了一群人在圍觀，楓火便擠了進去，看到一個人在和老闆玩一個小遊戲，便一舉發現老闆所用的小技巧。

遊戲內容：桌面上有N個硬幣(N<1001，必為偶數)，輪流拿一個硬幣，如果挑戰者拿的總價值比較高，則可獲得獎金 7億 。且硬幣價值不超過1000(<=1000)，不用懷疑，楓火的國家硬幣有1到1000元，缺點就是沒有紙鈔，呵呵！

## 輸入說明

第一行

有一個數，N。

第二行到第N+1行皆有一個數

為硬幣的價值。

## 輸出說明

老板是先拿硬幣的，請輸出老闆要先拿左邊還是右邊才會必勝呢？

如果是右邊 輸出"right"

反之輸出"left"

## 範例輸入 #1

```
6
7 
8 
9 
3 
2 
1
```

## 範例輸出 #1

```
left
```
