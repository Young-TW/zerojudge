# b855. 一封信

## 內容

有一天小明收到了一封奇怪的恐嚇信，信裡面只有一大堆的數字，仔細一看小明還發現角落有寫著小小的KM，翻到背面才發現信中要他依據裡面數字往東邊或北邊前進，不然他之前玩過的每個遊戲的記錄檔會完全消失，小明已經玩過很多種遊戲，每種遊戲平均都花了5000元，是一名重度課金的玩家，因此他不想失去他任何遊戲的紀錄檔，所以他決定要乖乖照著信中的數字走，但是小明不想要離家裡太遠，請你求出他最後停留的地點和起點的直線距離的平方最小值為何?

## 輸入說明

第一行有一個數字T代表有幾筆測資，每筆測資中第一行有一個數字N，代表接下來有幾個數字，每個數字皆以換行隔開。


20%測資滿足 T ≤ 10, N ≤ 10, K ≤ 10


40%測資滿足 T ≤ 10, N ≤ 100, K ≤ 100


60%測資滿足 T ≤ 10, N ≤ 1000, K ≤ 1000


80%測資滿足 T ≤ 10, N ≤ 1000, K ≤ 10000


100%測資滿足 T ≤ 50, N ≤ 1000, K ≤ 20000

## 輸出說明

對每筆測資輸出一個數字代表最小的距離平方，每個數字皆以換行隔開。

## 範例輸入 #1

```
1
3
1
4
7
```

## 範例輸出 #1

```
74
```
