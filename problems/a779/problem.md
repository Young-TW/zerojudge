# a779. 1. Reversal of Field

## 內容

回文是指一串單字、片語或數字組成的文字，而這串文字由左至右讀和由右至左讀是一樣的 (不含標點符號) 。請你寫一個程式判斷輸入字串是否為回文。

## 輸入說明

輸入多個字串。

## 輸出說明

請重複一次輸入字串，如果是回文請輸出"-- is a palindrome" (不含引號) ，不是則輸出"-- is not a palindrome" (不含引號) 。

## 範例輸入 #1

```
Madam, I'm Adam.
12345321
```

## 範例輸出 #1

```
Madam, I'm Adam.
-- is a palindrome
12345321
-- is not a palindrome
```
