# e421. 質數日 Again!

## 內容

Caido是生活在未來的未來人，他看了看日曆


「嗯，今天是西元2000000年1月13號，剛好是質數日呢!」


他很好奇西元2000000年~2999999年還有哪些質數日，於是開始拿起筆在紙上算


隔天早上，你經過他家，你看到Caido在算數學，你問他在算什麼，他說：「我在算2000000年~2999999年的質數日。」


聽完之後，你在心裡暗自竊笑，這麼多數字怎麼可能算出來。Caido似乎聽到你在想什麼，他瞪著你說：「你竟然敢這樣質疑我，那你自己來算，你算不出來，我就不跟你當朋友。」你心裡想著自己什麼時候跟Caido當過朋友了，不過基於你善良的心，你還是決定幫Caido一把。

## 輸入說明

(無輸入)

## 輸出說明

輸出格式為yyyyyyy/mm/dd


其中y代表年，m代表月，d代表日，不足補0

## 範例輸入 #1

```
(空氣)
```

## 範例輸出 #1

```
(答案被Caido吃掉了)
```
