# n761. 10209 - Is This Integration

## 內容

在下圖中，你可以看到一個正方形 ABCD，其中 AB = BC = CD = DA = a。以 A、B、C、D 四個頂點為中心，並以 a 為半徑，畫出了四條弧線。以 A 為中心畫的弧線從鄰近的頂點 B 開始，到另一個鄰近的頂點 D 結束。所有其他弧線也以類似的方式畫出。通過這種方式，形成了三種不同形狀的區域。你需要確定這些不同形狀區域的總面積。



![image](ShowImage?id=4038)

## 輸入說明

輸入檔包含若干行浮點數 a (0 ≤ a ≤ 10000)，每行表示正方形的一邊長度。輸入以文件結尾作為終止。

## 輸出說明

對於每行輸入，輸出一行包含三個浮點數，表示三種類型區域的總面積（對應於上圖中的不同圖案填充區域）。 這三個數字應保留三位小數。第一個數字表示條紋區域的面積，第二個數字表示點狀區域的總面積，第三個數字表示其餘區域的面積。

## 範例輸入 #1

```
0.1
0.2
0.3
```

## 範例輸出 #1

```
0.003 0.005 0.002
0.013 0.020 0.007
0.028 0.046 0.016
```
