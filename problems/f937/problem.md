# f937. 循環節

## 內容

給定 2 個整數 n, m 請計算 n/m 為循環小數時,
不循環長度 a, 及循環長度 b
1/22 = 0.0(45)
藍色為不循環, 紅色為循環節。
0 < n < m < 10000000

## 輸入說明

每組測資的第一列有一個整數 T , 表示接著有 T 列測資。


每一列測資有 2 個整數 n, m


請計算 n/m 循環小數的不循環位數長度 a, 及循環長度 b

## 輸出說明

印出 a b

## 範例輸入 #1

```
2
2 61
1 22
```

## 範例輸出 #1

```
0 60
1 2
```
