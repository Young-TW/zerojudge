# i762. 大排序

## 內容

給你 $n$ 個 $32$ 位元的無號整數，請你將它們由小到大排序。


不過因為 $n$ 可能會很大，所以會給你一個生成輸入和輸出的函式。


以下為函式的連結 (只有提供 C++)。

## 輸入說明

第一行會輸入兩個整數 $n, \text{seed}$。


- $1\leq n\leq 5\times 10^7$
- $0\leq \text{seed}<2^{32}$

## 輸出說明

請參考連結提供的作法生成陣列後，對整個陣列進行排序，再呼叫連結提供的輸出函式。

## 範例輸入 #1

```
1 0
```

## 範例輸出 #1

```
23333332
```

## 範例輸入 #2

```
48763 48763
```

## 範例輸出 #2

```
1184188198
```

## 範例輸入 #3

```
50000000 12345678
```

## 範例輸出 #3

```
2140514395
```
