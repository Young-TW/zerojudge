# k986. 12172 - Matchsticks

## 內容

火柴棒是用來表達數字不錯的工具。一種常見表達數字的方式如下:



![image](ShowImage?id=3353)


如同你在一般電子錶上看到的數字。

給你一些火柴棒，你能排出一些數字來。本題要問你的是: 使用所有的火柴棒，你能排出最小和最大的正整數是多少?

## 輸入說明

輸入的第一列有一個整數T，代表以下有幾筆測資。


接下來有T列，每列有一個整數 n (2 <= n <= 100)，代表你有的火柴棒數量。

## 輸出說明

對每組測資輸出一列。含有2個整數(以一空格分開)，代表你能排出的最小及最大的數字。


這2個數都是正整數，並且不會有前導0 (例如: 01這樣的數字)。

## 範例輸入 #1

```
4
3
6
7
15
```

## 範例輸出 #1

```
7 7
6 111
8 711
108 7111111
```
