# e524. 106 彰雲嘉區複賽 - Q4 變位字判斷

## 內容

四、變位字判斷 ( 15分 )


   


    丹布朗的「達文西密碼」一書中提到以下的變位字謎：


O, Draconian devil!   Oh, lame saint!


其隱含的變位字為 Leonardo da Vinci!   The Mona Lisa!


變位字是將一段文字裡面的相同字母重新排列成另一段意義不同的文字，標點符號如逗號、句號、連字號、空格、驚嘆號、問號等可以不相等，而且變位字只要字母(包括A~Z、a~z)相同即可，大小寫也可以不同。舉例來說，“Evil eye.”和“Eye-Live! ”兩者是變位字，但“Abba”和“Aba”兩者則不是變位字，因為第二段文字少了一個b。


    寫一個程式可以檢查兩行文字是否為變位字。

## 輸入說明

輸入資料中第一列為一整數n，代表接下來有n組測試資料。


每組測試資料有兩行文字，每行文字的長度不超過50個字元。

## 輸出說明

檢查這兩行文字是否為變位字，若是變位字則輸出1，若不是變位字則輸出0。

## 範例輸入 #1

```
2
Abcba-Daba Kbbcaa!
Caba. Bad, Bakaacbb.
Abc, Ddeff-Gh!
Bdd. Cae. Fgh.
```

## 範例輸出 #1

```
1
0
```
