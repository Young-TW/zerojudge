# b883. 崩潰 の OωO學長！

## 內容

西元２０１６年０９月２６日（上午），雖說"OωO學長"在假日睡醒的時間很不穩定，但是在上課日卻都是固定時間起床的，明明在上課日起床時間的誤差大約 ±3 分鐘，假日的誤差卻可能到 ±2 小時，這媲美Ｘ檔案的神秘現象，"OωO學長"卻不以為意。在"OωO學長"準備要出門時，"OωO學長"的家人正好在客廳看電視，"OωO學長"隱約聽到電視新聞氣象播報說道：「颱風沒機會影響北部地區... ...」，聽到這消息的"OωO學長"心中產生半憂半喜的複雜心情。半憂的部分當然就是為了明天是否會放颱風假而擔憂，畢竟現在外頭天氣忽晴忽陰絲毫沒有即將狂風暴雨的跡象，不過現在時間還早，或許下午有翻盤的機會也不一定？那麼這樣子"OωO學長"會產生半喜部分就講不通了，到底為什麼"OωO學長"會產生半喜的心情呢？


 


　　西元２０１６年０９月２６日（中午），據說 板橋高中 似乎是有這麼一個規定：「學生使用班級電子講桌禁止用於學術用途之外。」，不過以「練習英文文法」和「增進資訊安全知識」為由的"OωO學長"和另一名"Y學長"正使用電子講桌玩一款名為「Hacker Experience」的駭客遊戲，雖然這遊戲是全英文的遊戲，不過經過假日的歷練之後，"OωO學長"和"Y學長"總算是知道遊戲大致上該如何進行，不過"OωO學長"和"Y學長"在談論的事情卻與這款遊戲一點關係也沒有，"OωO學長"和"Y學長"主要都是在討論明天會不會放颱風假的議題。"Y學長"很好奇地問"OωO學長"為什麼要這麼在意颱風的事情？原來"OωO學長"是為了明天的「板橋高中 AP 課程」而煩惱，因為"OωO學長"很榮幸地在下週可以幫各位學弟妹們出題，而這週「板橋高中 AP 課程」的進度是到 if 指令的運用，所以理當然地出題時的範圍可以包含 if 指令，但是如果明天的「板橋高中 AP 課程」因颱風假而停課，那麼學弟妹們就不會學到 if 指令，下週的考試也就不能出要用到 if 指令的題目，這樣子對於已經大致想好要出什麼題目的"OωO學長"來說可不是什麼好消息，因為"OωO學長"準備出的題目就必須使用的 if 指令，這也就是早上"OωO學長"聽到「颱風沒機會影響北部地區... ...」會半喜的原因，因為如期上課的話就不用重新設計題目了。只不過"snail老師"剛剛在「板橋高中資訊社」的ＦＢ社團裡貼文（非「板橋高中資訊社」的各位大大應該是看不到），大致上是講說如果明天受颱風影響導致停班停課的話那後天相同進度的班也跟著停止上課一次。"OωO學長"抱持著希望發了留言問說：「這樣我幫忙出題不就不能有 if 了!?  OAO」，結果只換來"snail老師"的一句話：「是的。」，於是"OωO學長"就崩潰了，現在只能祈禱明天不要放颱風假了... ...


 


　　西元２０１６年０９月２６日（傍晚），"OωO學長"與一名"Bo學長"一起吃晚餐，雖然兩人待會的目的地不同，但還是一起吃晚餐，怎麼這樣說來感覺好邊緣啊，不過那不是重點，「嗡...」，這並不是小動物或昆蟲翅膀震動的聲音，是"OωO學長"手機震動所發出的聲音，由於"OωO學長"對於這次手機震動並沒有任何想法來得知手機會震動的理由，所以就拿出手機查看，沒想到竟然是補習班發的簡訊!?大致內容是說受颱風影響本週三數學課停課一次。"OωO學長"拿著手機給"Bo學長"看並說補習班準備神預言星期三會停班停課，"Bo學長"不以為意地說那只是補習班自己也想放假而以吧。這時"OωO學長"藉由補習班的簡訊忽然發覺一件事情，"OωO學長"先是對"Bo學長"說"snail老師"在ＦＢ社團貼文的事情，然後開始講貼文之中存在很大的 bug ，那就是如果星期二沒放颱風假的話，明天的「板橋高中 AP 課程」就會如期上課，那假設星期三放颱風假怎麼辦？沒想到"Bo學長"竟然回說明天一定會放假的啦～於是"OωO學長"與"Bo學長"之間有關颱風的話題就此被終止了。


 


　　西元２０１６年０９月２６日（晚上），今天是本學期「板橋高中 進階程式設計課程」的開課日，今天的主題是檢討總計 4 場的「板橋高中 １０５資訊能力競賽名賽」之中的所有題目，其中因為"OωO學長"的實力實在是太強了，所以負責講當中的 2 場考試，總共合計 8 題，這對於懶惰無極限的"OωO學長"來說是個極其困難完成的任務，不過"OωO學長"還是順利地講解完畢。由於"OωO學長"中午在"snail老師"的貼文下留言，所以在場就有人問"OωO學長"如果明天真的放颱風假"OωO學長"會出什麼題目？"OωO學長"學長表示說雖說自己是有準備備用題啦，但是那是個幾何數學題，"OωO學長"怕出了之後參加「板橋高中 AP 課程」的各位學弟妹們完全不會解，還因此大受打擊從此對程式設計失去信心。那個題目大致如下：如下圖，有一 ∆ABC ，其中 AB線段長 = b ， AC線段長 = c ， M 為 BC線段 之中點， 且 AM線段長 = a ， P、Q 分別是在 AB線段 及 AC線段 上之兩點，且 AP線段長 = p ， AQ線段長 = q ，現以 A、P、Q 三點作一圓，其圓交 AM射線 於 R，求 AR線段長 = ？

　　西元２０１６年０９月２６日（晚上），在場的各位學長們都嫌"OωO學長"出得太難了，不過在場有一名「板橋高中 １０５資訊能力競賽名賽」排行第 3 的"baluteshih學弟"提供了以下解法：先用 中線定理 求出 BC線段長 ，再用 餘弦定理 求出 cos(∠BAC) 、 cos(∠BAM) 、 cos(∠CAM) ，先假設 AR線段長 = x ，又再用 餘弦定理 求出 PQ線段長 ，以及 PR線段長 與 x 的比例關係， QR線段長 亦同，最後透過 托勒密定理 建立等式求出 x 。看到這樣的解法"OωO學長"也驚呆了，因為他實在沒想過這可以搞得這麼複雜，於是"OωO學長"就對"baluteshih學弟"表示說這既然是出給學弟妹們寫得題目當然不會這麼狠啊，"OωO學長"也表示反正這也是沒題目出才用的備用題嘛，而且也是明天要先放颱風假才會有可能拿出來當題目的。說那時遲那時快，不偏不倚地就是在這個時候，北北基宣布明天停止上班上課！於是"OωO學長"真的崩潰了... ...


 


　　西元２０１６年０９月２７日（上午），"OωO學長"一直百思不得其解，為什麼昨天的新聞明明就說：「颱風沒機會影響北部地區... ...」，那為什麼北北基還是停班停課了呢？事後"OωO學長"在網路上找到了答案，"OωO學長"看著網路上備份的新聞說道：「颱風梅姬會影響北部地區... ...」，真相終於大白了，原來一切都只"OωO學長"在跟自己搞烏龍而已，不過該逃得也逃不過，"OωO學長"還是必須重新設計題目了。因為"OωO學長"實在是太懶惰了，所以原本就沒打算要在今天重新設計題目，於是在什麼事都沒有做的情況下今天就這麼過了，不過今天還是驗證了一件事情，那就是補習班的神預測出錯了，放颱風假的是今天而不是明天。


 


　　西元２０１６年０９月２８日（上午），今天完美驗證了補習班的神預測，因為今天北北基又停班停課，對於窗外無風無雨的景色"OωO學長"表示非常不能諒解，於是"OωO學長"決定把心頭中難以宣洩的怒火發洩到即將重新設計的題目上，於是以含有大量廢話作為特色的「OωO學長系列題」誕生了，這也造就了學弟妹們現在的煩惱。

## 輸入說明

輸入只有一行，每一行包含 5 個正整數 a、b、c、p、q ，分別代表題目中所述的幾何圖形之對應邊長，其中 a、b、c、p、q < 105，並且測資保證輸入不會有不合理的幾何圖形。

## 輸出說明

請輸出一個數字，代表題目敘述的幾何圖形中 AR線段 之線段長，測資保證答案為 < 231 的正整數。

## 範例輸入 #1

```
5 8 6 4 3
```

## 範例輸出 #1

```
5
```
