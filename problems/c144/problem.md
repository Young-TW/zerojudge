# c144. 105北二2加法填空題

## 內容

![image](ShowImage?id=968)


為了方便評測，請輸出個位數由上而下三個數字最小的一組，若仍有多組可能，再選十位數的三個數字最小的，依序往左直到唯一的一組。

## 輸入說明

輸入資料共有3列資料，每列都有n個(1<=n<=9)阿拉伯數字或「?」。如果是「?」，則表示是要填空的位置。

## 輸出說明

輸出一組正解。 


例如：
輸入為 5?4 + 09? = ?0?
填空後可以是  504 + 096 = 600 ，也可以是  514 + 090 = 604
而我們規定選第2組，因為個位數分別為 460及404所以第2組較小。

## 範例輸入 #1

```
範例一
?72
3?8
47?
範例二
4??975
398?9?
?01374
範例三：
1??
1??
3??
```

```
範例一
072

範例二

範例三
```
