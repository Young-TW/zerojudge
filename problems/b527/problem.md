# b527. 板條編年史 (一)：紅圓茵可與大魔導 nanj 的出現

## 內容

茵可，作為板條高中的一個遊戲設計師，平常最喜愛的活動就是－－寫遊戲。雖然時間日復一日的過去，但做為一個遊戲設計師總是會找到有趣的樂子，例如：科學實驗證明，將手指戳入肛門按摩直腸壁，可以刺激神經有效停止打嗝；產生 O(1) Lucky Sort 演算法，讓七十二億人都驚呆了！諸如此類，但茵可還是覺得很無聊，期待著那一天可以發生一些神奇事件。


板條賈姆曆 609 年的某一天，茵可前往板條高中的路上，忽然靈機一動，想要製作一款嶄新的遊戲－－柏油的路 (Path of Asphalt，簡稱 PoA)，在 PoA 中有一條柏油路，道路的盡頭是一個村莊，稱為可茵城。村莊中央有一個城堡，城堡內住著一位大魔法師－－紅圓茵可。而 PoA 的遊戲是控制一名角色，剛誕生的主角必須穿過這條柏油路才能到可茵城中。


想好構想之後，茵可便開始動工這個遊戲，儘管還沒有人可以幫忙他測試遊戲，但起碼可以先寫出來就夠了。


於是茵可開始把可茵城的架構 coding 出來，才剛把可茵城的外牆架設完後，忽然風聲大作、雷聲隆隆，茵可房間內刮起狂風，將桌上的遊戲攻略、Introduction to Algorithms、金蛇圍巾吹得七零八落，眼前忽然現出一片耀眼的寶藍色，接著茵可就失去了意識 ……


「嗚 …… 這裡是哪裡？」茵可緩緩睜開眼睛，頭痛欲裂。眼前的景物到底是哪裡？只見到起床的地方是一片乳白色沙灘，天空飄舞著彩色的氣球，遠處山上有著鮮紅的外牆環繞，而在山頂上座落著一座憂鬱藍的哥德式堡壘。眼前一條柏油路就一直延伸到堡壘處，道路兩側種滿了蓊鬱的樹林，離最近的一棵樹掛著一個牌子，上面寫著「柏林大道」。


熟悉的景色讓茵可直打冷顫，道：「這不是我寫的遊戲嗎？」


「不是，這裡是板擦界。」茵可背後冷不防出現一道聲音，一驚！轉頭過去卻空無一物。


「在這裡。」聲音從上面傳來，茵可馬上抬頭，卻發現一道黑影籠罩著視線，然後就被一個人壓住了。那個人是茵可原本要加的一個 NPC 角色－－大魔導 nanj，是紅圓茵可的師父，但是茵可根本還沒把那一段程式碼寫進去。


「為什麼 ...... 我明明還沒加這個角色啊，為什麼你會出現？」茵可疑惑的問，但大魔導 nanj 無視這個問題：「今天起你就是我的徒弟啦！」


「蝦？」茵可傻眼。


「因為我叫『nanj 紅圓』，那麼按照以往慣例，你的名字就叫做『紅圓茵可』吧！」nanj 一臉開心的說。


故事在此就要開始「茵可＝紅圓茵可」的見習魔法師之路！板條高中與可茵城之謎就要逐漸展開。因為大魔導 nanj 是爆搜大神，在這裡大家將會迎來第一個見習魔法師的試煉，nanj 降下了數個大小不一的門，每個門的大小可以表示為一個正整數 N，見習魔法師學到第一個魔法就是把兩個門互相合成，變成一個比較大的門且大小為前兩個門的和，而第一個試練就是要把這些門最終變成兩個一樣大的門。


但為了避免門會變得大到佔去整座可茵城的通路，只要門的大小超過 10007，門的大小就會自動變小 10007，直到不大於 10007。

## 輸入說明

輸入只有一筆測試資料，測試資料第一行有一個整數 M，代表門的個數，接著第二行有 M 個數字 Ni，代表門的大小。


1 <= M <= 20


1 <= i <= M


1 <= Ni <= 10007


有不超過 50% 的測資，M <= 10


有不超過 80% 的測資，M <= 15


有不超過 100% 的測資，M <= 20

## 輸出說明

對於每筆測試資料，根據題目敘述，如果最後兩個門一樣大，則輸出「YES」（不含引號）；否則，輸出「NO」（不含引號）。

## 範例輸入 #1

```
3
1 3 4
```

## 範例輸出 #1

```
YES
```
