# j234. 補考日的愛校服務

## 內容

全校補考日這天，衛生組長需要有人協助愛校服務。為了減少同學們的舟車往返，他決定要請第一節有參加補考的同學留下來幫忙，但是這些同學中難免有人第二節也要補考，就沒有辦法參與了。因此，組長只能請必須參加第一節補考，但是不用參加第二節補考的人來幫忙。


給你第一節及第二節的補考名單，請問有幾個人可以留下來幫忙？

## 輸入說明

輸入有三行。第一行含有兩個整數 𝑛, 𝑚 (1 ≤ 𝑛, 𝑚 ≤ 50)，分別代表第一節及第二節的補考人數。第二行含有 𝑛 個名字，是第一節補考的名單，第三行含有 𝑚 個名字，則是第二節補考的名單。

## 輸出說明

輸出一個整數，代表有多少人必須參加第一節的補考，但是不用參加第二節的補考。

## 範例輸入 #1

```
4 5
John David Mary Jacob
Simon David John Isaac Benjamin
```

## 範例輸出 #1

```
2
```

## 範例輸入 #2

```
3 8
Rat Ox Tiger
Rat Ox Tiger Rabbit Dragon Snake Horse Goat
```

## 範例輸出 #2

```
0
```
