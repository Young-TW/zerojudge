# o182. 最大連續子序列和

## 內容

給定 n 個正整數，請輸出這 n 個正整數中總和最大且不超過 k 的連續子序列和。 


連續子序列和 : [ ai +a(i+1) +...... +aj ] ( i<=j )

## 輸入說明

第一行有兩個正整數 n , k  ( 1 <= n <= 100 ， 1 <= k <= 100000 )


第二行有 n 個正整數  a1 ~ an  (  1 <= ai <= k ， 1 <= i <= n )

## 輸出說明

輸出一個正整數，表示總和最大且不超過 k 的連續子序列和

## 範例輸入 #1

```
3 8
3 4 5
```

## 範例輸出 #1

```
7
```

## 範例輸入 #2

```
7 20
8 5 7 6 4 2 3
```

## 範例輸出 #2

```
20
```
