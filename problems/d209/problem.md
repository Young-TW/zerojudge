# d209. 古代神祕文字

## 內容

有一位姜姓農夫偶然在田裡挖到了一塊石碑，上頭刻著詭異的文字。

經過農夫姜的研究，發現這張石碑是失落的羅力帝國的神祕記載，小時候就對傳說中的羅力帝國懷抱著憧憬的農夫姜決定放下鋤頭，窮盡一生努力來推導羅力的奧祕。

經過多年的研究，農夫姜發現這種文字是一種複雜的拼音文字，字母一共有3000種，且拼出來的單字符合一種神奇的規律，即：每個字母都有自己的編號，所有單字中，字母編號皆為順序排列(嚴格遞增)，且單字最長1000個單字。於是農夫姜想為這種文字編寫一個字典，即依長度遞增及字典遞增順序排列每個單字，依其順序給予一個編號，。

請你寫一個程式，讀入一個羅力文單字，求出其編號，由於答案會很大，只要輸出末七位即可。

## 輸入說明

有多筆測資，請以EOF判斷結束。

每筆測資第一行有一個整數n(1≤n≤1000)表示這個單字的長度，

下一行有n個以空白分隔的整數ki(0≤ki<3000)，為這個單字的組成字母。

## 輸出說明

對每筆測資輸出該單字編號的末七位。

## 範例輸入 #1

```
1
0
1
2999
2
0 1
2
19 56
3
12 34 56
```

## 範例輸出 #1

```
0000001
0003000
0003001
0059847
8312382
```
