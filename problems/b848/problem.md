# b848. 一窩螞蟻

## 內容

有一窩神奇的螞蟻，裡面有n隻公螞蟻和m隻母螞蟻，每天，每隻公螞蟻"或"母螞蟻會生出2隻公螞蟻(每天只有性別數較多的螞蟻會生公螞蟻)，公螞蟻會變成母螞蟻，母螞蟻會變成公螞蟻，求K天後分別有幾隻公螞蟻和母螞蟻?

## 輸入說明

第一行輸入一個數字T，代表之後有T筆測資，每一筆測資中分別輸入N, M, K三個數字，每個數字會以一個空格隔開。


20%測資滿足 T ≤ 10, N ≤ 10, M ≤ 10, K ≤ 10


40%測資滿足 T ≤ 10, N ≤ 100, M ≤ 100, K ≤ 100


60%測資滿足 T ≤ 10, N ≤ 10000, M ≤ 10000, K ≤ 10000


80%測資滿足 T ≤ 10, N ≤ 1000000, M ≤ 1000000, K ≤ 1000000


100%測資滿足 T ≤ 100, N ≤ 2147483647, M ≤ 2147483647, K ≤ 2147483647

## 輸出說明

針對每一筆測資輸出相應的公螞蟻數量和母螞蟻數量(因為數字會很大，所以只要輸出MOD1000007的值即可)，兩個數字以一個空白隔開，每一筆測資都要輸出一個換行。

## 範例輸入 #1

```
1
5 10 3
```

## 範例輸出 #1

```
160 65
```
