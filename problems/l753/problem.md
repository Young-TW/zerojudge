# l753. 數學運算式(3)

## 內容

續上兩題，數學運算式(1)、數學運算式(2)，這次小明遇上了更加進階的問題，請你編寫一段程序，協助他解決問題。

## 輸入說明

第一行為一個整數n，n是由很多整數連起來組成的，不過沒有空格。


第二行為一個整數t，代表有多少個s。


第二行為多個符號s，為"+" 或"-"  或 "*" 或 "/"，分別代表加減乘除。以空格隔開，順序為位置順序(由左邊起)。


第三行為一個整數a，代表將t個s連接在n的不同位置所成立的算式的計算結果。

## 輸出說明

輸出將t個s連接在n的不同位置所成立的算式，詳見範例輸出，請謹記有需要時加括號。

## 範例輸入 #1

```
55456
2
+ *
325
```

## 範例輸出 #1

```
55+45*6=325
```

## 範例輸入 #2

```
50503
2
+ *
300
```

## 範例輸出 #2

```
(50+50)*3=300
```
