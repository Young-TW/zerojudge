# e809. 1.字母排序 (Letters)

## 內容

TOI練習賽_2019/11_a1.潛力組1.字母排序 (Letters) {試題連結}


問題敘述


小蕭的老師出了一道題目，要求他對一個字串進行字母排序，並且抽問他某 些位置上的字母為何。因為按照 A, B, C, …, Y, Z 的順序對小蕭來說太容易了， 所以老師還特別自定了字母的順序規則。小蕭需要你的幫助，撰寫一個程式來快 速回答老師的抽問。

## 輸入說明

第一行為老師自定的字母排序規則，包含 N (0 < N<=26) 個不重複的大寫英 文字母。第二行為待排序的字串，包含 S (1<=S<=5×10^6 ) 個大寫英文字母。第三 行有一個正整數 Q，接著 Q 行為抽問的字母位置 X（1<=X<= S）。

## 輸出說明

對每筆資料請輸出 Q 行，每行為該位置上之大寫字母。

## 範例輸入 #1

```
CBA
ABCABCABC
4
2
1
5
8
```

## 範例輸出 #1

```
C
C
B
A
```

## 範例輸入 #2

```
ZAD
ZZZZZAAAADDD
3
1
6
12
```

## 範例輸出 #2

```
Z
A
D
```
