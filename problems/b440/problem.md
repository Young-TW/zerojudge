# b440. 【我愛Möbius】之互質對

## 內容

給定$n$和$m$，請你統計有序對$(a,b)$的個數，其中$1\leq a\leq n,1\leq b\leq m$且$a$與$b$互質。

$a$與$b$互質的定義是：$a$與$b$的最大公約數等於$1$。

## 輸入說明

第一行是一個正整數$T(1\leq T\leq 10^4)$，代表測資筆數。

接下來$T$行，每行兩個正整數$n(1\leq n\leq 10^7)$和$m(1\leq m\leq 10^7)$。

## 輸出說明

對於每筆測資，輸出互質對的個數。

## 範例輸入 #1

```
2
3 4
10000000 10000000
```

## 範例輸出 #1

```
9
60792712854483
```
