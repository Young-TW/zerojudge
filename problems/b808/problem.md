# b808. 12005 - Find Solution

## 內容

看以下的方程式：

![image](ShowImage?id=976)

現在給你 c ，請算出 (a, b) 有多少種可能的解。 a,b 都必須是正整數。

## 輸入說明

輸入含有多筆測資，每筆測資一列，含有一個整數 c ( 0 < c <= 1014 )。 當 c=0 時表示輸入結束。

## 輸出說明

對每組測資先輸出 c ,空一格後再輸出(a, b) 有多少種可能的解。以第一筆範例測資為例: (a,b)共有8個可能的解: (1, 2039), (2, 680), (5, 227), (14, 76), (76, 14), (227 5), (680, 2)  (2039, 1)

## 範例輸入 #1

```
1020
400
0
```

## 範例輸出 #1

```
1020 8
400 2
```
