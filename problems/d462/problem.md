# d462. 幸运的朋友(求幂系列题6)

## 內容

“我们分手吧！...”

    老Z坚决不肯。

    “...除非你能告诉我a的n次幂是多少，我给你一小时时间思考。”

    老Z大脑爆了。五天后，

    “锦瑟无端五十弦，一弦一柱思华年。

    庄生晓梦迷蝴蝶，望帝春心托杜鹃。

    沧海月明珠有泪，蓝田日暖玉生烟。

    此情可待成追忆，只是当时已惘然。”

如果它发生在你身上，你怎么解决呢？

 

    ——幸运的是，你的朋友会编程帮你。

## 輸入說明

a n i k（1<a<100,1<=n,i,k<=10000)

## 輸出說明

a的n次幂从第i位起连续k位,不够则输完。

## 範例輸入 #1

```
2 64 6 8
```

## 範例輸出 #1

```
74407370
```
