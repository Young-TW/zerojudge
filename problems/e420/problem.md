# e420. 灰姑娘的新衣

## 內容

灰姑娘是一個可憐的小女孩，常常被繼母和姐姐欺負


有一天，皇宮邀請全城的的人去皇宮跳舞，繼母和姐姐都去了，但灰姑娘要留在家裡做家事


灰姑娘正準備要曬衣服，結果衣服不小心掉到地上，這時一個男人走出來，他說：「你好，我叫 Caido，請問你掉的是金衣服還是銀衣服呢?」


灰姑娘覺得很奇怪，家裡怎麼會有一個男人


「請...請問你是神仙教母嗎?」


「不，我是 Caido，請問你掉的是金衣服還是銀衣服?」


「呃，我掉的是普通的衣服」


「你很誠實，三件衣服都送你」


灰姑娘原本想報警，但看到眼前的男人這麼大方，就取消這個念頭了


Caido 總共送他很多種類的衣服，例如上衣、褲子...，每種衣服也有很多顏色


現在給你每種種類衣服的顏色，想要問你幾個問題

## 輸入說明

每行為一種顏色，
當輸入為 "@" 時，代表換下一種衣服
當輸入為 "#" 時，代表輸入結束

## 輸出說明

輸出共有幾種搭配方法，顏色相同視為同一種，答案在 int 範圍內
再輸出什麼顏色的衣服最多，若有多種則全部輸出(按字串大小由小到大，以逗點隔開)
再輸出「完美搭配」的衣服(至少有一組)，完美搭配是指每種衣服顏色都一樣，若有多種則全部輸出(按字串大小由小到大，以逗點隔開)

## 範例輸入 #1

```
red
blue
yellow
yellow
@
light blue
white
yellow
blue
gold
@
brown
blue
brown
brown
#
```

## 範例輸出 #1

```
30
blue,brown,yellow
blue
```
