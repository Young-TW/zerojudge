# d786. 三、平均值

## 內容

已知一n個元素的整數數列，計算出這組數列的平均值，四捨五入至小數第2位。

## 輸入說明

第一行的數字，代表有幾組測試資料，第二行開始的每一行即為一筆測試資料。
每一筆測試資料以空格分開數字：
第一個整數為數列長度n，其範圍為[1,100]的整數；
接下來的n個整數，其範圍為[-10000, 10000]的整數。

## 輸出說明

對每一筆測試資料，以一行輸出平均數，並四捨五入至小數第2位。

## 範例輸入 #1

```
2
5 2 4 6 8 10
3 52 30 61
```

## 範例輸出 #1

```
6.00
47.67
```
