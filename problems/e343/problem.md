# e343. BMI 計算

## 內容

「身體質量指數」這個概念，是由19世紀中期的比利時統計學家及數學家凱特勒（Lambert Adolphe Jacques Quetelet）最先提出。它的定義如下：



![image](ShowImage?id=1340)
 𝑤 = 體重，單位：公斤; ℎ = 身高，單位：公尺; BMI = 身高體重指數，單位：公斤/平方公尺

## 輸入說明

輸入第一行含有一個整數 𝑤，代表體重，單位公斤。第二行含有一個浮點數 ℎ，代表身高，單位公尺。

## 輸出說明

輸出 BMI 值，四捨五入到小數點以下一位數。

## 範例輸入 #1

```
72
1.78
```

## 範例輸出 #1

```
22.7
```

## 範例輸入 #2

```
55
1.58
```

## 範例輸出 #2

```
22.0
```
