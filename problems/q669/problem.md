# q669. pA. 聽說今年人很少

## 內容

聽說今年報名的人很少(╥﹏╥)
身為友善出題者決定出個友善題，題目是這樣的：


依據答題成績我們將排列名次。
其中比較特別的是同分的狀況，當同分時之後名次將從累計名次開始算起。
例如有兩個第一名，則之後名次從第三名開始算起。


最後我們將選出 5 個優勝和 5 個(或更多)佳作。
因為優勝只能剛好 5 人，如果遇到第 5 名同分，將會根據答題時間決定該第 5 名將為優勝或是佳作。
不過因為佳作可以增額，如果遇到第 10 名同分，則都可以獲頒佳作獎狀。


請撰寫一個程式，判斷你將屬於哪一類別：


Top 5Maybe Top 5Top 10Thank You必定優勝依答題時間不同
有機會為優勝或是佳作必定佳作下次再加油

## 輸入說明

第一行有一個正整數 N，代表總共有 N 個人
1 ≤ N ≤ 25


接著有 N 行，每行有一個非負整數 ai 代表成績
0 ≤ ai ≤ 100，其中 a0 是你的成績

## 輸出說明

顯示是否可能獲得優勝或佳作，
分為四種可能類別「Top 5」、「Maybe Top 5」、「Top 10」、「Thank You」
