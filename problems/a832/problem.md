# a832. 2、数列变换

## 內容

小X看到堆成山的数列作业十分头疼，希望聪明的你来帮帮他。考虑数列
a=[a1,a2,...,an]
    定义变换
f(a,k)=[a2,a3,...,ak,a1,ak+2,ak+3,...,a2k,ak+1...]
    也就是把a分段，每段k个（最后如果不足k个，全部分到新的一段里，见样例），然后将每段的第一个移动到该段的最后一个。现在，小X想知道如下表达式的结果。
f(f(f(...f([1,2,3,...,n],1),2),3)...n)

## 輸入說明

输入一行包含一个整数n。

## 輸出說明

输出一行包含n个整数，表示最终的数列。

## 範例輸入 #1

```
4
```

## 範例輸出 #1

```
4 2 3 1
```
