# c735. magicpower

## 內容

有兩個魔力強大的魔法師


法師A會等級1~N的魔法


其中每個等級的熟練度為a1, a2, ..., aN


法師B會等級1~M的魔法


其中每個等級的熟練度為b1, b2, ..., bM


有天他們決定一起施魔法


如果A施放了等級x的魔法，B施放了等級y的魔法


他們就會讓第x+y個魔法塔注入ax*by的能量


魔法塔的能量可以累加


並且他們可以施放任意多次魔法


不過同一個人只能施放同一等級的魔法一次


他們很好奇如果用最佳的策略施法


哪個魔法塔的能量會最高，最高的能量為多少

## 輸入說明

第1行有2個整數N,M


第2行有N個整數a1, a2, ..., aN


第3行有M個整數b1, b2, ..., bM


 


90% 測資中


N, M <= 1000


100% 測資中


N, M <= 200000


1 <= ai, bi <= 1000

## 輸出說明

第一行輸出兩個整數，用空格隔開


第一個數字為能量最高的魔法塔編號


如果有多個能量相同的塔 輸出編號較小的


第二個數字為能量最高的魔法塔能量

## 範例輸入 #1

```
範例輸入一
3 3
3 2 1
1 2 3

範例輸入二
5 5
1 2 3 4 5
5 4 3 2 1
```

## 範例輸出 #1

```
範例輸出一
4 14

範例輸出二
6 55
```
