# c226. 公關的大雞腿

## 內容

在某個學校某個社團中，有位出了名喜歡大雞腿的公關，當他沒吃到雞腿的時候就會很生氣。


而今天他實在是餓扁了，恨不得想要大吃一頓，現在他面前有編號為1 ~ N-1的N-1隻雞腿，編號為i的飽足度為i，而公關的肚子容量恰好為N


但吃雞腿前有一個規定，舉例來說:假設目前吃的雞腿的編號為k，那麼他下次只能吃k+1號雞腿。


給予一個N，輸出能讓公關的肚子不多不少剛好填滿的吃雞腿方法數

## 輸入說明

題目有多筆測資，每次給予正整數N，N <= 10 ^ 18


N = 0時代表輸入結束

## 輸出說明

輸出總共飽足度剛好等於N的方法數

## 範例輸入 #1

```
3
4
5
9
25 
0
```

## 範例輸出 #1

```
1
0
1
2
2
```
