# n773. 10424－Love Calculator

## 內容

有一天我問 Saima 她有多愛我。她的回答是「71.43%」。我既驚訝又震驚，無法理解為什麼她不說100%，而是給出這樣一個特定且奇怪的百分比。看到我驚訝、震驚和緊張的表情，她大笑起來，告訴我她愛我勝過世上任何東西，這只是個愚蠢有趣的愛情計算遊戲。接著她解釋了這個計算方法。在這個問題中，你需要寫一個程序，使任何人都能快速計算兩個人之間的愛情程度（當然這只是個愚蠢的遊戲）。


規則： 你會得到兩個名字。這兩個名字中可能包含空格或其他非字母字符，如 $ & % 等。但只有從 a 到 z 或 A 到 Z 的字母會參與愛情計算。每個字母都有一個特定的數值，這些數值從 1 到 26 按字母順序遞增。這樣，a = 1, b = 2, c = 3, ..., z = 26。大寫字母和小寫字母的值相同。然後將這些數字相加，直到結果為一位數為止。例如，考慮名字 ‘bcz’，這裡 b = 2, c = 3 和 z = 26。因此，總和是 (2+3+26) = 31 = (3+1) = 4。然後這兩個數字的比率以百分比形式表示，將是最終結果。


記住：結果不能超過 100%。計算比率時要小心避免這個問題。

## 輸入說明

輸入將包含兩個名字。每個名字最多不超過 25 個字符。文件結束符將指示輸入的結束。

## 輸出說明

對於每對名字，您的程序將計算這兩個人之間的愛情並給出結果。結果需要顯示到小數點後兩位。所有結果都必須在新行中顯示。

## 範例輸入 #1

```
saima
shanto
Pakistan
India
USA
USSR
```

## 範例輸出 #1

```
71.43 %
100.00 %
100.00 %
```
