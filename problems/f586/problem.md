# f586. 數字D×D

## 內容

在駒王學園有位高中生名叫兵藤一誠
現代的日本高中生基本上都有特殊能力(擁有超強領導能力的魯X修、身為戰略型魔術師的司XX也)
一誠當然也不例外
一誠的特殊能力「洋服崩壞」，這是一個可以拿來剝除蔬菜水果外皮的特殊能力
雖然這個能力看起來沒什麼用


直到某次數學課考試：把數字質因數分解
沒想到對數字使用洋服崩壞後，數字就自動質因數分解了，但考試規定要30分鐘才能交券
所以沒事做的一誠就把這些數字分類:
1->1
若無平方數因數->(-1)^k(k為質因數個數)
否則->0

## 輸入說明

第一個數字T代表有幾筆測資(T<=1000)
接下來T行每行有一個數字N(N<=1000000)

## 輸出說明

輸出N分類的結果為1、0還是-1

## 範例輸入 #1

```
5
1
2
3
4
39
```

## 範例輸出 #1

```
1
-1
-1
0
1
```
