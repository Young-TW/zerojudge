## 輸入說明

第一行數字n,m代表有n個提示和m個詢問地點，接下來n行每行有對數字(x,y)代表編號(由1到n)的提示位置，接下來m行每行有對數字(a,b)代表詢問地點位置(0<n,m,x,y<=1000)，請參考範例輸入

## 輸出說明

根據每個地點請輸出距離(Euclidean distance)最近的提示編號，如果有多個提示距離相同，請輸出編號最小的，請參考範例輸出

## 範例輸入 #1

```
2 1
1 1
3 3
3 2
```

## 範例輸出 #1

```
2
```
