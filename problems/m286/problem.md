# m286. 真分數與帶分數

## 內容

給你兩個正整數n,m(中間以空白間隔，且1<=n,m<=2147483647)，n是分子,m是分母，如果分數可以整除，請輸出整除後的整數結果，如果不能整除，格式請輸出成真分數或帶分數(本題為簡化題目，先不用化成最簡分數)。
舉例來說，若輸入的n=10 , m=5 ，則輸出「整數 2」，若輸入的n=5 , m=10，則輸出「真分數 5/10」，若輸入的n=6, m=4 則輸出「帶分數 1 2/4」 (1 2/4就是1又4分之2，請注意輸出項目之間的空白問題，多一空白或少一空白都不會AC)

## 輸入說明

輸入只有一行為兩個正整數n,m(中間以空白間隔，且1<=n,m<=2147483647)，n是分子，m是分母。

## 輸出說明

如果分數可以整除，請輸出整除後的整數結果，如果不能整除，格式請輸出成真分數或帶分數(本題為簡化題目，不用化成最簡分數)，請看範例輸出。

## 範例輸入 #1

```
10 5
```

## 範例輸出 #1

```
整數 2
```

## 範例輸入 #2

```
5 10
```

## 範例輸出 #2

```
真分數 5/10
```

## 範例輸入 #3

```
6 4
```

## 範例輸出 #3

```
帶分數 1 2/4
```
