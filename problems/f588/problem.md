# f588. 數字D×D 番外篇

## 內容

終於到可以交卷的時間
一誠交完考卷後，就收拾書包準備回家
這時老師說：先別那麼急嘛，給你一個回家作業
一誠：什麼回家作業？
老師：給你一個數字N，請問有多少數字小於N並且和N互質
一誠：阿不就歐拉函數？
老師：阿那個...剛剛是你聽錯了，我要問的是φ(1)+φ(2)+...+φ(N-1)+φ(N)為多少 φ()代表歐拉函數

## 輸入說明

第一個數字T代表有幾筆測資(T<=10)
接下來T行每行有一個數字N(N<2^31)
聽說和數字D×D 續的測資一樣

## 輸出說明

φ(1)+φ(2)+...+φ(N-1)+φ(N)為多少

## 範例輸入 #1

```
5
1
2
3
4
39
```

## 範例輸出 #1

```
1
2
4
6
474
```
