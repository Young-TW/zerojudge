# a679. Interval Overlapping

## 內容

Background 

無
The Problem計算一維空間區間重疊的 pair 個數。

重疊的定義：兩個區間有無限多的共同點

## 輸入說明

多筆測資。
每組第一行有一個整數 N，
接著有 N 行，每行有兩個整數 L, R 代表區間 [L, R]

(0 < N < 30,000, -32767 < L, R < 32767)

## 輸出說明

對於每組輸出一行 "Case testcase: " 後接一個整數代表重疊個數。
