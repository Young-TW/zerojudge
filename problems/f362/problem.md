# f362. 彼得的誤差率

## 內容

「欸，你想好了沒阿？」


「你覺得我們要怎麼點？」


「我不知道耶，挑便宜一點的點吧。」


「這每個都沒寫價位，那我們要怎麼點，我哪知道哪個便宜。」


「阿現在海產店都寫時價阿。」


「...沙魚450，軟絲850，花蟹750，小的450，再小一點350，大沙母850...海瓜子180，奶油貝250，生蠔180，木瓜螺350，小花龍1250，水姑娘1680......象蚌650。」


「你誰啊？」


「彼得，熟的人都叫我海龍王彼得，你們等下只要照著我講的數字點，絕對不會吃虧。」


------------------------------------------------------------------------------------


「欸，怎樣？」


「欸，真的有便宜到耶，我覺得大概有省500塊。」


「對阿！」


「這餐真的超厲害的。」


「彼得。」


「不用叫我彼得，熟人都叫我海龍王-彼得，你們叫我海龍王彼得就可以了。」


「謝謝。」


「是不是有便宜到？」


「真的，真的有差。」


「跟著我就不會吃虧。走，我帶你們去逛逛。」


------------------------------------------------------------------------------------


「欸，海龍王...彼得，你為什麼那麼懂時價？」


「我懂得不只是時價，我懂的是海。我這麼問你吧，你知道黑鮪魚一尾要賣多少錢嗎？」


「不知道阿。」


「黑鮪魚下個月必漲30%。」


「為什麼？」


「我有線人告訴我：下個月呢，菲律賓的船隊，將會在南海動作頻頻，到時候就會跟我們的漁場有所重疊，影響到我們漁民的作業。那個時候，黑鮪魚，必漲！」


「你到底是誰啊？」


「圈內人都這麼叫我：東太平洋-漁場時價分析師，海龍王-彼得。」


「所以你也是分析師喔。」


「你聽我這麼一講，就知道時價有多重要了吧。你一個月薪水有多少錢？」


「三萬二。」


「三萬二嘛，那你剩下的錢會怎麼處理？」


「就可能先存起來。」


「這是一種非常浪費的行為阿。錢呢，放在銀行，非常的浪費，錢應該要拿來投資，不是拿來存的。像我剛剛這麼一操作，一進一出，兩個禮拜，就賺超過你兩個月的薪水。你用勞力賺錢，是一種非常沒有效率的方法，錢-必須拿來滾錢，這才是真正的作法。」


「這麼厲害喔。」


「哀，只要你懂海，海就會幫助你。」


「這個，你真的會投資耶。」


「業界老一輩的人，都會叫我：東太平洋-漁場時價分析師-兼操盤手，海龍王-彼得！」


「可是我現在只是上班族阿，阿我這樣哪有時間研究海，我沒有時間研究，這我要怎麼投資？」


「這你只要跟對老師，老師呢，就會帶你一起獲利，一起往賺錢的方向前進。」


「那我要哪裡找老師？」


「阿呀，我剛剛都忘記跟你講，其實前面那些全都是我的小名。我真正的名字是---東太平洋-漁場時價分析師-兼操盤手-暨洋流講師，海龍王---彼得！」


「阿你是講師喔。」


「喔，北太平洋的洋流，我是略懂，那我專攻的項目呢，剛好就是黑潮。下個月呢，我跟我的學生，剛好要進場黑鮪魚的部分，也再找一些有緣人啦，看是不是要一起入場。」


「可是我真的沒錢耶，我可能就先不投資了。」


「欸，這樣就不對了，我剛剛跟你講的觀念不就是：你越存錢，就會越窮嗎？小額，趕快拿來滾雪球，小錢也能賺大錢。好啦，沒關係啦，這是我的名片啦，如果你想通了，就打給我，我先去忙囉。記住一件事：你只要懂海，海就會幫助你。」

為什麼彼得這麼懂投資呢，因為他懂得計算「誤差率」。


什麼是誤差率呢？計算必有誤差，海洋投資家們定義誤差率為：| ( x -n ) / x |，x為正確答案，n為算出來的答案。


你才剛入門，想要計算自己的誤差率，但你其實不知道，你在算誤差率的過程中已經被你的誤差率影響了，所以算出來的誤差率有誤差，到底要怎麼知道真正的誤差率呢？

## 輸入說明

每行給你一個二位小數k(|k|≤1000)


代表你自己算出來的誤差率

## 輸出說明

輸出你真正的誤差率，四捨五入到小數第二位


若沒有答案，輸出"NULL"


若有多組答案，由小到大輸出，四捨五入後的結果相同視為同一數，須合併

## 範例輸入 #1

```
0.00
1.00
0.25
```

## 範例輸出 #1

```
1.00
0.62
0.21 0.50
```
