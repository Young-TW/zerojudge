# d343. 10578 - The Game of 31

## 內容

31點是一款老千專門用來騙錢的遊戲,

一開始我們有一副24張牌,包含點數1~6的牌,每個點數的牌有4張

兩個玩家A,B從A開始先從牌堆裡抽一張牌(所有未被抽出的牌皆可)

抽出的牌點數從0開始累計,接下來就ABABABA的交錯抽直到抽出點數

最後累計一超過31的人就輸了,給你A,B玩家玩到一半時以抽出的牌

 請你計算出當兩個玩家都採用最佳策略時,最後誰會勝出

## 輸入說明

輸入包括幾行，每一行都包含一個序列表示遊戲玩到一般目前以抽出的牌。第一個數字是玩家Ａ所拿的，第二個數字是玩家Ｂ所拿的，等等。假設兩個玩家都使用完美的遊戲策略，你要判斷完成這個遊戲，確定誰一定會獲勝。

## 輸出說明

對每個遊戲，輸出一行包括輸入和誰必勝（’A’,’B’），中間要有一個空格。

## 範例輸入 #1

```
356656
35665
3566
111126666
552525
```

## 範例輸出 #1

```
356656 B
35665 B
3566 A
111126666 A
552525 A
```
