# o230. 11348 - Exhibition

## 內容

一些朋友有相同的興趣，他們都在收集郵票。有一次，他們決定舉辦一個展覽。展覽帶來了一些收入，現在他們不知道該如何分配這些收入。他們決定根據以下規則來分配收入：“第 i 位朋友將獲得的總收入百分比等於他擁有的獨特郵票種類的部分。”
當某種郵票只有一個人擁有時，這種郵票被稱為獨特郵票。

## 輸入說明

第一行包含一個整數 K (0 < K ≤ 100)，表示測試次數。每個測試案例由一個正整數 N (0 < N ≤ 50) 描述，表示朋友的數量。接下來是 N 行整數，每行對應一位朋友的郵票收藏。該行的第一個整數是 M — 一個人擁有的郵票數量 (0 < M ≤ 50)。接下來是 M 個整數 A (0 ≤ A ≤ 10000) — 郵票的種類。

## 輸出說明

對於每個測試案例，輸出格式如下：“Case i: a1% a2% a3% . . . an%”。其中 i 是測試案例編號，ai 是第 i 位朋友獲得的收入百分比。四捨五入到小數點後6位。

## 範例輸入 #1

```
1
3
3 1 2 3
2 4 5
3 4 2 6
```

## 範例輸出 #1

```
Case 1: 50.000000% 25.000000% 25.000000%
```
