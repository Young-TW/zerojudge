# n759. 10945－Mother bear

## 內容

不幸的是，對於我們懶惰的“英雄”們來說，堅果是由一隻名為戴夫的邪惡熊種下的，他們正落入了他的陷阱。戴夫不僅僅是一隻熊，他是一隻會說話的熊，但他只能理解回文的句子。當拉里茫然不知所措時，瑞安發現了這一點，但需要一個方法來確保他的句子是回文的。所以他拿出了他信賴的iPod，幸運的是裡面有你為此目的寫的程式……或者是嗎？



![image](ShowImage?id=4035)

## 輸入說明

你將會得到許多句子。你必須確定它們是否是回文，忽略大小寫和標點符號。每個句子只包含字母A-Z、a-z、'.'、','、'!'和'?'。輸入的結束將是一行包含單詞“DONE”的行，不應進行處理。

## 輸出說明

對於每個輸入，如果它是一個回文，輸出“You won't be eaten!”


如果它不是一個回文，輸出“Uh oh..”。

## 範例輸入 #1

```
Madam, Im adam!
Roma tibi subito motibus ibit amor.
Me so hungry!
Si nummi immunis
DONE
```

## 範例輸出 #1

```
You won't be eaten!
You won't be eaten!
Uh oh..
You won't be eaten!
```
