# d999. 清空倉庫遊戲

## 內容

OpenChan 與 Kitty 兩個人被指派到落日倉庫，負責將倉庫內所有的貨箱搬出清空，以方便新貨品的規劃與進駐。堆積如山的貨品讓工作變得疲累又無聊，為了尋求刺激，於是兩人想出了一個賭博的遊戲，輸的人要奉獻出全部的工資來請客。
遊戲規則如下： 
已知倉庫中貨箱的總數量為 P，OpenChan 與 Kitty 兩人事先約定有N 個數字(A1、A2、...、An，其中包含1)，代表每人每次允許從倉庫中搬走的貨品數量的數字，若由OpenChan先搬，兩個人輪流搬出貨品，最後剛好將倉庫貨品清空的人獲得勝利，假設兩人都使出了洪荒之力，請問最後誰會獲勝？

## 輸入說明

測資第一行一個數字 N (N<=20)，第二行有N個整數，代表A1、A2、....、An
第三行一個數字 K，代表以下有 K 筆測資，第四行以下有 K 行，每行一個整數，代表不同測資的 P 值 (P<=1000000)。

## 輸出說明

每筆測資一行輸出，如果OpenChan會贏，輸出"OpenChan"，如果Kitty 會贏則輸出 "Kitty"，如範例。

## 範例輸入 #1

```
3
1 3 5
3
10
11
12
```

## 範例輸出 #1

```
Kitty
OpenChan
Kitty
```
