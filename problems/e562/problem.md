# e562. 11401 - Triangle Counting

## 內容

給你n根長度為1、2 ...、n的棍子。
您必須選擇其中的任何3根棍子並且拼成一個三角形。
您可以製作多少個不同的三角形？
請注意，如果兩個三角形至少有一對長度不同的邊，則它們將被認為是不同的。



![image](ShowImage?id=4049)

## 輸入說明

每一行有一個正整數n (3 ≤ n ≤ 1000000)。n代表有n跟長度為1到n的棍子。
如果n = 0代表輸入結束。

## 輸出說明

對於每組輸入，輸出可以製作多少個不同的三角形。

## 範例輸入 #1

```
5
8
0
```

## 範例輸出 #1

```
3
22
```
