# b477. NOIP2014 Day2.3.解方程

## 內容

已知多项式方程：

\[
a_0 + a_1x + a_2x^2 + \cdots + a_nx^n = 0
\]

求这个方程在$[1,m]$内的整数解。

## 輸入說明

输入共n+2行。

第一行包含2个整数n、m，每两个整数之间用一个空格隔开。

接下来的n+1行每行包含一个整数，依次为$a_0,a_1,a_2,\dots,a_n$。

## 輸出說明

第一行输出方程在[1, m]内的整数解的个数。

接下来每行一个整数，按照从小到大的顺序依次输出方程在[1, m]内的一个整数解。

## 範例輸入 #1

```
2 10
1
-2
1

2 10
2 
-3
1

2 10
1
3
2
```

## 範例輸出 #1

```
1
1

2
1
2

0
```
