# g502. 109北二5.簡單的數學式計算任務

## 內容

有一天佛列德接受到一個整數的數學計算任務，其數學式子包括正整數以及加法(+)和乘法(*)運算子(Operators)。
由於此運算任務將來有可能常發生，佛列德必須要有一個程式讀寫運算式並利用電腦得出結果。

## 輸入說明

輸入僅有一行，代表需要讀入並計算的運算式，所有參與運算的正整數值最大可為 230。
運算子和數值間並沒有空格，輸入長度最多可達 1500 字元。

## 輸出說明

輸出只有一行，包含一個整數，表示這個運算式的值。
注意：當答案長度多於 5 位時，只輸出最後 5 位(前導 0 不輸出)。

## 範例輸入 #1

```
1000*90+2*4520+960
```

## 範例輸出 #1

```
0
```

## 範例輸入 #2

```
100*200+456+500*500
```

## 範例輸出 #2

```
70456
```
