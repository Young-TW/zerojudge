# e005. 王老先生有塊地

## 內容

王老先生有塊三角形的土地(ABC)，希望能放牧一些牛羊，王老先生要在土地的周圍佈上木樁並拉起鐵絲圍欄，
由於地質關係(原因不明)，木樁只能打在座標點上，請幫忙計算一下王老先生總共需要打上多少根的木樁？
以下圖為例： A(3,11)、B(3,1)、C(13,1)，共需要30根木樁。

![image](ShowImage?id=1266)

## 輸入說明

多筆測資，每筆測資一行，六個整數 A(x,y)、B(x,y)、C(x,y)，測資保證ABC一定可以構成三角形。( -10000000<= x,y,x <=10000000 )，最後 0 0 0 0 0 0 結束。

## 輸出說明

每筆測資輸出一行，木樁的總數。

## 範例輸入 #1

```
3 11 3 1 13 1
13 3 3 1 13 1
0 0 0 0 0 0
```

## 範例輸出 #1

```
30
14
```
