# e638. 10176 - Ocean Deep! Make it shallow!!

## 內容

Ocean deep
    I’m so afraid to show my feelings,
    I have sailed a million ceilings
    In my solitary room
Ocean deep


上面的幾行來自Cliff Richard的一首非常流行的歌曲。
有個人名字叫Rampell-Stilt-Skin，幾天前有人殺了他，而您是解決這個謎團的偵探。
這個傢伙的問題在於他總是將自己的訊息和感受隱藏在海底 (我的意思是遙不可及很難懂)。 
他寫了一些日記，其中包含一些訊息，和一個大二進制數 (可能有多達10000個數字)。 
如果該數字可被 質數 131071 整除，則該訊息為正確的，否則該訊息為錯誤的。 
在此問題中，你需要判斷大二進制數是否能被131071整除。

## 輸入說明

包含多組輸入，每組輸入包含一個二進制數。
輸入可能有多行，每行不超過100位數字，每組輸入以"#"結尾。

## 輸出說明

對於每組二進制數字
如果數字可被131071整除，輸出"YES"
否則，輸出"NO"

## 範例輸入 #1

```
0#
1010101#
```

## 範例輸出 #1

```
YES
NO
```
