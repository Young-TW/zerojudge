# m139. pD. 草莓園

## 內容

在猴猴共和國的成功街上，有個著名的草莓園，猴猴們都喜歡進去草莓園裡採草莓。採草莓的方式如下:草莓園內由前至後一共有$N$顆草莓，每顆草莓都有自己的甜度與新鮮度，第$i$顆的甜度為$a_i$，而甜度則是長在越後面的越新鮮，因此第$i$顆的新鮮度是$i$。採完草莓後需要自己將草莓一顆一顆的分裝進一些盒子裡，一個盒子最多只能裝下$K$顆草莓。然後計算每盒草莓的價格，一盒草莓的價格為盒裡最新鮮的草莓的新鮮度，這次草莓的花費就是每盒草莓價格的總和。


今天小東想要來採草莓，小東的預算總共是$D$塊錢，因為小東喜歡吃甜的草莓，所以他希望在符合預算的前提之下採到的草莓的甜度總和在越大越好。請幫忙小東計算在合理的採集與分裝，甜度總和最大是多少。

## 輸入說明

第一行包含三個正整數$N,K,D$，分別代表的意義題目所述。
第二行有$N$個正整數$a_i$，第$i$個數$a_i$代表第$i$顆草莓的新鮮度。


測資限制


輸入皆為正整數
$K \leq N \leq 500$
$a_i \leq 10^6,1 \leq i \leq N$
$D \leq 10^6$

## 輸出說明

輸出一行包含一整數，代表的意義題目如所述。

## 範例輸入 #1

```
7 2 8
2 1 5 4 6 7 9
```

## 範例輸出 #1

```
18
```

## 範例輸入 #2

```
7 1 9 
2 4 3 6 5 7 9
```

## 範例輸出 #2

```
13
```
