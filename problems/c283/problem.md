*****************************************


你會拿到N堆, 每堆有M張牌每次可以從每堆最後一張牌(有的話) 挑選2張牌是同花色or同點數


M張牌的輸入依序是最上層的牌到最後一張牌(底牌)


求全部拿完的機率

## 輸入說明

輸入2個整數N, M
2≤N≤8, 1≤M≤13
輸入N堆 每堆M張牌

第一個字代表數字
A,2~9,T,J,Q,K
第二個字代表花色
'S'代表黑桃
'C'代表梅花
'D'代表方塊
'H'代表愛心

## 輸出說明

輸出拿完的機率

## 範例輸入 #1

```
2 1
AS
2S
3 2
AS 2S 
AS AS
2D AS
0 0

```

## 範例輸出 #1

```
1.000000
0.166667
```
