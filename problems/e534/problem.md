# e534. 01587 - Box

## 內容

Ivan在一家生產重型機械的工廠工作。
他的工作很簡單：敲開了不同大小的木盒，交給機器包裝起來，以交付給客戶。
每個木盒都是長方體。 Ivan使用六個矩形木頭製成一個木盒。每個矩形木頭用於木盒的一側。



![image](ShowImage?id=4048)


Joe為Ivan運送包裝好的木盒。
Joe是一個經常犯錯誤的人，他常常帶來無法組裝的木盒。
因為Joe不相信Ivan，所以每次Ivan向Joe解釋他犯的錯，他都不相信自己帶錯木盒。
幸運的是，Joe崇拜與電腦相關的所有事物，並真誠地相信電腦永遠不會犯錯誤。
因此Ivan要求您寫一個程式，給定六個矩形木頭的大小，該程式可以告訴您是否可以用它們製成一個木盒。

## 輸入說明

輸入包含多組測資。
每組測資六行。每行包含兩個整數w和h (1≤ w、h ≤ 10000)，代表一個矩形木頭。
矩形木頭的寬度w和高度h以毫米(mm)為單位。

## 輸出說明

對於每組測資。
如果可以用六個給定的矩形木頭製作木盒
請輸出"POSSIBLE"
如果無法製作木盒
請輸出"IMPOSSIBLE"

## 範例輸入 #1

```
1345 2584
2584 683
2584 1345
683 1345
683 1345
2584 683
1234 4567
1234 4567
4567 4321
4322 4567
4321 1234
4321 1234
```

## 範例輸出 #1

```
POSSIBLE
IMPOSSIBLE
```
