# e304. a probability problem

## 內容

在一顆半徑為$r$的球面上隨機選四個點，考慮該四點為頂點之四面體，球心在四面體內部的機率$\alpha$為何?


請輸出$\lfloor \alpha \times 10000 \rfloor \pmod m$

## 輸入說明

輸入有$T$行，每行有一實數$r$, 及一正整數$m$，代表球的半徑以及模數。


$T = 100$


$0.0 \leq r \leq 92.3$


$10 \leq m \leq 9230$

## 輸出說明

輸出$\lfloor \alpha \times 10000 \rfloor \pmod m$

## 範例輸入 #1

```
1.0 30
```

## 範例輸出 #1

```
20
```
