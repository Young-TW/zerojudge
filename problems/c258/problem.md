# c258. kevin 愛畫圓

## 內容

kevin 在x軸上畫了一堆圓


告訴你每個圓中心的x座標跟半徑


請你告訴kevin有幾對圓是相離的


內離外離定義：https://zh.wikipedia.org/wiki/圆#兩圓位置關係

## 輸入說明

輸入 一個整數 n ( 1 <= n <= 5 * 10 ^ 5)


代表有 n 個圓


接著輸入 n 行


每行兩個數字 x, r (-10 ^ 9 <= x <= 10 ^ 9, 1 <=  r <= 10 ^ 9)


代表 圓的 x 座標跟半徑

## 輸出說明

輸出有幾對圓是相離的

## 範例輸入 #1

```
4
9 6
8 2
1 4
5 4
```

## 範例輸出 #1

```
2
```
