# n732. 11960 - Divisor Game

## 內容

史蒂夫正在玩一個數字遊戲。他隨機選擇一個正數 N，並找到不大於 N 的擁有最多約數的最大正數。隨著 N 變得越來越大，史蒂夫在計算約數時會越來越難以避免出錯，所以他請你編寫一個程式。


你認為僅僅找出約數是非常簡單的任務，並建議你可以解決史蒂夫的原始問題。

## 輸入說明

給你一個測試數量 T (T ≤ 50000)。每個測試在單行中指定一個數字 N (1 ≤ N ≤ 10^6)。

## 輸出說明

你需要找到不大於 N 且擁有最多除數的最大數字。對於每個測試，輸出一行包含該遊戲答案。

## 範例輸入 #1

```
3
1
10
37
```

## 範例輸出 #1

```
1
10
36
```
