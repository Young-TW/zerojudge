# r635. P1. 海島的連續寶藏

## 內容

海盜阿銀在一條長長的海岸線上挖到一排寶箱，每個寶箱有正負不同的金幣（可能負，代表債務）。阿銀想知道在不同目標值下，有多少段相連的寶箱使得總和恰好等於那個目標值 — 他要把這些段賣給收藏家。你的任務是回答多個目標查詢。

給定長度為 n 的整數序列 a[1..n]，以及 q 個目標 t_i。對每個目標 t，求序列中 和等於 t 的子區間數量（子區間索引為連續）。

## 輸入說明

n q
a1 a2 ... an
t1
t2
...
tq

- 
1 ≤ n ≤ 2000

- 
1 ≤ q ≤ 2000

- 
-10^9 ≤ ai ≤ 10^9

- 
-10^15 ≤ ti ≤ 10^15

## 輸出說明

對每個 t_i 輸出一行，為對應的子區間數量。

## 範例輸入 #1

```
5 3
1 2 -1 2 1
2
3
0
```

## 範例輸出 #1

```
4
3
0
```
