# a785. 7. It's All Greek To Me

## 內容

隨著人工智慧的增強，無論是口頭還是書寫，人類跟電腦的互動漸漸趨向自然(人類)化，這種科技仰賴能夠理解並生成自然語言句法( syntax )的演算法。

請寫一個程式生成希臘根字-後綴字對的縮寫，母音的縮寫依據下表，方便起見，希臘字母可用羅馬文字拼寫。

![image](ShowImage?id=500)

根字以a, e, o結尾( 對應到表左方 )，後綴字已a, e, i, y, o, ou, w開頭( 對應到表上方 )，縮寫的結果在兩母音的交界處，例如：a+o=w

## 輸入說明

每行輸入包含根字和後綴字，兩字用"-"分開，輸入END結束。

## 輸出說明

輸出縮寫後的字對。

## 範例輸入 #1

```
eggua-omen
marture-omeny
END
```

## 範例輸出 #1

```
egguwmen
marturoumeny
```
