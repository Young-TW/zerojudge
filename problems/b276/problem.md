# b276. 4和7

## 內容

萌萌哒doge突然想吃药了！

现在有一排格子，从左向右标号为0到m。doge最初在0号格子中。

一共有n堆药，第i堆药有a[i]粒，被放在b[i]格子里。

每次，萌萌哒doge可以跳到它右边4格或右边7格的位置。求它最多能吃到的药的个数。注意，doge不必跳到格子m，而是可以随时结束游戏。

## 輸入說明

第一行为两个正整数n,m，分别代表药的堆数与格子坐标的最大值。
接下来n行，每行两个正整数a[i],b[i]，分别代表某堆药的粒数和位置。

## 輸出說明

输出一个非负整数，最多能吃到的药的个数。

## 範例輸入 #1

```
3 13
100 4
10 7
1 11
```

## 範例輸出 #1

```
101
```
