## 輸入說明

第一行有1個正整數N，代表序列長度(1<=N<=10^6)


第二行有N個整數M，代表整個序列(-10^9<=M<=10^9)(序列中的數字不會重複)


第三行有1個正整數K，代表要求序列中第K大的數字(1<=K<=N)

## 輸出說明

請輸出這串數字中第 K 大的數字。

## 範例輸入 #1

```
3
5 2 3
2
```

## 範例輸出 #1

```
3
```

## 範例輸入 #2

```
6
8 7 6 5 4 9
2
```

## 範例輸出 #2

```
8
```

## 範例輸入 #3

```
3
-1 -2 0
1
```

## 範例輸出 #3

```
0
```
