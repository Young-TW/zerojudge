# k152. 文文與她的距離 (楚河漢界篇)

## 內容

文文和珊珊是青梅竹馬，從小一塊兒長大，雖然感情很好，但難免偶爾還是會鬧彆扭。


這次小倆口又吵架了...


好死不死，文文和珊珊不但是同班同學還坐同一張桌子。於是他們在桌子的中央畫了一條「楚河漢界」，彼此互不侵犯。



![image](ShowImage?id=2793)


珊珊：你超過界限了，離我遠一點！


文文：誰稀罕靠近妳？妳也離我遠一點！


於是兩人分別向桌子的兩邊移動。


給你文文和珊珊的英文名字，請模擬他們的座位圖。

## 輸入說明

輸入只有一行，含有兩個以空白隔開的英文名字，名字的長度不會超過 10 個字元。

## 輸出說明

輸出包含三個直線字元「|」，左右兩邊為桌子的邊緣，中間則是「楚河漢界」。在楚河漢界的兩側各有 10 個字元的空間，第一個名字放左邊空間靠左，第二個名字放右邊空間靠右。

## 範例輸入 #1

```
Adam Eve
```

## 範例輸出 #1

```
|Adam      |       Eve|
```

## 範例輸入 #2

```
Xi Biden
```

## 範例輸出 #2

```
|Xi        |     Biden|
```
