# j186. 棒棒糖事件 (蛀牙篇1)

## 內容

為了讓孩子們開心地上課，蝸牛老師每天都會帶一些棒棒糖到班上，再依當天出席的人數平均分給所有的小朋友。


可是這個班的小朋友實在是太喜歡棒棒糖了，如果分完後還有剩的棒棒糖，全班就會搶成一團，根本無法上課！於是蝸牛老師在發棒棒糖之前，必須把多出來的棒棒糖先吃掉。


一個星期後，蝸牛老師蛀牙了！牙醫師問蝸牛老師到底吃了多少棒棒糖，蝸牛老師自己也搞不清楚。給你這個星期每天蝸牛老師所帶的棒棒糖的數量及小朋友的出席人數，請你算一算蝸牛老師總共吃了多少棒棒糖。

## 輸入說明

輸入共有五行，每行含有兩個非負整數，依序代表星期一到星期五每天蝸牛老師所帶棒棒糖數量及小朋友出席人數。

## 輸出說明

請輸出一個整數，代表蝸牛老師這星期吃掉幾支棒棒糖。

## 範例輸入 #1

```
23 10
99 9
88 11
95 12
51 13
```

## 範例輸出 #1

```
26
```

## 範例輸入 #2

```
10 5
0 11
11 0
52 17
19 20
```

## 範例輸出 #2

```
20
```
