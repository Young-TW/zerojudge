## 輸入說明

測試資料只有一行,有兩個數字 n 及 m,
表示棋盤大小為 n × m,
其中1 ≤ n ≤ 8,1 ≤ m ≤ 8 且 n × m ≤ 42。
這兩個數字之間用空格(white space)隔開。

## 輸出說明

輸出資料為一個正整數,表示一開始在左上角的炮能吃到最多顆卒的數目。

## 範例輸入 #1

```
範例輸入一：
3 5

範例輸入二：
4 3
```

## 範例輸出 #1

```
範例輸出一：
7

範例輸出二：
5
```
