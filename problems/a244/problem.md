## 輸入說明

第一行有一個正整数N，

代表接下來有N行每行有三個正整數 a , b , c     

 ( 1 <= b , c <= 2147483647 )

 ( 1 <= a <= 4 )

## 輸出說明

如果 a = 1  請輸出  b+c 

如果 a = 2  請輸出  b-c 

如果 a = 3  請輸出  b*c 

如果 a = 4  請輸出  b/c  

結果請用整数輸出

## 範例輸入 #1

```
4
1 2 3
2 2 3
3 2 3
4 2 3
```

## 範例輸出 #1

```
5
-1
6
0
```
