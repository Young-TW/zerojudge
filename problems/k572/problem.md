# k572. pC. 關於冷氣調溫這件事

## 內容

每個人都會有自己覺得舒適的溫度範圍，
但當所有人都處在同個空間時，只能夠盡可能將空調設定為最多人所能接受的適宜溫度。


如果有多個最多人所能接受的適宜溫度，
為了節能省電，則會選擇溫度最大者。


舉例來說，假設有 3 個人，
喜歡的溫度範圍分別為 [19度 ~ 25度], [23度 ~ 26度], [18度 ~ 20度]


溫度181920212223242526編號0 AAAAAAA 編號1     BBBB編號2CCC      
 


可以注意到，19、20、23、24、25 這五個溫度都有最多人(此題為兩人)喜歡，
而 25 度是其中最大者，因此最後會將冷氣設定為 25 度。


給定 N 個人，與各自喜歡的溫度區間 [L : R]
請找出最多人喜歡的溫度，當同時有多個時則印出最大者。

## 輸入說明

第一行有一個正整數 N，代表總人數
1 ≤ N ≤ 100


接著有 N 行，每行有兩個整數 L 和 R，
代表喜歡的溫度區間 [L, R]
-109 ≤ L ≤ R ≤ 109

## 輸出說明

最多人喜歡的溫度，
當同時有多個時，則印出最大者。

## 範例輸入 #1

```
3
19 25
23 26
18 20
```

## 範例輸出 #1

```
25
```

## 範例輸入 #2

```
2
-1000000000 999999995
-999999995 1000000000
```

## 範例輸出 #2

```
999999995
```
