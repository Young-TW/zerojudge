# f012. 奶油機器人

## 內容

![image](ShowImage?id=1530)


奶油機器人(Butter Robot)，是一款專門用來遞奶油的機器人，他有一自動化的程式，當他所在的格子為白色時，他會噴出火焰，使得格子燒焦變成黑色，並向右轉 $90$ 度，前進一格；所在的格子為黑色時，他會塗上奶油，使得格子變成白色，並向左轉 $90$ 度，前進一格


現在機器人在一個全白的平面上，且起始座標為 $(0,0)$，並面向上方，請問機器人經過 $n$ 步後停在哪裡?

## 輸入說明

每行一個整數 $n$，代表機器人走 $n$ 步 $0≤n<2^{48}$

## 輸出說明

輸出 $(X,Y)$，$X$ 為 $x$ 座標，$Y$ 為 $y$ 座標

## 範例輸入 #1

```
1
2
3
5
10
100
1000
2147483647
2147483648
2200000000
```

## 範例輸出 #1

```
(1,0)
(1,-1)
(0,-1)
(-1,0)
(-1,-1)
(0,2)
(8,6)
(-41297588,-41297561)
(-41297588,-41297562)
(-42307516,-42307490)
```
