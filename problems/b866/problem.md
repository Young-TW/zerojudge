# b866. Liebig

## 內容

所謂限制因子，無論它是生物或非生物因子，凡是能阻止生物生長及分佈的，就稱之限制因子。


賴必格(1803~1873)因此提出最小量定律，定律表明當限制因子降低到該生物的最小需求量以下時，此因子就會成為限制該生物生長的關鍵因子。


然而並不是所有的因子都只以最低限度來限制生物生長，例如溫度對每一種生物來說都有其最適範圍，也就是同一種因子包含上界和下界的限制。在地球面臨氣候變遷之下，外星移民成為最佳選擇，各方專家集結並開始尋找適合的星球，其中生物學家對於每一種想要轉移的動植物都有其各種因子的最適範圍，想請問有沒有可能存在星球能夠同時讓這些動植物成長。生物能夠成長的條件就是生長環境中每一種因子的數值都落於其最適範圍內。

## 輸入說明

每組測試資料包含數比測資，每筆測試資料第一行有兩個整數n、m代表想要轉移的動植物數量、有多少種限制因子，接下有n行，第i(0≤i<n)行包含2*m個非負整數，其中第1和第2個整數代表第1種因子的最適範圍區間，第3和第4個數代表第2種因子的最適範圍區間，以此類推


100%測試資料滿足

1 ≤ n ≤ 100

1 ≤ m ≤ 100

0 ≤ L ≤ R < 231，L,R ∈ Ν ∪ {0}

## 輸出說明

輸出一行，有沒有可能存在這樣的星球，有則輸出Yes，沒有則輸出No

## 範例輸入 #1

```
2 2
1 2 2 5
1 5 5 7
1 1
1 1
2 1
1 1
2 2
```

## 範例輸出 #1

```
Yes
Yes
No
```
