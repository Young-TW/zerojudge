# e167. 演唱會記行 - 排隊買周邊

## 內容

俗話說：錢沒有不見，只是變成你喜歡的形狀


難得都來到演唱會了，怎麼可以不買周邊呢？


想當初第一次排隊買周邊是在日本東京巨蛋，光排隊就就排了8個小時呀~~


科普一下，由於結帳口非常多，最多可能開到2X~3X個結帳口，所以其實隊伍不是一個人一個人排的，是一坨人一坨人排的


雖然結帳口非常多，還是要排很久呀(累...)，如果要買周邊，記得要提早到喔，9:30開賣，差不多6:00就要去排了，加油~~~


到達現場後，為了縮短排隊的時間，快速的找到隊伍的最後尾是非常重要的！！！


小粉絲已經掌握了周邊開賣地點了，你能幫助小粉絲快速的找到隊伍最後尾嗎？

## 輸入說明

第一行為兩個整數n、m，代表地圖的長寬，n = 0、m = 0代表輸入結束 5 <= n、m <= 100


接下來n行為整張地圖，0代表空地，1代表排隊人潮 接下來一行為周邊結帳口的座標x、y


0 <= x < n, 0 <= y < m


注意！！！隊伍可能會不連續，因為工作人員為了避免檔到重要出入口，會將隊伍切割


每個隊伍一定只有一個最後尾

## 輸出說明

對每一組測試資料，請輸出隊伍的最後尾的座標
