# e097. 指數練習

## 內容

有一個數為  a ^ m  假設此數可表示為  b ^ h * c
今已知 a, m, h 請算出  b, c

## 輸入說明

每個測資的第 1 列有 3 個數字  T  A  M


T 表示此測資有 T 列，


A 表示每個  a  的最大值，  A <= 109


M 表示每個  m  的最大值。  M <= 104


每列有 3 個數字  a  m  h


a <= A , m <= M

## 輸出說明

請輸出 b  c


0 < b, c < 263 


b, c 若有多組解，請輸出  b  最大的那組。

## 範例輸入 #1

```
2 1000 100
108 5 10
15 3 2
```

## 範例輸出 #1

```
6 243
15 15
```
