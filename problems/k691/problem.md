# k691. 肥余歷險記---(反數學城5)(BOSS!!!!!!!!!)

## 內容

經過肥余辛辛苦苦的打怪 他終於引起了反數學BOSS的注意!!


反數學BOSS向肥余發出PK 如果肥余勝利 反數學BOSS就會放肥余回數學城


反之 肥余就要留在反數學城一輩子!!!!!!!!!


反數學城的BOSS 精通所有運算(+/-/*/除/**/sqrt)


請你幫幫忙，幫肥余打敗反數學BOSS!

## 輸入說明

第一行為測資個數


測資第一行是一個算式

## 輸出說明

輸出答案


sqrt和除法算式取整數

## 範例輸入 #1

```
6
5+5
4-0
5*4
3/3
6**2
sqrt 9
```

## 範例輸出 #1

```
10
4
20
1
36
3
```
