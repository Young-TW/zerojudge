# a217. caps lock的災難

## 內容

電腦的每個鍵盤都具有他們各自的重要功能，

要是哪個鍵突然出錯了都會帶來巨大的困擾。

今天假設有個人叫小明，他的電腦的caps lock鍵壞掉了，

所以他打出來的英文字全都是小寫，

這時小明為了證明他每個星期學寫程式是很有用的，

他決定寫個程式，可以自動把句子轉成正確的大小寫格式。

請你同樣寫個程式完成這件事。

p.s.這只是考試題目在嘴砲，事實上你可以用shift鍵取代caps lock的大寫功能，

或是這種時候根本就應該儘快把電腦拿去修。

## 輸入說明

每行為一筆單獨之資料，僅有小寫字母和空格和,.?! 

每行不超過1000個字元

## 輸出說明

每筆輸出一行， 

把每個句子第一個英文字母變大寫，

單獨的 i 也要變成大寫。

## 範例輸入 #1

```
yo! i am so fat. macbook air so cool!
me, too. battle!
who am i?
```

## 範例輸出 #1

```
Yo! I am so fat. Macbook air so cool!
Me, too. Battle!
Who am I?
```
