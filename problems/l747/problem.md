# l747. 連線

## 內容

小明在玩連線遊戲，但由於線連得太密，點又太多，所以他分不清哪些東西有透過不過的線和點連在一起。請你你編寫一段程序，協助他解決問題。

## 輸入說明

第一行為一個整數T，代表接下來有T行。 1<=T<=21。


接下來有T行，為字串s和字串p，以空格隔開。代表s和p有透過線連在一起。


最後一行為字串l和字串o，以空格隔開。

## 輸出說明

輸出只有一行，如果l和o有透過其他字串連在一起，則輸出 格式 : "True 最短連接步數"。


否則輸出"False"。

## 範例輸入 #1

```
4
tom john
mike john
paul tom
alex mike
alex tom
```

## 範例輸出 #1

```
True 3
```

## 範例輸入 #2

```
4
1 2
5 3
4 3
6 7
7 1
```

## 範例輸出 #2

```
False
```
