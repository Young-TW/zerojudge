# i921. 11984 - A Change in Thermal Unit

## 內容

測量溫度和溫差是許多研究和應用中的常見任務。 不幸的是，存在不止一個測量溫度的單位。 這有時會帶來很多混亂。 兩種流行的測量單位是攝氏 (C) 和華氏 (F)。 F 從 C 的轉換由以下公式給出：


F = 9/5*C + 32


在這個問題中，您將得到一個以 C 為單位的初始溫度和以 F 為單位的溫度升高。
在計算新的溫度後，您必須以 C 為單位呈現。

## 輸入說明

第一行有一個數字 T (≤ 100)，代表測試資料數量


接下來 T 行，每行包含兩個整數 C、d (0 ≤ C, d ≤ 100)


C 代表初始攝氏溫度


d 代表華氏溫度的增加

## 輸出說明

對於每個測試資料，四捨五入到小數點後兩位數後


輸出測試編號和以攝氏度為單位的新溫度

## 範例輸入 #1

```
2
100 0
0 100
```

## 範例輸出 #1

```
Case 1: 100.00
Case 2: 55.56
```
