# o169. 10523 - Very Easy !!!

## 內容

大多數時候，BUET計算機科學與工程專業的學生們都會面對一些虛假的、困難的和非常複雜的公式。因此，有時即使是面對簡單的問題，他們也會想得很複雜，從而使問題變得更加難以解決。然而，“BUET PESSIMISTIC”隊的成員們是唯一的例外。正好相反，他們將每個困難的問題都當作簡單的問題來對待，因此在任何比賽中都表現不佳。今天，他們嘗試解決一個序列問題，但因為將其視為困難問題而失敗。讓我們來幫助他們。

## 輸入說明

只需嘗試確定以下序列的答案



![image](ShowImage?id=4140)


給定整數N和A的值（1 ≤ N ≤ 150，0 ≤ A ≤ 15）。

## 輸出說明

對於輸入的每一行，你的正確程式應該為每對N和A的值分別輸出序列和的整數值。

## 範例輸入 #1

```
3 3
4 4
```

## 範例輸出 #1

```
102
1252
```
