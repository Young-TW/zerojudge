# g297. DD想測量

## 內容

DD有兩杯尿杯，一杯是n毫升，一杯是m毫升，因為明天要尿液檢查，所以幼祥老師要求DD準確量出k毫升的尿，但是不能用工具測量，假設DD可以無限供應尿液，請算出DD最少要交換幾次尿杯，若無法，請輸出:「老師請不要鬧!」(不含引號)。


範例一解釋 : 


1.將9毫升尿杯尿滿，並將4毫升的尿杯用9毫升的尿杯倒滿，9毫升的尿杯剩五毫升，交換一次。


2.將4毫升的尿杯倒掉，再用９毫升的尿杯倒滿，9毫升的尿杯剩一毫升，交換兩次。


3.將4毫升的尿杯倒掉，把9毫升的尿杯的一毫升倒至4毫升的尿杯，4毫升的尿杯剩一毫升，交換三次。


4.將9毫升的尿杯尿滿，並將4毫升的尿杯用9毫升的尿杯倒滿，9毫升的尿杯剩六毫升，交換四次。

## 輸入說明

第一行有一整數P，代表P筆測資。


每筆測資有n,m,k三數。

## 輸出說明

請算出DD最少要交換幾次尿杯。

## 範例輸入 #1

```
1
4 9 6
```

## 範例輸出 #1

```
4
```

## 範例輸入 #2

```
1
4 10000 97 
```

## 範例輸出 #2

```
老師請不要鬧!
```
