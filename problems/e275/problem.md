# e275. Xor and Parity

## 內容

如果整數 x 以二進位表示時有奇數個 1，就定義 x 的 parity 為 1，否則為 0。


現在有 n (n <= 100000) 個非負整數 a1, a2, ..., an，請問有多少數對 (i, j) 滿足 i < j 且 ai xor aj 的 parity 為 0？

## 輸入說明

輸入有多行，以 EOF 結束。


每筆測資第一行為一個正整數 n (n <= 100000)，第二行為 a1, a2, ..., an (0 <= ai < 2^60)，以空格隔開。


單一測資檔 n 的總和 <= 1000000。

## 輸出說明

輸出題目要求的答案。

## 範例輸入 #1

```
2
1 2
3
1 2 3
```

## 範例輸出 #1

```
1
1
```
