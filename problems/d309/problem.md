# d309. Put a banana in your ear!

## 內容

人類是群居性的生物，當然在社會形成的過程中需要領導者的出現，來統領族人向前邁進，看見未來的道路與希望。

從歷史可以看得出來，在東方，統領者稱之為皇帝；在西方則被稱為國王，這些在當初被推舉出來、眾望所歸的王者都有共同的特點：渾身散發著強大的王霸之氣、虎軀一震便是喝退來犯敵軍，一夫當關萬夫莫敵，在動盪不安的混亂時代中，王者們逐鹿中原，就看誰能有本事一統天下，領導群龍無首的國家、解放人民於水深火熱之中..............................

不過今天要討論的沒那麼嚴肅也沒那麼偉大以上東西看看就好(￣▽￣＃)﹏﹏

回歸正題，首先你要先知道一件事情。

香蕉王─查理‧獨角有言：「想要我的banana嗎？想要的話就自己去找吧！我把所有的banana都藏在那裏了！」

現在我們有N(N < 2^31)名外星人來到了附中，想要找到隱藏在附中鼎鼎有名的舊北桶並找出香蕉，然後 put a banana in their ears.

只可惜路途遙遠，他們需要推舉出一名帶路的隊長，又怕隊長在附中這塊如此危險的環境中壯烈犧牲(例如跌進新北池爬不出來之類)，所以再推選出一名副隊長。

接著剩下的N-2人再表明自己的心意，看誰有意願繼續朝向這塊未知領域繼續冒險...

例如N = 4時，會先挑出1名隊長1名副隊長，剩下的2名外星人都可以有去或不去的兩種選擇。

不過這群外星人的數學實在是不太好，根本算不出來他們最後出發時的隊形會有哪些種組合，身為想要幫助他們找到埋藏在舊北桶香蕉的你，能夠幫他們的忙嗎？

噢噢因為可能的方法實在是太多種了，所以只要印出這個數字 Mod 1000000007就好了。

## 輸入說明

一個正整數N，代表有N名外星人來到附中

我們保證： 

至少30%的測試資料中N<1000000

至少60%的測試資料中N<100000000

所有的測試資料中中N<2^31

## 輸出說明

依題意計算隊形的組合數後，除以1000000007的餘數

## 範例輸入 #1

```
3
```

## 範例輸出 #1

```
12
```
