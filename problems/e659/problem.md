# e659. 11878 - Homework Checker

## 內容

您的弟弟剛剛完成了"不大於一百的整數加減"的作業，並要求您檢查答案。
每個問題(連同您弟弟計算出的答案)的格式為"a + b = c"或"a - b = c"。
其中a和b是老師準備的數字(保證它們是非負整數，而不大於100)。
c是您的弟弟計算出的答案，並且c是一個不大於200的非負整數，或者是一個字元"?"(代表你弟弟不會算)。

## 輸入說明

包含多行輸入，最多100行。
每行包含一個問題和您弟弟的答案，格式如上所述。
每行中沒有空格字元。數字不會有前導零。

## 輸出說明

對於所有輸入，輸出一個整數，代表您弟弟答案正確的題目數量。

## 範例輸入 #1

```
1+2=3
3-1=5
6+7=?
99-0=99
```

## 範例輸出 #1

```
2
```
