# e635. 12908 - The book thief

## 內容

2014年2月18日，Red Matemtica在其Twitter帳戶(@redmatematicant)上提出了以下數學難題：
"當Anita讀偷書賊(作者：Markus Zusak)這本書時，她將從1開始的所有頁碼相加。她得到總和等於9000，但她發現他在相加過程中少加一頁頁碼。這個數字是多少？這本書有幾頁？"
以這個有趣的難題作為起點，現在您需要解決的問題是：
給定一個正整數s (1 ≤ s ≤ 10^8)，表示Anita獲得的結果，請你找出被遺忘的頁碼以及書的總頁數。

## 輸入說明

輸入包含多組測資。
每組測資一行，每一行包含一個正整數s (1 ≤ s ≤ 10^8)。
如果s = 0代表輸入結束。

## 輸出說明

對於每組測資，輸出兩個正整數，分別表示被遺忘的頁碼以及書的總頁數。

## 範例輸入 #1

```
1
2
3
4
5
6
9000
499977
49999775
0
```

## 範例輸出 #1

```
2 2
1 2
3 3
2 3
1 3
4 4
45 134
523 1000
5225 10000
```
