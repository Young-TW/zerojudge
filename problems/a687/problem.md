# a687. 軍方密碼

## 內容

最近恐怖分子勢力擴大，造成人心惶惶，有一天呱呱國軍方攔截到一組恐怖分子的密電，發現邪惡的首領Giu想要發射飛彈轟炸呱呱國首都，為了避免入亡國危機，軍方在飛彈發射前找到了解除碼要解除飛彈，但他們發現解除碼被Giu加密成他自創的Giugiu碼(簡稱G碼)，並拆成兩部分，你的任務就是把G碼A和G碼B兩部分合在一起找出解除碼，及時解除飛彈。

	G碼是一組由A到Z組成的26進位數，A代表0，B代表1，以此類推，Z代表25。例如：
 
	BA             = 1*26^1 + 0*26^0                  = 26，
	GOD          = 6*26^2 + 14*26^1 + 3*26^0 = 4397，
	BA + GOD = GPD                                     = 4423。

## 輸入說明

輸入多組測資，每組測資中，第一行為一個三位以內(含)G碼，表示接下來密碼的位數，如果為A(0)則代表輸入結束，無需輸出此行。
接下來兩行為G碼A和G碼B。

## 輸出說明

共n行，每行輸出每組密碼相加後的答案，以G碼格式輸出。

## 範例輸入 #1

```
F
EDCBA
DECAF
```

## 範例輸出 #1

```
HHEBF
```
