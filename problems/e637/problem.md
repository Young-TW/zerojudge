# e637. 10465 - Homer Simpson

## 內容

聰明的荷馬·辛普森 (Homer Simpson)喜歡吃Krusty漢堡。
荷馬需要m分鐘的時間能吃一個Krusty漢堡。 
然而，荷馬也喜歡阿普(Apu)的Kwik-e-Mart新型漢堡。
荷馬需要n分鐘的時間能吃一個Kwik-e-Mart漢堡。
在t分鐘的時間內，您需要找出荷馬可以不浪費任何時間吃的最大漢堡數。
如果他一定會浪費掉時間，那麼...他可以喝啤酒。



![image](ShowImage?id=4060)

## 輸入說明

輸入包含多組測資。
每組測資包含三個整數m，n，t (0 < m，n，t < 10000)。
m，n，t如題目所述。

## 輸出說明

對於每組測資，輸出荷馬不喝啤酒可以吃的最大漢堡數。
如果一定會喝啤酒，則多輸出他喝酒的時間，並用一個空格隔開。
荷馬最好不要喝太多啤酒。

## 範例輸入 #1

```
3 5 54
3 5 55
```

## 範例輸出 #1

```
18
17
```
