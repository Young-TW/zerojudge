# c498. 三、咒術契合(Spell)

## 內容

傳說中，如果兩名法師詠唱咒語的時間相同，就有機會發動合體咒術。


　　我們可以知道，兩種咒術的契合度將會是「26-(同時詠唱出的字母的距離)」的總和。舉例來說，以a為0，z為25，那麼如果c和x同時分別被兩位法師詠唱到，這兩個字母提供的契合度就是26-|2-23|=5。


　　但法師們可沒那種閒功夫記下冗長的咒語，他們各自選擇了一條沒那麼長的咒語來反覆詠唱以提升契合度。


　　雖然法師們大可以兩個人都重複念同一種咒語來達到最高的契合度，但不同屬性咒術的咒語可就沒辦法長的一樣了，這迫使當兩位法師想契合不同屬性的咒術難度增高，必須進行更長的詠唱。
　　現在給你兩個法師欲詠唱的咒語A,B和重複詠唱的時間T，請輸出這次合體咒術的契合度。

## 輸入說明

每筆測試資料只有一行，會有兩個字串A,B和一個正整數T以空格隔開。

## 輸出說明

對於每一筆測資，輸出這次合體咒術的契合度。

## 範例輸入 #1

```
//輸入範例1
owo oao 10

//輸入範例2
cplusplus pcshic 45
```

## 範例輸出 #1

```
//輸出範例1
194

//輸出範例 2
743
```
