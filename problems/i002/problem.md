## 內容

![image](ShowImage?id=2720)


原題


$\text{Waimai}$：「給你一個問題。」


$\text{W̴a̸i̷m̸a̷i̴}$：「有一個數列 $f$。」


$\text{W̶̨͉̳̤̼̓̇͂̃a̷̭͕͕̍͜i̴͙̗̣͎͓̊̚m̴̛̱̹̤͂̓̒͜͠a̷̭͑ḯ̶̮̉̔}$：「已知這個數列前 $n$ 項 $f_1\sim f_n$ 的數值。」


$\text{W̴̧̛̯̖͚͖̹̞̙͔̮̜̬͆̑͑̉̌̒̏̍͠ȧ̴̪͎̯̯͈̙͔͑̈́̓̎̽͜͝i̴̥̭͐͛̀̉̄͆̔̍̿m̵̬̘͂̈́ą̵̢̘̣͕̪͉̗̝̲̘͚̻̞̏̅̂̆̓͛̐̔i̸̤̙̮̬̼͎͇͍̖͗̋͂̃̈̆̄̈́͒ͅ}$：「當 $i>n$ 時，$f_i = \sum\limits_{j = 1}^n a_j\times f_{i - j}$。」


$\text{Ẉ̶̢̨̨̢̨̡̡̧̢̛̳͎̫͔͓͍̮̦̺̜̭͎̱̞̼͇̪̤̤̲͇̰̭͈̯̪̪̰̜̞̻̮̬̰͈̱̋̿͌̋́̋̍͆͆̇̊̓͋̓͋̑̂̓̅̊͊́̄͒͋̀̇̎̈́̋̊͋̈́̌̓̽͜͝͝ą̷̥̱̪͕̪̟̦̩̣͙̗̺͕͈̻̺̞͉͔̼̖̘̪̍̿̚͜ͅi̴͓̭͎̻͈̙͇̣͖̮̞͍͚̮̖̖̭͔̱̟͎̹̘̲̯͕͚̩̳͉͓̯̰̟͎̙͈͋̾̅͆̉̈̉̑̓̈́̇̈́̈́́̉͊̏̍̄̇̌̆̏̚̕͘͘̚̚m̵̢̼̖̦̗̟̉̌̀̀̏̀́̈̾̀̐̕͘ͅą̴̧̨̡̢̢̹̦̲͔̳̲̮̣̙̭͔̩͖̻̩̱̺̺̝̜͙̻̬̟͉̥̝̩͔͕̙̺͚̻̥̬̭̝͎̤̫͕͍̪͂̃̑̓̄̈͆͜͝į̴̧̙͙̖̲͙̱̫̪̜̐͆̐͗͑́̾̂͆̑̊̋̒̎̈́̏̆̅́͑̉̾̉͑̇͐͛͊̍̍͗̂̒͆͌̚̕̚͠͝͝͝}$：「請你求出 $f_k$。」

$\text{ }$：「不過因為答案可能會很大，請 $\bmod 998244353$ 後再輸出。」

## 輸入說明

第一行有兩個正整數 $n, k$ 代表已知 $f$ 的前 $n$ 項，和想要求 $f_k$。


第二行輸入 $n$ 個整數 $f_1\sim f_n$。


第三行輸入 $n$ 個整數 $a_1\sim a_n$。


- $1 \leq n \leq 3\times 10^4$
- $1 \leq k \leq 10^9$
- $0 \leq f_i, a_i < 998244353$

## 輸出說明

輸出一個整數代表 $f_k\ (\bmod 998244353)$。

## 範例輸入 #1

```
3 2
2 3 5
5 5 1
```

## 範例輸出 #1

```
3
```
