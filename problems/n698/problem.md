# n698. 超鐽杯的R值

## 內容

超鐽杯很喜歡玩音遊，他立志在20歲的時候要把R值提升到17.43。為了讓自己更有動力實現目標，他向他的朋友瓶頸哥發誓，如果自己沒有達到目標就請瓶頸哥吃拉麵來當作對自己的懲罰。


 


現在來到了超鐽杯的20歲生日，請你寫個程式幫他計算他應該要請瓶頸哥吃幾碗拉麵。


 


懲罰的程度會隨著與目標的距離而有所差異，如下(懲罰不會疊加)：


如果超鐽杯最終R值達到17.43，則不必受到任何懲罰。


如果超鐽杯最終R值未達17.43，則要請瓶頸哥吃1碗拉麵。


如果超鐽杯最終R值未達17.00，則要請瓶頸哥吃3碗拉麵。


如果超鐽杯最終R值未達16.50，則要請瓶頸哥吃7碗拉麵。


如果超鐽杯最終R值未達16.00，則要請瓶頸哥吃10碗拉麵。


如果超鐽杯最終R值未達15.50，則要請瓶頸哥吃15碗拉麵。


如果超鐽杯最終R值未達15.00，則要請瓶頸哥吃20碗拉麵。

## 輸入說明

輸入為一個浮點數R，代表超鐽杯20歲時的R值。(R>=0.00 且 R<=17.43)

## 輸出說明

請輸出一個整數，代表超鐽杯應該要請瓶頸哥吃幾碗拉麵。

## 範例輸入 #1

```
17.20
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
16.72
```

## 範例輸出 #2

```
3
```

## 範例輸入 #3

```
17.43
```

## 範例輸出 #3

```
0
```
