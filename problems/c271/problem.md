# c271. 疊疊樂(Easy)

## 內容

你會得到一堆箱子，每個箱子有各自的重量，以及負重


每個箱子等高等寬但有些不一樣長


我們希望可以疊很多箱子，而且長度由下而上是非遞增的


當箱子包含自身以上的箱子總重超過負重，箱子就會壞掉


 


關於負重的補充，{}左至右相當於上到下放置箱子


Example #1:


箱子重量{2, 4, 8}, 所需負重(2, 6, 14)


Example #2:


箱子重量{2, 2, 3}, 所需負重(2, 4, 7)

## 輸入說明

第一行包含整數T，代表有T筆小測資


每筆小測資中，第一行包含整數N(1≤N≤1000)，接下來N行包含Wi,Si,Li (1≤Wi,Si,Li ≤109 )，分別代表重量，負重，長度

## 輸出說明

對於每一筆小測資，輸出最多可以疊幾個箱子

## 範例輸入 #1

```
2
2
2 4 2
8 10 3
2
2 4 2
8 9 3
```

## 範例輸出 #1

```
2
1
```
