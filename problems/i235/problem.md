# i235. Hehe排列組合

## 內容

求 $1 \text{ ~ } N$ 所組成的 $2^N$ 個子集合中，有多少個子集合的加總是 $K$ 的倍數 ($0$ 也是 $K$ 的倍數)


我們定義空集合的加總為 $0$


由於答案很大，請輸出符合條件的子集合數，$\bmod 998244353$

## 輸入說明

輸入為兩個正整數 $N, K$


$1 \leq N \leq 10^{18}$


$1 \leq K \leq 150$

## 輸出說明

輸出答案 $\bmod 998244353$

## 範例輸入 #1

```
5 5
```

## 範例輸出 #1

```
8
```

## 範例輸入 #2

```
1 100
```

## 範例輸出 #2

```
1
```
