# d763. 好多线段

## 內容

爱斯基摩人给了你一堆线段，他想要问这些线段能组合出几种多边形呢?

## 輸入說明

有多组测试资料

每组测试资料占一行，有一些以一个空白分开的正整数(不超过100个)，并且行末不会有多余的空白

若该组测试资料只含有0 ，请结束程式且不要输出任何数字

(所有资料均不会大于1000000)

## 輸出說明

对每组测试资料输出一个数

代表这些线段能组合出几种多边形(多边形为三边或以上)

## 範例輸入 #1

```
3 5 6 9 4
2 3 4 10
7 8 9
1 2 10 
7 8
0
```

## 範例輸出 #1

```
12
1
1
0
0
```
