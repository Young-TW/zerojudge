# s087. Automated Telephone Exchange

## 內容

在聖彼得堡，電話號碼的格式為“XXX-XX-XX”，其中前三位數字代表自動電話交換機（ATE）的索引。每個 ATE 都剛好有 10000 個唯一的電話號碼。

彼得剛買了一間新公寓，現在他想安裝一條電話線。他認為，如果一個電話號碼所代表的算術表達式等於零，那麼這個電話號碼就是吉利的。例如，電話號碼 102-40-62 是吉利的（102 - 40 - 62 = 0），而號碼 157-10-47 則不是吉利的（157 - 10 - 47 = 100 ≠ 0）。

彼得知道他家使用的 ATE（自動終端設備）的索引號。為了了解他拿到幸運號碼的機率，他想知道他家的 ATE 有多少個幸運號碼。

## 輸入說明

輸入檔包含一個整數 n，即 Peter 的 ATE 的索引（100 ≤ n ≤ 999）。

## 輸出說明

輸出一個整數－Peter 的 ATE 擁有的幸運電話號碼的數量。

## 範例輸入 #1

```
196
```

## 範例輸出 #1

```
3
```

## 範例輸入 #2

```
239
```

## 範例輸出 #2

```
0
```
