# h862. 超級老鼠測試

## 內容

貓咪是個天才兒童，他在一個月大的時候就學會數數、六個月大的時候就學會乘法跟除法、一歲時學會寫程式、一歲又六個月時養了可愛的拉不拉多、一歲又十個月時養了可愛的吉娃娃、兩歲時發明了「倒墨水」的遊戲，而現在要講的，是貓咪在國小二年級的故事。


貓咪體育課要玩「抓老鼠」的遊戲，總共要抓 $n$ 趟老鼠，在第 $i$ 趟時有老師準備好的 $a_i$ 隻老鼠，因為老師規定貓咪在第一趟之後的每一趟都要比前一趟抓的老鼠數目多，所以貓咪得想辦法讓 $a_i$ 減少或增加，貓咪可以用 $1$ 單位的力氣去抓新老鼠，讓 $a_i$ 增加 $1$，或是用 $1$ 單位的力氣將老鼠送給拉不拉多與吉娃娃，讓 $a_i$ 減少 $1$。貓咪在每一趟抓新老鼠或送老鼠的次數不限，但老鼠的數量 $a_i$ 不能小於 $0$。請問貓咪最少要花多少單位的力氣才能完成「抓老鼠」的任務呢？

## 輸入說明

第一行有一個正整數 $n$，代表要抓 $n$ 趟老鼠。


第二行有 $n$ 個非負整數 $a_1\sim a_n$，$a_i$ 代表老師在第 $i$ 天準備給貓咪的老鼠數量。


- $1\leq n\leq 2\times 10^5$
- $0\leq a_i \leq 10^9$

## 輸出說明

輸出一個整數 $X$，代表貓咪如果在第一趟之後的每一趟都要比前一趟抓的老鼠數目多，最少需要花 $X$ 單位的力氣。

## 範例輸入 #1

```
5
0 2 1 1 5
```

## 範例輸出 #1

```
4
```

## 範例輸入 #2

```
5
157965481 134394366 394057443 615081232 457094533
```

## 範例輸出 #2

```
181557816
```
