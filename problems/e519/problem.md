# e519. 10633 - Rare Easy Problem

## 內容

N是一個隨機數，由於某種原因，它至少是兩位數。
有一個莫名其妙的人John Doe，他對N進行了一次運算：他將最後一位數字切掉以形成新的數字M，然後找到N-M。
這件事讓他有一種難以言喻的興奮。最後他會告訴你N-M。
由於對這個數字背後的迷人故事感到興奮，您立下目標要找出原來的N。
喔！對了，順帶一提，John後來被老虎吃掉了。

## 輸入說明

輸入包含多行，每種情況下一行。
每行包含一個正整數N-M (10 <= N-M <= 10^18)。
N-M = 0代表輸入結束。

## 輸出說明

對於每種情況，請按順序輸出N的可能值。
每個可能值間請用空格分隔。

## 範例輸入 #1

```
18
0
```

## 範例輸出 #1

```
19 20
```
