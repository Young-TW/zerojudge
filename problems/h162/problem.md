# h162. 最少的操作數

## 內容

給定一長度為 n 的整數陣列 arr (index從0開始)，每次操作可以選擇任意一個i位於[0,n) 使得arr[i]變為任意的正整數。


求最少需要幾次操作可以使得arr變為非嚴格遞增的序列。

## 輸入說明

多筆測資


每筆測資有兩行


第一行是一個整數n代表陣列的長度(1<=n<=10^5)


第二行包含n個整數，以空格隔開，代表陣列對應的值(1<=arr[i]<=n)

## 輸出說明

輸出最少的操作數

## 範例輸入 #1

```
5
5 4 3 2 1
6
4 1 5 2 6 2
```

## 範例輸出 #1

```
4
3
```
