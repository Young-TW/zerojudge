# o773. 喵-Base64

## 內容

開頭給你 e (Encode) 或是 d (Decode)
後面會跟著一個字串


請編碼或是解碼它們

## 輸入說明

多筆測資


開頭是一個字元 e 或是 d 
然後會有一個字串 s (1 <= s.Length <= 100000)

## 輸出說明

若是 e 請編碼字串並輸出
若是 d 請解碼字串並輸出

## 範例輸入 #1

```
e Hello, World!
d SGVsbG8sIFdvcmxkIQ==
```

## 範例輸出 #1

```
SGVsbG8sIFdvcmxkIQ==
Hello, World!
```
