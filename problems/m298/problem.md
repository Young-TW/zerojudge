# m298. 12004 - Bubble Sort

## 內容

檢查以下程式碼，該程式碼是在計算 Bubble Sort 的交換次數。



![image](ShowImage?id=3620)


當陣列大小為 n，且陣列元素a[]分別為1~n的值之隨機分佈時，若呼叫findSwaps()無限多次，則count的平均值應為多少？

## 輸入說明

第一行有一個整數 T (≤ 1000) 代表有幾組測試資料


接下來T行，每行有一個整數 n (1 ≤ n ≤ 10^5)

## 輸出說明

對於每組測試資料


輸出測試資料編號，以及count的期望值


如果期望值是整數，請直接輸出


否則請輸出最簡分數"p/q"，其中 p 和 q 互質

## 範例輸入 #1

```
2
1
2
```

## 範例輸出 #1

```
Case 1: 0
Case 2: 1/2
```
