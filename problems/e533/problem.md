# e533. 11942 - Lumberjack Sequencing

## 內容

你是說?伐木工人的故事？讓我想想......
伐木工人莫過於非常粗魯，而且大鬍子，其中工頭往往是專橫而頭腦簡單的人。
工頭喜歡按伐木工人的鬍鬚長度，十人一組排列，以刁難伐木工人。
但是不同身高的伐木工人常常改變他們的位置，這讓工頭感到非常困擾。
因此，工頭實際上必須以公分為單位測量伐木工人的鬍鬚，以查看每個人是否都按順序排好隊。
您的任務是寫一個程式來幫助工頭，確定伐木工人是否從最短到最長的鬍鬚或從最長到最短的鬍鬚正確地排列。

## 輸入說明

輸入第一行包含一個整數N (0 < N < 20)，N是要處理的組數。
接下來N行，每行包含10個小於100的不同正整數。

## 輸出說明

輸出第一行為標題"Lumberjacks:"
之後根據每組鬍鬚長度：
如果有按照工頭的喜好順序排隊
輸出"Ordered"
否則輸出"Unordered"

## 範例輸入 #1

```
3
13 25 39 40 55 62 68 77 88 95
88 62 77 20 40 10 99 56 45 36
91 78 61 59 54 49 43 33 26 18
```

## 範例輸出 #1

```
Lumberjacks:
Ordered
Unordered
Ordered
```
