# e598. 傑出校友表揚

## 內容

蝸牛老師所任教的優質高中即將舉行校慶，校慶大會中將表揚傑出校友。傑出校友的認定方式非常簡單，每位校友在加入校友會時都會得到一個 9 位數的校友編號，如果有一個校友的編號無法被任何其他可能的校友編號所整除，那麼他就是傑出校友。


給你一位校友的編號，請判斷他是不是傑出校友。

## 輸入說明

輸入只有一行，含有一個 9 位數的正整數，代表該校友的編號，編號的第一位數不會是 0。

## 輸出說明

如果該校友為傑出校友，請輸出「yes」，否則請輸出「no」。

## 範例輸入 #1

```
123456789
```

## 範例輸出 #1

```
yes
```

## 範例輸入 #2

```
200000000
```

## 範例輸出 #2

```
no
```
