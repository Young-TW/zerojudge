# q369. 4. 洛聖都逃亡計畫

## 內容

你是洛聖都（Los Santos）最臭名昭著的街頭賽車手，在一場瘋狂的銀行搶案後，你必須甩開瘋狂追捕你的警察，並設法逃到藏身點！
　　洛聖都的街道如同迷宮，某些路段被警方封鎖，而你只能在開放的道路上移動。你需要找到一條路，從你現在的位置 S 開車逃到安全屋 T，避免被警方圍捕！
　　洛聖都是一個 $N×M$ 的城市地圖，由以下符號組成：


-  . （道路）：你可以自由行駛的街道。
-  # （封鎖區）：警方封鎖的道路，你無法通過。
-  S （起點）：你目前的位置。
-  T （安全屋）：你的目標地點。

你只能沿著上下左右四個方向駕駛（不能斜向移動）。請判斷你是否能從 S 開車到 T，並輸出 "mission passed respect+" 或 "wasted"。

## 輸入說明

第一行包含兩個整數 $N, M$（$1\le N, M\le 1000$）代表城市地圖的大小。
　　接下來的 $N$ 行,每行包含一個長度為 $M$ 的字串，描述整個城市的道路狀況。（保證 S 和 T 各出現恰好一次）

## 輸出說明

若能順利開車到達安全屋，輸出 "mission passed respect+"。
　　若所有道路都被封鎖而無法抵達安全屋，輸出 "wasted"。

## 範例輸入 #1

```
5 5
S...#
.##.#
...#.
#.#.T
#....
```

## 範例輸出 #1

```
mission passed respect+
```

## 範例輸入 #2

```
3 3
S##
.#.
##T
```

## 範例輸出 #2

```
wasted
```
