# j056. 11650 - Mirror Clock

## 內容

Little Onuvob 已經學會了看掛鐘。他經常大聲宣佈時間。他的房間裡有一個大鐘。 時鐘的對面還有一面鏡子。 幾天前，他的母親注意到有時 Onuvob 會報錯時間。 她開始擔心，經過一番調查，她發現，Onuvob 從時鐘上看時間沒有問題，問題是他有時會從鏡子裡的時鐘上看。 所以她告訴他"Onuvob，你不應該從鏡子裡的時鐘上看時間。"
"但是為什麼媽媽？" 好奇的 Onuvob 問道。
"因為它給出了錯誤的時間"母親回答道。
現在 Onuvob 看了看那兩個鐘，意識到他媽媽說的是真的。 兩個時鐘不匹配。 他想了想，喊道："但是，媽，我們可以根據鏡子裡的時鐘計算出真正的時間，不是嗎？"

## 輸入說明

輸入的第一行包含一個整數 T (T < 1000)，即測試資料的數量。


每組測試資料都包含 Onuvob 從鏡子中的時鐘讀取的時間。


時間將採用以下格式：
HH:MM，小時和分鐘都是整數，並且是有效時間。

## 輸出說明

對於每組測試資料，輸出正確的時間 HH:MM

## 範例輸入 #1

```
2
12:00
10:09
```

## 範例輸出 #1

```
12:00
01:51
```
