# e389. 壞掉的計算機

## 內容

小崴的計算機 , 電腦, 跟手機都壞了!!! (慘?)


心急的他，馬上就寫出了一個計算機程式!!


(不要問我他是怎麼做到的 XD 我也不知道電腦壞了要怎麼寫程式 QAQ)


 


P.S. 感謝 evan85910907 提醒，已修改題目敘述! :)

## 輸入說明

每一行一個計算式 , EOF 結尾 , 


各包含:


數字(有可能有小數)


+ - * / ^ ()  //--- ^ 是指 次方 ex: 2^3=8


!  //---階乘(此題只需考慮非負整數的階乘)


sin() cos() tan() 


ln() log() 


abs()


sqrt()


e //--- 2.71828182846


pi //--- 固定這樣寫 , 3.141592653589...


 


網址(google)


他能做到的你的程式都要能做到!


範例輸入:


google 計算機

## 輸出說明

答案 (保證有答案) 到小數點第五位(四捨五入)


 一行一個~

## 範例輸入 #1

```
sqrt(abs(log(ln(tan(cos(sin((1+2-3*4/5!))))))))^e*pi

```

## 範例輸出 #1

```
0.96286
```
