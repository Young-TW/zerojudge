# o337. YTPizza

## 內容

今天是一年一度的YTPizza派對舉辦日，由廚師大黑及助手凰駁蟲準備料理。


由於廚師要處理太多訂單加上助手的搞耍，不小心做出了$N$個有重複種類的配料（每種配料有一個專屬代號）。


因為他很珍惜每一份配料，所以他決定把重複的配料收回倉庫裡下次用。


請幫忙他收起重複的配料後，按原本順序輸出處理完的配料單。


如果使用cin、cout，請在主程式中加上以下兩行：
ios::sync_with_stdio(0);
cin.tie(0);

## 輸入說明

第一行為$N$


第二行有$N$個數字，表示原本的配料表$a_i$


 


\(1\le N\le 10^6\)


\(-10^6\le a_i\le 10^6\)

## 輸出說明

輸出處理完的配料單，每一個配料代號以空格隔開

## 範例輸入 #1

```
5
4 3 2 3 9
```

## 範例輸出 #1

```
4 3 2 9
```

## 範例輸入 #2

```
9
10 89 10 76 10 89 89 12 90
```

## 範例輸出 #2

```
10 89 76 12 90
```
