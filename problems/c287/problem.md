## 輸入說明

每筆輸入n代表分成的時段,w代表所需單位時段數
1<=n<=300000
1<=w<=n
n=w=0時結束
接下來有
n個m(0<=m<=100)


代表每個時段的機率

## 輸出說明

第一行輸出符合的個數t
再來t行 輸出範圍 


若t>1 輸出範圍下限從小到大

## 範例輸入 #1

```
10 3
30 20 40 50 60 80 10 20 90 70
0 0
```

## 範例輸出 #1

```
2
1 3
3 5
```
