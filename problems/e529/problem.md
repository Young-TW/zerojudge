# e529. 00482 - Permutation Arrays

## 內容

在許多電腦問題中，必須置換數據陣列。
也就是說，必須以某些指定順序重新排列數組中的數據。
置換任意數據數組的一種方法是使用索引數組指定置換，以指出元素在新數組中的位置。

## 輸入說明

第一行為一個整數T，代表Case數量。
每個Case前有皆有一個空白行做為開頭。
每個Case將包含兩行數字。
第一行是索引數組p，從1~n。此行中的數字將以某種方式排列。
第二行為數據數組x，可能包含浮點格式。

## 輸出說明

對於每個Case。
請將數據數組x根據索引數組p進行排序。
並且將排序好的數據數組x輸出，每個數據一行(如範例輸出)。
兩個連續Case由空白行分隔。

## 範例輸入 #1

```
1

3 1 2
32.0 54.7 -2
```

## 範例輸出 #1

```
54.7
-2
32.0
```
