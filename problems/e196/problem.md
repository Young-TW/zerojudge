# e196. 又是進德教育

## 內容

說到板橋高中著名的傳統(?)，就是進德教育了。


　　每個需要進德的學生，都需要在進入欲進德的地點時簽一次名，做當天指定的事(EX:打掃、聽演講、跳八段錦)，出來的時候再簽一次名，就能完成整個進德的流程。當然，該簽名卻沒有簽名的，就會被記警告。


　　請你寫一支程式，依序統計每個人簽名的時機，判斷一個已知有$\color{black}{\space N\space}$個人需要參加的進德教育中有多少人需要被記警告。


 


　　一個人只要簽名兩次，就肯定表示他進去又出來了(時機不是問題XD)，我們保證不會有人簽名三次(含)以上。欲進德學生們的編號將以$\color{black}{\space 1\sim N\space}$表示。

## 輸入說明

輸入首行有兩個$\color{black}{\space N,M(1\leq N \leq 2600,0\leq M \leq 2N)\space}$，代表這次進德教育有$\color{black}{\space N\space}$個人得參加、且接下來會有$\color{black}{\space M\space}$筆簽名紀錄。


接下來一行有$\color{black}{\space M\space}$個數字以空格隔開，依序代表學生的簽名紀錄。

## 輸出說明

輸出有多少人會被記警告。

## 範例輸入 #1

```
6 7
1 2 3 4 3 1 2
```

## 範例輸出 #1

```
3
```
