# k799. WEBWORK 的逆襲

## 內容

微積分期考終於考完了，大家都在討論成績。


小明：「你 webwork 多少分？」
中明：「100 分，你呢？」
小明：「我也 100 分。」
中明：「哈哈哈，這麼弱的 webwork 其實是雜魚吧。雜魚 webwork～，雜魚雜魚。」


webwork 聽到兩個人說自己弱實在是受不了了。


webwork：「我 要 黑 化 了！」
小明：「你黑化後還是敵不過我們啊，哈哈哈。」
webwork：「以黑化 🚬」
小明：「什麼，webwork 竟然進化成二階型態 WEBWORK 了。中明，你有信心能贏過巨大化後的 webwork 嗎？」
中明：「會贏喔。」


說出勝利宣言後，中明和 WEBWORK 經過 15 話的激戰，中明被 WEBWORK 分成了上明與下明。


WEBWORK：「現在只剩下你了，小明。」
小明：「請原諒我吧，我什麼都會做的。」
WEBWORK：「咦，你說什麼都會做，那你能做出這題嗎？」


進化後的 WEBWORK 題目變更難了，小明要算出 $\int_l^r [\ln(ax)]^{-\ln(bx)}dx$，WEBWORK 才會放過小明。


小明實在是算不出來這題，請你幫幫小明吧！

## 輸入說明

第一行輸入 $t$，代表詢問次數。


接下來 $t$ 行，每行輸入四個 $5$ 位小數 $a,b,l,r$。


- $1\leq t\leq 1000$
- $1\leq a,b\leq 10$
- $\frac{1}{a}<l<r\leq 20$

## 輸出說明

請輸出 $\int_l^r [\ln(ax)]^{-\ln(bx)}dx$ 是多少，與正確答案的相對誤差或絕對誤差 $\leq 10^{-7}$ 都會通過。

## 範例輸入 #1

```
2
1.00000 1.00000 1.50000 19.99999
3.14159 4.15411 10.00000 20.00000
```

## 範例輸出 #1

```
5.14799322900000001
0.04440736712345678987654321
```
