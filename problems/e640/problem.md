# e640. 造句

## 內容

噹噹噹噹~~~噹噹噹噹~~~(國小上課鐘聲)


老師：各位觀眾，上課啦！拿出上次考的考卷，我們要檢討造句的部分。


學生：老師，我全對耶！可以不要檢討嗎？


老師：可以呀，你出去罰站！！


學生：~!@#$%^&*()_+


老師：有人覺得自己造句很厲害的嗎？大聲說，說出來就不用檢討。


學生：選我選我~~~我可是江湖傳聞的造句怪呢！


老師：好！用"...從前..."造一個句子。


學生：剛剛那位同學從前門出去罰站。


老師：再給你一次機會，用"...如果..."造一個句子。


學生：假如果汁比汽水好喝，我會喝果汁。


老師：給我滾，通通滾出去，全班這個星期，每天造100個句子給我。


學生：(你自己叫我造的，在那邊哈喽)


---------------------------------------------


老師在情緒失控後有點後悔，因為這樣子她就要改一堆造句。


於是他決定先把不合格式的造句全部刪掉，以加快改作業速度。


至於不合語意的部分，他再想其他辦法。


你能幫助老師嗎？

## 輸入說明

輸入第一行有一個整數T，代表有幾個題目。


每個題目第一行為一空白行。


第二行為一個字串s，代表造句範例。


造句範例由"..."和中文字構成。


第三行為一個整數n，代表有幾句學生的造句


接下來n行，每行代表需要判斷的造句。


注意：造句內的標點符號只會有"，。？！"這四種。
不會有亂寫的情況，比方說全部都是逗點，或者沒意義句子。
造句範例小於5個單詞，每個單詞長度小於5。

## 輸出說明

對於每個題目，輸出題目編號


判斷學生的造句是否有符合範例造句
範例造句中的"..."，代表需要接字(標點符號不算)


如果符合
輸出"對"
如果不符合
輸出"錯"


每組輸出間請空一行。

## 範例輸入 #1

```
2

因為...所以...

系統因為要維修，所以不能提交程式。
因為因為我手抖，所以所以我寫了兩次。

...因為...所以...
4
我因為沒打註解，所以看不懂程式。
雖然這題有點難，但是我還是努力寫。
我喜歡咖啡因，為了喝咖啡，所以種咖啡豆。
因為因為我手抖，所以所以我又寫了一次。
```
