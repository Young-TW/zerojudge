# a352. 解电阻

## 內容

众所周知，在物理电学恒定电流中很重要一个内容就是：给出一个电路图，该图中只存在着理想导线和理想电阻，要求你求出这个电路中某两点间的电阻值。 

现在，我们回顾一下一些电学知识。

   ①    两个电阻串联：那么这个电路两端的等效电阻R有关系式R = R1+ R2。

   ②    两个电阻并联：那么这个电路两端的等效电阻R即为1/R = 1/R1+ 1/R2。        当然，还有更加诡异的电路了，比如惠根斯电桥。

       现在给你的任务就是，给你一个电路图，求出某两点之间的电阻值。

## 輸入說明

输入测资的第一行有两个以一个空格隔开的正整数N和M，分别表示节点数和介于节点与节点之间的电阻的个数。

         接下来有M行，每行有三个以一个空格隔开的正整数X，Y（X≠Y）和R，意义为节点X与节点Y之间有一个阻值为R（单位为国际单位欧姆，字母是Ω）的电阻。注意：两个节点间可以有多个电阻，如②所示的并联电阻。

         在输入测资的最后，有两个以一个空格隔开的正整数S和T（S≠T），表示节点S与节点T之间的电阻值。

         请参考范例输入。

## 輸出說明

只有一行，输出节点S和节点T之间的电阻值，保留至小数点后2位。       你的答案必须与标准答案完全一样才可以通过该测资，请不要输出多余信息。        题目保证，Ans∈（0,﹢∞）

## 範例輸入 #1

```
3 3
1 2 3
1 2 6
2 3 1
1 3
```

## 範例輸出 #1

```
3.00
```
