# c031. 00264 - Count on Cantor

## 內容

現代數學中有一個有名的證明（由Georg Cantor所提出的）：有理數是可數的。他使用一個圖表（Cantor's 列舉）列舉出有理數，如下圖所示：  



  
![image](ShowImage?id=191)

 在此圖中，第一項是1/1,第2項是1/2,第三項是2/1,第四項是3/1,第五項是2/2,以下依此類推。

## 輸入說明

輸入每筆資料1行，含有1個正整數n (1<=n<=10^7)。 對每行輸入，輸出在Cantor's 列舉圖中的第n項。

## 輸出說明

參考輸入格式

## 範例輸入 #1

```
3
14
7
```

## 範例輸出 #1

```
TERM 3 IS 2/1
TERM 14 IS 2/4
TERM 7 IS 1/4
```
