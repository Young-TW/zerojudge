# d541. “∧”形排列

## 內容

对于一个n的全排列A[i]，当且仅当存在某个x∈[1，n]，使得A[1]<A[2]<...<A[x]>A[x+1]>...>A[n]时，该排列称为“∧”形排列。

    例如，对于9的全排列，125789643是一个“∧”形排列，123456789也是，但256198743就不是。

    试求n的“∧”形全排列的个数。

## 輸入說明

以下有多笔数据，每行一个。

读入一个数n。(n<=2 000 000 000)

## 輸出說明

输出n的全排列中“∧”形排列的个数。

由于这个数可能很大，因此你只需输出它mod 1234567的值。

## 範例輸入 #1

```
3
```

## 範例輸出 #1

```
4
```
