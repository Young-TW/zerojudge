# d299. 程式設計師的面試問題

## 內容

程式設計師的面試到底有多困難？

請推理出以下10個字母所代表的數字( 0 ~ 9 )。

重複的字母為相同的阿拉伯數字。

　  FORTY

      　TEN

   +)  TEN

    ______

     SIXTY

請以XXXXX + XXX + XXX = XXXXX的格式輸出所有可能(這些X就是數字)。

請複製上述格式再將X改為數字,避免因為空白的關係而造成WA 。

## 輸入說明

本題沒有輸入

## 輸出說明

每組輸出為一行，格式為

XXXXX + XXX + XXX = XXXXX

(這些X就是數字)

## 範例輸入 #1

```
無
```

## 範例輸出 #1

```
(略)
```
