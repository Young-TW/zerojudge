# h572. 蝸牛的星座

## 內容

小明：你知道蝸牛老師是什麼星座的嗎？


小華：他那麼龜毛，一定是處女座！


給你蝸牛老師的生日，請問他是什麼星座的？


星座日期
Aries 牡羊座 3/21 ~ 4/20
Taurus 金牛座 4/21 ~ 5/21
Gemini 雙子座 5/22 ~ 6/21
Cancer 巨蟹座 6/22 ~ 7/22
Leo 獅子座 7/23 ~ 8/21
Virgo 處女座 8/22 ~ 9/23
Libra 天秤座 9/24 ~ 10/23
Scorpio 天蠍座 10/24 ~ 11/22
Sagittarius 射手座 11/23 ~ 12/22
Capricorn 摩羯座 12/23 ~ 1/20
Aquarius 水瓶座 1/21 ~ 2/19
Pisces 雙魚座 2/20 ~ 3/20

## 輸入說明

輸入含有兩個整數 𝑚, 𝑑 (1 ≤ 𝑚 ≤ 12, 1 ≤ 𝑑 ≤ 31)，代表生日的月和日。

## 輸出說明

請依題目所提供的星座日期，輸出蝸牛老師的星座名稱。

## 範例輸入 #1

```
8 10
```

## 範例輸出 #1

```
Leo
```

## 範例輸入 #2

```
12 25
```

## 範例輸出 #2

```
Capricorn
```
