# a882. B. 我灰灰的橡皮鴨是個恐怖的危機

## 內容

小鴨進場已屆倒數十二小時的階段了。在勘過場務之後你感到十分滿意，心想這樣龐大的鴨鴨陣一定能令人精神放鬆，心情愉悅，連作夢都會夢到小鴨，實在是太美好了，原作H先生一定會心存感激！

  展覽除了展場之外，最重要的當然就是小鴨了。雖然你十分放心你私雇的手下、為人老實敦厚的郭房布(有關房布的身世之謎，詳見「2013暑期快樂程設營練習賽系列」，在此不加贅述)，但想想鴨子訂購後自己看都沒看過一眼，畢竟讓人毛毛的，於是你在與長官報告過場務後，隻身前往私藏黃色小鴨的「明光橡皮場」。

  一進儲藏室你倏然大吃一驚，好幾隻素色髒髒灰灰、還沒油漆過的橡皮鴨散亂在地上，而你的部下房布卻醉倒在一旁，整間儲藏室都是酒味！你心想這下完蛋了，得趕快叫他起來漆鴨子才行，鴨子可是再不到半天就要展出了！但是不管你怎麼搖怎麼晃，房布總是像具死屍一般毫無反應。快被酒味薰昏的你又看了看地上，這才發現原來房布幾天來竟已喝了74瓶「酒精120%」，這才舉手投降。

  失去了房布這得力助手，你只好自己來油漆剩下的小鴨了。沉著冷靜的你」大略把小鴨分成了三種情況：已經完全漆好的、根本還沒開始漆的，以及漆到一半的，而你當然只需要處理後兩種就行了。但這時你又發現了件恐怖的事情：喝醉了的房布竟然把所有的油漆也當酒喝掉了！(阿們……)這下你除了得自己油漆之外，還得千里迢迢親自去買油漆來漆才行。時間緊迫的你當然得就近取材，而就你所知這工廠附近就只有「超大桶」一廠在賣油漆，於是自然決定是它了。因為「超大桶」的油漆實在太大桶了，所以漆一隻74 * 74 * 71.22 (m3) 的鴨子只需要兩桶就夠了！

  因為鴨子數量實在太龐大，加上你對今早的場務處理程式非常滿意，於是你打算再一次讓程式來解決你的危機！

## 輸入說明

輸入第一行有一個整數T，代表總共會有幾組測試資料。

  每組測資包含一行字串，其中的每個字元代表每隻鴨子的狀態。字元可能為”O”, ”X”或是”H”，分別代表全部塗滿、完全沒塗或塗了一半。

保證字串長度<=5000

## 輸出說明

對每組測試資料輸出一行，包含一個整數，代表總共需要幾桶「超大桶」油漆。

## 範例輸入 #1

```
3
OOXXHH
XOX
OXXXOXOHHOXOHX
```

## 範例輸出 #1

```
6
4
15
```
