# d316. Quadrangle！

## 內容

已知四边形ABCD的面积为S。E，F，G，H各点分别在四边形ABCD的AB，BC，CD，DA边上。

当 AE:EB=BF:FC=CD:GD=DH:HA=k 时，求四边形EFGH的面积。

## 輸入說明

每行一组数据：S，k（它们都是小于2的31次方的正整数）。

## 輸出說明

对于每组数据，输出四边形EFGH的面积。

如果面积是整数，就直接输出；

如果面积是分数，就按照 5 / 9 输出，分数要化简。

参考 Sample output.

## 範例輸入 #1

```
1 1
```

## 範例輸出 #1

```
1 / 2
```
