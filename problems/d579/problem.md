# d579. 兩條線

## 內容

「真是三條線點點點｜｜｜。。。」

生活中我們常常會發出這樣的感慨，同樣的，在實數的世界中，他們也會發出「兩條線」的感慨，此時它的值也會發生奇妙的改變，如下所示；

|105| = 105

|-270| = 270

|-0.752| = 0.752

之類的。總之，現在請你寫一個程式可以計算某個實數加上「兩條線」後的值。

## 輸入說明

輸入包含一堆實數N，每行一個。小數點後面不會超過4位精度，並且-10000 <= N <= 10000。

## 輸出說明

請見範例。輸出內不含任何空白，且一律輸出到小數點後4位。

## 範例輸入 #1

```
-100.057
874.1
-154.9877
```

## 範例輸出 #1

```
|-100.0570|=100.0570
|874.1000|=874.1000
|-154.9877|=154.9877
```
