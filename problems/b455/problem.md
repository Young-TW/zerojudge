# b455. 請輸出這張圖片的RGB數值（難度↓）

## 內容

![image](ShowImage?id=796)

## 輸入說明

## 輸出說明

請輸出本題的圖片：兩個整數 W H (W = H = 4)，是圖片的寬和高；接下來的 H 行，每行有 W*3 個整數，是每個像素的 RGB 值 (0 <= R, G, B <= 255)。

## 範例輸入 #1

```
無
```

## 範例輸出 #1

```
4 4
226 137 124 206 98 94 211 113 110 222 132 103
163 59 71 237 147 114 218 158 151 202 176 186
177 63 85 231 123 101 161 50 65 108 34 68
87 13 57 130 66 117 130 30 55 182 113 112
```
