# e658. 11614 - Etruscan Warriors Never Play Chess

## 內容

伊特魯里亞戰士的部隊組成如下。
在第一行中，有一個戰士；第二行中，有兩個戰士；第三行中，有三個戰士，依此類推。
通常，第i行包含i個戰士。
我們給定伊特魯里亞戰士的數量。您必須幫忙計算他們可以排幾行。
注意：如果還有一些剩餘的戰士，且他們不足以組成下一行，則該行不列入計算。
例如：擁有6、7、8或9名戰士，您都只能排成3行。

## 輸入說明

輸入的第一行包含一個整數T，代表測資數量。
對於每組測資，都有一個整數n (0 ≤ n ≤ 10^18)，代表伊特魯里亞戰士的數量。

## 輸出說明

對於每組測資，輸出他們可以排幾行。

## 範例輸入 #1

```
6
3
6
7
8
9
10
```

## 範例輸出 #1

```
2
3
3
3
3
4
```
