# e595. 11475 - Extend to Palindrome

## 內容

給定一個整數N，您的任務是使迴文的長度至少為N，任何迴文都可以，很簡單，對吧？
這就是之前您傳遞給經驗不足的隊友的想法。當比賽快結束時，您發現該問題仍未解決。(豬隊友)
程式的問題是生成的字符串通常不迴文。你沒有沒有足夠的時間從頭開始或更改他雜亂無章程式。
看到這種情況你頭很痛，所以你決定簡單地寫一些其他程式來獲取正確的輸出，並在其中添加足夠多的額外字符以使其成為迴文。
你的程式應以字串當輸入，並且產生最小的迴文，該迴文可以通過在其末尾添加零個或多個字元來形成。

## 輸入說明

本題有多行輸入。
每行將包含一個由大寫和小寫英文字母組成的非空字串。
字串的長度將小於或等於100000。

## 輸出說明

對於每行輸入，輸出添加最少數量的字母而形成的迴文字串。

## 範例輸入 #1

```
aaaa
abba
amanaplanacanal
xyz
```

## 範例輸出 #1

```
aaaa
abba
amanaplanacanalpanama
xyzyx
```
