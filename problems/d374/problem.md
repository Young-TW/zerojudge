# d374. 6. X^2 ≡ 1 (mod M)

## 內容

給你一個式子X^2 ≡ 1 (mod M) 和M ( 0＜X≦M )，請找出所有符合這個式子的X。 

例如：M=5，則X 可以等於1 或4；M=8 時，X 可以等於1, 3, 5, 7。
請你寫一個程式，針對每一個M，輸出能滿足這個式子的X。

## 輸入說明

每一個測試檔裡有一個整數即為M，你可以假設M 不會大於2147483647。

## 輸出說明

第一行為一個整數n，代表共有多少組解。
接下來的n 行則為所有滿足此式子的X，並由小到大輸出。

## 範例輸入 #1

```
15
```

## 範例輸出 #1

```
4
1
4
11
14
```
