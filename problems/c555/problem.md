# c555. 瓜有藤，樹有根

## 內容

瓜有藤，樹有根，多項式也有根。請求出給定之多項式方程式的根。

## 輸入說明

本題有多筆輸入


每筆輸入第一行有一個正整數N(1≦N≦16)，為多項式的次數


第二行有N+1個實數，分別是多項式由高次項到低次項的係數


所有係數均能用雙精度浮點數(double)精確儲存(有效位數不超過15位)

## 輸出說明

對於每筆測資，請輸出一個實數x代表此多項式方程式的根，若該多項式方程式有多個解，則輸出任一個解即可


解的定義:若|f(x)|<1e-10則視x為f的其中一個解


保證每個多項式方程式必定有解


若x的有效位數超出double的儲存範圍則會被截斷

## 範例輸入 #1

```
2
1.0 3 2
3
-1.23 0.41 6 -2
1
0.01 0.003
```

## 範例輸出 #1

```
-2.000000000000027
2.208630521496931
-0.3
```
