# f176. 湊不到錢大作戰

## 內容

現在有一個艱難的任務要賦予你，給你兩個錢幣，數值為 m,n ，請你幫忙找出小於等於k且最接近k且沒辦法被 m,n 湊到的數。

## 輸入說明

輸入第一行有一個整數t 表示接下來有幾筆測資(t<=100)


每組測資有兩行


第一行為整數 k (0<k<=1000000)


第二行為整數m n (m,n>0)


所有數皆可用int儲存

## 輸出說明

輸出小於等於 k 且最接近k且沒辦法被 m,n 湊到的數


如果找不到請輸出 "good"

## 範例輸入 #1

```
3
100
3 4
123
5 8
1000000
1 2
```

## 範例輸出 #1

```
5
27
good
```
