# h090. 運算式操作

## 內容

現在，有一個長度為|S|的字串S(|S|<=2*10^5)


S由1~9的數字組成


你可以選擇在S的任意位置插入+號(也可都不插入)，使其變成一個運算式


請求出所有可能產生的運算式結果的總和，除以998244353的餘數

## 輸入說明

一個字串S(|S|<=2*10^5)，包含1~9的數字

## 輸出說明

所有可能產生的運算式結果的總和，除以998244353的餘數

## 範例輸入 #1

```
1234
```

## 範例輸出 #1

```
1736
```

## 範例輸入 #2

```
31415926535897932384626433832795
```

## 範例輸出 #2

```
85607943
```
