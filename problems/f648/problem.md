# f648. 昀善的FB好友

## 內容

昀善喜歡和女生聊天，所以他的FB加了一堆女生好友並每天和他們聊天，不論他認不認識。有天昀善看見一個女生，那個女生深深吸引了昀善的注意，所以他想過去搭訕，但是他在FB上和太多女生聊過天了，他甚至不知道眼前這個人是不是他的FB好友，要是突然跑過去搭訕，可能會發生很尷尬的事情（曾吵過架或是被封鎖過之類的），想想就覺得可怕，所以他想請你幫忙判斷這個人是不是他的FB好友。
昀善會打開手機唸他的FB好友清單給你聽，但是太長了根本記不住！不過既然已經答應他了，也只好想辦法幫他了。他的好友名單中，有些人的名字甚至出現了兩次以上！但是這並不是一件好事，因為這代表昀善被這個人給封鎖了......但是也不用太難過，因為有的人名字雖然很像，但其實只要名字不是完全相同，就不是同一個人。例如：Shanjing和shanjing就不是同一個人。

## 輸入說明

測資為單筆輸入。


第一行有N個以空白隔開且只由英文字母組成的字串ni，代表昀善的FB好友清單。（0<N≤10^6，0<|ni|≤10）


第二行有Q個以空白隔開且只由英文字母組成的字串qi，代表昀善想問你的女生叫甚麼名字。（0<Q≤100，0<|qi|≤10）

## 輸出說明

對於昀善問的每個女生，請依據以下幾種情況回應（輸出）：


如果她是昀善的FB好友，請輸出"Yes"。（不含引號）


如果她不是昀善的FB好友，請輸出"No"。（不含引號）


如果昀善被她封鎖了，請輸出"Pathetic"。（不含引號）


每個回應（輸出）請以換行隔開。

## 範例輸入 #1

```
Jiayi AmandA amanda Zixuan Jiayi
Zixuan Shanjing Jiayi
```

## 範例輸出 #1

```
Yes
No
Pathetic
```
