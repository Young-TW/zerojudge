   共有七行八列。请找出其中的规律。国王想知道第i行第j列的数是多少，这样他就有办法应付宰相的要求了。

## 輸入說明

有多组测试数据，读至EOF结束。数据的组数不会超过64。每行2个数据，i、j。表明国王问你第i行第j列的麦子数是多少。

 

（1≤i≤7,1≤j≤8）

## 輸出說明

对于每一组数据，输出第i行第j列的麦子数是多少。这个数一定不超过2^63-1。

## 範例輸入 #1

```
1 1
1 2
1 3
1 4
2 1
```

## 範例輸出 #1

```
1
2
4
8
256
```
