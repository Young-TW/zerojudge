# k475. 4或7的倍數

## 內容

"你梔道嗎? 805就會是5的倍數，也是7的倍數 !"


"鬆鬆"


"鬆鬆"


"7*4=28"


"鬆鬆鬆"


最近大家流行一種遊戲，若話中有 4, 7的倍數 或 數字中有4或7的話，就要說一次"鬆"

## 輸入說明

輸入兩整數 a, b 一字串 c，表示 "若話中有 a 或 b 的倍數 或 數字中有a或b的話 就要說一次c "。


接下來會有多行字串


- 0<a,b<10
- 字串裡的數字不超過int範圍

## 輸出說明

輸出要回覆的內容

## 範例輸入 #1

```
4 7 鬆
你梔道嗎? 805就會是5的倍數，也是7的倍數 !
7*4=28
9410雖然不是4的倍數或7的倍數，還是要說一次"鬆"!!
```

## 範例輸出 #1

```
鬆鬆
鬆鬆鬆
鬆鬆鬆
```
