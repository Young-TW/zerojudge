# d456. 社辦合併

## 內容

一串電研 ( CIRC ) 和一串數創 ( DDC ) 的社辦終於要合併了！

但是這兩間社辦本來相隔很遠，要合併得搬遷很多物品，為了方便起見，我們決定直接把中間的其他社辦統統鏟掉。

現在用一個字串 S 代表所有的社辦配置，以及兩個字元代表電研和數創的社辦，請你輸出合併後的社辦配置以及被鏟掉的部份。

假設所有社辦如下：

abcxyzdef

而電研為 c ，數創為 d ，則要鏟掉 c 和 d 中間的 xyz ，所以合併後的社辦配置為：

abcdef

而被鏟掉的部份為：

xyz

其餘範例請見輸出輸入範例。

## 輸入說明

第 1 行有 1 個整數 T ( T < 100 ) ，代表有幾筆測試資料。

以下有 T 筆測試資料，每筆包含： 

第 1 行有 1 個字串 S ，長度不超過 10000 個字元，只包含大小寫字母、數字及空白。

第 2 行有 2 個字元， 分別代表電研社辦與數創社辦。

請注意，在 S 中可能會有重複的字元，但電研和數創只會各出現一次。

## 輸出說明

對每筆測試資料輸出2行，第1行代表代表將電研和數創合併後的社辦配置，第2行代表被鏟掉的社辦。

## 範例輸入 #1

```
3
11121113111
23
abcdefgh
ah
Aio56mnK7I 13j980J4poL
JK
```

## 範例輸出 #1

```
11123111
111
ah
bcdefg
Aio56mnKJ4poL
7I 13j980
```
