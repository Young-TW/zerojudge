# e183. 10940 - Throwing cards away II

## 內容

給你一疊有1~n編號的牌，1在最上面；而n在最下面。


在牌數大於1的時候請執行以下操作：


丟掉最上面的牌，並把現在最上面的的牌放到最下面。



求最後剩下的那張牌編號為?

## 輸入說明

每行的輸入包含一正整數 $n \leq 500000$，以$n = 0$結尾。
輸入的行數不大於500001。

## 輸出說明

對於每筆測資，請輸出最後剩下一張牌的編號。

## 範例輸入 #1

```
7
19
10
6
0
```

## 範例輸出 #1

```
6
6
4
4
```
