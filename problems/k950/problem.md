# k950. easy math

## 內容

很簡單，編程一條程序，判斷算式是使用四則運算+乘方的哪一種。如果多種結果皆行，那麼依次輸出。順序是加減乘除+乘方(+ - * /  ** )

## 輸入說明

輸入一共一行，共有若干個數字，除了最後一個以外皆是要運算的數字，最後一個是結果。

## 輸出說明

輸出只有一個符號，是四則運算+乘方的其中一個。

## 範例輸入 #1

```
2 2 4
```

## 範例輸出 #1

```
+
*
**
```

## 範例輸入 #2

```
5 5 1
```

## 範例輸出 #2

```
/
```

## 範例輸入 #3

```
1 2 3 6
```

## 範例輸出 #3

```
+
*
```

## 範例輸入 #4

```
8 4 4
```

## 範例輸出 #4

```
-
```

## 範例輸入 #5

```
2 3 8
```

## 範例輸出 #5

```
**
```
