# k621. [紅]括號匹配問題

## 內容

假設表達式中允許包含圓括號和方括號兩種括號，其嵌套的順序隨意，例如：


( [ ] ( ) ) 以及 [ ( [ ] [ ] ) ] 都是正確的匹配，而 [ ( ] ) 以及 ( ( ) ) ) 都是錯誤的匹配。


本題的任務是檢驗一個給定表達式中的括號是否正確匹配。

## 輸入說明

輸入一行字符s，只有圓括號和方括號，s的字符個數不多於500。

## 輸出說明

若表達式中的括號為正確匹配，則輸出「Right」；


否則就輸出「Wrong」。

## 範例輸入 #1

```
[([][])]
```

## 範例輸出 #1

```
Right
```

## 範例輸入 #2

```
[(])
```

## 範例輸出 #2

```
Wrong
```
