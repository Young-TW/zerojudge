# g220. D.華國探查（research）

## 內容

華國是一個方形的城市，左下角是(0,0)，右上角是(109,109)，裡面有許多店家，店家都位於整數點上。而現在你想要知道有多少組四間店家位於同一條直線上，這樣以後出遊時比較方邊安排行程，而不需要轉彎。

## 輸入說明

輸入第一行有一個正整數 n，代表店家數量。


之後有 n 行，每行有用空格隔開的兩個整數 xi,yi，代表第 i 間店家的座標。


 


測資限制


1. 1<=n<=5000。


2. 0<=xi,yi<=1e9。

## 輸出說明

請輸出一個整數，代表符合題目要求的組數。

## 範例輸入 #1

```
4
1 1
1 2
1 3
1 4
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
8
0 0
4 9
150 301
300000000 600000001
0 1
0 2
0 3
0 4
```

## 範例輸出 #2

```
6
```
