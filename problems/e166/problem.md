# e166. 演唱會記行 - 粉絲應援

## 內容

韓團粉絲最具代表的應該就是應援文化


從一開始的應援物，應援手幅，應援手燈，應援色


到幫偶像取暱稱，自製應援物，刷音源，應援口號


最後還會準備生日應援，廣告應援，慈善應援等等


最近也很流行"逆應援"，就是偶像回饋給粉絲的應援，很有趣吧^^


 


小粉絲難得參加了演唱會，粉絲團們這次想來個集體的大應援，獻給自己心愛的偶像


這次的應援為超級波浪舞，讓偶像再演唱會中場休息時，看到壯觀的波浪舞~~~


不過由於演唱會的座位，沒辦法同一粉絲團坐在一起，所以可能沒辦法展現完美的波浪舞


相鄰座位的波浪會儘量聚集，以製造最大波浪


你能幫助粉絲團，算出最大的波浪會有多高嗎？

## 輸入說明

每一行為一個整數n，代表演唱會場地有多少排座位，n = 0代表輸入結束
0 <= n <= 1000
接下來有n個數字代表第i排的粉絲們的波浪值p(他們當前能製造最高的波浪)
-10000 <= p <= 10000


注意！！！演唱會場地是圓形的！！！

## 輸出說明

輸出最大的波浪會有多高

## 範例輸入 #1

```
5
1 2 -4 4 -5
5
1 2 -1 4 -5
5
1 2 -4 4 5
0
```

## 範例輸出 #1

```
4
6
12
```
