# a845. B.奶罐小馬買東西

## 內容

奶罐小馬是一個熱心助人、酷愛辣妹卻腦袋空空的資訊系大學生。有一天，他到了福利社想要購買飲料，每一瓶飲料有不同的價錢。口渴的他非常想把每一瓶飲料（和辣妹店長）都買下來，但是受限於他胃的容量和荷包，他只能買兩瓶飲料。

　　身為一個資訊系的學生，他當然要利用電腦來協助他計算，他想要知道他購買某兩瓶飲料需要花多少錢呢？這樣他才能對他的零用錢做充分的規劃。但是他的電腦恰巧壞掉了，所以沒有辦法寫程式。聰明的你能幫幫他嗎？

## 輸入說明

輸入檔僅含一組測試資料。

  第一行有一個正整數N(1<=N<=200)，代表福利社有幾種飲料。

  第二行有N個正整數A0~An-1，代表第一瓶飲料的價錢為A0，第二瓶飲料的價錢為A1，第k+1瓶飲料的價錢為Ak。保證飲料價錢小於1000。

  第三行有一個正整數T(1<=T<=200)，代表奶罐小馬接下來要問你幾個問題

  接下來有T行，每一行包含兩個正整數x、y(0<=x.y<N)，代表他想問你購買第x+1和第y+1瓶飲料需要付多少錢。

## 輸出說明

對每個詢問請輸出一行包含一個正整數R，代表奶罐小馬要付多少錢才能買到第x+1和第y+1瓶飲料，也就是Ax+Ay

## 範例輸入 #1

```
5
7 3 2 4 1
3
3 0
1 4
0 2
```

## 範例輸出 #1

```
11
4
9
```
