# k921. 存錢箱

## 內容

小明和我們生活在不同的世界裡，在他的世界裡，任何面值的鈔票都有可能出現。他有一個存錢箱，請你編程一個程序，從大到小排序他的金錢。

## 輸入說明

輸入共有n行，為一個整數a和b，以空格隔開。a為面值，b為多少張。


以EOF為結束。

## 輸出說明

將輸入的金錢從大到小排序，以金錢的面值排序，並在最後輸出總和。詳見範例輸出。

## 範例輸入 #1

```
100 5
200 4
1000 10
456 7
900 5
```

## 範例輸出 #1

```
1000元鈔票共有10張
900元鈔票共有5張
456元鈔票共有7張
200元鈔票共有4張
100元鈔票共有5張
總共有18992元
```
