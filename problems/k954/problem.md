# k954. 一番賞問題 Extreme

## 內容

原題


嘅多和老鼠一起去抽一番賞，老闆看到這兩個人這麼聰明，於是就考了他們一個問題：「我這裡有A賞~Z賞，目前還剩下這麼多，請問你們如果把剩下的都抽光，A賞比裡面還有的其他賞都來得早抽光的方法數有幾種？」，這個問題嘅多和聰明的老鼠在 2 秒內就在頭腦裡面光速用算給出了解答，請寫一個程式打敗他們兩個。

## 輸入說明

輸入一個字串 s (|s| <= $10^6$)，包含 'a' ~ 'z' 的字元，代表剩下的賞的種類，保證 s 裡面至少有一個 'a'。

## 輸出說明

輸出答案對 1000000007 取模。

## 範例輸入 #1

```
cadd
```

## 範例輸出 #1

```
5
```
