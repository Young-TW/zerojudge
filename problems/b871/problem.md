# b871. 又是一些字串

## 內容

接續 b847: 一些字串，你又得到了一些字串，不過不是再要你求出現次數和頻率這麼簡單的問題了，這次的字串中間會有一些空格，請你搜尋看看有沒有字串出現的頻率超過1/2

## 輸入說明

第一行有一個數字T代表測資筆數，接下來每筆測資的第一行會有兩個數N,M分別代表會有幾行和每行幾個字串，在輸入完字串之後會有一個數字Q代表問題數量，每個問題分別有四個數字(a,b)(c,d)代表一個區間，請你求出此區間是否有字串出現頻率高於1/2



```
n,m<=1000
```

## 輸出說明

如果有找到符合條件的字串就輸出該字串，不然就輸出"NO!!"

## 範例輸入 #1

```
1
2 4
YES YES YES YES
YEE yee yes YES
2
0 0 1 3
0 0 1 1
```

## 範例輸出 #1

```
YES
NO!!
```
