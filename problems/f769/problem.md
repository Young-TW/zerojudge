# f769. 可樂磷酸

## 內容

臨末愛喝可樂，特別是含有磷酸的可樂(超愛的)，但可樂實在太多了，所以他找了他的朋友們幫他數共有幾瓶可樂。


今天，請你當他的朋友們，告訴他你看到了幾瓶可樂，又有幾瓶是他超愛的可樂磷酸呢?

## 輸入說明

每個測資點只有一個字串，字串中只會出現F(表示臨末的朋友)、C(不含磷酸的可樂)、CA(臨末最愛的可樂磷酸)。


(字串長度<=1500000)

## 輸出說明

請針對每個臨末的朋友回報他所看到可樂的數量及可樂磷酸的數量(數量間以空白隔開，每個朋友回報完應輸出換行)


//看到指的是他後面所出現的可樂數量，請見範例


//範例1說明:第一個朋友看到一瓶可樂磷酸及一瓶可樂，故輸出2 1(2瓶可樂 1瓶含磷酸)


                 第二個朋友看到一瓶可樂，故輸出1 0(1瓶可樂 0瓶含磷酸)

## 範例輸入 #1

```
FCAFC
```

## 範例輸出 #1

```
2 1
1 0
```

## 範例輸入 #2

```
FFFC
```

## 範例輸出 #2

```
1 0
1 0
1 0
```
