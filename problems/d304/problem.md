# d304. 複製貼上

## 內容

當我們在複製、貼上時 , 都希望按按鍵的次數越少越好

當初在學程式語言的時候 , 一直用 for 去跑星星 ?

現在請你回歸傳統使用暴力

 

一開始會有一顆星星 , 請你告訴我 最少的次數 以及有多少種方式 以及怎麼按

因為 C 離 Ctrl 比較近 所以 C 優先於 V

而且一複製就是複製全部

Ctrl + C 是複製  Ctrl + V 是貼上

## 輸入說明

每行會有一個 n ( 2 ≦ n ≦ 10000 )  代表你要複製到幾顆星星。

## 輸出說明

請輸出最少的按鍵次數、以及最少次數有幾種、以及方法。

輸出方法時 : 越早按 C 的排越前面輸出

測資中: ( 400 筆輸入而已)

次數不超過 10000 次

種類不超過 6000 種

## 範例輸入 #1

```
50
```

min : 12
way : 3
