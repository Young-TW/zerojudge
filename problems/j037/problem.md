# j037. 11854 - Egypt

## 內容

很久以前，埃及人發現，邊長為 3、4、5 的三角形的最大角為直角。


您必須確定其他三角形是否具有相似的屬性。



![image](ShowImage?id=2473)

## 輸入說明

每行有三個數字代表三角形的邊長 (邊長 < 30,000)


輸入 0 0 0 代表結束

## 輸出說明

如果三角形是直角三角形，輸出 "right"


否則，輸出 "wrong"

## 範例輸入 #1

```
6 8 10
25 52 60
5 12 13
0 0 0
```

## 範例輸出 #1

```
right
wrong
right
```
