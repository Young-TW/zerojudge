# g640. 璽羽的壽司

## 內容

璽羽獨自到一座島上並開了一家壽司店，在這座島上的居民認為壽司的價錢等同於其品質，所以島民在買壽司時會依據自己的預算挑選盡可能高價的壽司，若店內沒有符合自己要求的壽司便會直接離開。
由於璽羽是在有客人訂購時才會去買食材來製作壽司，所以每種壽司基本上可以視為無限量供應（但客人可能要等比較久）。
品質越高的壽司往往需要跑到越遠的地方收集食材，所以璽羽會以符合客人要求的壽司中品質最差的優先售出。
 


請寫一個程式計算璽羽賣掉的壽司的總價格。

## 輸入說明

第一行兩個正整數N、M，代表今天有N種不同品質的壽司與M個客人要訂購。(N<=10^6、M<=10^6)


第二行有N個以空白分隔的正整數代表每個壽司的售價。(皆小於2^31)


第三行有M個以空白分隔的正整數代表每個客人能接受的最低品質。(皆小於2^31)

## 輸出說明

請輸出一個正整數，代表今天璽羽今天總共會賣出價值多少的壽司。

## 範例輸入 #1

```
5 4
4 2 8 10 12
3 15 11 4
```

## 範例輸出 #1

```
20
```
