# a702. Cousin Primes

## 內容

如果 p 為質數，且 p + 4 也是質數，則我們說 (p, p+4) 是一對 Cousin prime。例如：(3, 7), (7, 11), (13, 17), (19, 23), (37, 41), (43, 47), (67, 71)…都是 Cousin primes。這個問題是要請你找出第 S 對  Cousin prime。

## 輸入說明

每筆測試資料一列，最多不會超過 10000 列。每列有1個整數 S（1 ≤ S ≤ 100000）。

## 輸出說明

每組測試資料輸出第 S 對 Cousin prime，以 (p1,<space>p2) 的格式表示。其中<space>代表空白字元。你可以放心的假設第 100000 對 Cousin prime 中的質數比 20000000 小。

## 範例輸入 #1

```
1
2
3
4
18888
```

## 範例輸出 #1

```
(3, 7)
(7, 11)
(13, 17)
(19, 23)
(2679037, 2679041)
```
