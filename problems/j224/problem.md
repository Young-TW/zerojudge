# j224. 寶可夢圖鑑

## 內容

文文最近入坑 Pokémon Go 開始抓寶可夢。每抓到一隻新的寶可夢，圖鑑上的寶可夢代號就會變成那隻寶可夢的彩色圖案。上方還會顯示文文一共抓到幾種不同的寶可夢。



![image](ShowImage?id=2527)


給你文文所抓到的寶可夢編號，請問他一共收集了幾種不同的寶可夢？

## 輸入說明

輸入只有 2 行，第一行含有一個整數 𝑛 (1 ≤ 𝑛 ≤ 200)，代表文文抓到了幾隻寶可夢。第二行含有 𝑛 個以空白隔開的整數，代表文文抓到的寶可夢編號，編號介於 1 到 925 之間。

## 輸出說明

輸出一個整數，代表文文抓到了幾種不同的寶可夢。

## 範例輸入 #1

```
11
3 1 4 6 2 8 7 9 5 3 1
```

## 範例輸出 #1

```
9
```

## 範例輸入 #2

```
3
149 149 149
```

## 範例輸出 #2

```
1
```
