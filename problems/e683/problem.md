# e683. 00621 - Secret Research

## 內容

在某個實驗室中，秘密研究的結果被完全加密。
我們將單個實驗的結果分為：
"positive result"(正面)，"negative result"(負面)，"experiment failed"(失敗)，"experiment not completed"(未完成)。
將加密結果構成一串數字S，可以採用以下形式之一：
正面結果：S = 1 or S = 4 or S = 78
負面結果：S = S35
實驗失敗：S = 9S4
實驗未完成：S = 190S
例如：實驗結果S35代表，如果我們將35添加到一個數字的右方，則整串數字表示此實驗為負面結果。
您需要寫一個程式來解密給定的數字序列。

## 輸入說明

第一行有一個整數n，代表加密結果的數量。
接下來n行，每行包含一個數字字串，代表加密的實驗結果。

## 輸出說明

對於每個加密實驗結果，請按照以下輸出：
正面結果："+"
負面結果："-"
實驗失敗："*"
實驗未完成："?"
如果不能確定實驗結果，則輸出上述列表中的第一個匹配的選項。

## 範例輸入 #1

```
4
78
7835
19078
944
```

## 範例輸出 #1

```
+
-
?
*
```
