# b570. 為什麼你們都喜歡撞來撞去的？

## 內容

板燒國是一個大國，板燒國裡有許多城市，每個城市都單獨在一座島上，城市與城市之間有橋連接。甲哥是一名恐怖分子，他很討厭板燒國，他覺得這國家的人都在撞來撞去真是太不爽了。所以他決定要拆散這個國家，於是他想把城市與城市之間的橋給炸掉。他想知道炸掉幾座橋之後，板燒國被分成了幾塊，互相連通的城市屬於同一塊。

## 輸入說明

第一行有兩個正整數n, m (n<=10000, m<=1000000)，代表有n個城市以及m座橋，城市編號為1~n。
 接下來有m行，第1行是編號1的橋，第2行是編號2的橋，以此類推。每行有兩個正整數a, b，代表這座橋連接a, b兩座城市。
 接下來有一個正整數q(q<=m)，代表要毀掉q座橋。
 接下來q行每行有一正整數表示要毀掉編號為幾的橋。
 (已經炸掉的橋不會復原)

## 輸出說明

每炸掉一座橋就輸出一次目前板燒國被分成了幾塊。

## 範例輸入 #1

```
3 3
1 2
2 3
1 3
3
1
2
3
```

## 範例輸出 #1

```
1
2
3
```
