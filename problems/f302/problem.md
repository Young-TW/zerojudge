# f302. 沒有 0 的阿拉伯數字

## 內容

目前世界上最廣為使用的數字文字俗稱「阿拉伯數字」，但是數學老師說阿拉伯數字始創於印度，但當時沒有 0。這時候文文就覺得很納悶，沒有 0 的阿拉伯數字是要怎麼用啊？9 的下一個數字要如何表示呢？


文文在網路上 Google 了一下還是找不到答案，於是他提出了一個簡單的解決方案：跳過所有有 0 的數字。例如 9 的下一個數字是 11，19 的下一個數字是 21，99 的下一個數字是 111。


現在，給你一個「沒有 0 的阿拉伯數字」數字，請把它轉換成我們現在慣用的數字。

## 輸入說明

輸入含有一個「沒有 0 的阿拉伯數字」數字，由數字 1 ~ 9 組合而成。

## 輸出說明

請輸出現在慣用的十進位數字系統中相對應的數字。轉換後的數字不會超過 2147483647。

## 範例輸入 #1

```
19
```

## 範例輸出 #1

```
18
```

## 範例輸入 #2

```
111
```

## 範例輸出 #2

```
91
```
