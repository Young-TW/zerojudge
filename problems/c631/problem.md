# c631. 記憶力試驗

## 內容

窗外，蟬鳴作為夏日的白噪音而存在。
教室裡的學生無不振筆疾書，似乎一停筆便在人生中成為失敗者。
你屏住呼吸，不自覺慢下動作。


許多日子之前，你被告知參加一場記憶力的試驗。
你必須記下一本書的內容，而試驗將抽出其中幾段，請你默寫。
仔細一看，這本書足足有 50,000,000 個字元呢！
於是，確信自己記不住所有內容的你決定在這場試驗中作弊，邊看題目邊翻書抄寫。


多年以後，你會遇到比翻書還要更像作弊的事，
但此時的蟬聲唧唧，應該難以從記憶中抹去了吧。

## 輸入說明

第一行是一個不超過 50000000 的正整數 N ，代表內容長度。


第二行為書本內容，由 ASCII 33~126 的非空白字元組成。


接下來詢問若干個段落，左界為L，右界為R (1 <= L <= R <= N)，請讀到 EOF 結束。

## 輸出說明

輸出詢問的段落。

## 範例輸入 #1

```
18
asworlddkjhellodlk
11 15
3 7
```

## 範例輸出 #1

```
hello
world
```
