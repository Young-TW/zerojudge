# f376. 芝麻街的團購

## 內容

Elmo 揪芝麻街的好友們團購餅乾。他們所住的芝麻街是一條直線，就像美國很多城市一樣，他們的門牌號碼就是房子的座標，兩棟房子的距離就是門牌號碼的差。為了節省大家的時間，他們需要決定把餅乾寄到誰家，好讓每個人去那個人家裡取貨時所走的距離總和最短。

## 輸入說明

輸入只有二行。第一行含有一個整數 𝑛 (1 ≤ 𝑛 ≤ 100000)，代表參與團購的人數。第二行含有 𝑛 個整數，代表參與團購的所有人的門牌號碼 𝑎 (1 ≤ 𝑎 ≤ 109)，每兩個號碼間以一個空格隔開。

## 輸出說明

輸出餅乾應該寄送的門牌號碼。如果有超過一個以上的地點同樣可以有最短的距離，則輸出門牌號碼最小的那一個。

## 範例輸入 #1

```
7
2 7 4 13 3 1 5
```

## 範例輸出 #1

```
4
```

## 範例輸入 #2

```
4
83 119 46 5
```

## 範例輸出 #2

```
46
```
