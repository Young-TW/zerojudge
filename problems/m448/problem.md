# m448. n顆星星

## 內容

印出n顆星星

完成此題後，可接續完成下列題目。
c418. Bert的三角形 (1)
c419. Bert的三角形 (2)
c420. Bert的三角形 (3)

## 輸入說明

輸入只有一行，一個正整數 n ( 1<=n<=100 )

## 輸出說明

輸出一行有n顆星星(*)

## 範例輸入 #1

```
1
```

## 範例輸出 #1

```
*
```

## 範例輸入 #2

```
2
```

## 範例輸出 #2

```
**
```

## 範例輸入 #3

```
5
```

## 範例輸出 #3

```
*****
```
