# e616. Aggressive cows

## 內容

X棠指考戰士的穀倉有$N(2\leq N \leq 100000)$個隔間，它們的位置為$x_1, x_2, x_3 ... x_N (0 \leq x_i \leq 1,000,000,000)$


他的$C(2\leq C \leq N)$隻牛非常討厭這個穀倉，所以變得具有侵略性，會隨意攻擊旁邊的牛，為了保持牛隻的健康，他決定將牠們關在小隔間裡，而他希望牛之間的最小距離要最大。


請輸出最大的最小距離為何。

## 輸入說明

第一行有兩個整數$N, C$。


接下來$N$行有一個正整數，代表隔間的位置。

## 輸出說明

輸出最大的最小距離。

## 範例輸入 #1

```
5 3
1
2
8
4
9
```

## 範例輸出 #1

```
3
```
