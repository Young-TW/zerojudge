# j031. 11934 - Magic Formula

## 內容

給你一個二次函數
f(x) = ax^2 + bx + c
給您一個除數 d 和一個極限值 L。


想請問有多少函數值 f(0), f(1), ......, f(L) 能被 d 整除？

## 輸入說明

輸入由多組測試資料組成


每個測試資料一行，包含數字 a b c d L (−1000 ≤ a, b, c ≤ 1000, 1 < d < 1000000, 0 ≤ L < 1000) 組成


如果輸入 “0 0 0 0 0” 則結束

## 輸出說明

輸出題目所需答案

## 範例輸入 #1

```
0 0 10 5 100
0 0 10 6 100
1 2 3 4 5
1 2 3 3 5
0 0 0 0 0
```

## 範例輸出 #1

```
101
0
0
4
```
