# b524. 先別管這個了，你聽過yee嗎？

## 內容

叭叭叭叭叭叭叭叭～叭叭～～叭叭叭叭叭叭叭～yee。yee由一個y和兩個e組成，實在是太yee了。你有一個只包含"y"和"e"的字串，而且"e"的數量是"y"的兩倍，你可以交換相鄰的兩個字母，請問要交換多少次才能把該字串變成yeeyeeyee......的形式？

## 輸入說明

每組測資有多筆輸入，一筆輸入一行，其中只會有e跟y，且e的數量為y的兩倍，一行最多100000個字元，最多1000行，檔案以EOF做結尾。

## 輸出說明

輸出要交換幾次，一個輸出一行。

## 範例輸入 #1

```
yee
eye
eey
eyeeey
```

## 範例輸出 #1

```
0
1
2
3
```
