# c784. PA. 木棍上的螞蟻

## 內容

現在長度為L的一根木棍上有n隻螞蟻,給定螞蟻的初始位置及初始方向(L為左,R為右),
每隻螞蟻會以每秒1單位的速度向前行走,然而遇到另外一隻螞蟻時,兩隻螞蟻會相撞並同時轉向.
註:兩隻螞蟻同初始位置且同初始方向時,他們是一起前進的好朋友,但遇到一隻相向的螞蟻時,只有一隻會轉向

請問:t秒後還有幾隻螞蟻在木棍上?

## 輸入說明

每個測資點有一個 $T$ $\left( 1 \leq T \leq 10 \right)$,接著有$T$筆測資,


第一行包含$L$ $\left(0 \leq L \leq 11615641 \right)$,$n \left( 0\leq n\leq L \right)$,$t\left( 0\leq t\leq L \right)$. 接下來的$n$行每行包含一個整數$x\left( 0\leq x\leq L \right)$及一個字元$y$代表第$k$隻螞蟻的初始位置及初始方向.

## 輸出說明

對每筆測資，請輸出還有幾隻螞蟻停留在木棍上

## 範例輸入 #1

```
1
10 5 3
3 R
8 R
4 L
6 L
9 L
```

## 範例輸出 #1

```
4
```
