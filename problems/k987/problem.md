# k987. 11549 - Calculator Conundrum

## 內容

愛莉絲有一台老式計算機，共可顯示 n 位數字， 她很無聊地輸入一個數字 k 後，不斷按著「平方」鍵，直到數字溢位，當數字溢位的時候，螢幕上會顯示最前面的 n 位數字，且會有一個表示錯誤的旗標出現。愛莉絲把錯誤的旗標清除後，繼續按著平方鍵…，她想知道：給定 n 與 k，不斷重覆按平方鍵之後，螢幕上顯示最大的數值為何？

## 輸入說明

輸入資料的第一列會有一個整數 t (1 <= t <= 200)表示測試資料的組數。


每組資料有兩個整數 n (1 <= n <= 9) 與 k (0 <= k < 10n)，n 表示螢幕顯示的位數，k 表示初始值。

## 輸出說明

```
請輸出每組測試資料的答案。
```

## 範例輸入 #1

```
4
1 6
2 99
3 45
5 12
```

## 範例輸出 #1

```
9
99
992
99162
```
