# e177. 圍棋進階 - 打劫

## 內容

佐為：打劫，提子!!!


阿光：可惡~~看我把它吃回去


佐為：阿光，你不能這樣，這是打劫，你要隔一手才能吃


阿光：蛤~~~怎麼這樣QQ


 


"打劫"，在提掉對方一顆子之後，儘管我方所落之子只剩一氣，對方也不能立刻提回，必須要在其它地方下子，等到下一手才能提這顆子


此規則如果不這樣硬性規定，那麼雙方即可以反覆提子，對局無法進行下去


由於阿光是初學者，你能幫阿光判斷當前局棋有哪些點正在發生打劫嗎？

## 輸入說明

第一行有一個數字n，代表阿光想判斷的棋盤部份的長寬


n = 0輸入結束，5 <= n <= 19


接下來n行，每行n個字元i代表棋盤狀況


i = '.'代表沒棋子，i = 'O'代表黑棋，i = 'X'代表白棋


注意：(1, 1)為最左下角座標，(n, n)為最右上角座標

## 輸出說明

每行由"Case i:"開頭，代表第幾個Case


接下來輸出正在發生打劫的座標


如果有多組座標


請按照x由小到大排


如果x相同


請按照y由小到大排


如果沒有發生打劫


輸出Peace!!

## 範例輸入 #1

```
5
.XOO.
X.XXO
.XOO.
..XOX
.XO.O
5
.XO..
X.XOO
.XO..
...OX
..OX.
0
```

## 範例輸出 #1

```
Case 1:
Peace!!
Case 2:
2 4
5 1
```
