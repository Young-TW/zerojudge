# f670. FJCU_109_Winter_Day1_Lab5 連通塊數量

## 輸入說明

第一行給定兩個數字 $N, M(0\leq N \leq 10, 0\leq M \leq \frac{N(N-1)}{2})$，表示有 $N$ 個點和 $M$ 條邊，第 $2$ 到 $M+1$ 行，每行有兩個數字 $s_i, t_i$，代表 $s_i$ 和 $t_i$ 之間有一條邊，此圖為無向簡單圖。

## 輸出說明

輸出共有幾塊連通塊，連通塊內的所有點都彼此相通。

## 範例輸入 #1

```
7 5
1 2
2 3
3 4
2 4
5 6
```

## 範例輸出 #1

```
3
```
