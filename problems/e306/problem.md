# e306. Xor 運算 Again!

## 內容

給你一正整數$L$，請找出滿足下列條件的$(a, b)$有幾對?


$a + b \leq L$


$a + b = a \oplus b$


定義 $\oplus$ 為XOR運算。等同於 C/C++ 裡的 ^ 運算子

## 輸入說明

輸入只有一行，有一二進制正整數L(沒有前導零)。


$1_{10} \leq L_{10} \leq 2^{923,000}_{10}$

## 輸出說明

輸出有幾對$(a,b)$滿足條件，由於答案可能很大，請取$10^9 + 7$的餘數。

## 範例輸入 #1

```
第一筆:
10

第二筆:
1111111111111111111

第三筆:
1
```

## 範例輸出 #1

```
第一筆:
5

第二筆:
162261460

第三筆:
3
```
