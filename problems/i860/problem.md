# i860. 13171 - Pixel Art

## 內容

Pixel Art 是一種藝術形式，包括從小方塊或用單一顏色填充的像素。


由於計算機必須代表圖像的限制，它在 80 年代變得非常流行，但即使在今天，它仍然在復古愛好者中流行。


當您擁有所有必要的顏色時，很容易複製這種類型的圖像。
挑戰在於只使用原色：洋紅色(magenta)、黃色(yellow)和青色(cyan)。

## 輸入說明

第一行有一個數字 T 代表測試資料數量


接下來 T 行，每行包含 3 個整數 m、y、c 和 1 個字串 s


3 個整數 m、y、c 代表 洋紅色(magenta)、黃色(yellow)、青色(cyan) 數量


1 個字串 s 代表圖像的 pixel 序列，(1 ≤ s長度 ≤ 100000)


s只包含: magenta (M), yellow (Y), cyan (C), red (R), black (B), green (G), violet (V) and white (W)

## 輸出說明

對於每組測試資料輸出一行


請判斷是否圖像可以由擁有顏色完成


如果可以，請輸出 YES 以及洋紅色(magenta)、黃色(yellow)、青色(cyan) 剩餘數量


否則，請輸出 NO

## 範例輸入 #1

```
3
1 1 1 MCY
3 2 1 MYWWCR
1 1 1 RV
```

## 範例輸出 #1

```
YES 0 0 0
YES 1 0 0
NO
```
