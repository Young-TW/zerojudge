# m924. 賴著不走！抗議啦！

## 內容

天龍國舉行總統大選，金華里共有里民 N 人，每個人都會進投票所投票，然後返家。但選民張大媽因抗議都更計畫補償太少，決定利用此次選舉表達抗議，也就是說張大媽打算進入投票所之後就賴著不走了。
因為投票所走道一次只能通行1人，屬於先進後出的模式，里長牛哥可以說是一個頭兩個大。請問里長牛哥可以有幾種安排里民進出投票所的方案，能夠避開張大媽的抗議擋道，讓每個里民投完票並各自返家。

![image](ShowImage?id=3743)

## 輸入說明

多筆測資(<=10000筆)，每筆測資一行，
一個整數N (包含張大媽)。
(2<= N <=10000)

## 輸出說明

每筆測資一行輸出，代表可行的方案數。
數字很大，所以 mod 1000000007 後輸出。

## 範例輸入 #1

```
2
3
4
```

## 範例輸出 #1

```
2
5
14
```
