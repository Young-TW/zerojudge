# e546. 12650 - Dangerous Dive

## 內容

最近在Nlogonia發生了地震，此地震並未對首都的建築物造成太大影響。
但是科學家發現，此地震影響了水壩外牆，現在水壩外牆的地下部分存在嚴重的結構故障。
如果不及時修復，可能會導致水壩崩塌，以至於淹沒整個首都。
修理必須由潛水員在極其困難和危險的條件下進行維修。由於這關乎著整座城市的生命，所以很多居民自願參加這一危險的任務。
按照危險任務的傳統，每個潛水員在任務開始時都會收到一張帶有ID的小卡片。
任務結束時，志願者會交還這些小卡片，將其放置在存儲庫中。
雖然現在水壩危機已經解除，但不幸的是，似乎有些志願者沒有從任務中返回。
根據放置在存儲庫中的小卡片，您被雇用來計算哪些志願者為拯救城市而犧牲。

## 輸入說明

輸入包含多組測資。每組測資由兩行組成。
第一行包含兩個整數N和R (1 ≤ R ≤ N ≤ 10000)
N代表前往任務的志願者人數，R代表從任務返回的志願者人數。
志願者的ID號碼編號從1到N。
第二行包含R個整數，表示從任務返回的志願者(至少有一名志願者返回)。

## 輸出說明

對於每組測資，輸出犧牲者的ID號碼，並且由小到大排列。
在每個號碼後皆有一個空格。
如果每個志願者都返回，則輸出"*"。

## 範例輸入 #1

```
5 3
3 1 5
6 6
6 1 3 2 5 4
```

## 範例輸出 #1

```
2 4
*
```
