# f673. FJCU_109_Winter_Day2_Lab1 樹高

## 內容

給定一棵二元樹，有 $N$ 個點，$0$ 為樹根，問這棵樹的高度。

## 輸入說明

第一行有一個數字 $N(10\leq N\leq 30)$ 代表有 $N$ 個點，接下來 $N$ 行，每行有三個數字 $u, a, b$ 代表 $u$ 的左子節點是 $a$，右子節點是 $b$，$a,b=-1$ 代表左/右子樹為空，$0\leq u,a,b < N$。

## 輸出說明

請輸出這個樹的高度。

## 範例輸入 #1

```
8
0 1 3
1 6 2
2 4 -1
3 7 5
4 -1 -1
5 -1 -1
6 -1 -1
7 -1 -1
```

## 範例輸出 #1

```
3
```
