# e285. Power of Cryptography (強化版)

## 內容

給你兩個整數 n（n >= 1）和 p（p >=1），你必須寫一個程式來計算出 p 的正 n 次方根。在這個問題裡，p 皆可表成 kn 的形式，其中 k 為整數。（k也就是你的程式所要求的）

## 輸入說明

每組測試資料2列，第1列有1個整數 n（1 <= n <= 100000），第2列有1個整數 p（1 <= p <= 101000000）。 並且存在一個整數 k，（1 <= k <= 4294967295），使得 kn=p。


單一測資檔不超過 20000 組測試資料，且大小不超過 35MB。

## 輸出說明

每組測試資料請輸出 k。

## 範例輸入 #1

```
3
27
7
4357186184021382204544
```

## 範例輸出 #1

```
3
1234
```
