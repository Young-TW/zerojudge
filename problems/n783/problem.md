# n783. 10919 - Prerequisites?

## 內容

大一新生Freddie選擇了修讀k門課程。為了滿足學位要求，他必須從每個類別中選修課程。您能根據他的課程選擇保證Freddie能夠畢業嗎？



![image](ShowImage?id=4073)

## 輸入說明

輸入包含多個測試案例。對於每個案例，第一行輸入包含1 ≤ k ≤ 100，即Freddie選擇的課程數量，以及0 ≤ m ≤ 100，即類別數量。接下來的一行或多行包含k個四位數的整數，每個整數表示Freddie選擇的一門課程的編號。每個類別由一行表示，該行包含1 ≤ c ≤ 100，類別中的課程數量，0 ≤ r ≤ c，該類別必須選修的最少課程數量，以及c個四位數的課程編號。每個課程編號都是四位數的整數。相同的課程可以滿足多個類別的要求。最後一個測試案例後面有一行包含‘0’。

## 輸出說明

對於每個測試案例，如果Freddie的課程選擇滿足學位要求，輸出一行包含‘yes’；否則，輸出‘no’。

## 範例輸入 #1

```
3 2
0123 9876 2222
2 1 8888 2222
3 2 9876 2222 7654
3 2
0123 9876 2222
2 2 8888 2222
3 2 7654 9876 2222
0
```

## 範例輸出 #1

```
yes
no
```
