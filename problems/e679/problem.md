# e679. 11115 - Uncle Jack

## 內容

親愛的傑克叔叔願意將他收藏的CD贈送給他的侄子們。
在CD中，您可以找到非常罕見的Hard Rock，古典音樂，Reggae等專輯；每張CD都是唯一的。
上週，他正在聽他最喜歡的一首歌《沒人管的傻瓜》，突然意識到，他需要謹慎的在許多侄子中分發CD。
因為到目前為止，他還沒有決定要發出去的CD的總數和侄子的數量。而且實際上，有可能某個侄子根本不會發到CD。
基於CD的總數和侄子的數量，請幫助親愛的傑克叔叔，計算他有幾種方式可以發給這些侄子。

## 輸入說明

包含多組測資，每組測資一行。
每行有兩個整數N (1 ≤ N ≤ 10)和D (0 ≤ D ≤ 25)，分別代表侄子數量和CD數量。
如果N = D = 0代表輸入結束。

## 輸出說明

對於每組測資，輸出有幾種發放CD的方式。

## 範例輸入 #1

```
1 20
3 10
0 0
```

## 範例輸出 #1

```
1
59049
```
