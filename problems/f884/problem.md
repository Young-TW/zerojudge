# f884. 柏恩的奶茶

## 內容

柏恩愛喝奶茶，他每天一定會不多不少地喝到4杯奶茶，不然他會暴氣。而每天早上起床時，柏恩的心情值都會變成1（很不爽欸！），所以他會透過喝奶茶來提升自己的心情值，每當他喝下一杯奶茶，他的心情值就會成長為該杯奶茶的甜度倍！（而且在他喝完4杯奶茶後，心情值會再額外+1！）
然而柏恩每喝完一杯奶茶，下一杯的甜度就必須比剛剛那杯多1，不然喝起來就會變得沒味道；但是也不能夠比剛剛那一杯多超過1的甜度，不然他的腸胃無法適應，會害他拉肚子。
由於甜度這麼剛好的4杯奶茶並不容易出現，所以柏恩必須特別委託老闆幫他調製，然而他早忘了今天喝的四杯奶茶甜度是多少了，所以他跑來告訴了你他今天喝完後的心情值，想請你幫他算算看他今天到底喝了甜度多少的奶茶。

## 輸入說明

測資為多筆輸入，直到EOF。


每行一個正整數H，代表柏恩今天喝完奶茶後的心情值。（H<264）

## 輸出說明

針對每個H，請以空白分隔並輸出柏恩今天喝的4杯奶茶的甜度於一行。

## 範例輸入 #1

```
121
25
```

## 範例輸出 #1

```
2 3 4 5
1 2 3 4
```
