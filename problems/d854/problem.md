# d854. NOIP2001 1.一元三次方程求解

## 內容

有形如：ax3+bx2+cx+d=0这样的一个一元三次方程。给出该方程中各项的系数（a,b,c,d 均为实数），并约定该方程存在三个不同实根（根的范围在-100至100之间），且根于根之差的绝对值>=1。
　　要求由小到大依次在同一行输出这三个实根（根于根之间留有空格），并精确到小数点后2位。 提示：记方程f(x)=0，若2个数x1和x2，且x1<x2，f(x1)*f(x2)<0,则在（x1,x2）之间一定有一个根。

## 輸入說明

一行4个数，分别表示题目中的a,b,c,d四个数。

## 輸出說明

一行三个数，精确到小数点后2位，分别表示题目中的三个根。

## 範例輸入 #1

```
1 -5 -4 20
```

## 範例輸出 #1

```
-2.00 2.00 5.00
```
