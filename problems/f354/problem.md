# f354. 好多好多 ~ 好多正方形

## 內容

(參考 e895. 好多正方形 、 e896. 好多好多正方形 )
在一條長為 L 的通道上堆放正方形的貨品，正方形的邊長不可超過 L，且邊長皆為整數(最小邊長為 1)，
貨品也不能相互堆疊。每次都必須把長度 L 的整個通道堆滿。
這次倉庫管理員希望能將貨品堆成左右對稱的型態，試問有多少種的堆放方式？

![image](ShowImage?id=3723)

## 輸入說明

多筆測資(<=10000筆)，每筆測資一行，
一個整數 L (<=1,000,000)
EOF 結束輸入

## 輸出說明

每筆測資一行輸出，即長度為 L 的擺放方案數。
(數字很大，輸出除以 10007 的餘數即可)

## 範例輸入 #1

```
1
2
3
4
```

## 範例輸出 #1

```
1
2
2
4
```
