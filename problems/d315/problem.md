## 輸入說明

每行依序輸入n跟x

其中1<=n<=1031-1；1<=x<=231-1

當n=0時輸入結束，不須處理這組輸入。

## 輸出說明

對每行輸入 輸出sin(x)+sin(2*x)+sin(3*x)...+sin(n*x)的答案

答案以小數點以下4位輸出(%.4f的型態)

## 範例輸入 #1

```
1 60
3 60
2147483647 30
0 0
```

## 範例輸出 #1

```
0.8660
1.7321
3.2321
```
