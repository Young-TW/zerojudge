# q877. 國文&程式?!

## 內容

日前HJ寫114年會考國文題，發現只錯一題(A++)，於是他就信心爆棚(HJ今年國二)，決定將國文與程式結合，這題就這樣誕生了。
以下題目改寫自114年國文會考38,39題

公子光之父曰吳王 諸樊。諸樊弟三人：次曰餘祭，次曰夷眛，次曰季子札。諸樊知季子札賢而不立太子，次傳三弟，欲卒致國於季子札。諸樊既死，傳餘祭。餘祭死，傳夷眛。夷眛死，當傳季子札。季子札逃不肯立，吳人乃立夷眛之子僚為王。公子光曰：「使以兄弟次邪，季子當立；必以子乎，則光真嫡嗣，當立。」 ——改寫自司馬遷《史記》

請問公子光與僚誰傳位順序較前?

## 輸入說明

第一行為正整數n(n<10)
接下來有n行
每行有三種可能
1.a之父曰b
2.a弟m(m<10)人，次曰b，次曰c ...共m人(保證這m+1人之父不會之後才出現)
3.a之子曰b
最後一行有一詢問o,p誰傳位順序較前?
(假設輩分最大者為王，且傳位按輩分傳)

## 輸出說明

輸出o或p傳位順序較前者名字，保證不會有兩者為同輩且祖父不同的情形。

## 範例輸入 #1

```
3
公子光之父曰諸樊
諸樊弟三人，次曰餘祭，次曰夷眛，次曰季子札
夷眛之子曰僚
公子光 僚
```

## 範例輸出 #1

```
公子光
```
