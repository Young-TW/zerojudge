# g352. 函數的秘密

## 內容

$G(p)$ 代表 $p$ 在十進位中前綴 $1$ 的數量。


例如：$G(123) = 1$, $G(1198) = 2$, $G(23432) = 0$


請你計算


$$\sum_{i=1}^{N} G(i) \bmod 998244353$$

## 輸入說明

測資共兩行


第一行為一個整數$X$


第二行為一個$X$位數的整數$N$


$1 \leq X \leq 1000$

## 輸出說明

輸出答案 $\bmod 998244353$

## 範例輸入 #1

```
1
9
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
2
12
```

## 範例輸出 #2

```
5
```

## 範例輸入 #3

```
8
12345678
```

## 範例輸出 #3

```
4691357
```
