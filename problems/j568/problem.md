# j568. 無預警小考 ( 上 )

## 內容

"要上數學言果了!"


學生最害怕的就是所謂看老師心情的 " 無預警小考"


最近老師教到了配方法解一元二次 但出題卻相當麻煩


現在請寫一個程式 幫助老師出題


在本題，只要先幫助老師輸出題目即可


考量到學生解題速度和時間 老師最多出100題就好了

## 輸入說明

輸入第一行為一個整數t (t<=10000), 代表老師要出幾張考卷，接下來每筆測資第一行有一整數n,代表每張考卷的題目數量。接下來輸入每題的三個係數 ( 分別為x^2,x,常數項 )。係數均小於10000。


*x^2的係數不會是0

## 輸出說明

每張考卷第一行請輸出 "請使用配方法解下列一元二次方程式" 並換行。


輸出題號，題目後空一行。


|係數|若為1則不輸出1。


係數若為0則x也不輸出。


每張考卷的結尾請輸出 "考試要加油口屋" 並換行。


請見範例測資。

## 範例輸入 #1

```
1
5
2 3 4
1 2 1
1 -2 -3
1 4 1
29 9410 1919
```

## 範例輸出 #1

```
請使用配方法解下列一元二次方程式

1. 2x^2+3x+4=0

2. x^2+2x+1=0

3. x^2-2x-3=0

4. x^2+4x+1=0

5. 29x^2+9410x+1919=0

考試要加油口屋
```
