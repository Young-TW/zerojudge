# e165. 誰是小隊長

## 內容

n個標兵，站在一條直線上，位置為x1,…,xn，其中xi都是不同的整數。請從中決定k個小隊長，使得所有標兵至其最近的小隊長的距離(位置相差的絕對值)總和最短。請計算最短總距離。

## 輸入說明

第一行輸入一正整數t(1≤t≤10)表示共有t筆測試資料。對每筆測試資料，第一列包含兩個整數n與k, n<=1000, k<=10。第二列包含n個整數Xi,所有Xi均在int範圍內。

## 輸出說明

對每筆測試資料，輸出答案於單獨一列。

## 範例輸入 #1

```
1
5 3
-1 0 2 7 9
```

## 範例輸出 #1

```
3
```
