# l720. 數字組合

## 內容

小明手上有一些數字，他想按照規律求出這些後t個或-t個數字，請你編寫一段程序協助他解決問題。


規律例子 (b): 234→345→456 如此類推。


規律例子 (s): 456→345→234 如此類推。

## 輸入說明

輸入第一行為一個整數l，代表該數字的長度。


第二行為一個整數n，代表該數字。


第三行為一個字母s，和一個整數h， 以空格隔開。


如字母s為b，代表求第h大。


如字母s為s，代表求第h小。

## 輸出說明

輸出只有一行，代表求出的數字。

## 範例輸入 #1

```
3
345
b 2
```

## 範例輸出 #1

```
567
```

## 範例輸入 #2

```
3
678
s 3
```

## 範例輸出 #2

```
345
```
