## 輸入說明

每一行有三個數字n、k、target，代表錢袋數量，需要k袋錢總和剛好為target


1 <= n <= 20，1 <= k <= n，0 <= target <= 2 * 10^7 


接下來一行，每行n個數字i，代表錢袋內部金額


0 <= i <= 10^6

## 輸出說明

如果可以使用k袋錢總和剛好為target


輸出 YES


否則 


輸出 NO

## 範例輸入 #1

```
5 2 3
1 2 3 4 5
5 3 3
1 2 3 4 5
5 4 10
1 2 3 4 5
5 1 0
1 2 3 4 5
```

## 範例輸出 #1

```
YES
NO
YES
NO
```
