<主要題敘>


假設一隊Shadow Bolts(簡稱ACSB,Aircraft the Shadow Bolts)有N名成員，而每位成員都會經過一位名為Mr.J的NLR技師所發明之儀器測量出他們的Wing Power Level(簡稱WPL)為多少，接著求出這隊ACSB的WPL平均值為多少，然後將每位成員的WPL扣掉平均值後求平方相加起來，最後再除以隊員人數後開根號便是這隊ACSB的戰力離散概況；接著，Woona便會幫這隊ACSB安排一場特別訓練，使該隊每位成員的WPL變為原本的a倍加b，此時戰力離散概況又會改變，所以要再幫他們算一次戰力離散概況來看看訓練成效為何。


舉例來說，有一隊ACSB有5名成員，ACSB分別是1 2 3 4 5，
此時先算出原本的戰力離散概況:


平均值 = 3
((1 - 3)^2 + (2 - 3)^2 + (3 - 3)^2 + (4 - 3)^2 + (5 - 3)^2)/5
=2
2^(0.5) = 1.14
此時戰力離散概況便是1。
而經過訓練，假設每位成員的WPL變為原本的2倍加3，此時每位成員的WPL則為:
5 7 9 11 13，訓練後的戰力離散概況也會跟著改變。

## 輸入說明

一開始會有一個整數 T，代表有T隊ACSB要測量 (1≦ T ≦ 10)
每筆測資由一個N開始，表示這隊ACSB的成員數量 ( 1 ≦ N ≦ 10000 )
下一行有N個0到100之間的整數表示每個成員的WPL


接下來有一個Q 表示接下來有幾組訓練的詢問 ( 0 ≦ Q ≦ 100000 )
接下來有Q行，每行有兩個數字a和b為訓練效果的係數 (－10 ≦ a, b ≦ 10 )

## 輸出說明

對於每筆測資，先輸出原本的戰力離散概況，再分別輸出原始WPL經過該訓練後的新戰力離散概況，所有輸出皆保留兩位小數。各組資料後請輸出一行空白。
