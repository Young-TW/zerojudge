# e173. 圍棋入門 - 真眼假眼

## 內容

阿光：佐為，這個眼是真眼還是假眼呀？


佐為：假的


阿光：真的假的？


佐為：真的


阿光：!@#$%^&*()_


 


"眼"，由同一方棋子佔有最近4個相鄰點


"真眼"，除了上述條件，對角4個相鄰點至少要有3個亦為同一方棋子佔有


否則，則容易被對手破壞，容易被破壞的點稱之為"假眼"


就邊緣而言，最近及對角共5個相鄰點須為同一方棋子佔有才是真眼


就角落而言，最近及對角共3個相鄰點須為同一方棋子佔有才是真眼


今天佐為想訓練阿光，你能幫阿光判斷佐為指的點是真眼還是假眼嗎？

## 輸入說明

第一行有一個數字n，代表佐為的考試棋盤部份的長寬


n = 0輸入結束，5 <= n <= 19


接下來n行，每行n個字元i代表棋盤狀況


i = '.'代表沒棋子，i = 'O'代表黑棋，i = 'X'代表白棋


接下來有一個整數q，代表佐為的詢問次數


1 <= q <= 100


接下來q行，每行兩個整數x、y代表棋盤座標


1 <= x,y <= n


注意：(1, 1)為最左下角座標，(n, n)為最右上角座標

## 輸出說明

如果此點為真眼


輸出 Real!


如果此點為假眼


輸出 Fake!


如果此點不是眼位


輸出 Interesting...

## 範例輸入 #1

```
5
XX...
X.X..
XXXO.
OOO.O
.O.O.
4
1 1
4 2
2 4
5 5
0
```
