# b569. 總之就是一串燈泡

## 內容

目前有n個燈泡由左至右排成一排，從最左邊開始燈泡編號為1, 2, ..., n。編號為1的燈泡每1秒會變換一次狀態(亮->暗 或 暗->亮)，編號為2的燈泡每2秒會變換一次狀態，編號為k的燈泡每2k-1秒會變換一次狀態。給你起始狀態以及目前狀態，求下一秒的狀態。

## 輸入說明

輸入共兩行，第一行是初始狀態，第二行是目前狀態。每一行有一個只包含0跟1的字串，由左邊開始第1個0或1代表編號1的燈泡的狀態，第2個0或1代表編號2的燈泡的狀態以此類推。0代表燈泡是暗的，1代表燈泡是亮的。每一行長度不超過10000000。保證兩行一樣長。

## 輸出說明

輸出下一秒的狀態。

## 範例輸入 #1

```
00000
00000

----------------
10111
11111
```

## 範例輸出 #1

```
10000

----------------
01111
```
