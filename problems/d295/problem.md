# d295. 算算算....Normal

## 內容

本題也是在算矩形內有幾個矩形，但本題有一點不同就是矩形內會有一個虧格，什麼是虧格呢？虧格就是，多個矩形變成一格，也是一個矩形。

## 輸入說明

每筆測資皆有兩行。

第一行有八個數字，x1, y1, x2, y2, x3, y3, x4, y4，為大矩形的周圍四點的座標。

第二行有八個數字，x5, y5, x6, y6, x7, y7, x8, y8，為虧格的周圍四點的座標。

## 輸出說明

有幾個矩形。

## 範例輸入 #1

```
0 0 9 0 9 9 0 9
3 3 6 3 6 6 3 6
```

## 範例輸出 #1

```
1192
```
