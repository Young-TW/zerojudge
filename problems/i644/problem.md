# i644. 列舉八皇后問題所有解

## 內容

西洋棋得棋盤中你可以放置8個皇后而且彼此都不衝突（就是都不能吃到對方）。請你寫一個程式來輸出所有這樣可能的安排。


為了把棋盤標準化，我們定義棋盤最左上角的位置為（1,1）。所以下圖黑色方塊的位置為（4,6），代表第4列（ROW），第6行（COLUMN）。


  
![image](ShowImage?id=2289)

## 輸入說明

本題無輸入

## 輸出說明

輸出每一種解答一列，在這裡只輸出列的位置，行的位置則以這8個數字出現的順序表示。例如：若以某解答8個皇后的位置分別為(1,1)(5,2)(8,3)(6,4)(3,5)(7,6)(2,7)(4,8)來說。請輸出：15863724


總共有92組解答，請按照字典順序由小到大排列。請先輸出這是第幾組解答，再輸出答案。格式請參考輸出範例。
