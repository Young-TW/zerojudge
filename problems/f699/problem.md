# f699. 品嘉的茶葉蛋

## 內容

品嘉愛吃茶葉蛋，只要有茶葉蛋出現在他面前，他就會忍不住衝上前狼吞虎嚥地吃掉它。
現在品嘉有張地圖，記錄著茶葉蛋的位置，品嘉想請你幫他算算看每個茶葉蛋距離他有幾步遠。
 


品嘉吞茶葉蛋辛酸照片流出。

## 輸入說明

每個測資點僅單筆輸入。


第一行有兩正整數N、M，代表地圖大小為N×M。（N,M<=100）


接下來有N行，每行有M個數字Pi，Pi的值可能為0~2其中一個數字，代表意義如下：


若其值為0，代表此處為可通行的路。


若其值為1，代表此處有一顆茶葉蛋。


若其值為2，代表此處有障礙物，無法通行。

## 輸出說明

針對每顆茶葉蛋，請輸出若品嘉從起點(0,0)出發，要走幾步可以吃到它。（由步數近到遠輸出並以換行隔開）


但若品嘉無法吃到任何茶葉蛋，請輸出"嘉油！"（不含引號）。

## 範例輸入 #1

```
2 3
0 1 0 
0 2 1 
```

## 範例輸出 #1

```
1
3
```

## 範例輸入 #2

```
3 3
0 0 0
0 2 2
0 2 1
```

## 範例輸出 #2

```
嘉油！
```
