# r709. 進階的三角形迷宮

## 內容

在一座古老的遺跡中，你發現了一張奇怪的「三角形迷宮」地圖。
迷宮由許多層組成，第i 層有 i 個房間，因此整個迷宮總共有1+2+⋯+N個房間，稱為 第 N 階三角形迷宮。

你從第 1 層進入迷宮，想要知道：
走完整個迷宮所有房間的「權重總和」是多少？

每一層的房間有不同的權重：

- 
第 i 層的每一個房間權重都是 i^2 + i。

請你輸出這個總和。

## 輸入說明

一行整數 N

1≤N≤10^6

## 輸出說明

輸出總權重（整數）。

## 範例輸入 #1

```
3
```

## 範例輸出 #1

```
50
```

## 範例輸入 #2

```
1
```

## 範例輸出 #2

```
2
```

## 範例輸入 #3

```
5
```

## 範例輸出 #3

```
280
```
