## 輸入說明

測資範圍
4 < N ≤ 100000
0 < M < 16
Σ(開始調度前，各驛站馬匹數量總和) = N*M

1 ≤ i, j ≤ N
0 ≤ Cij ≤ 100
0 ≤ Lij ≤ 1000



子題配分
30%：N ≤ 50
70%：無特別限制

3 4 2 160
1 3 2 121
