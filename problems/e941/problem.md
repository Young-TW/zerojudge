# e941. pB. 分數式運算

## 內容

給定一行分數的運算式，請輸出對應的運算結果。

## 輸入說明

輸入運算式 s，其中分數與運算符號間皆由空格隔開。

對於 s，
出現的分數有 N 個（1 ≤ N ≤ 5），分數格式為 X/Y，X 和 Y 皆為正整數（1 ≤ A, B ≤ 100）。
運算符號則只有 + 和 -。

## 輸出說明

運算結果需表示為最簡分數，分數格式為 X/Y，若有負號則請加在分子

## 範例輸入 #1

```
1/2 + 1/3 - 1/6
```

## 範例輸出 #1

```
2/3
```

## 範例輸入 #2

```
1/3 - 1/2
```

## 範例輸出 #2

```
-1/6
```

## 範例輸入 #3

```
2/4 - 1/2
```

## 範例輸出 #3

```
0/1
```
