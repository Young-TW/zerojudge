# g396. 老鼠的噩夢

## 內容

令 $f(x)$ 為 $x$ 在二進位中最長連續 $1$ 的數量。


例如：


$f(14) = f(\text{"}\textbf{111}\text{0"}) = 3$


$f(13) = f(\text{"}\textbf{11}\text{01"}) = 2$


$f(5) = f(\text{"}\textbf{1}\text{01"}) = 1$


給你一個二進位的整數 $n$, 求 $\sum_{i=1}^{n} f(i) \bmod 998244353$

## 輸入說明

輸入只有一行，為一個長度為 $m$ 的二進位整數 $n$


$1 \leq m \leq 200$

## 輸出說明

輸出答案 $\bmod 998244353$

```
1
```
