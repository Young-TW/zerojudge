# e170. 粉絲中毒 - 花絮周邊TV影集

## 內容

看了那麼多V Live和飯拍影片，小粉絲還是不滿足


小粉絲決定連花絮影片、周邊影集、各種TV影片都拿來看


在其中，小粉絲有一個習慣，他習慣把所有影片都打分數，分數為1~100分


今天小粉絲不想只挑兩部影片來看，這樣太少了>/////<


他決定把影片排成一排，選定一個頭一個尾(頭尾可相同)，然後把排好的影片從頭到尾都看


不過這所有影片分數相加一定要是100分或者100的倍數


這樣小粉絲的心情才會100分


由於影片數量太多了，你可以幫助小粉絲，讓他知道他有幾種選擇方法嗎？

## 輸入說明

每一行為一個整數n，代表有多少個影片


0 <= n <= 50000，n = 0代表輸入結束沒影片可看


接下來有n個數字p，代表排好的影片分數


每個數字代表第i部影片的分數


1 <= p <= 100

## 輸出說明

```
輸出小粉絲共有幾種挑選方法
```

## 範例輸入 #1

```
5
1 2 3 4 5
5
1 2 98 99 100
5
100 100 100 100 100
0
```

## 範例輸出 #1

```
0
4
15
```
