# b974. 3.任務二->誰是音樂達人

## 內容

這關是要各組派出一位代表猜歌，此遊戲規則如下:


1.最先猜對的得一分


2.得最高分的獲勝


說到計分，活動那時是用紙本紀錄，但allem40306學長想要寫程式紀錄，他邀請你一起來寫，以下是程式他所要做的

## 輸入說明

第一行有正整數n和m，0<n,m<=20，代表接下來有n題及m組，接下來有n行，每行第一個字串代表正確答案，接下來有不超過m個以"x s"形式存在的數字(x<=20)和字串(|s|<=100)，代表第x隊及他們所答的答案，請參考範例輸入

## 輸出說明

請輸出勝利的隊伍(分數最高的獲勝，如果有兩組以上同分，請依隊伍編號大小輸出，隊伍編號間空一格)及最終結果(依編號大小輸出分數)，請參考範例輸出

## 範例輸入 #1

```
3 5
twice 1 once 2 orange 3 twice
cheer 3 up 4 baby 5 baby 2 cheer
up 2 aoa 3 2pm 4 up
```

## 範例輸出 #1

```
2 3 4
1 0
2 1
3 1
4 1
5 0
```
