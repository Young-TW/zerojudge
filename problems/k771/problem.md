# k771. 請認真做好值日!

## 內容

小明是今天的值日生，但是他做事很懶，因此隨便做了幾下就準備回家。不料值日組長小光在門口攔住他，要他認真做好值日才能走。小明央求說自己要做功課，很趕時間。小光見他可憐，便讓他從幾件工作裡挑幾件來做，每完成一項就有積分，積分要達到一個標準才能走。小明想在他最少的時間內達到標準，請幫幫他。

## 輸入說明

第一行有三個數字n,m,o，n是小明最多可用的時間，m是要達到的分數，o是有多少件工作。


接下來有o行，每行有兩個數字x,i。x是完成需要的時間，i是完成可以獲得的分數。

## 輸出說明

輸出只有一行，分別是需要的時間和可獲得的分數，以空格隔開。

## 範例輸入 #1

```
8 10 4
4 3
2 4
5 6
3 2
```

## 範例輸出 #1

```
7 10
```
