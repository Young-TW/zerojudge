# l722. 字串排列

## 內容

小明手上有一些字串，他想改變他們的排列，請你編寫一段程序，協助他。


舉例，字串為book，請看下方表格。


 n / tt1t2t3t4n1 bboboobookn2ooboboobokn3oooooboobkn4kkokookoob

## 輸入說明

輸入第一行為一個整數l，代表該字串的長度。


第二行為字串s。


第三行一個整數n，代表字串s的第n位。


第四行為一個整數t，詳見範例輸入。

## 輸出說明

輸出只有一行，為字串p，詳見範例輸出。

## 範例輸入 #1

```
4
book
2
3
```

## 範例輸出 #1

```
obo
```

## 範例輸入 #2

```
4
book
1
2
```

## 範例輸出 #2

```
bo
```

## 範例輸入 #3

```
4
book
1
1
```

## 範例輸出 #3

```
b
```

## 範例輸入 #4

```
3
abc
2
3
```

## 範例輸出 #4

```
bac
```
