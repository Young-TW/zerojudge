# e655. 10852 - Less Prime

## 內容

令n為 100 ≤ n ≤ 10000 的整數。
請找出一個質數x，x ≤ n，使(n - p*x)為最大值。
其中p為整數，且p*x ≤ n < (p + 1)*x。

## 輸入說明

輸入的第一行包含一個整數M，代表測資數量。
每組測資一行，每行包含一個數字N (100 ≤ N ≤ 10000)。

## 輸出說明

對於每組測資，輸出質數x。

## 範例輸入 #1

```
5
4399
614
8201
101
7048
```

## 範例輸出 #1

```
2203
311
4111
53
3527
```
