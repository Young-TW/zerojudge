# d401. B-成績單

## 內容

海綿寶寶有一天跟派大星出去玩，一起去玩堆沙子，海綿寶寶提出一個建議看誰堆出最高又最多的沙堆，於是兩個人堆了好多堆的沙子，一共堆了滿山滿谷沙堆，這要怎麼辦勒，是誰贏阿???身為他們的好友章魚哥，要幫他們找出誰輸誰贏，章魚哥要他們丈量每堆的高度，交給章魚哥，但是章魚哥很爛，請你們幫幫他，寫一個程式來判斷誰贏，判斷的規則是章魚哥說1個數字K，代表找出他們第K高的來決定。

## 輸入說明

第一行有N(0<N<=1000000)代表它們堆了幾堆，接下來的N行會給兩個數字a,b(a=1 or a=2 0<b<10000000)a=1代表是海綿寶寶堆的a=2代表是派大星堆的，b是這堆的高度(CM)，接下來的有一個數字K代表是章魚哥說的數字，保證有第K堆高的。

## 輸出說明

請輸出誰贏，贏了幾公分，以空白隔開，見範例。

## 範例輸入 #1

```
3
1 2
1 3
2 1
1
```

## 範例輸出 #1

```
1 2
```
