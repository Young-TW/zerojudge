# f351. 接觸不良的空白鍵

## 內容

文文的電腦鍵盤上的空白鍵有點接觸不良，按一個空白鍵常常都會跑出很多個空白，還得用倒退鍵把多餘的空白刪除才能繼續下一個字。他午夜前就要上傳一篇報告，現在急得像熱鍋上的螞蟻，要麻煩你幫忙把文件中多餘的空白刪掉。

## 輸入說明

輸入只有一行，含有文文所輸入的文字，行頭及行尾沒有空白。

## 輸出說明

將所有的連續空白轉換成一個單一空白後輸出。

## 範例輸入 #1

```
The quick  brown   fox    jumps     over      the       lazy        dog.
```

## 範例輸出 #1

```
The quick brown fox jumps over the lazy dog.
```

## 範例輸入 #2

```
Banqiao    Senior  Highschool
```

## 範例輸出 #2

```
Banqiao Senior Highschool
```
