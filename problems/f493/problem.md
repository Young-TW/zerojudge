# f493. 水窪問題

## 內容

給你一個地圖N*M，水坑是W，地是#，求最大/最小連續水漥多大、總共有幾個水漥


水漥是連續的水坑


N,M<=1000

## 輸入說明

第一行為N,M


之後N行為地圖

## 輸出說明

輸出最大水漥、最小水漥、總共有幾個水漥

## 範例輸入 #1

```
4 4
W#WW
#W#W
#W#W
W#W#
```

## 範例輸出 #1

```
4 1 5
```

## 範例輸入 #2

```
5 5
W#WWW
WW#W#
#W#WW
WWW#W
####W
```

## 範例輸出 #2

```
8 7 2
```
