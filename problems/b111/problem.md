## 輸出說明

預期一個字元編碼的長度。精確至小數點後兩位，小數點後第三位四捨五入。

## 範例輸入 #1

```
4
a 0.1
b 0.1
? 0.3
8 0.5
6
* 0.3
b 0.3
< 0.05
H 0.25 
( 0.05 
h 0.05
```

## 範例輸出 #1

```
1.70
2.25
```
