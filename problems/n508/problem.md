# n508. 區間分割演算法練習

## 內容

校內接下來會辦幾場演講，校長想請你寫個程式計算最少需要用到幾間教室才能順利辦完所有演講。

## 輸入說明

每一筆測資第一行會有一個正整數N，代表總共有幾場演講。（N <= 100000）


接下來N行會有每一場演講的資訊，格式為 "Lecture [演講編號]: [開始時間]-[結束時間]" 。

## 輸出說明

請輸出一個整數C，代表最少需要用到幾間教室。

## 範例輸入 #1

```
3
Lecture 1: 9:10-10:00
Lecture 2: 9:30-11:30
Lecture 3: 10:00-11:00
```

## 範例輸出 #1

```
2
```
