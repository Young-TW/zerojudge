# m309. 外星時間

## 內容

有一顆外星人的星球，他們的時間單位稱為外星小時，外星分，外星秒。
和地球不一樣的是，他們的1外星小時=32外星分，1外星分=16外星秒。
(地球的是1小時=60分，1分=60秒)
而1外星秒=地球的4秒。
現在給你n個外星時間，請你換算成n個地球時間(超過24小時不用換算成天)。

## 輸入說明

輸入的第一行為數字n(1<=n<=100)，代表接下來會有n行的外星時間。
每一行的外星時間有三個整數，依序為外星小時h、外星分m、外星秒s，三數字以空白隔開(0<=h,m,s<=100)。

## 輸出說明

根據每一行的外星時間，請輸出轉換後的地球時間，格式為「小時:分:秒」，請看範例輸入輸出。

## 範例輸入 #1

```
4
0 0 1
0 1 0
24 0 0
48 0 0
```

## 範例輸出 #1

```
0:0:4
0:1:4
13:39:12
27:18:24
```
