# e680. 11947 - Cancer or Scorpio

## 內容

Alice和Michael是一對年輕夫婦，正計劃生育他們的第一個孩子。
他們的願望是他們的兒子Nelson在一個特殊的日子出生。
Alice在網路上進行了搜尋，發現妊娠期為40週。
也就是在最後一個月經週期的第一天開始計算這四十週。
Michael對占星術以及十二星座非常有興趣，他想知道他的兒子會是什麼星座？
Michael調查了每個星座相對應的日期範圍如下：



![image](ShowImage?id=4062)


Alice和Michael對你尋求幫助，希望你計算他們的兒子Nelson的出生日期，以及他的星座。

## 輸入說明

輸入第一行包含一個整數N (1 ≤ N ≤ 1000)，代表測資數量。
每組測資一行，每行包含八位數字，格式為MMDDYYYY，這些數字代表最後一個月經週期的第一天。

## 輸出說明

對於每組測資，輸出測資編號，出生日期，格式為MM/DD/YYYY和星座名稱。
注意：需要考慮閏年。

## 範例輸入 #1

```
2
01232009
01232008
```

## 範例輸出 #1

```
1 10/30/2009 scorpio
2 10/29/2008 scorpio
```
