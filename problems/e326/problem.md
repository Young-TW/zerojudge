# e326. 小崴的大數MOD挑戰!!

## 內容

對 輸入的 a,b,c,d


算出 (a^b) % (c^d) !!!


 


//----- 2019/7/18 感謝 vincent97198 提醒，已加強測資!! :)

## 輸入說明

a b c d 四個數 用空格 分開 (一組)


(a,b<=100  /  c,d<=1000)

## 輸出說明

(a^b) % (c^d)


每三位 空一格

## 範例輸入 #1

```
3 100 2 100
```

## 範例輸出 #1

```
503 611 859 755 855 824 366 132 007 889
```
