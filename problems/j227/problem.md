# j227. 我愛蝸牛大贈獎

## 內容

蝸牛飲料公司推出收集瓶蓋大贈獎活動。活動期間集滿印有「我、愛、蝸、牛」等文字的瓶蓋各一枚，就可以兌換一個蝸牛公仔。


給你文文所收集到的瓶蓋上的文字，請問他可以兌換幾個公仔？

## 輸入說明

輸入的第一行含有中獎所需的所有單字。第二行含有文文所有的瓶蓋上的單字。 每兩個單字間以一個空白隔開。

## 輸出說明

輸出文文可以兌換的公仔數量。

## 範例輸入 #1

```
i love snail
i love snail snail snail
```

## 範例輸出 #1

```
1
```

## 範例輸入 #2

```
one for all
all for one one for all
```

## 範例輸出 #2

```
2
```
