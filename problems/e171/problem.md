# e171. 粉絲心願 - 參加音放節目

## 內容

音樂放送節目，簡稱音放


韓國有五大音樂放送節目


分別為星期三的Show Champion，星期四的M!Countdown，星期五的音樂銀行(Music Bank)，星期六的音樂中心(Music Core)，星期日的人氣歌謠(Inkigayo)


每個歌手都會參加音放節目相互較勁，每個節目也有個別的計分方式，如果有幸成為第一名，將會對歌手的生涯產生莫大的影響


在音放節目中，粉絲也是非常重要的，除了表演過程中的應援，還有重要的粉絲投票，讓自己喜歡的歌手們，拿下當周的一位


身為小粉絲，一定要去參加呀~~~~


在現場，工作人員都會算好要進入會場的粉絲群人數，讓最大群的粉絲先行入場，以加快入場動作


現在工作人員有多餘的票卷，他想發給來到現場卻沒有票的粉絲們


工作人員想盡量讓最大群的粉絲人數增加，這樣入場就能加快很多


你可以幫助工作人員計算當前最大粉絲群人數嗎?

## 輸入說明

每一行為一個整數n，代表有多少粉絲，n = 0代表沒有粉絲輸入結束


1 <= n <= 10000


接下來有n個數字代表第i位粉絲手上票卷號碼p


1 <= p <= 1000000，票卷號碼有可能相同


如果粉絲手上沒有票卷，以0表示


接下來有一個整數k，代表工作人員手上的多餘票卷數量

## 輸出說明

當前能進去的最大群粉絲人數

## 範例輸入 #1

```
5
1 2 0 0 5
0
5
1 2 0 0 5
1
5
1 2 0 0 5
2
0
```

## 範例輸出 #1

```
2
3
5
```
