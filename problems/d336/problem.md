# d336. 一即是全、全即是一

## 內容

「鍊成陣的基本是『圓』的力量，『圓』代表著力量的循環。」

 

Edward和Alphonse在無意間發現了一個破損的鍊成陣，雖然無法確定其是否能使用，不過他們兄弟倆卻對這鍊成陣的基本架構有些了解；鍊成陣是由一個圓所環繞起來，圓上寫有一串由0與1構成的二進位制數字，而且只有在這串數字能被3整除時，鍊成陣才有啟動的可能性。

給你這一串數字，請問鍊成陣是否可能成功發動？

## 輸入說明

第一行有一個數字t(t<=10000)代表有t筆測資，

接下來的t行，每一行有一串二進位制數字，長度不超過9000。

## 輸出說明

對每一行二進位制數字輸出，

如果鍊成陣有發動的可能，輸出"Yes"，反之則輸出"No"。

## 範例輸入 #1

```
3
1
11
10
```

## 範例輸出 #1

```
No
Yes
No
```
