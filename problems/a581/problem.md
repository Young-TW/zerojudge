# a581. 祖靈的煩惱  ˊ^ˋ

## 內容

小小高中生 "祖靈"，上課的時候總是愛在底下玩 "Defender II"。

老師對此感到非常的不滿意，於是決定要給他一個方程式。

p*e-x + q*sin(x) + r*cos(x) + s*tan(x) + t*x2 + u = 0 

而要找的x的範圍是 0<=x<=1

老師給了他很多組的p,q,r,s,t,u，要祖靈告訴她到底答案是什麼。

上課都沒在聽的祖靈當然不會啊!!

請你們幫幫他!!

好讓祖靈通過老師的試煉XD繼續上課玩 "Defender II" !!!!!!

## 輸入說明

每一筆測資有六個整數，分別是p,q,r,s,t,u。

範圍： 0 <= p,r <= 20 ，  -20 <= q,s,t <= 0 。 

總共有2000筆測資，以EOF終止程式。

## 輸出說明

解出方程式中的 x (小數點下第四位，第五位四捨五入)。如果在0<=x<=1=而無解輸出 "No solution" (不含引號)

## 範例輸入 #1

```
0 0 0 0 -2 1
1 0 0 0 -1 2
1 -1 1 -1 -1 1
```

## 範例輸出 #1

```
0.7071
No solution
0.7554
```
