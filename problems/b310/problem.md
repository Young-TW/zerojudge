# b310. 英靈召喚

## 內容

決定好Servant後，接下來就要開始召喚啦。因為你才剛成為魔法師，自身的魔力很少，而召喚儀式需要耗費很大的魔力，所以你決定使用大地的魔力。大地每一秒鐘都會放出魔力，但是每次放出的魔力量並不一定，沒有被蒐集的魔力會散發到空氣中，並不會保存下來。你需要建造一個魔法陣來蒐集大地釋放的魔力，蒐集到足夠的魔力候你就可以召喚你的Servant了，但是蒐集魔力的期間是很危險的，光維持魔法陣你就已經用盡全力了，若是被其他魔法師攻擊根本無力防禦或反擊，然後就GG。所以你希望蒐集魔力所耗的時間越少越好。你現在知道接下來N秒每一秒大地會放出的魔力，你最少需要幾秒才能把所需的魔力蒐集完畢?
 
蒐集魔力的時間必須是連續的。

## 輸入說明

第一行有兩個整數N、M(0<N<=106，0<=M<=109)，M為所需的魔力
下一行有N個數字A1~AN，數字Ai代表第i秒大地會放出的魔力(0<=Ai<=109)

## 輸出說明

請輸出最少要花幾秒才能蒐集完魔力
若N秒內無法蒐集完所需的魔力則輸出"GGGGGZ"(不含雙引號)

## 範例輸入 #1

```
6 8
1 3 2 4 0 3
```

## 範例輸出 #1

```
3
```
