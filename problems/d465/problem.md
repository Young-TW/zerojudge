# d465. 不幸的你(求幂系列题7)

## 內容

“我们分手吧！...”

    老Z坚决不肯。

    “...除非你能告诉我a的n次幂是多少，我给你三小时时间思考。”

    老Z大脑爆了。九天后，

    “锦瑟无端五十弦，一弦一柱思华年。

    庄生晓梦迷蝴蝶，望帝春心托杜鹃。

    沧海月明珠有泪，蓝田日暖玉生烟。

    此情可待成追忆，只是当时已惘然。” 

如果它发生在你身上，你怎么解决呢？

    

    ——你只有找你的朋友编程帮你。（幸好他会）

## 輸入說明

a n i k（1<a,n<10000,1<=i,k<=30000）

## 輸出說明

a的n次幂从第i位起连续k位,不够则输完。

## 範例輸入 #1

```
    3 536 135 56
```

## 範例輸出 #1

```
    77353152383916167705006185104046144992000730527283912844
```
