# i840. 字元矩陣轉置

## 內容

請將所給字元矩陣轉置後輸出。

## 輸入說明

一個 5 × 5 的字元矩陣。

## 輸出說明

轉置後的字元矩陣。

## 範例輸入 #1

```
ABCDE
FGHIJ
KLMNO
PQRST
UVWXY
```

## 範例輸出 #1

```
AFKPU
BGLQV
CHMRW
DINSX
EJOTY
```

## 範例輸入 #2

```
SIRHO
NSTAM
ASANE
IMND!
LADS!
```

## 範例輸出 #2

```
SNAIL
ISSMA
RTAND
HANDS
OME!!
```
