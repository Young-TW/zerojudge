# b311. 強襲堡壘

## 內容

你現在已經有強大的Servant了，接下來當然是要開始戰鬥了阿。這時你發現，其他Master已經建好數個堡壘，而且每個堡壘會吸收大地的魔力而越來越強大。而一個強度為k的堡壘，你的Servant就需要花k單位時間去摧毀它。在你正在摧毀其中一個堡壘時，其他堡壘會越來越強，但是正在被摧毀的堡壘不會變強。因為你不知道敵方Master會躲在哪個堡壘裡面，考慮到最差的狀況，你必須將所有堡壘摧毀掉才能幹掉敵方Master。你詳細的調查過後，你現在知道每一個堡壘目前強度跟其每一單位時間會增加的強度。你至少需要花多少單位時間才能將所有堡壘摧毀完畢。

你的Servant很強，所以不用考慮失敗的問題。
你的Servant很強，在堡壘跟堡壘間移動所花的時間可以無視。
你很弱，所以無法幫助你的Servant摧毀敵方堡壘。

## 輸入說明

第一行有一個整數N(0<=N<=100000)，表示有N座堡壘要摧毀。
接下來N行每行有兩個非負整數分別代表一個堡壘的目前強度與每單位時間增加的強度。
除了N以外，其他數字皆不超過100。

## 輸出說明

輸出至少要花幾單位的時間才能摧毀所有堡壘。
數字可能會很大，請輸出 MOD 100000007後的結果。

(不用擔心會打到天荒地老，因為你的Servant很強，所以時間單位大概是10-10000秒等級的。)

## 範例輸入 #1

```
5
0 9
1 5
2 4
6 3
3 0
```

## 範例輸出 #1

```
37
```
