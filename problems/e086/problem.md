# e086. 電腦閱卷 2

## 內容

承上題  c680: 電腦閱卷 1 大家都已經練習了比對答案卡來計算成績。

實際的情境是，讀卡機讀取答案卡時，卡片上除了考生的答題外，還有科目代號及學生編號，以方便製作成績單。
本題將提供考試科目的資料表及學生的基本資料，請您於利用提供的資訊於閱卷後產生成績單。
成績單的排序方式：   order by 學生編號 , 科目編號。 

 這是原題目的評分標準：
假設本張試卷有 20 題，可能有下列情況：
1)  答案卷畫了 25 題，超出部分不理會。
2)  答案卡畫了 15 題，不足部分不計分。
3)  答案卡畫了 20 題，裡面有空白 (= 號是空白)，本題不計分。

## 輸入說明

測資的第 1 列有 2 個數字  n  m
接著會有 n 列科目的資料，格式為： 科目編號  科目名稱  標準答案
接著會有 m 列學生的資料，格式為： 學生編號  姓名
接這會有 n * m 張卡片，格式為：  學生編號  科目編號  學生答案

## 輸出說明

成績計算方式如下：
假設當卷題數為 T
1)  每答對 1 題，請給 100 / T 的分數。
2)  空白題不計分。

## 範例輸入 #1

```
2 3
001 eng AAAAA
002 math BBBBBBBBBB
001 Tom
002 Peter
003 David
001 001 AAA=A
001 002 AAABBBCCC=
002 001 CCAAA=A
002 002 AADBBBCCC=DDD
003 001 AAAAA
003 002 BBBBBBBBBB


```

## 範例輸出 #1

```
001,Tom,001,eng,80
001,Tom,002,math,30
002,Peter,001,eng,60
002,Peter,002,math,30
003,David,001,eng,100
003,David,002,math,100
```
