# e157. UJ去運動

## 內容

Da Da 是一隻可愛的烏龜，但題目和他沒關係。


 


重點是UJ被Da Da的主人—ID邀請去打球，但肥宅UJ最討厭流汗了！！！


但UJ又沒辦法直接拒絕ID，於是UJ出了一道題目希望轉移ID的注意力，題目如下：


 


有一顆氦原子在真空橢圓燒杯的焦點上，你可以假設氦原子不受到重力、凡得瓦力、及旋轉的作用，只有水平方向的速度。


給你燒杯的長軸及短軸的長度，請問：在氦原子第t次經過原位置（一開始放的那個焦點）前，總共移動了多少距離。


 


你是ID的小幫手，你也希望UJ多出門運動，所以你決定寫一個程式幫助ID解決這個問題

## 輸入說明

第一行表示n筆測資


第二行有三個正整數a,b,t


分別表示燒杯的長軸，短軸，要求的第t次


第接下來的n行有兩個正整數 v,degree


分別表示 氦原子的初速及射出方向與長軸的夾角(度)。


n<10000


a,b,v<2^31


1<=degree<=90


t<10

## 輸出說明

若有小數請輸出四捨五入到小數點後第2位

## 範例輸入 #1

```
1
10 5 1
12 1
```

## 範例輸出 #1

```
20
```
