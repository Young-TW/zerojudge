# a172. 覆蓋數列

## 內容

因為本人沒梗,所以就不囉嗦了
給你一串數字, 問最少要用幾條不重疊的嚴格遞增數列才能覆蓋整條數列

## 輸入說明

有多筆測資, 每筆第一行, 有一個數字 N, 代表接下來有 N 個數字, (1 ≦ N ≦ 100000)

下一行, 有 N 個數字 P(1  ≦ P ≦ 2147483647)

## 輸出說明

輸出一個數字，表示總共需要幾條不重疊的嚴格遞增數列才能覆蓋整條數列

## 範例輸入 #1

```
5
1 2 3 4 5
5
5 4 3 2 1
7
1 3 5 8 2 6 7
```

## 範例輸出 #1

```
1
5
2
```
