# e162. 01636 - Headshot

## 內容

俄羅斯輪盤，現在與一位朋友一起玩這場遊戲，然而我們已經知道左輪手槍中的填彈資訊，剛聽到朋友按下扣板，但是沒有死。現在輪到你，如果你希望下一槍沒有子彈，請問最好要直接按？還是隨機轉一圈再按？

## 輸入說明

輸入包含多個測資，每筆測資都只有由0或1組成的字串si，代表左輪手槍裡的子彈。0代表無子彈、1代表有子彈。


輸入保證至少有一個0。


2 <= si.size <= 100

## 輸出說明

如果要隨機轉一圈，請輸出 "ROTATE"


如果要直接開槍，請輸出 "SHOOT"


兩者皆非請輸出"EQUAL"

## 範例輸入 #1

```
0011
0111
000111
```

## 範例輸出 #1

```
EQUAL
ROTATE
SHOOT
```
