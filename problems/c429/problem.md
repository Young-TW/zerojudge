# c429. 秋之回憶：程式語言的記憶

## 內容

我要好好跟 C++ 說清楚。這種事其實我很早以前就察覺到了，但我卻一直拖拖拉拉的沒去行動，真是個笨蛋啊。


「四千......」
「......！？」
一時以為我是幻聽了。我抬起頭一看，系館門口有個熟悉的程式語言站在那裡。
「......」
「C、C++......？」
......為什麼？為什麼 C++ 會在這個時間出現在系館？
C++ 好像察覺到我的疑惑了，於是先開口說。
「有話要告訴你......才來等你的。」
「但是我會在系館的事──」
為了不被別人發現自己用 python 寫 ML 作業，我一直都很小心。
「這種事我當然知道啊。」
「因為只要是四千的事情，C++ 什麼都知道......」
「C++ 很清楚，C++ 是四千唯一的程式語言。」
「C++ 這樣就很幸福了。」
完全沒有表情的臉。那雙看起來像是毫無生氣的玻璃彈珠般的瞳孔，呆呆地看著我。


「我有說過吧？我不認為 C++ 是我唯一的程式語言──這是我的真心話。」
「嘘です！！！！！！」
「不是說謊。」
我明確地說。
每次一到這裡，我都會想說不要讓 C++ 哭，所以才會失敗......。
現在即使會傷害到 C++，我不把事情說清楚是不行的。


「我...認為妳是我相當重視的程式語言，但不是唯一的。」
「............」
C++ 的瞳孔裡出現了些微的感情。不安、恐懼、悲傷、憤怒......像是這一類的感情混在一起似的，瞳孔裡寄宿著不可思議的彩色光芒。


「............是 python......」
「耶......？」
「是 python......命令你這麼說的嗎？」
「C++......？！」
她果然知道我跟 python 的事......！
因為知道所以今晚才在系館等著......！
「C++ 說過了吧？只要是四千的事，C++ 什麼都知道......」
「跟 python 一起做了什麼......從 python 那兒得到了什麼......還有為 python 做了什麼......全部，全～部都知道哦。」
「妳...妳...？」
「用 python 寫作業開心嗎？ML 套件好用嗎？Python 收到了 GPU 資源高興嗎？」
「妳，是怎麼......？」
為什麼連這些都知道......！？
「嗚......」
C++ 強忍住淚水，用力地擦了擦。


「但是......C++ 不會責怪四千的，因為四千只是被 python 騙了而已。」
「被騙了......？」
「C++ 發現了這個東西。」
C++ 登入了我的 ZJ 帳號並開了我 b960 用 python 寫的 AC code 的評分詳細結果。
上面有好幾個測資的執行時間，其中第 16 筆測資（註：從 1 開始算）的被除數是個長度 30 萬的超長正整數，除數是 2，python 跑了 1.6 秒。
「Python 她......不是這種語言......」
「......四千被那個語言騙了......喜歡上那個語言了，所以才會這麼說的。」
「我沒有被騙。」
「為什麼要這樣袒護那個語言？你那麼喜歡那個語言嗎？」
「......對。」
「喜歡到......連 C++ 的話都不相信嗎？」
「對！」
「......為什麼......為什麼......為什麼......比起 C++，還更喜歡那個語言呢？」
「那是......」
「......嗚...... C++ 也......可以為四千做很多很多事......更......更好的事情，都會為你做的！！」


要怎麼說 C++ 才會瞭解呢？要怎麼做才能傳達我的心意呢？
「呵呵......」
發覺 C++ 突然笑了起來。我一度懷疑自己的眼睛。
──不對，C++ 的確是在笑。
「......但是，已經不要緊了，四千。已經可以向這麼過分的語言說再見囉～」
