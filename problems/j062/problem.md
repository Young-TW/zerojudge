# j062. 11470 - Square Sums

## 內容

你知道正方形中的正方形嗎？ 這樣講起來你可能不太懂，但請看一下。
假設你有一個大小為 5 × 5 的整數正方形，如下圖:



![image](ShowImage?id=2478)


您可以從上表中看到三個同心正方形。
在這個問題中，您必須找到每個同心正方形的數字之和。
對於上述情況，總和是:


5 + 3 + 2 + 7 + 9 + 1 + 4 + 5 + 6 + 1 + 1 + 1 + 4 + 5 + 6 + 3 = 63
7 + 4 + 2 + 3 + 4 + 3 + 4 + 5 = 32
2 = 2

## 輸入說明

輸入文件中有多組測試資料


每個測試資料第一行為一個整數 n (n ≤ 10)，該整數決定了方格的維度


接下來的 n 行中的每行包含 n 個整數，如果 n = 0 代表輸入結束

## 輸出說明

每行輸出都以“Case x:”開頭，其中 x 為案例編號


然後你必須輸出 ⌈(n/2)⌉ 個空格分隔的數字，代表從外到內同心正方形的數字的總和

## 範例輸入 #1

```
5
5 3 2 7 9
1 7 4 2 4
5 3 2 4 6
1 3 4 5 1
1 4 5 6 3
1
1
0
```

## 範例輸出 #1

```
Case 1: 63 32 2
Case 2: 1
```
