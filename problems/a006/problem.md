# a006. 一元二次方程式

## 內容

求一元二次方程式 ax2+bx+c=0 的根

## 輸入說明

每組輸入共一行，內含三個整數 a, b, c 以空白隔開。

## 輸出說明

Two different roots x1=?? , x2=??


Two same roots x=??


No real root


PS: 答案均為整數，若有兩個根則大者在前

## 範例輸入 #1

```
1 3 -10
```

## 範例輸出 #1

```
Two different roots x1=2 , x2=-5
```

## 範例輸入 #2

```
1 0 0
```

## 範例輸出 #2

```
Two same roots x=0
```

## 範例輸入 #3

```
1 1 1
```

## 範例輸出 #3

```
No real root
```
