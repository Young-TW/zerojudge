# d875. 4. 窮舉的階梯問題

## 內容

有一段距離 L 

而你每次走的距離只能是前一次的±1，或者是不變

而一開始你只能走 1，到達終點的時候，你要恰好走 1 到達

例如: L=10

1 1 2 3 2 1

例如: L=9

1 2 3 2 1

請你輸出最少步數的走法

答案可能很多，請輸出序列最小的那個

## 輸入說明

每一行都有一個正整數 L ( L ≦ 2147483647 )

## 輸出說明

請輸出最少步數的走法 (序列最小的那組)

## 範例輸入 #1

```
9
10
```

## 範例輸出 #1

```
1 2 3 2 1
1 1 2 3 2 1
```
