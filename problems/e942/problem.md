# e942. pC. 數字排列

## 內容

給予 N 個相異的正整數，請按大小順序輸出所有可能的排列組合。

## 輸入說明

第一行有一個正整數 N （1 ≤ N ≤ 9），代表有幾個相異的正整數。
第二行則有 N 個相異的正整數 x（0 ≤ x ≤ 4,294,967,295）。

## 輸出說明

按照大小順序，輸出所有可能的排列組合。

## 範例輸入 #1

```
4
30 10 20 40
```
