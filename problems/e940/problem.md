# e940. pA. 猜單字遊戲

## 內容

現在有一列 * 表示尚未被猜中的字，
如果過程中猜中字母，則該字母出現的所有位置都必須寫上該字母。
現在給定一個答案字串及過程中每個步驟所猜的字母，請印出所有遊戲過程。

## 輸入說明

第一行有一個字串 s（1 ≤ s 字串長 ≤ 10），代表答案字串。
第二行有一個整數 n（1 ≤ n ≤ 26），代表猜測次數
接下來有 n 行，每行有一個字元 c，代表該次所猜字母。

（本題所有出現字元與字串，皆由小寫字母 a - z 所構成）

## 輸出說明

所有遊戲過程

## 範例輸入 #1

```
hangman
6
a
n
g
p
h
t
```

## 範例輸出 #1

```
*******
*a***a*
*an**an
*ang*an
*ang*an
hang*an
hang*an
```
