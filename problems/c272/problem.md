# c272. 趙哥的養成計畫I

## 內容

俗話說的好! 妹子的心不是一天就可以獲得的
趙哥為了要獲得妹子的愛
他決定去幼稚園當老師
希望從小陪養妹子與他的感情
讓妹子更有機會喜歡上他
來到了幼稚園
好不容易通過面試後
他來到了他將來要帶的班的門口
「是天使班~~」趙哥在心中吶喊
「裡面肯定有很多像天使般的妹子」趙哥打從心底高興著
進到了教室，趙哥打開學生資料表
總共有n個學生 (學生編號=0~n-1)
開始很糟糕得把每一個學生評分
妹子分數有高有低，都是大於0的數字
而男生的分數也又高有低，但都是<=0的數字
算計著怎麼排位子比較好的趙哥
把他剛剛的分數算入內
他想到了一個算法
稱之為爽度算法
這算法是為了計算接下來趙哥這一學期上課的爽度
算法如下:
從講台開始第i個學生所得的爽度等於該學生的評分*(m-min(i,m+1)+1) (因為趙哥有近視所以第m個位子是趙哥所能看到的最遠距離
但因為怕被投訴
所以趙哥要求爽度不得超過q

## 輸入說明

第一行有兩個整數n,m,q
第二行有n個整數以空白隔開代表第0個學生到第n-1個學生各自的分數
第三行到第2+q行有一個整數qs

## 輸出說明

請輸出所有<=qs的爽度最優解
每輸出一個qs需換行
如果低於qs沒有其他解則輸出 "No Solution!"

## 範例輸入 #1

```
Input1:
2 1 2
10 -10
10
-9

Input2:
3 2 5
2 4 -1
6
8
-10
6
7
```

## 範例輸出 #1

```
Output1:
Case #1
10
Case #2
-10


Output2:
Case #1
3
Case #2
8
Case #3
No Solution!
Case #4
3
Case #5
7
```
