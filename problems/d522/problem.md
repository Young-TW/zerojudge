# d522. 走棋盘

## 內容

有一个n*n的棋盘，每个方格里都有着相应的数字。你从左上角出发，每次可以向上下左右四个方向最多移动k格，并且要求你每次到达的方格里的数字必须大于上一次所在方格的数字。现在要求你走过的方格的所有数之和最大，问这个最大和是多少。

## 輸入說明

输入数据首先为两个正整数n、k（1<=n<=100,0<=k<=n） 不换行。

接下来的n行(包括n,k这一行)，每行有n个integer范围的整数，表示地图中的数。

## 輸出說明

输出数据只有一行，为最大的和。

## 範例輸入 #1

```
2 2 2 3
4 5
```

## 範例輸出 #1

```
11
```
