# c643. 55555

## 內容

「55555 我什麼都沒有又什麼都不會｡ﾟ(ﾟ´Д｀ﾟ)ﾟ｡」四千如是哭著。


這題很簡單，給定一個正整數 $\color{black}{n}$，請幫四千找一個能被 $\color{black}{5^n}$ 整除的 $\color{black}{n}$ 位數正整數。


對了，在意識到自己沒有妹妹後，四千變得非常討厭配對的數 aka 偶數，所以你找的 $\color{black}{n}$ 位數正整數的每一位都必須是奇數。如果有多個這種正整數存在，請給出最小的那個，因為四千喜歡小的。

## 輸入說明

輸入的第一行包含一個正整數 $\color{black}{T}$，代表接下來有幾筆測試資料。


接下來的 $\color{black}{T}$ 行，每一行有一個正整數 $\color{black}{n}$，代表你要幫四千找的那個正整數是 $\color{black}{n}$ 位數。


- $\color{black}{1 \leq T \leq 555}$
- $\color{black}{1 \leq n \leq 55555}$

## 輸出說明

對於每筆測試資料，輸出一行，就是你幫四千找的那個正整數。如果不存在這樣的正整數，請輸出 "-1" (不含引號)。

## 範例輸入 #1

```
3
2
3
112
```

## 範例輸出 #1

```
75
375

```
