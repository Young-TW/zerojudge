# d092. 算式也可以比大小！？

## 內容

一日，老師教到了數字的比大小，此時，某K想到了一種新的比較方式，如下： 

a+b=c 

但若a>b時，答案就變成[>c]， 
若a=b時，答案就變成[=c]， 
若a<b時，答案就變成[<c]。 

而[>c] > [=c] > [<c]。 

例如： 
1+1=[=2] 
2+0=[>2] 
0+2=[<2]

## 輸入說明

首先會先輸入要比較的n組算式(n<=1000)，之後再輸入兩個正整數a和b，數字最大不會超 
過2^31-1。若n等於0即跳出。

## 輸出說明

將a+b加起來後排序(大->小)，並輸出。

## 範例輸入 #1

```
5
1 5
5 1
3 3
4 5
5 6
0
```

## 範例輸出 #1

```
<11 <9 >6 =6 <6
```
