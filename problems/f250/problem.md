# f250. ugly ++

## 內容

d129加強版 
ugly的定義參考 https://zerojudge.tw/ShowProblem?problemid=d129
但因為有人直接 post答案， 修改一下題目要求


給 t(0<t<=10^3), 接下來 t個正整數n {0<n<=10^4} 求第 n 個 ugly數u[n]，註u[10000] < 10^18

## 輸入說明

第一列一個正整數  t {0<t<=10^3}


接下來 t個正整數n {0<n<=10^4} 求第 n 個 ugly數u[n]，註u[10000] < 10^18

## 輸出說明
