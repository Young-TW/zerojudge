# e793. an easy gcd problem

## 內容

冬天的某一天午後，陽光如同薄紗蓋在エミリア衣上，溫暖的微風吹拂窗簾，外頭傳來スバル的吵鬧聲擾亂了她的思緒...
她閉上眼，追尋著題目的突破口，將意識放逐於無盡的黑洞中。


「妳需要停下腳步，リア」帕克的聲音迴盪在耳中。「可是...」


「該怎麼判斷?」「妳想的太複雜了，需要化簡...」


スバル的一聲叫喊，將エミリア從睡夢中吵醒。


她望向窗外，太陽已經沉入樹叢中，用盡力氣發出最後一點微薄的光芒。


時間不早了，リア選擇回到被窩中，因為答案已浮出水面。



請用C++實作可以求出兩數的最大公因數的函式，此程式不能使用分支條件if..else.., 任何除了<cstdio>函式庫, #define, &&, ||, /, while, do..while, goto, break, continue, for, 三元運算子, switch..case, operator。

## 輸入說明

輸入有多行，每行包含兩個數字$a, b$。


保證$a, b$可以用long long儲存。


以$a = b = 0$結束。

## 輸出說明

輸出$a, b$的最大公因數。

## 範例輸入 #1

```
1 1
923 1
6 3
0 0
```

## 範例輸出 #1

```
1
1
3
```
