# f282. 松鼠的願望

## 內容

有一隻松鼠，它非常的愛吃堅果，可是它只喜歡吃由小到大已經排列好的堅果，不然它的強迫症就會發作讓它不得不先把堅果排好再吃。可是一袋的堅果實在太多了，所以它委託你寫一個程式幫它把堅果排好，讓他可以好好的享用他的大餐。

## 輸入說明

每一行有一個整數 n 代表堅果的大小(0<n<=100)

## 輸出說明

輸出排列好的堅果給松鼠


格式請參考範例輸出

## 範例輸入 #1

```
1
2
3
4
5
```

## 範例輸出 #1

```
1 2 3 4 5
```

## 範例輸入 #2

```
1
1
2
2
3
```

## 範例輸出 #2

```
1 1 2 2 3
```
