# i855. 01260 - Sales

## 內容

CozyWalk Co. 的 CEO Mr. Cooper，會收到每天公司的銷售報告


對於銷售列表 A = (a1, a2, . . . , an)，Mr. Cooper 會額外在維護一個列表 B = (b1, b2, . . . , bn−1)


在第 i 天 (2 ≤ i ≤ n)，bi−1 = { ak 的數量，使得 ak ≤ ai (1 ≤ k < i) }


假設 A = (20, 43, 57, 43, 20)，則 B = (1, 2, 2, 1)

## 輸入說明

第一行有一個數字 T 代表測試資料數量


每筆測試資料，第一行有一個整數 n (2 ≤ n ≤ 1000)，代表 A 銷售列表長度


接下來一行有 n 個數字 ai (1 ≤ ai ≤ 5000 且 1 ≤ i ≤ n)，代表 A 銷售列表數字

## 輸出說明

對於每組測試資料


請輸出B列表數字總和

## 範例輸入 #1

```
2
5
38 111 102 111 177
8
276 284 103 439 452 276 452 398
```

## 範例輸出 #1

```
9
20
```
