# i959. 11078 - Open Credit System

## 內容

在開放的學分制系統中，學生可以選擇任何他們喜歡的課程，但存在一個問題。 有些學生比其他學生年級更高。
課程的教授發現了相當多的來自高年級的學生（好像他們是來參加必修課程的）。
教授想為新生給一點福利。因此，他將舉辦分班考試（基本上是智商測試），以評估學生之間的差異程度。


他想知道高年級學生比任何低年級學生獲得的最高分數。


例如，如果一個高年級學生得到 80 分，一個低年級學生得到 70 分，那麼這個數值就是 10。注意我們不想要絕對值。


你可以幫助教授找出一個快速的解決方案嗎?

## 輸入說明

第一行有一個數字 T ( T < 20)，代表測資數量


每組測資首行有個數字 n (2 ≤ n ≤ 100,000)，代表參加課程人數


其中第 i 個整數是第 i 個學生的分數。 所有整數絕對值小於 150000


如果 i < j，則第 i 個學生年級比第 j 個學生大

## 輸出說明

對於每組測資，輸出教授要求

## 範例輸入 #1

```
3
2
100 20
4
4 3 2 1
4
1 2 3 4
```

## 範例輸出 #1

```
80
3
-1
```
