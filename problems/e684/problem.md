# e684. 00897 - Anagrammatic Primes

## 內容

一個數字只能被自己和1整除，且大於1的數，我們稱它為質數。 
例如：23是質數，而35不是質數，因為35 = 7×5。
如果重新排列此數字，其性質可能會更改。
例如：35不是質數，而53是質數。
對於此問題，無論您如何重新排列此數字，此數字皆為質數，我們稱之為"anagrammatic prime"。 
例如：數字113為一種，他的所有排列113、131、311都是質數，因此我們說113是一個"anagrammatic prime"(131和311也是)。

## 輸入說明

輸入的每一行將包含一個數字n (n < 10000000)。
如果n = 0代表輸入結束。

## 輸出說明

對於輸入中的每個數字，輸出最接近n且比n大，並且位數和n一樣的"anagrammatic prime"。 
如果沒有請輸出"0"。

## 範例輸入 #1

```
10
16
900
113
8000000
0
```

## 範例輸出 #1

```
11
17
919
131
0
```
