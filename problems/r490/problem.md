# r490. 3. 商品包裝地

## 內容

給定 $N$ 筆長度為 13 的商品條碼。條碼前 3 碼為產地，第 13 碼為檢驗碼 $C$。

令 $S_{\text{odd}}$​ 為條碼前 12 位中奇數位（位置 1,3,…,11）的總和。

令 $S_{\text{even}}$​ 為條碼前 12 位中偶數位（位置 2,4,…,12）的總和。

若一個條碼滿足以下條件則視為合法：

$$(S_{\text{odd}} + 3 \times S_{\text{even}}) \% 10 + C = 0 \text{ 或 } 10$$

請統計所有合法條碼中最多的產地（前 $3$ 碼）及其對應數量。

保證答案唯一，且至少有一組有效條碼。

## 輸入說明

輸入的第一行是一個整數 $N$ ($1 \le N \le 100$)。 接下來的 $N$ 行，每行是一個長度為 $13$ 的商品條碼字串。

## 輸出說明

輸出包含一行，為出現次數最多合法條碼的產地（$3$ 位字串）和其對應的數量，兩者之間以一個空格隔開。

## 範例輸入 #1

```
4
4710018000102
0309876543216
4711234560012
3001111111118
```

## 範例輸出 #1

```
471 2
```
