# a858. 數三角形

## 內容

給定一個大小為$N$的完全圖，其中的每條邊可能是紅色或黑色。任三個不同的點都可以形成一個三角形，請問三邊同色的三角形有幾個？

## 輸入說明

輸入的第一行為一個正整數$N$。接下來的$N$行，每行有$N$個數字，其中第$i$行的第$j$個數字$c_{ij}$代表邊$(i,j)$的顏色，紅色用$1$表示，黑色用$2$表示，而當$i=j$時則用$0$表示。


- $3 \leq N \leq 1000$。
- 對於所有的$1 \leq i, j \leq N$，保證有$c_{ij}=c_{ji}$。

## 輸出說明

請輸出同色三角形的數目。

## 範例輸入 #1

```
4
0 1 2 1
1 0 1 1
2 1 0 2
1 1 2 0
```

## 範例輸出 #1

```
1
```
