# k678. 肥余歷險記---(反數學城1)

## 內容

肥余成功走出他迷路的路口，但是! 肥余沒有戰斗能力，他只能去數學城的商店購買木劍。


又但是！ 他現在不在數學城 他在反數學城 。


肥余只能打敗反數學怪獸 才能走出反數學城!!!


肥余遇到在反數學城的第一隻怪獸是 ----- 反加法獸!!!

## 輸入說明

第一行為小怪數量(反加法獸)


然後重覆(第一行) 次:


輸入一個加法算式


和答案

## 輸出說明

答對輸出"yes"


答錯輸出"no"

## 範例輸入 #1

```
3
1+1
2
2+4
5
5+8
4
```

## 範例輸出 #1

```
yes
no
no
```
