# c640. 滿滿的糖果屋   #1

## 內容

暑假快到了，王老師打算買些糖果獎勵小朋友。
糖果屋有各式各樣的糖果。打算只買一種糖果。

王老師發現帶來的錢，全部買單價 3元, 5元 或 7元
的糖果都會剩下 2 元，你能推算出王老師可能帶多少錢嗎。

有若干組測資
每組測資有 2 行
第一行有 2 個數字 M k , 帶來的錢不超過 M 元， k 是剩下的錢。
第二行是若干種糖果的單價。

## 輸入說明

M k
p1 p2 ... pn

 


1 < n < 5
0 < k < min(pi)

## 輸出說明

M > max(p)

## 範例輸入 #1

```
500 2
3 5 7
```

## 範例輸出 #1

```
107 212 317 422
```
