# a264. 骰子疊疊樂

## 內容

骰子！疊疊樂！！ 

 

![image](ShowImage?id=73)

![image](ShowImage?id=74)
 

要怎樣才能用最少的骰子疊成一疊

使得表面的總和恰好為 n 呢 ～

噢你在透明的桌面上，所以最下面的點數也算在表面

## 輸入說明

多組輸入，以EOF作為結束
每組輸入為一個正整數 n (1<=n<=1000000000)

## 輸出說明

一個數字，代表至少要用幾個骰子才能疊出 n
無解請輸出 -1

## 範例輸入 #1

```
50
7
32
```

## 範例輸出 #1

```
3
-1
2
```
