# f455. 詠竣的哀鳳12

## 內容

哀鳳推出了第12代，許多人都搶著想買到，詠竣也不例外。然而詠竣的錢並不夠，所以他需要去當臨時工來賺錢。詠竣能拿來打工的時間有限，但是有太多地方在招臨時工了，所以他希望你幫他寫程式計算他最多能賺到多少錢。喔對了，詠竣又小又快（體型小且移動速度快），有學過「神炎光速移動」，所以當他完成一份工作後能夠無縫接軌地直接開始下一份工作，而且他不需要休息！
不過這只是暫時的事情，詠竣對於人生還是希望有點規劃，所以當他完成這次的臨時打工計畫後，他還想到這些工作中時薪最高的地方應徵正式員工，所以請你順便找出時薪最高的工作。

## 輸入說明

測資為單筆輸入。


第一行有兩個以空白隔開的正整數T、N，代表詠竣有T小時能拿來打工且有N個地方在招臨時工。（T<=10^5、N<=10^4）


接下來N行中，每行有一個字串S和兩個正整數M、H，皆以空白隔開，S代表工作的內容，該份工作需要花費H小時，完成後薪水為M。（M、H<=2×10^5）

## 輸出說明

第一行請輸出詠竣最多能賺到多少錢。


第二行請輸出N個工作中時薪最高的工作內容。（若答案不只一個，以先輸入的為優先）

## 範例輸入 #1

```
10 3
印製鯊魚T恤 20 5
用下巴清潔紅土 20 4
賣蘇菜湯 38 10
```

## 範例輸出 #1

```
40
用下巴清潔紅土
```

## 範例輸入 #2

```
20 3
表演喝下可樂混威士忌後嘔吐 10 20
擔任某高一女孩的男友 10000 22
過來阿姨這邊 500 1
```

## 範例輸出 #2

```
500
過來阿姨這邊
```
