## 輸入說明

測試資料的第 一行數字為 n，其值為 2至 6的整數 ，表示棋盤大小為 n×n。 從第二行開始為棋盤第一列 由左 到右的數字 ，依此類推， 輸入值皆為 0或 1。

## 輸出說明

輸出資料為一個正整數或零，表示 最多可以找到的 「1」的個數 。

## 範例輸入 #1

```
4
1111
1111
1101
0101
```

## 範例輸出 #1

```
11
```
