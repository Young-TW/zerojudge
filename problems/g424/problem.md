# g424. PF.抱ㄌㄌ

## 內容

![image](ShowImage?id=1878)


熊頭喜歡ㄌㄌ，所以他想抱ㄌㄌ回家


最近$n$天都會有ㄌㄌ出沒，抱回家就能獲得$a_i$的幸福值


每天熊頭都可以選擇要不要抱ㄌㄌ回家


可是因為有萬惡的IBF存在，熊頭最多只能連續抱走$k$個ㄌㄌ


如果連續$k+1$天都抱ㄌㄌ回家，會被IBF抓走


熊頭不知道要怎麼抱才好，請你寫個程式幫熊頭判斷在不被IBF抓走的情況下，最多能獲得多少幸福值


 
![image](ShowImage?id=1880)

## 輸入說明

第一行有兩個整數$n,k$表有n個ㄌㄌ，最多連續取k個ㄌㄌ


第二行有$n$個整數$a_i$表抱走第i個ㄌㄌ的幸福值


 


保證所有測資符合


$1 \le n,k \le 10^5$


$0 \le a_i \le 10^9$

## 輸出說明

輸出不被IBF抓走的最大幸福值

## 範例輸入 #1

```
6 2
3 5 1 6 2 4
```

## 範例輸出 #1

```
18
```

## 範例輸入 #2

```
10 3
23 54 12 20 48 32 59 74 15 23
```

## 範例輸出 #2

```
301
```
