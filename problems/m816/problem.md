# m816. 991 Safe Salutations

## 內容

任何稍微迷信的人都清楚地知道，當4個人進行交叉的握手時，會發生可怕的事情。


你，一位勇敢的計算機科學家，被賦予了一項任務，即通過為這些人提供一組可行的握手方式，讓組中的每個人，避免任何這樣的交叉，來減輕這些人的負擔。以下的圖示展示了三對人之間的情況：



![image](ShowImage?id=3734)

## 輸入說明

輸入包含多筆測資。每筆測資包含一個整數 n，表示共有幾對人要進行握手，其中 1 ≤ n ≤ 10。

## 輸出說明

對於每筆測資，請輸出一個整數，表示在共有 n 對人進行握手中，沒有交叉的握手方式有多少種。在測資之間請輸出一個空白行。

## 範例輸入 #1

```
3
4
```

## 範例輸出 #1

```
5

14
```
