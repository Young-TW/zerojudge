# a513. 最大值

## 內容

給你一序列，找出最大值。

## 輸入說明

第一行有一正整數T，代表有幾組測試資料

接著每筆第一行有兩正整數n、m

n為輸入有多少個數字，m為處理資料數，n <= 30000，m <= 10000 

接下來有m行

若為1，必須再輸入一值，並放至序列中

若為2，必須顯示序列最大值，並刪除該數

## 輸出說明

以下面第二筆測資舉例

n = 3，m = 10

輸入有3、2、1三個數字

接著有10行

若為2，輸出最大則3，依此類推...

像1 8，代表輸入一個數為8的給序列

最後再輸出剩餘的序列:由大到小輸出 

若序列為空需輸出It's empty!

最後輸出剩餘序列若為空也需輸出It's empty!
