# s573. 多項式 - 判斷正負

## 內容

給一個無限多項的式子，我們可以重新排列成以下 : 

![image](ShowImage?id=5186)

請判斷第 $n$ 項是正還是負。

## 輸入說明

輸入一個正整數  $t$ ($t\le 10^6$)，代表有 $t$ 筆測資。

每筆測資輸入一個數字 $n$，保證 $n$ 會在 int 範圍且 $n$ 為非負整數。

## 輸出說明

對於每筆測資 $n$，輸出第 $n$ 項是正還是負。

如果是正，輸出 $+$；否則輸出 $-$。

## 範例輸入 #1

```
5
0
1
2
3
7
```

## 範例輸出 #1

```
+
-
-
+
-
```
