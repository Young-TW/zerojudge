# c635. 基礎排序 #1-2 ( 位置排序 )

## 內容

有若干行測資。
每一行測資有 10 至 20 個用逗號分隔的數字。
請分別對數列中奇數位置及偶數位置的數字加以排序，
並將結果輸出。

## 輸入說明

4806,39740,87462,16241,69980,93315,6786

## 輸出說明

4806,16241,6786,39740,69980,93315,87462

## 範例輸入 #1

```
4806,39740,87462,16241,69980,93315,6786
```

## 範例輸出 #1

```
4806,16241,6786,39740,69980,93315,87462
```
