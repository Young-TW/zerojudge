# d885. NOIP2007 1.統計數字 番外篇

## 內容

某次科研调查时得到了n个自然数，每个数均不超过1500000000（1.5*109）。已知不相同的数不超过100000个，现在需要统计这些自然数各自出现的次数，并按照自然数从小到大的顺序输出统计结果。

## 輸入說明

每組输入包含n+1行：

      第1行是整数n，表示自然数的个数。

      第2~n+1行每行一个自然数。

## 輸出說明

每組输出包含m行（m为n个自然数中不相同数的个数），按照自然数从小到大的顺序输出。每行输出两个整数，分别是自然数和该数出现的次数，其间用一个空格隔开。

## 範例輸入 #1

```
8
2
4
2
4
5
100
2
100
```

## 範例輸出 #1

```
2 3
4 2
5 1
100 2
```
