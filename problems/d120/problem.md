# d120. 10699 - Count the factors

## 內容

寫一個程式算出一個正整數有多少個不同的質因數。例如：45=3*3*5，所以45有2個質因數（3和5）。

## 輸入說明

每組測試資料一列。含有1個正整數 n（ 1 < n <= 1000000）。

              若 n=0 代表輸入結束。

## 輸出說明

對每組測試資料輸出一列，n有多少個不同的質因數。輸出格式請參考Sample  Output。

## 範例輸入 #1

```
7
8
45
289384
930887
692778
636916
747794
238336
885387
760493
516650
641422
0
```

## 範例輸出 #1

```
7 : 1
8 : 1
45 : 2
289384 : 3
930887 : 2
692778 : 5
636916 : 4
747794 : 3
238336 : 3
885387 : 2
760493 : 2
516650 : 3
641422 : 3
```
