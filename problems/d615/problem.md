# d615. 計算機運算

## 內容

![image](ShowImage?id=328)

 這是大家熟悉的計算機

 現在給一運算式

 請用此計算機算出運算式答案

## 輸入說明

第一行是一個整數 T ( T ≤ 20 )

 代表接下來有 T 組測試資料 

 每組測試資料為一行運算式 ( 字元長度不超過 100,0000 )

 運算式包括數字 0 ~ 9 和 + - * / ( 數字不超過 231-1 )

 運算元與運算子之間以空白相隔

 保證運算過程中不會出現小數

## 輸出說明

每行請輸出運算式答案 n

 n 不大於 263-1

## 範例輸入 #1

```
2
1 + 3 - 5 * 8 / 2 + 4
1 + 2 / 1 * 9 / 27
```

## 範例輸出 #1

```
0
1
```
