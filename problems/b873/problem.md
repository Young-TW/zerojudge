# b873. 又是一堆按鈕

## 內容

接續 b844: 一堆按鈕，這次你又看到了一堆按鈕，按鈕也像之前一樣會連動，不過不是直接影響後面所有按鈕，一樣按一下每個按鈕都會把0變1，把1變0，還有每一個按鈕都有對應到另一個按鈕，例如A對應到B，所以一旦A被按下的同時，B也會被按下去，A就彈起來了，請問你最後這些按鈕會一直不停地按下去還是最後會停下來。 如果A對應到A，A按下去的同時，A就彈起來了。

## 輸入說明

第一行是一個數字T代表測資筆數，每筆測資開頭都有數字N代表按鈕數量(1~N)，之後有N個數字分別代表1~N所對應到的按鈕。之後有一個數字Q代表問題數量，每個問題都會有一個數字K，請問按下K之後是會停止還是一直不停。



```
n，q<=10000
```

## 輸出說明

如果會停請輸出-1，不停輸出1，不一定的話就輸出0。

## 範例輸入 #1

```
1
3
2
3
1
3
1
2
3
```

## 範例輸出 #1

```
1
1
1
```
