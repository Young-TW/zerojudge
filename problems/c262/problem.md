# c262. 在巨人群中救回同伴是否搞錯了什麼

## 內容

以下防雷塊
-----------------------------------------------------------
阿爾敏:【糟了，米卡莎被帶走了】
艾連:【可惡!快追!】
因為一連串令人難過的事情
結果米卡莎被抓走了
所以 現在為了要討回米卡莎
必須穿越巨人群跑到米卡莎那邊
艾連:【把米卡莎(老婆)給我還來！！！】
 



![image](ShowImage?id=995)


經過了一番地獄般的追逐戰
最後....
請你算出米卡莎是否能被帶回來
----------------------------------------------------------
 為了迅速追到米卡莎
要利用巨人來快速移動
因為有些巨人非常兇猛
有些則是奇行種
如果途中遇到boss級巨人甚至會被吃掉
因此艾連將兩個巨人之間的移動危險度
和自身的能力值量化
只要自身能力值小於某次移動的危險值
就會成為食物
所以只有選擇最佳方法才有希望救到米卡莎

## 輸入說明

每筆測資 第一行輸入 n(1<=n<=500)代表n個巨人，編號1~n


m(0<=m<=n*(n-1)/2)代表m筆資料


和 integer q 最大能力負荷


接下來m行資料


輸入a,b(1<=a,b<=n)代表巨人編號


和integer c 為巨人a和巨人b間移動的危險值


有時候數據會有bug 所以要以最後更新的資料為標準


輸出三個0結束

## 輸出說明

假設米卡莎在巨人n 


艾連在巨人1


如果米卡莎得救輸出"Save"


不然輸出"GG"

## 範例輸入 #1

```
7 3 70
3 2 50
1 3 60
2 7 120
7 3 130
3 2 50
1 3 60
2 7 120
0 0 0
```

## 範例輸出 #1

```
GG
Save
```
