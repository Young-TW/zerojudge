# f821. nAnB ( 正常版 )

## 內容

正常版的 nAnB 遊戲, 出題跟答題都必須是一組字元不重複的數字組合。


本題正是循此規則而設計。


每個測資的第一列為 N, m


N 是此題的正確答案, 長度介於 4 ~ 8


m 是詢問的筆數。請就每筆詢問, 輸出您比對的答案。

## 範例輸入 #1

```
9038 5
5192
8642
0652
5167
0749
```

## 範例輸出 #1

```
0A1B
0A1B
0A1B
0A0B
0A2B
```

## 範例輸入 #2

```
36217840 3
60931758
24105739
87519260
```

## 範例輸出 #2

```
0A6B
0A6B
2A4B
```
