# q370. 5. 年輕人不講５德

## 內容

你有完成去年校內賽第五題「n371. 5. 隔山打牛」了嗎？
　　一頓操作下，馬老師發現被打的永遠是他自己，他知道你是有備而來，於是提醒你：「傳統功夫應點到為止，這年輕人不講武德，來騙～來偷襲，我 69 歲的老同志。這好嗎？這不好。以後不要再犯這樣的聰明，小聰明啊。」
　　你決定要回覆馬老師一句話，這句話由一串很長的數字組成，為了體現你的敬意，你希望這句話不要再出現任何「$5$」，並且對於任一連續區間的各位數字總和，其個位數也不能為「$5$」。

## 輸入說明

輸入只有一行，包含一個整數 $x$（$1\le x\le 10^{2^{20}}$），代表你打算回覆馬老師的話。（$x$ 不含前綴 $0$）

## 輸出說明

若 $x$ 的任一位數是「$5$」，或是任一連續區間總和之個位數為「$5$」，就輸出「耗子尾汁，好好反思」；否則請輸出「謝謝朋友們」。

## 範例輸入 #1

```
134086
```

## 範例輸出 #1

```
耗子尾汁，好好反思
```

## 範例輸入 #2

```
1021337
```

## 範例輸出 #2

```
謝謝朋友們
```
