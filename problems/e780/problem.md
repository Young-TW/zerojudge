# e780. 禮物拋接遊戲

## 內容

![image](ShowImage?id=1433)


相信你曾經玩過接禮物的遊戲，在遊戲中你需要使用 L 和 R 鍵，以操控正下方的小精靈位置，並接取上方小精靈所拋接下來的禮物。

遊戲過程中的指令分為三種，
　"L"：正下方小精靈向左移一格，即 x - 1
　"R"：正下方小精靈向右移一格，即 x + 1
　"Gift 數字y"：正上方小精靈從 y 位置丟一個禮物下來

假設正下方小精靈遊戲一開始位在正中間，即 x = 0 的位置，
請撰寫一個程式，計算正下方小精靈可以接到幾個禮物。

## 輸入說明

輸入第一行為整數，代表總共有 N 個指令
接下來有 N  行指令，分別為 "L", "R", 整數 "Gift 數字y" 三種任一

## 輸出說明

正下方小精靈可以接到的禮物個數

## 範例輸入 #1

```
12
L
L
L
Gift -3
R
Gift -2
R
L
Gift 5
L
Gift 1
R
```

## 範例輸出 #1

```
2
```
