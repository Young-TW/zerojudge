# d783. 11818 - Mouse & a Cheese

## 內容

SOHA和TARA最近發明了一種新的遊戲叫“Mouse and Cheese”。顧名思義，這遊戲是一隻老鼠標尋找一塊奶酪。這個遊戲在一個 3x3的盤面，如下圖所示。每個格子有獨特的數字1至9之間的整數。該盤面包含12支棒子（圖中藍色的線）。

最初老鼠和奶酪放置在兩個不同的格子。上圖的例子中，老鼠放在8這個格子和奶酪則是放在3這個格子。

遊戲的規則：這是一個雙人遊戲，玩家輪流移去。SOHA是先動的。在每次動作中，玩家可以選擇一個棒子把它移去。每個動作後，如果老鼠可以達到的格子含有奶酪不碰到任何殘留在盤面上的棒子，那麼該玩家被宣布為獲勝者。如果兩者都是完美的玩家，那會是誰贏了？

 

遊戲開始前，可能會有一些棒子先被移去。您將獲得的坐標這些棍棒的數量、鼠標和奶酪的格子編號。棒子是用坐標表示。左下角的網格是原點（0，0）。圍繞在這奶酪的兩個支棒子點座標是，2 0 2 1”和“2 1 3 1”。請注意，棒子的座標可以任何順序排列。於是“2 0 2 1”可定為“2 1 2 0”。

## 輸入說明

第一行是一個整數T(T<1000)，表示有T筆測試資料。每一筆測試資料都會有三個整數S,C和R。分別表示老鼠的位置(S)、奶酪的位置(C)和幾隻棒子再一開始就移去(R)。接下來會有R行，每行都有四個整數，表示棒子的兩個端點座標。

重點和範圍

1 <= S,C <= 9

S != C

0 <= R <= 12

對於每筆測試資料，所有被移去的棒子會截然不同。

## 輸出說明

對於每筆測試資料，輸出的第一筆案件，從一開始，其次是哪個玩家獲勝。如果老鼠可以在比賽開始前達到奶酪，輸出“No Cheese!”代替。其他請看範例。

## 範例輸入 #1

```
3
1 2 1
1 1 1 0
1 4 0
7 2 2
1 2 1 3
2 2 2 3
```

## 範例輸出 #1

```
Case 1: No Cheese!
Case 2: SOHA
Case 3: TARA
```
