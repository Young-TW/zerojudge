# c289. 來自深淵【壹】採藥記

## 內容

「啊啊阿!!..」
Riko不小心滑了一跤,就這樣跌落小坡
「沒事吧!Riko」
「雷格...好痛..QQ」
「我看看...手和膝蓋都流血了阿,幸好沒有骨折」
「我沒事的,雷格 我們繼續走吧」
雷格看像Riko微微顫抖的四肢
「不行,我去找草藥給你塗點」
「我說我沒事的啦,你看」
Riko說罷就站了起來
然後又軟腳了..
「我記得二層這裡有一些【珍珠草】,我去採一點回來別亂動」
「好...」
【珍珠草】
由多顆球形組織組成
通常都生長在 深界二層【誘惑之森】
球形組織總共有94種
每種都長得很不一樣
雖說是球形組織,但大多數仍有稜有角
這種草可以生成所謂的【塵】
用以治療外傷為主
不過得到【塵】的條件為 兩株草的頭有相同的球形組織
互相碰觸後生成
因為除此之外的部份無法輕易的剝離
只能從頭開始一顆一顆生成

雷格走了很久..
終於找到了【珍珠草林】
這裡有許多的珍珠草
但雷格只有兩隻手,他沒辦法帶太多,也不需要
如果可以能找出能生成最多【塵】的兩株
就可以重返旅途了

因為球形組織形狀實在是太過奇特
所以 以下以ASCII 33~126碼來表示

![image](ShowImage?id=1011)

## 輸入說明

每筆測資輸入1個interger n(1<=n<=100000)
接下來有n個【珍珠草】


左邊代表頭,右邊是尾


雷格所採的【珍珠草】最多也只有50個球形組織
輸出0結束

## 輸出說明

對於每筆測資
輸出任選兩株【珍珠草】
所能產生最大【塵】的【塵】的數量

## 範例輸入 #1

```
5
AC
AA
T44
236
62
3
#3$2%
3$2%
#3%2^&
0
```

## 範例輸出 #1

```
1
2
```
